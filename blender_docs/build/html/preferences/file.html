«Файл» -- File¶

Вкладка «Файл» -- File в «Уподобаннях Користувача» -- User Preferences дозволяє вам сконфігурувати уподобання авто-збереження та задати стандартні файлові шляхи для blend-файлів, рендерених зображень та інше.

Шляхи Файлів -- File Paths¶

Локації для різних зовнішніх файлів можуть бути задані наступними опціями:

Шрифти -- Fonts
Стандартна локація для пошуку файлів шрифтів.
Текстури -- Textures
Стандартна локація для пошуку зображеннєвих текстур.
Вивід Рендера -- Render Output
Де будуть зберігатися рендерені зображення/відео.
Скрипти -- Scripts
Додаткова локація для пошуку скриптів на Python. Дивіться нижче Scripts Path.
Звуки -- Sounds
Стандартна локація для пошуку файлів звуку.
Тимчасове -- Temp
Локація, де будуть зберігатися тимчасові файли.
Кеш Рендера -- Render Cache
Локація, де зберігатимуться кешовані зображення рендера.
Гілки перекладу -- I18n Branches
Шлях до каталогу /branches вашої локальної копії svn-перекладу, що дозволяє перекладати інтерфейс користувача.
Редактор Зображень -- Image Editor
Шлях до зовнішньої програми, що використовується для редагування зображень.
Відтворювач Анімації -- Animation Player
Шлях до зовнішньої програми, що використовуватиметься для відтворення рендерених анімацій.

Ghi chú

Якщо такі теки не існують, то вони не будуть створені автоматично.

Шлях Скриптів -- Scripts Path¶

By default Blender looks in several directories (OS dependent) for scripts. By setting a user script path in the preferences an additional directory is looked in. This can be used to store certain scripts/templates/presets independently of the currently used Blender Version.

Всередині цієї визначеної теки, мають бути створені конкретні підтеки, щоб вказати, де має шукати ці користувацькі файли Blender. Структура цієї теки є дзеркальною відносно структури теки зі скриптами, що знаходиться у каталогу інсталяції Blender'а:

  • scripts
  • add-ons
  • modules
  • presets
  • camera
  • cloth
  • interface_theme
  • operator
  • render
  • ...
  • startup
  • templates Не усі ці теки повинні бути присутніми.

Cảnh báo

Переконайтеся, що ви маєте відповідні права для запуску виконуваних файлів за визначеним шляхом. На MS-Windows, наприклад, якщо опція «Запускати цю програму від імені адміністратора» -- "Run this program as an administrator" увімкнена для виконуваних файлів, то це призведе до збою відкриття редактора через обмеження в контролі облікового запису користувача -- OS User Account Control. Запуск програм з підвищеними привілеями є потенційно небезпечним!

Авто Виконання -- Auto Execution¶

Скрипти Python (включаючи вирази драйверів) не виконуються стандартно з міркувань безпеки.

Авто Виконання Скриптів Python -- Auto Run Python Scripts
Ви можете увімкнути ігнорування цих питань безпеки та дозволити автоматичне виконання скриптів.
Виключені Шляхи -- Excluded Paths
Blend-files in these folders will not automatically run Python scripts. This can be used to define where blend-files from untrusted sources are kept.

Збереження і Завантаження -- Save & Load¶

Відносні Шляхи -- Relative Paths
Стандартно, зовнішні файли використовують відносний шлях -- relative path.
Стискати Файл -- Compress File

Стискає blend-файл при збереженні.

Ця опція буде ущільнювати ваші файли кожного разу, коли Blender зберігатиме їх. Щільні сітки, великі запаковані текстури або багато елементів у вашій сцені призводитимуть до створення великого blend-файлу.

This option may slow down Blender when you quit, or under normal operation when Blender is saving your backup files. Using this option traces processor time for file size.

Завантажувати Інтерфейс Користувача -- Load UI
Стандартна устава -- завантажувати розставу вікон (дивіться детальніше тут -- Screens) зі збереженого файлу. Це може бути змінено окремо при завантаженні файлу з панелі «Відкрити Файл Blender» -- Open blend-file Оглядача Файлів -- File Browser.
Фільтрувати за Розширенням Файлів -- Filter File Extensions

При активації цього регіон з файлами в Оглядачі Файлів буде показувати тільки відповідні файли (тобто, blend-файли при завантаженні повної устави Blender). Вибір типів файлів може бути змінене у регіоні файлів.

Фільтр файлів за розширенням.

Ховати Файли з Крапкою/Блоки Даних -- Hide Dot File/Data-blocks
Ховає файл, ім'я якого починається з крапки . (у системах Linux та Apple файли з . сховані).
Ховати Останні Локації -- Hide Recent Locations
Ховає панель «Останнє» -- Recent в Оглядачі Файлів -- File Browser, яка показує теки, до яких останнім часом здійснювався доступ.
Ховати Системні Закладки -- Hide System Bookmarks
Ховає системні закладки для тек в Оглядачі Файлів -- File Browser.
Показати Мініатюри -- Show Thumbnails
Показує мініатюри зображень та фільмів в Оглядачі Файлів -- File Browser.
Зберігати Версії -- Save Versions

Кількість версій, створюваних для одного і того ж файлу (для резерву).

Ця опція вказує Blender'у зберігати зазначену кількість збережених версій вашого файлу у вашому поточному робочому каталогу, коли ви вручну зберігаєте файл. Такі файли версій будуть мати розширення: .blend1, .blend2 тощо з прирістним номером відповідно до кількості заданих вами версій. Старіші файли будуть іменовані з вищим номером. наприклад, зі стандартним значенням цієї устави як 2, ви будете мати три версії вашої файлу: *.blend (ваше останнє збереження), *.blend1 (ваше друге перед останнім збереження) та *.blend2 (ваше третє перед останнім збереження).

Останні Файли -- Recent Files
Кількість файлів, показуваних на у підменю «Файл > Відкрити Останнє» -- File ‣ Open Recent в Оглядачі Файлів.
Зберегти Передогляди Зображень -- Save Preview Images
Передогляди зображень та матеріалів в Оглядачі Файлів -- File Browser створюються на вимогу. Для збереження цих передоглядів у вашому blend-файлі увімкніть цю опцію (за рахунок збільшення розміру вашого blend-файлу).

Авто Збереження -- Auto Save¶

Зберегти Сесію -- Keep Session
Automatically loads the quit.blend file after re-starting Blender. This file is always saved on quit, regardless of this option. See Recover Last Session.
Авто Збереження Тимчасових Файлів -- Auto Save Temporary Files
Enables Auto Save. Tells Blender to automatically save a backup copy of your work-in-progress files to the Тимчасовий каталог.
Часолік -- Timer
This specifies the number of minutes to wait between each Auto Save. The default value of the Blender installation is 2 minutes. The minimum is 1, and the Maximum is 60 (save every hour).

Редактор Тексту -- Text Editor¶

Табуляції як Пробіли -- Tabs as Spaces
При натисканні Tab табуляції записуються як клавіатурні пробіли.
Автор -- Author
Ім'я, що буде використовуватися в експортованих файлах, коли їх формат підтримує таку функцію.
