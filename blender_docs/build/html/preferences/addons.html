«Додатки» -- Add-ons¶

Вкладка «Додатки» дозволяє вам управляти вторинними скриптами, що називаються «Додатки» -- "Add-ons", які розширюють функціональність Blender'а. У цій вкладці ви можете шукати, інсталювати, вмикати та вимикати Додатки.

Вкладка «Додатки» -- Add-ons в Уподобаннях Користувача.

Пошук -- Searching¶

Blender поставляється з деякими корисними Додатками, вже готовими для вмикання. Але ви можете також додавати свої або цікаві додатки, що можна знайти у веб.

Фільтрування -- Filtering¶

Підтримуваний Рівень -- Supported Level

Додати Blender'а розділені на дві групи, залежно від того, хто написав/підтримує їх:

  • Офіційне -- Official: Додатки, що написані розробниками Blender'а.
  • Спільнота -- Community: Додатки, що написані людьми зі спільноти Blender'а.
Категорії -- Categories
Додатки діляться на категорії, які є областями використання Blender'а.

Вмикання та вимикання¶

Вмикається та вимикається додаток, поставленням або зняттям стягу на позначці, справа від його імені, як показано на ілюстрації.

Enabling an add-on.

Функціональність додатку повинна стати негайно ж доступною. Якщо Додаток не активується при увімкненні, то перевірте вікно Консолі -- Console window на наявність помилок, що могли трапитися.

Інформація про додаток¶

Ви можете клацнути на стрілці зліва від імені додатку та розгорнути його панель для перегляду більше про нього інформації, зокрема, про його локацію, опис та посилання на документацію. Тут ви можете також знайти кнопку для повідомлення про ваду, конкретно щодо цього додатку.

Mẹo

Збереження уподобань щодо додатків

Якщо ви хочете, щоб певний Додаток вмикався кожен раз при запуску Blender'а, то після його увімкнення необхідно клацнути на кнопці «Зберегти Устави Користувача» -- Save User Settings.

Уподобання щодо додатку¶

Індивідуальна активація¶

Додатки, що активують або змінюють кілька клавіатурних скорочень, мають спеціальну систему активації. Наприклад, з додатком "UI: Pie Menu Official" для кожного меню передбачене поле вибору для активації цього меню та його клавіатурного скорочення.

With Pie menus, First you activate the add-on. This activates the "Add-ons Preferences Submodule Activation". You then need to expand the Add-ons Preferences, then you will see the list of Pie Menu types you can choose from. From here you can individually activate the menus you like to use. If the menu conflicts with another favorite, there's no need to activate it. You can activate any combination and save as user settings so your activations are available next time you start Blender.

Заголовок -- Header¶

Інсталяція з файлу

Для додатків, які ви знайшли на веб, або написали самі, щоб вони стали показуватися у цьому списку, слід спершу їх інсталювати, клацнувши на кнопці «Інсталювати з Файлу» -- Install from File... та вказати на відповідний файл .zip або .py.

Now the add-on will be installed, not automatically enabled. The search field will be set to the add-on's name (to avoid having to look for it). Enable the add-on by turning on the checkbox.

Mẹo

User-Defined Add-on Path

Ви можете також створити особистий каталог, що міститиме нові додатки, та сконфігурувати ваш шлях до цих файлів на вкладці «Файл» -- File в редакторі «Уподобання Користувача» -- User Preferences. Для створення особистого каталогу скриптів:

  1. Створіть порожній каталог у бажаній вами локації (наприклад, my_scripts).
  2. Додайте у ньому, my_scripts, підкаталог з іменем addons (це повинно бути саме це ім'я, щоб Blender розпізнавав його).
  3. Відкрийте вкладку «Файл» -- File «Уподобань Користувача» -- User Preferences.
  4. У полі устави «Скрипти» -- Scripts вкажіть шлях до вашого каталогу зі скриптами (наприклад, my_scripts).
  5. Збережіть Уподобання Користувача та перезапустіть Blender для розпізнавання цієї нової локації з додатками.

Тепер, коли ви інсталюватимете додатки, ви можете вибирати в лівому регіоні Оглядача Файлів для опції «Цільовий Шлях» -- Target Path значення «Уподобання Користувача» -- User Pref (і вони установлюватимуться по шляху, заданому у вкладці «Файл» -- File).

Blender буде копіювати ново інстальовані додатки у цей каталог, визначений вами в Уподобаннях Користувача.
