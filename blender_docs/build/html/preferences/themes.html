«Теми» -- Themes¶

Вкладка «Теми» -- Themes дозволяє вам налаштовувати вигляд та кольори інтерфейсу.

Кольори для кожного редактора можуть бути установлені окремо у відповідних їм категоріях для кожного доступного контролера. Зауважте, що ці зміни показуються у реальному часі на вашому екрані. Крім того, такі деталі, як розмір точки у 3D Огляді -- 3D View або Редакторі Графів -- Graph Editor також можуть бути змінені.

Для збереження теми у Blender'і використовується система передустав. Це зберігає тему у файл формату XML у підтеці ./scripts/presets/interface_theme/ одного з каталогів конфігурації -- configuration directories.

Blender поставляється з невеликим набором тем.

Це приклад теми Elsyiun.
