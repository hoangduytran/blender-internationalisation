«Увід» - Input¶

У вкладці «Увід» ви можете налаштовувати те, як Blender реагує на дії миші й клавіатури, а також визначати власну розкладку клавіш.

Взаємодія¶

Передустави Взаємодії -- Interaction Presets
Передустави, що дозволяю Blender'у діяти, як інша програма по ваших особистих уподобаннях.

Миша -- Mouse¶

Емулювати 3-кнопкову Мишу -- Emulate 3 Button Mouse

Blender can be configured to work with pointing devices which do not have an MMB (such as a two-button mouse, Apple's single-button mouse, or laptop touch-pad). The functionality of the three mouse buttons will then be emulated with key/mouse button combinations as shown in the table below.

Скорочення для підтримуваного устаткування мишей¶
3-кнопкова Миша 2-кнопкова Миша Миша Apple
LMB LMB LMB (кнопка миші)
MMB Alt-LMB Alt-LMB (клавіша Option/Alt + кнопка миші)
RMB RMB Cmd-LMB (клавіша Command/Apple + кнопка миші)

Комбінації Миша/Клавіатура, які вказуються у цьому підручнику, можуть виражатися комбінаціями, вказаними у цій таблиці. Наприклад:

  • MMB перетяг стає Alt-LMB перетягом.
  • Shift-Alt-RMB стає Shift-Alt-Cmd-LMB для одно-кнопкової миші.
Безперервний Захоп -- Continuous Grab

Ця функція використовується для запобігання проблемі, коли така дія, як захоплення або панорамування огляду, обмежується межами вашого екрану.

Це робиться шляхом завертання миші у межах огляду.

Ghi chú

Завертання вказівника підтримується тільки пристроями відносного уводу (миша, трекбол, трекпад).

Графічні планшети, однак, типово використовують абсолютне позиціонування і ця функція буде вимикатися при їх використанні.

Це виявляється для кожної дії, а тому присутність планшета не вимикає «Безперервний Захоп» -- Continuous Grab для уводу вказівником миші.

Поріг Перетягу -- Drag Threshold
Кількість пікселів, на яку елемент Інтерфейсу Користувача має переміститися, щоб бути розпізнаним Blender'ом.
Виділяти за допомогою кнопки -- Select With
Ви можете обрати, яка кнопка миші буде використовуватися для виділення (інша буде вживатися для розміщення 3D курсора).
Подвійний Клацок -- Double Click
The time in ms for a double-click to be recognized.

Ghi chú

Опція емулювання миші доступна тільки, якщо для опції «Виділяти за допомогою кнопки» -- Select With задано значення «Справа» -- Right.

Emulate Numpad¶

Клавіші цифроблоку (окремої цифрової частини на клавіатурі справа) -- Numpad -- доволі часто використовуються у Blender'і й не є тими ж клавішами, що й звичайні цифрові клавіші. Якщо ви маєте клавіатуру без Цифроблоку (наприклад, на лептопі), то ви можете вказати Blender'у обробляти звичайні цифрові клавіші як клавіші Цифроблоку. Просто увімкніть «Емулювати Цифроблок» -- Emulate Numpad.

Маніпуляція Оглядом -- View Manipulation¶

Стиль Обльоту -- Orbit Style

Дає на вибір, як Blender буде працювати при стандартному обльоті, обертанні 3D Огляду -- 3D View при утримуванні натисненою MMB.

Поворотний стіл -- Turntable

Обертає огляд, зберігаючи горизонт горизонтальним.

Це веде себе, як гончарний круг або патефон, де ви маєте дві доступні осі для обертання, і світ, здається, має краще визначення того, що є «Верх» та «Низ» у ньому.

Недоліком використання стилю Turntable є те, що ви втрачаєте деяку гнучкість при роботі з вашими об'єктами. Однак, ви отримуєте почуття «Верху» та «Низу», яке може допомогти, якщо відчуваєте дезорієнтацію.

Трекбол -- Trackball
Є менш обмежувальним, дозволяючи будь-яку орієнтацію.
Стиль Зумування -- Zoom Style

Choose your preferred style of zooming in and out with Ctrl-MMB.

Масштаб -- Scale
Стиль Scale залежить від того, де ви спершу клацнули в огляді. Для відсування утримуйте Ctrl-MMB при перетягу миші від краю екрану до його центру. Для присування утримуйте Ctrl-MMB при перетягу миші від центру екрану до його краю.
Продовження -- Continue
Стиль зумування Continue дозволяє вам керувати швидкістю (а не значенням) зумування, рухаючи мишу від точки початкового клацку Ctrl-MMB. Рух миші уверх або управо від початкової точки клацку дасть відсування, рух вниз або вліво -- присування огляду. Чим далі ви порухаєте, тим швидше буде здійснюватися зумування. Напрямки зумування можуть бути змінені радіокнопками «Вертикально» -- Vertical та «Горизонтально» -- Horizontal, а також перемиканням опції «Інвертувати Напрям Зумування» -- Invert Zoom Direction.
Наїзд-Від'їзд -- Dolly
Зумування стилем Dolly працює подібно до стилю Continue, за винятком того, що швидкість зумування буде постійною.
Вісь Зумування -- Zoom Axis

Вісь, яка використовується для зумування за MMB.

Вертикально -- Vertical
Рух миші уверх відсуває, а рух вниз -- присуває огляд.
Горизонтально -- Horizontal
Рух миші вліво присуває, а рух управо -- відсуває огляд.
Інвертувати Напрям Зумування -- Invert Zoom Direction
Інвертує напрямок зумування для стилів Dolly та Continue.
Інвертувати Напрям Зумування Коліщам -- Invert Wheel Zoom Direction
Інвертує напрямок зумування коліщам миші.

Редактор Розкладки Клавіш -- Keymap Editor¶

Редактор Розкладки Клавіш дозволяє вам змінювати стандартні клавіатурні скорочення. Ви можете змінювати розкладки клавіш для кожного з редакторів Blender'а.

Keymap editor.

Передустави Розкладки клавіш
Список попередньо визначених розкладок клавіш.
  1. Виберіть бажану розкладку клавіш та клацайте на білі стрілки зліва для відкриття дерева розкладки клавіш.
  2. Виберіть, який Увід буде керувати функцією.
    • Клавіатура -- Keyboard: Лише клавіша або поєднання клавіш E, Shift-E.
    • Миша -- Mouse: Клацок лівою/середньою/правою її кнопкою. Може комбінуватися з клавішами-модифікаторами Alt, Shift, Ctrl, Cmd.
    • NDOF: Movement from a 3D Mouse.
    • Підправка -- Tweak: Клацок і перетяг. Може також комбінуватися з клавішами-модифікаторами.
    • Увід тексту -- Text input: Використовуйте цю функцію для уведення тексту.
    • Часолік -- Timer: Використовується для керування діями на основі часового періоду. наприклад, стандартно, Animation Step використовує "Timer 0", Smooth View використовує "Timer 1".
  3. Змініть скорочення бажаним вам чином. Просто клацніть на увід скорочення та уведіть нове скорочення.

Якщо ви хочете відновити стандартні устави для розкладки клавіш, то просто клацніть на кнопці «Відновлення» -- Restore у правому верхньому куті для цієї розкладки клавіш.

Mẹo

Instead of deleting the default keymap to create your custom one, you can just add a new Preset for both the mouse and keyboard.

Експорт/Імпорт конфігурації клавіш¶

In some cases, you may need to save your configuration in an external file (e.g. if you need to install a new system or share your keymap configuration with the community). To do this, simply press the Export Key Configuration button found in the header. After doing so, the File Browser will open to choose where to store the configuration. The Import Key Configuration button installs a keymap configuration that is on your computer but not in Blender.

Експортована розкладка клавіш буде містити тільки розкладки клавіш і категорії, що були модифіковані користувачем. Крім того, додатки можуть реєструвати розкладки клавіш для їх відповідних функцій, однак, ці розкладки клавіш не експортуються, допоки не були змінені користувачем. Цей експортований файл може вважатися як «різниця у розкладках клавіші -- "keymap delta", а не повним експортом розкладки клавіш.
