«Правка» -- Editing¶

Ці уподобання керують тим, як різні засоби будуть взаємодіяти з вашим уводом.

Пов'язати Матеріали З -- Link Materials To¶

Example for a mesh.

Для належного розуміння цієї опції вам необхідно зрозуміти, як Blender працює з Об'єктами. Майже все у Blender'і організоване в ієрархію блоків даних. Блок даних -- data-block -- можна розглядати, як контейнер для певних фрагментів інформації. Наприклад, блок даних «Об'єкт» -- Object містить інформацію про локацію Об'єкта, тоді як блок даних «Дані Об'єкта» -- Object Data чи "ObData" містить інформацію про сіть.

Матеріал може бути пов'язано двома різними способами:

Матеріал пов'язаний з блоком «Дані Об’єкта» -- Object Data (зліва) та з блоком «Об'єкт» -- Object (справа).

Дані Об'єкта -- Object Data
Будь-який створений матеріал буде створюватися як частина блоку даних «Дані Об’єкта».
Об'єкт -- Object
Будь-який створений матеріал буде створюватися як частина блоку даних «Об'єкт».

Read more about Blender's Data System.

Нові Об'єкти -- New Objects¶

Увійти в Режим Правки -- Enter Edit Mode
Якщо увімкнено, то автоматично буде активуватися Режим Правки -- Edit Mode після створення вами нового об'єкта.
Вирівняти За -- Align To
Світ -- World
Нові об'єкти вирівнюються за координатами світу.
Огляд -- View
Новий об'єкт вирівнюється за координатами огляду.

Касування -- Undo¶

Глобальне Касування -- Global Undo

Це дає змогу Blender'у зберігати зроблені дії, коли ви не знаходитеся у Режимі Правки -- Edit Mode. Наприклад, дублювання Об'єктів, зміна устав на панелях або перемикання між режимами.

Cảnh báo

Хоча вимкнення цієї опції заощаджує пам'ять, однак воно зупиняє функціонування панелі відновлення -- Redo Panel, а також запобігає зміні опцій інструментів у деяких випадках.

Для типового використання найкраще зберігати цю опцію увімкнену.

Кроки -- Steps
Кількість доступних кроків касування.
Ліміт Пам'яті -- Memory Limit
Максимум використання пам'яті у мегабайтах (значення 0 -- без обмежень).

Read more about Undo and Redo options.

Нарисний Олівець -- Grease Pencil¶

Радіус Стирача -- Eraser Radius
The size of the eraser used with the Grease pencil.
Манхеттенська Відстань -- Manhattan Distance
Мінімальна кількість пікселів, на які миша повинна пересунутися горизонтально або вертикально, щоб це переміщення було записане. Зменшення значення цього має працювати краще для кривих ліній.
Euclidean Distance
Мінімум відстані, на яку миша повинна перейти, перш ніж переміщення запишеться.
Спростити Штрих -- Simplify Stroke
Це вмикає після-обробний етап спрощення штриха для вилучення близько половини поточних у ньому точок. Це має значення тільки, коли не рисуються прямі лінії.
Стандартний Колір -- Default Color
Стандартний колір для нових шарів Нарисного Олівця.

Read more about Grease Pencil.

Відтворення -- Playback¶

Дозволити Від'ємні Кадри -- Allow Negative Frame
Курсор Часу -- Time Cursor може бути установлений у від'ємні кадри мишею або за допомогою клавіатури. При використанні «Вжити Діапазон Передогляду» -- Use Preview Range це також дозволяє відтворення.

Редактор Вузлів -- Node Editor¶

Припуск Авто-зсуву -- Auto-offset Margin
Припуск для використання при автоматичному зсуванні вузлів -- offsetting nodes.

Редактори Анімації -- Animation Editors¶

Видимість F-Кривої -- F-Curve Visibility
Opacity that unselected F-Curves stand out from the Graph Editor.

Ключкадрування -- Keyframing¶

У багатьох ситуаціях анімація керується ключкадрами -- keyframes. Стан значення (наприклад, локація) записується у ключкадр, а анімація між двома ключкадрами інтерполюється Blender'ом.

Видиме ключування -- Visual Keying
Коли об'єкти використовується примусами, то значення властивості об'єкта фактично не змінюється. «Візуальне Ключування» -- Visual Keying додаватиме ключкадри для такої властивості об'єкта зі значенням на основі візуального трансформування, отриманого від примусу.
Вставляти Лише Необхідне -- Only Insert Needed
Це буде вставляти ключкадри тільки, якщо значення властивості змінилося та відрізняється.
Авто Ключкадрування -- Auto Keyframing
Стандартно вмикає «Авто Ключкадр» -- Auto Keyframe для нових сцен.
Попереджувати про Авто Ключування -- Show Auto Keying Warning
Показує попередження у правому верхньому куті 3D Огляду -- 3D View при пересуві об'єктів, якщо «Авто Ключкадр» -- Auto Keyframe увімкнуто.
Вставляти Лише Доступне -- Only Insert Available
Це буде додавати кадри тільки у канал F-Кривих, що вже існує.

Стандарти Нової F-Кривої -- New F-Curve Defaults¶

Інтерполяція -- Interpolation
Керує стандартною інтерполяцією -- Interpolation для ново створених ключкадрів.
Держаки -- Handles
Керує стандартними держаками -- Handle для ново створених F-Кривих.
XYZ у RGB -- XYZ to RGB
Колір для X, ігрек або Z анімаційних кривих (локація, масштаб або обертання) є таким же, як і колір для осей X, Y та Z.

Трансформації -- Transform¶

Підтверджувати Відпуском -- Release confirms
Dragging LMB on an object will move it. To confirm this (and other) transform, an LMB is necessary by default. When this option is activated, the release of LMB acts as confirmation of the transform.

Колір Перекриття при Ліпці -- Sculpt Overlay Color¶

This color button allows the user to define a color to be used in the inner part of the brushes circle when in sculpt mode, and it is placed as an overlay to the brush, representing the focal point of the brush influence. The overlay color is visible only when the overlay visibility is selected (clicking at the eye to set its visibility), and the transparency of the overlay is controlled by the alpha slider located at the Option tab ‣ Overlay panel in the Tool Shelf.

Дублювати Дані -- Duplicate Data¶

The Duplicate Data checkboxes define what data is copied with a duplicated Object and what data remains linked. Any boxes that are checked will have their data copied along with the duplication of the Object. Any boxes that are not checked will instead have their data linked from the source Object that was duplicated.

Наприклад, якщо ви маєте тут увімкнений стяг «Сіть» -- Mesh, то створиться повна копія сітьових даних з новим Об'єктом, і кожна сіть буде діяти незалежно для кожного дубліката. Якщо ви залишите стяг «Сіть» вимкненим, то після дублювання коли ви змінюватимете сіть одного з об'єктів, то ці зміни автоматично будуть відображатися на дубльованому Об'єкті.

The same rules apply to each of the checkboxes in the 'Duplicate Data' list.
