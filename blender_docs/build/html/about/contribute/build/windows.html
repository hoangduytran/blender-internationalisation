Cài đặt trên MS-Windows -- Installation on MS-Windows¶

Chuyển đổi các tập tin RST thành các trang HTML đẹp. Có hai cách để thi hành quá trình xây dựng.

Trình Duyệt Tập Tin -- File browser

Thi hành bản make.bat trong thư mục C:\blender_docs.

Mẹo

Tạo đường tắt trên mặt bàn làm việc màn hình tới công cụ make.

Cửa Sổ Bàn Giao Tiếp -- Command Prompt

  1. Hoặc mở một cửa sổ bàn giao tiếp và chuyển sang kho lưu trữ bằng lệnh cd C:\blender_docs.

  2. Xây dựng bằng cách sử dụng lệnh sau:

    make

Quá trình xây dựng có thể mất vài phút lần đầu tiên (hoặc sau bất kỳ thay đổi lớn nào), nhưng lần sau bạn xây dựng nó thì thường chỉ mất vài giây thôi.

Ghi chú

Nếu bạn gặp dòng báo lỗi kết thúc với thông điệp TypeError: an integer is required (got type str) : Lỗi trong Kiểu Dữ Liệu : đòi hỏi một số nguyên (song lại nhận được kiểu chuỗi ký tự), bạn có thể cần phải cài đặt một phiên bản cũ hơn của phần mềm Babel (Thư viện Quốc tế hóa của Python).

Để làm điều này, chỉ cần thi hành:

pip install sphinx "babel<2.0"

Duyệt Thảo Bản Hướng Dẫn Sử Dụng Cục Bộ -- Viewing the Local Manual¶

Khi tài liệu đã được xây dựng xong, tất cả các tập tin HTML sẽ được định vị trong thư mục ~/blender_docs/build/html. Hãy thử mở build/html/contents.html trong trình duyệt trang mạng của bạn và đọc bản hướng dẫn sử dụng:

Tiếp tục với bước tiếp theo: Biên Soạn.
