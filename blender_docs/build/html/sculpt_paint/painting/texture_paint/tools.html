Інструменти -- Tools¶

Open the Tool Shelf in the 3D View or UV/Image Editor.

Пензель -- Brush¶

Brush settings.

За допомогою цієї панелі ви можете створювати пензлі, кожен зі своїми унікальними уставами (як колір та ширина).

Типи пензлів -- Brush Types¶

Рисування Текстури -- Texture Draw¶

Звичайний пензель; малює смугу кольору.

Заповнення -- Fill¶

Може використовуватися для заповнення зображення кольором пензля.

Поріг Заповнення -- Fill Threshold
ToDo <2.79. (2D only)

Ghi chú

Перевизначення -- Overrides

Для проекційного текстурування це буде зневажати деякі опції проекційного малювання на моделі. Це означає, що загороджені, тилові та відкинуті за нормаллю грані завжди будуть заповнюватися, незалежно від опцій, активований на панелі «Проекційне Малювання» -- Project Paint.

Маскування -- Masking

Використовуйте режим виділу «Грань» для ізоляції граней.

Маска -- Mask¶

Функція маски розкладає зображення на сіть та використовує інтенсивність цього зображення для відмаскування певних частин сіті у ході малювання. Опції маски можуть бути знайдені на панелі «Маска» -- Mask у вкладці «Гнізда» -- slots tab.

Пом'якшення -- Soften¶

Змішує краї між двома кольорами.

Напрям -- Direction
Пом'якшення -- Soften
Blur filter. ToDo 2.72.
Різкішання -- Sharpen

The Sharpen tool enhances the contrast of the image as you paint over it.

Поріг Різкості -- Sharp Threshold
Цей поріг буде застосовувати різкішання тільки до тих пікселів, що відрізняються на більше, ніж це значення порогу, від їх навколишніх пікселів.
Режим Розмиву -- Blur Mode

Тип ядра розмиву, що керує тим, як сусідні пікселі виважені, коли розраховується ефект розмиву.

Гаусове -- Gaussian
«Гаусове» буде відбирати пікселі здебільшого ближче до центру пензля.
Коробка -- Box
«Коробка» відбирає усі навколишні пікселі однаковою мірою.

Розмазання -- Smear¶

При клацанні вами бере кольори під вказівником та змішує їх у напрямку руху вами мишею. Подібно до інструмента "smudge" у Gimp.

Клонування -- Clone¶

Копіює кольори з вказаного зображення на активне зображення. Задання клонування, курсор клонування визначається за допомогою Ctrl-LMB. При 2D малюванні клон-трафарет може переміщатися перетягом RMB.

Клонувати з гнізда малювання -- Clone from paint slot

Показується фонове зображення при виборі цього пензля; використовуйте повзунок «Сила» -- Strength для керування тим, як проявляється фонове зображення.

Гніздо Джерела Клонування -- Source Clone Slot
При використанні пензля клонування це дозволяє вам вибирати зображення як джерела клонування.

Загальне -- Common¶

Більшість пензлів мають загальні устави.

Колір -- Color

Колір пензля Дивіться детальніше тут -- Відбірник кольору.

Натискання S на будь-якій частині зображення відбирає її колір та задає його як колір пензля.

Переверт -- Flip (іконка циклу) X
Обмінює колір переднього і заднього (фону) планів.
Радіус -- Radius
Радіус пензля у пікселях.
Сила -- Strength

Наскільки потужно застосовується пензель.

Затухання за Проміжком -- Space Attenuation (іконка замка)
Дає затухання сили пензля відповідно до інтервалу.
Чутливість Натиску (іконка з рукою та випуклістю в синій лінії)
Перемикач справа від цих трьох устав вмикає та вимикає чутливість натиску на планшеті для керування силою їх впливів.
Змішання -- Blend

Задає спосіб застосування малювання поверх підлежачого кольору. Дивіться детальніше тут -- Color Blend Modes.

  • Додання Альфа -- Add Alpha: робить зображень більш безпрозорим при малюванні.
  • Стирання Альфа -- Erase Alpha: робить зображення прозорим при малюванні, дозволяє фоновим кольорам та нижньо-рівневим текстурам проглядатися крізь. При «малюванні» вами проявляється псевдо фон шахівниці. Використання кінця стирача стилуса планшета буде перемикатися на цей режим.
  • Світимість -- Luminosity
  • Виключення -- Exclusion
  • Яскраве світло -- Vivid light
  • Шпилькове світло -- Pin light

Mẹo

Щоб бачити ефекти режимів змішання Erase та Add Alpha у Редакторі UV/Зображень -- UV/Image Editor, ви повинні увімкнути показ альфа каналу, клацнувши на кнопці Display Alpha або Alpha-Only. Прозорі (без альфа) області будуть показані з шаховим фоном.

Накопичення -- Accumulate
Це дозволяє штриху накопичуватися, подібно до того, як працює розбризкувач.
Альфа -- Alpha
Безпрозорість показу зображення клонування.
Вжити Градієнт -- Use Gradient

A gradient can be used as color source. ToDo 2.72. See Віджет колірного градієнта. To apply the gradient with the Fill brush click LMB and drag to define depending on the Gradient Fill Mode the gradient line, or radius, if radial gradient is used.

Режим -- Mode
Натиск -- Pressure
Will choose a gradient color from the color ramp according to the stylus pressure.
Затиск -- Clamp
Will alter the color automatically by the distance covered by the brush and as specified. by Gradient spacing. With Clamp it uses the last color of the color ramp after the specified.
Повтор -- Repeat
Similar to Clamp. After the last color it resets the color to the first color in the color ramp and repeating the pattern.
Режим Заповнення Градієнтом -- Gradient Fill Mode
Лінеарно -- Linear, Радіально -- Radial

Плиткування -- Tiling¶

Завертає штрих на інший бік зображення, коли пензель виходить за межі протилежного боку полотна. Дуже зручно для зроблення безшовних текстур.
