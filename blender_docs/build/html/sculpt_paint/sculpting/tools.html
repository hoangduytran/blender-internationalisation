Пензель -- Brush¶

Тип Пензля -- Brush Type¶

Орієнтир -- Reference

Mode:Sculpt Mode
Panel:Tool Shelf ‣ Tools ‣ Brush
Menu:Brush ‣ Sculpt Tool

Пензлі ліплення.

Кулька -- Blob
Штовхає сіть назовні або всередину сферичною формою з уставами, що керують величиною щипання на краю цієї сфери.
Глина -- Clay C
Подібно до пензля «Прорис Ліпки» -- SculptDraw, але включає устави для коригування площини, на якій пензель діє.
Смуги Глини -- Clay Strips
Подібно до пензля «Глина» -- Clay, але вживає кубічний тест для визначення області впливу пензля, а не сферу.
Складка -- Crease Shift-C
Створює різкі вдавлення або видавлення, уштовхуючи або виштовхуючи сіть, одночасно защемлюючи вершини.
Насип/Углиб -- Fill/Deepen
Працює подібно до пензля «Плющення» -- Flatten, але переносить тільки вершини, що нижче площини пензля, угору. Інверсія до пензля «Скребок» -- Scrape -- це режим «Углиб» -- Deepen, що уштовхує вершини над площиною пензля вниз.
Плющення/Контраст -- Flatten/Contrast Shift-T
Пензель «Плющення» -- Flatten виштовхує вершини у напрямку площини ліплення Sculpt plane. Інверсія пензля Flatten -- пензель «Контраст» -- Contrast, який виштовхує вершини уверх або вниз від площини ліпки Sculpt Plane.
Захоп -- Grab G
Used to drag a group of points around. Unlike the other brushes, Grab does not modify different points as the brush is dragged across the model. Instead, Grab selects a group of vertices on mouse-down, and pulls them to follow the mouse. The effect is similar to moving a group of vertices in Edit Mode with proportional editing enabled, except that Grab can make use of other Sculpt Mode options (like textures and symmetry).
Роздув/Здув -- Inflate/Deflate I
Подібно до SculptDraw, за винятком того, що вершини у режимі «Роздув» -- Inflate зміщуються у напрямку своїх нормалей.
Шар -- Layer L

Цей пензель подібний до SculptDraw, за винятком того, що висота зміщення шару задана. Це створює вигляд суцільного шару при рисуванні. Цей пензель не рисує поверх себе; штрих пензля само перетинається. Відпуск кнопки миші та починання нового штриха скине глибину та дасть малювання поверх попереднього штриха.

Стало -- Persistent
ToDo <2.70.
Задати Базу Сталості -- Set Persistent Base

Маска -- Mask M

Lets you select mesh parts to be unaffected by other brushes by painting vertex colors. The mask values are shown as gray-scale. I.e. the darker a masked area is, the less effect sculpting on it will have. See also the options of the Маска -- Mask menu.

Інструмент Маски -- Mask Tool

Пензель маски має два режими:

Рисування -- Draw
Рисує маску.
Згладження -- Smooth Shift
Натискання Shift при активному пензлі «Маска» перемикатиме маску у режим згладження.
Поштовх -- Nudge
Рухає вершини у напрямку штриха пензля.
Прищемлення/Розщемлення -- Pinch/Magnify P
Виштовхує вершини у напрямку центра пензля. Інверсна устава його -- режим «Розщемлення» -- Magnify, в якому вершини уштовхуються від центру пензля.
Оберт -- Rotate
Обертає вершини у межах пензля у напрямку руху курсора. Початковий напрямок перетягу є нульовим кутом і, обертаючи навколо центру, ви можете створювати ефект вихору.
Скребок/Піки -- Scrape/Peaks
Пензель «Скребок» -- Scrape працює подібно до пензля «Плющення» -- Flatten, але тільки переносить вершини, що вище площини, вниз. Інверсія пензля Scrape -- режим Peaks, що дає уштовхування вершин, що вище площини, уверх від цієї площини.
Прорис Ліпки -- SculptDraw X
Рухає вершини всередину або назовні, базуючись на усередненій нормалі вершин, що вміщуються у межах нарисованого штриха пензля.
Згладження -- Smooth S
Як випливає з назви, усуває нерівності на ділянці сіті у межах впливу пензля шляхом згладження позицій вершин.
Зміїний Гак -- Snake Hook K

Виштовхує вершини вздовж руху пензля для створення довгих, змієподібних форм.

Щипок -- Pinch
The Snake hook brush tends to loose volume along the stroke, with pinch > 0.5 its possible to sculpt shapes without loosing volume.
Граблі -- Rake
A factor to support grabbing the mesh with rotation following the cursor's motion.
Палюх -- Thumb
Подібно до пензля «Поштовх» -- Nudge, він розпластує сіть в області пензля, одночасно рухаючи її у напрямку штриха пензля.
Спрощення -- Simplify
Цей пензель стягує короткі ребра (як визначено розміром деталізації), незалежно від того чи увімкнена опція Collapse Short Edges. Цей пензель не діє, якщо не увімкнена динамічна топологія. Його можна знайти тільки у меню «Пензель > Інструмент Ліпки» -- Brush ‣ Sculpt Tool.

Загальні опції¶

Радіус -- Radius
Ця опція керує радіусом пензля, який вимірюється у пікселях. F дозволяє вам змінювати розмір пензля інтерактивно, тягнучи мишу, та підтверджуючи LMB (текстура пензля повинна бути видима всередині його кільця). Уведення номера після натискання F дозволяє вам задати числове значення розміру. На розмір пензля може впливати чутливість натиску, вмикається відповідною іконкою, якщо використовуєте Graphics Tablet.
Сила -- Strength

Керує тим, наскільки кожен застосування пензля впливатиме на модель. Наприклад, вищі значення для пензля SculptDraw призводять до додання заглиблень у модель більш швидко, а для пензля Smooth -- більш швидке згладження моделі. Ця устава не доступна для пензлів Grab, Snake Hook та Rotate.

Ви можете змінювати силу пензля інтерактивно, натиснувши Shift-F в 3D Огляді, порухавши мишу та підтвердивши LMB. Ви можете увести значення сили числовим уводом після натискання Shift-F. На силу пензля може впливати чутливість натиску, яка вмикається відповідною кнопкою, якщо використовується підтримуваний планшет.

Mẹo

Якщо діапазон сил здається, що не підходить до моделі (наприклад, якщо навіть найнижчі значення сили дають надто великі зміни на моделі), то ви можете масштабувати модель (у Режимі Правки -- Edit Mode, тільки не у Режимі Об'єкта -- Object Mode). Більші розміри будуть робити вплив пензля меншим, і навпаки.

Автозгладження -- Autosmooth
Задає величину згладження, що застосовується до кожного штриха.
Нормалізація Вагомостей -- Normal Weight Ctrl

Змушує пензель рухатися вздовж нормалі поверхні. Особливо корисно з пензлем Grab, може тимчасово вмикатися утримуванням натисненою Ctrl. Наприклад, пензель «Захоп» може використовуватися для виштовхування западини (отвору) у сіть при увімкненому Normal Weight.

Застосовується з пензлями Grab та Snake Hook.

Панель «Симетрія/Блокування» -- Symmetry/Lock¶

Орієнтир -- Reference

Mode:Sculpt Mode
Panel:Tool Shelf ‣ Tools ‣ Symmetry/Lock
Menu:Sculpt ‣ Symmetry/Lock
Дзеркалення -- Mirror
Mirror the brush strokes across the selected local axes. Note that if you want to alter the directions the axes point in, you must rotate the model in Edit Mode and not in Object Mode.
Радіально -- Radial
Ці устави дозволяють вжити радіальну симетрію по бажаних осях. Числа тут визначають, скільки разів штрих буде повторено у межах 360 градусів навколо центральних осей.
Розтушовка -- Feather
Скорочує силу штриха, де він перекриває площини симетрії.
Блокування -- Lock
Ці три кнопки дозволяють вам блокувати будь-яке модифікування/деформування вашої моделі вздовж обраних локальних осей у ході її ліплення.
Плиткування -- Tiling
Використання цієї опції дозволяє вам безшовно плиткувати ваші штрих вздовж заданих осей. Це дозволяє створювати повторювані взірці.
Зсув Плиткування -- Tile Offset
Стандартний розмір плитки установлено як одна одиниця Blender'а -- BU. Цей зсув дозволяє змінити розмір плиткування вздовж заданих осей.
