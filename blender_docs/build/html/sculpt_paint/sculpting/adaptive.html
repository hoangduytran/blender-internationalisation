Адаптивне Ліплення -- Adaptive Sculpting¶

Динамічна Топологія -- Dynamic Topology¶

Орієнтир -- Reference

Mode:Sculpt Mode
Panel:Tool Shelf ‣ Tools ‣ Dyntopo
Hotkey:Ctrl-D

Dynamic topology (aka dyntopo) is a dynamic tessellation sculpting method, adds and removes details on-the-fly, whereas regular sculpting only affects the shape of a mesh.

Це дає змогу виліплювати складні форми з простої сіті, а не тільки просто додавати деталі на змодельовану базову сіть.

Динтопо можна увімкнути або вимкнути за допомогою стягу у заголовку однойменної панелі на Полиці Інструментів або скороченням Ctrl-D. З активною динамічною топологією більшість пензлів будуть підподіляти сіть у ході нанесення штриха ліплення.

Тип Деталізації -- Detail Type

Динтопо використовує три різні методи деталізації для створення динамічної детальності на об'єкті.

Відносна Деталізація -- Relative Detail
This method uses a detail size based on the number of pixels, and in turn will create topology in that size. Zoom out big details, zoom in small fine details.
Постійна Деталізація -- Constant Detail
Для збереження детальності однорідною по всьому об'єкту можна скористатися типом «Постійні Деталі». Така деталізація базується на відсотку від однієї одиниці Blender'а -- BU.
Деталізація Пензлем -- Brush Detail
Дає більше контролю над топологією, за допомогою цього методу ви можете створювати топологію на основі поточного розміру пензля. Ви можете збільшувати або зменшувати топологію, просто змінюючи розмір самого пензля. Такий розмір деталізації базується на розмірі власне пензля, де 100% створює топологію за розміром кільця самого пензля.
Detail Size/Percentage, Resolution Shift-D

Значення кожного Типу Деталізації задається тут. Залежно від використовуваного Типу Деталізації ця властивість буде показуватися як кількість пікселів (px) або відсоток.

Вибірка Розмір Деталізації -- Sample Detail Size (іконка з піпеткою)
With Constant Detail Size it is possible to sample the detail value of a certain mesh area by clicking the pipette icon next to the detail setting and then clicking on the area.
Метод Уточнення Деталізації -- Detail Refine Method

При використанні Динамічної Топології використовується певний метод, що визначає, як буде оброблятися топологія. Задання цієї опції визначатиме, який з цим методів буде вживатися при зміні топології.

Subdivide Edges
Just like the Subdivide tool, this method will only subdivide topology to match the detail given.
Collapse Edges
When topology is too dense, and is smaller than the detail given, edges will be collapsed to fit the detail size appropriately.
Підподіл і Стягнення -- Subdivide Collapse
Цей метод комбінує попередні два методи, підподіляючи ребра, менші, ніж розмір деталізації, та стягуючи топологію.
Згладжене Відтінення -- Smooth Shading
Toggles whether mesh faces are smooth or flat-shaded. In dynamic-topology mode all faces have the same type of shading.
Оптимізація -- Optimize
Якщо ліплення починається уповільнюватися при увімкненій динамічній топології, то скористайтеся кнопкою «Оптимізація» -- Optimize для перерахування ієрархії межових об'ємів -- BVH для ліплення.
Однакова Детальність -- Detail Flood Fill
При використанні режиму методу «Постійна Деталізація» -- Constant Detail ця опція стає доступною та дозволяє заповнити увесь об'єкт однорідною детальністю, базуючись на цьому розмірі деталізації.
Напрям -- Direction
Визначає, в якому напрямку модель буде симетризована.
Симетризація -- Symmetrize
Uses direction orientation to symmetrize. Since Dyntopo adds details dynamically it may happen that the model goes asymmetric, so this a good tool for that.

Multiresolution Modifier¶

The Multiresolution Modifier is needed to sculpt. The modifier will subdivide the mesh. The more subdivision the more computing will be needed. With the Blender stack non-destructive data, multi-resolution sculpting will help when you have a clean topology base mesh.

When sculpting with multiple resolutions you have the ability to sculpt in different levels of subdivision, this mean you can sculpt some details in subdivision level 1 and add more details in subdivision 2 and go back to subdivision 1 correct some mistakes. While this workflow is often used, the Multiresolution Modifier has some limitations. You may end up with some mesh distortions. As an advice, add as most details as possible before adding more subdivisions. Clay brush, SculptDraw work better with multi-resolution sculpting to sculpt secondary forms.

  • Перейти уверх на один рівень багатороздільності -- PageUp
  • Перейти вниз на один рівень багатороздільності -- PageDown
  • Задати рівень багатороздільності -- від Ctrl-0 до Ctrl-5
