Кнопки перемикання та радіокнопки¶

Стяги та перемикачі¶

Toggle buttons.

Ці кнопки використовуються для активації або дезактивації опцій. Використовуйте LMB для зміни їх стану.

On checkboxes a tick is shown when the option is activated. Toggle buttons are used to set an on/off status. When state is on, they appear pressed (dark). Clicking this type of button will toggle a state but will not perform any operation. Some toggle buttons have an icon version for each state.

Перетягування¶

Для зміни багатьох значень за раз ви можете клацнути, утримувати та перетягувати з натисненою LMB мишею по кількох кнопках. Це працює для стягів, перемикачів та для вибору значення радіокнопки.

Mẹo

Для кнопок шарів (тип перемикачів) часто корисно утримувати натисненою Shift для вмикання або вимикання показу кількох шарів одночасно.

Радіокнопки¶

Radio buttons.

Радіокнопки (radio button) використовуються для вибору з кількох «взаємовиключних» опцій.

Циклування¶

Use Ctrl-Wheel, while hovering with the mouse over it, to cycle between the options. Cycling works also for number buttons and select menus.
