Кнопки з числом¶

Кнопки з числом.

(груповані або одиничні).

Кнопки з числом (number buttons) або кнопки для вказування числових значень -- це кнопки, що містять числові значення.

Кнопки з числом можуть ідентифікуватися за наявністю трикутників на них, що вказують вліво (◂) та вправо (▸) по боках кнопки. Другий тип повзунків з числом має на своєму фоні смужку та використовується для значень у діапазоні, наприклад, відсоткових значень. Обидва типи мають заокруглені кутки. У більшості випадків вони містять ім'я та двокрапку, за якими слідує число. Це числом може редагуватися кількома способами:

Прирістні кроки
Для зміни значення покроково клацайте LMB по маленьких трикутниках (лише кнопки з числом).
Перетягування
Для зміни значення у широкому діапазоні утримуйте натисненою LMB та тягніть мишу вліво або вправо. Утримуйте натисненою клавішу Ctrl для прилипання змін до дискретних кроків або клавішу Shift для більш точного уводу, у ході перетягування.
Увід тексту

Press LMB or Return to edit the value as a text field.

При уведенні значень вручну кнопка цього типу працює подібно до будь-якого іншого поля уводу тексту:

  • Press Return or LMB outside the field to apply the change.
  • Натиснення Esc або RMB дасть скасування уводу значення.
  • Натиснення Tab дасть перехід до наступної кнопки з числом, а Ctrl-Tab -- до попередньої.
  • Натиснення Minus при нависанні курсора миші над кнопкою дає зміну значення на від'ємне.

Одночасне редагування кількох значень¶

Одночасне редагування кількох значень.

Number buttons can edit multiple values at once (object scale or render resolution for example). This can be done by clicking on the button and dragging vertically to include buttons above/below. After the vertical motion you can drag from side to side, or release the LMB to type in a value.

Ліміти¶

Most Number Buttons have two types of "limits" imposed on them. The first of these is a "soft limit", this means that the property cannot surpass the value of the "soft limit" without having to LMB and input the value with the numpad. The second is the "hard limit", this is the value that cannot be surpassed even by LMB and inputing a value.

Вирази¶

Ви можете також уводити у кнопки з числом математичні вирази, як 3*2 замість 6. або 5/10+3. Можуть використовуватися навіть константи, як pi (3.142), або функції, як sqrt(2) (квадратний корінь з 2).

Вирази як Драйвери¶

Ви можете хотіти, щоб ваш вираз повторно обчислювався після його уводу. Blender підтримує таку можливість за допомогою драйверів -- Drivers (функція системи анімації).

Вираз, що починається з #, має спеціальне використання. Замість обчислення значення та відкидання виразу додається драйвер для потрібної властивості з уведеним виразом.

Драйверний вираз #frame -- це швидкий спосіб доступу до карти, розкладки значень у поточному кадрі, але також підтримуються більш складні вирази, як приклад, #fmod(frame, 24) / 24.

Це просто зручне скорочення для додання драйверів, які також можуть додаватися через меню за RMB.

Одиниці виміру¶

Як і вирази, ви можете змішувати одиниці виміру (units) із числами; для того, щоб це працювало, одиниці необхідно задати в уставах сцени scene settings.

Для використання одиниць просто після числа додатково впишіть її абревіатуру або повне ім'я англійською.

Приклади включення дійсних одиниць:

  • 1cm
  • 1m 3mm
  • 1m, 3mm
  • 2ft
  • 3ft/0.5km
  • 2.2mm + 5' / 3" - 2yards

Ghi chú

Using Units

  • Коми є необов'язковими (для десяткових розрядів -- 1,000 -- обчислюється однаково, що й 1000).
  • Ви можете змішувати між собою метричні -- metric та імперські британські -- imperial одиниці, хоча показуватися результатна одиниця буде лише в одній з цих систем.
  • Множити англійських імен одиниць також розпізнаються, а тому можуть використовуватися як meter, так і meters.
