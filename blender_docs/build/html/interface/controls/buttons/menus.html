Меню¶

Blender використовує ряд різних меню для доступу до опцій та знарядь.

Меню у заголовку редакторів¶

Кнопки меню Редактора Інфо -- Info Editor.

Більшість заголовків -- headers редакторів мають набір меню, що розміщується зразу справа після першої іконки вибірника типу редактора -- Editor Type. Меню заголовку використовуються для конфігурування відповідного редактора та доступу до його інструментів. Усі записи меню показуються з відповідними скороченнями клавіш, якщо такі для них передбачені.

Згортання меню¶

Інколи корисно одержати деякий додатковий простір у заголовку, що можна зробити згорнувши меню і це можна, викликавши контекстне меню заголовку клацком RMB на ньому та увімкнути там опцію «Згорнути Меню» -- Collapse Menus.

Клацок ПКМ на будь-якому з меню заголовку.

Доступ до згорнутих меню через іконку згортання.

Меню вибору значень¶

Меню вибору режиму 3D-Огляду -- 3D View mode.

The Select menu or short selector lets you choose between a set of options. They can show a text and/or an icon. The options are shown in a pop-up. The selected option is then shown as active.

Спливні меню¶

Спливне меню Відтінювання Оглядвікна -- Viewport Shading.

Спливні меню на час показу перекривають інші елементи інтерфейсу. Вони викликаються або кнопкою, іконка якої містить справа два трикутники, що вказують уверх і вниз, або клацком відповідної кнопки миші в її поточній позиції.

Якщо вміст спливного меню надто великий, щоб вміститися на екрані, то показ його скорочується з вказівкою на це маленькими індикаторними трикутниками. При наведенні на них вказівника миші вони вмикають прокрутку такого спливного вікна.

Наприклад, кнопка «Відтінювання Оглядвікна» -- Viewport Shading викликає спливне меню з доступними опціями відтінення.

Вибір мишею
Клацайте LMB на бажаному елементі.
Вибір цифровою клавішею
You can use the number keys or numpad to input an item in the list to select. For example, Numpad1 will select the first item and so on.

Спливні меню можуть бути переміщені перетягом за їх смужку з назвою.

Скорочення¶

  • Використовуйте коліща миші Wheel, коли воно над спливним вікном, для зміни вказування елемента.
  • Клавіші зі стрілками можуть використовуватися для переходу для вказування елемента.
  • Кожен запис меню має підкреслений символ, натиснення відповідника якого на клавіатурі буде активувати цей запис.
  • Number keys or numpad can be used to access menu items. (Where 1 is the first menu item, 2 the second, etc. For larger menus Alt-1 the 11th... up to Alt-0 the 20th).
  • Press Return to activate the selected menu item.
  • Натиснення Esc , пересування курсора миші далеко від меню або клацок LMB дає вихід з цього спливного меню та його закриття.

Контекстне меню¶

Контекстне (context) меню -- це спливне меню, що викликається клацком RMB. Нижче наводяться лише загальні опції:

Слово «Одиничне» -- Single означає, що задається або отримується значення з одиничної кнопки під курсором миші. Слово «Усе» -- All, відповідно, -- значення комбінованих там кнопок.

Скинути Усе/Одинично до Стандартного Значення -- Reset All/Single to Default Value(s)
Замінює поточне значення стандартним, аналогічно діє натиснення Backspace.
Скид -- Unset
ToDo.
Копіювати Шлях Даних -- Copy Data Path
For scripting -- Copies the path of the Python property, relative to the data-block.
Копіювати На Виділене -- Copy To Selected
Копіює значення властивості на відповідну властивість виділеного об'єкта. Випадком використання є ситуація, коли контекст Редактора Властивостей -- Properties editor є пришпиленим.
Додати Скорочення -- Add Shortcut
Дозволяє вам визначити скорочення клавіш або кнопок миші та асоціювати його з цим контролером. Для визначення цього скорочення ви повинні спершу пересунути курсор миші над спливною кнопкою, що з'явиться, і коли з'явиться «Натисніть клавішу» -- "Press a key", натиснути та/або клацнути потрібні клавіші/кнопки скорочення.
Змінити Скорочення -- Change Shortcut
Дозволяє вам перевизначити наявне скорочення для цього контролера.
Вилучити Скорочення -- Remove Shortcut
Відв'язує, вилучає наявне скорочення для цього контролера.

Онлайнова Довідка з Python -- Online Python Reference
Контекстно-чутливий доступ до онлайнової довідки Python API Reference.

Меню «Спеціальне» -- Specials¶

The Specials pop-up menu contains a context-sensitive list of operators. It is opened by a button with a down arrow on dark background or with W in most editors giving quick access to tools sensitive to the editor's mode.

Кругові меню¶

Кругове (pie) меню -- це меню, чиї елементи розподілені по колу навколо курсора миші. Щоб вони були доступні, кругові меню повинні бути активовані у редакторі Уподобання Користувача -- User Preferences на вкладці «Додатки» -- Add-ons ‣ UI ‣ Pie Menus Official.

Кругове меню опцій відтінення.

Взаємодія¶

Кругове меню викликається натискання клавіші, яка задається в Уподобаннях для Додатку кругових меню -- Add-on Preferences.

Releasing the key without moving the mouse will keep the menu open and the user can then move the mouse pointer towards the direction of a pie menu item and select it by clicking. Releasing the key, after moving the mouse towards a pie menu item, will cause the menu to close and the selected menu item to activate.

Дисковий віджет у відкритому круговому меню в його центрі показує поточний напрямок для вибору елемента. Обраний елемент також підсвічується. Кругове меню може спрацьовувати й по напрямку для вибору елемента, якщо ми знаходиться всередині дискового віджета у центрі меню.

Кругове меню підтримує для елементів клавішні прискорювачі у вигляді підкреслених літер. Також для вибору елементів можуть бути використані цифрові клавіші.

Якщо дане кругове меню має під-меню, то це позначається іконкою із плюсом.

Дивіться устави Кругового меню -- Pie menu settings.
