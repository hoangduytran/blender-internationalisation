Регіони -- Regions¶

Редактор підподіляється на регіони -- region.

Головний Регіон -- Main Region¶

Принаймні один регіон є завжди видимим. Він називається головним регіоном -- main region і є найбільш помітною частиною редактора.

Кожен редактор має певне призначення, а тому головний регіон та наявність додаткових регіонів є відмінними між редакторами. Дивіться конкретну документацію про кожен редактор у главі «Редактори» -- Editors.

Заголовок -- Header¶

Заголовок -- header -- це маленька горизонтальна смуга зі світлішим сірим фоном, яка може знаходитися вгорі або внизу області. Усі редактори мають заголовок, що діє як контейнер для меню та загально використовуваних інструментів. Меню -- Menus та кнопки будуть змінюватися при зміні типу редактора, виділення об'єкта та режиму.

Заголовок -- Header редактора 3D Огляд -- 3D View.

Якщо ви пересунете мишу над областю, то заголовок її редактора змінює своє забарвлення на дещо світліший відтінок сірого. Це означає, що він «сфокусований» або «у фокусі». Усі скорочення, які ви натискаєте, відтепер впливають на вміст цього редактора. Заголовок може бути схованим за допомогою Alt-F9.

Полиця Інструментів -- Tool Shelf¶

Полиця Інструментів -- Tool Shelf стандартно знаходиться з лівого боку та містить устави інструментів. Натискання T перемикає видимість Регіону Полиці Інструментів.

Панель Оператора -- Operator Panel¶

Панель Оператора -- Operator panel -- це регіон, що є частиною Полиці Інструментів, та містить тільки одну панель. У 3D Огляді -- 3D View він показує властивості останньо виконаного оператора -- last operator, а також в Оглядачі Файлів -- File Browser він показує опції імпорту/експорту.

Регіон «Властивості» -- Properties Region¶

Регіон «Властивості» -- Properties Region стандартно знаходиться з правого боку. Він містить Панелі -- Panels з уставами для об'єктів у межах цього редактора та для самого редактора. Натискання N перемикає видимість Регіону «Властивості».

Впорядкування¶

Прокручування¶

Регіон може прокручуватися вертикально та/або горизонтально перетягуванням з утримуваною натисненою середньою кнопкою миші MMB. Якщо регіон немає жодного рівня зумування, то він може прокручуватися за допомогою коліщатка Wheel, при нависанні миші над ним.

Зміна розміру і ховання¶

Зміна розміру регіонів працює так само, як і для -- Області -- Areas, за допомогою перетягування їх границь.

Для схову регіону зменшуйте його розмір до самої границі. Схований регіон залишає маленький знак плюса (дивіться ілюстрацію нижче). Клацок LMB на ньому розкриває схований регіон.

Регіони Полиця Інструментів -- Tool Shelf та Властивості -- Properties мають скорочення для перемикання між схованим та показаним станами.

Ховання та показування Заголовка -- Header.¶

Позиція¶

Для перевертання позиції регіону з одного боку на інший натискайте F5, коли курсор миші знаходиться над ним.

Позицію заголовку також може бути перевернуто за допомогою клацку RMB на ньому та вибору відповідного пункту у спливному меню. Якщо заголовок знаходиться вгорі, то текст пунктів буде читатися зверху вниз, а якщо заголовок внизу - то знизу уверх.
