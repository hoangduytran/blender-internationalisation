Екрани -- Screens¶

Спливне меню блоку даних Екран -- Screen.

Screens are essentially predefined window layouts. Blender's flexibility with areas lets you create customized working environments for different tasks such as modeling, animating, and scripting. It is often useful to quickly switch between different environments within the same file. See area controls for how to move frame borders, split and consolidate areas.

Меню блоку даних з екранами, що дозволяє вам вибирати ці розстави, розміщене у заголовку Редактора Інфо.

Контролери¶

Розстава Екрану -- Screen Layout
Список доступних розстав екрану. Дивіться нижче Стандартні екрани.
Додання -- Add +
Клацок ЛКМ на кнопці «Додання» -- Add дає створення нової розстави на основі поточної.
Видалення -- Delete X
Ви можете видаляти обраний екран за допомогою кнопки «Видалення» -- Delete.

Gợi ý

Стандартно, кожна розстава екрану «запам'ятовує» останню сцену -- scene, в якій вона використовувалася. Вибір іншої розстави екрану буде перемикати на потрібну розставу та переходити на її пам'ятну сцену.

Скорочення¶

Для циклічного переходу між екранами використовуйте Ctrl-Right та Ctrl-Left.

Ghi chú

На macOS вам можливо необхідно буде вимкнути ці скорочення для "Mission Control" у ваших уподобання комп'ютера. Це може бути знайдено у System Preferences ‣ Keyboard ‣ Shortcuts.

Стандартні екрани¶

3D View Full -- 3D Огляд Повністю:
 3D Огляд на повний екран, використовується для передогляду вашої сцени.
Animation -- Анімація:
 Для створення рухів акторів та інших об'єктів, зміни форми або кольору тощо.
Compositing -- Компонування:
 Комбінування різних частин сцени (наприклад, фону, акторів, спеціальних ефектів) та їх фільтрування (наприклад, корекція кольору).
Default -- Стандарт:
 Стандартна розстава, використовувана Blender'ом для нових файлів. Корисна для моделювання нових об'єктів.
Game Logic -- Ігрова Логіка:
 Планування і програмування ігор у Blender'і.
Motion Tracking -- Простеження Руху:
 Used for motion tracking with the Movie Clip editor.
Scripting -- Скриптування:
 Документування вашої роботи та/або написання власних скриптів для автоматизації Blender'а.
UV Editing -- Правка UV:
 Розпластання проекції сіті об'єкта у 2D для керування розкладанням текстури на його поверхні.
Video Editing -- Правка Відео:
 Нарізання та монтування послідовностей анімації.

Збереження і заміщення¶

The screen layouts are saved in the blend-file. When you open a file, enabling the Load UI in the File Browser indicates that Blender should use the file's screen layouts and overriding the current layout. See Load UI.

Власний набір розстав екрану може бути збережений як частина запускового файлу -- Запусковий Файл -- Startup File.

Додаткові розстави¶

Як ви ставатимете більш досвідченим у роботі з Blender'ом, розгляньте питання додання деяких інших розстав екрану, відповідних для вашого робочого процесу, що могли б вам допомогти підвищити вашу продуктивність. Наприклад, це можуть бути:

Modeling -- Моделювання:
 Чотири 3D Огляди -- 3D Views (зверху, спереду, збоку і перспектива), Редактор Властивостей -- Properties editor для редагування.
Lighting -- Освітлення:
 3D Огляди -- 3D Views для переміщення освітлювачів, Редактор UV/Зображень -- UV/Image editor для показу результату зображування, рендерингу, Редактор Властивостей -- Properties editor для рендерингу, зображування та властивостей і контролерів ламп.
Materials -- Матеріали:
 Properties editor for Material settings, 3D View for selecting objects, Outliner, Library script (if used), Node Editor (if using Node-based materials).
Painting -- Малювання:
 Редактор UV/Зображень -- UV/Image Editor для малювання зображень текстур, 3D Огляд -- 3D View для малювання безпосередньо на об'єкті у режимі виділення UV граней -- UV Face Select, три міні 3D Огляди -- 3D Views внизу збоку, що заповнені фоновими орієнтувальними картинками, Редактор Властивостей -- Properties editor.
