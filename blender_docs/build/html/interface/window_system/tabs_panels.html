Вкладки й Панелі -- Tabs & Panels¶

Вкладки -- Tabs¶

Горизонтальний заголовок вкладок.

Вкладки «Інструменти» -- Tools (поточно вибрана), «Створення» -- Create і так далі.

Tabs are overlapping sections in the user interface. The Tabs header can be vertical (Tool Shelf) or horizontal (Properties Editor, User Preferences).

Switching/Cycling¶

Вертикальні вкладки можуть перемикатися за допомогою коліща Wheel у межах заголовку вкладок та Ctrl-Wheel - у межах регіону з вкладками.

You can also cycling through tabs with Ctrl-Tab and Shift-Ctrl-Tab.

Панелі -- Panels¶

Панелі Полиці Інструментів -- Tool Shelf.

Оранжевим: Заголовки Панелей.

Найменша організаційна одиниця в інтерфейсі користувача -- це панель -- panel. Панелі можуть бути згорнуті для схову їх вмісту. Вони використовуються у Редакторі Властивостей -- Properties Editor, але також у регіонах редакторів «Полиця Інструментів» -- Tool Shelf та «Властивості» -- Properties.

На ілюстрації показано три панелі: «Трансформації» -- Transform, «Правка» -- Edit та «Історія» -- History. Панель «Правка» -- Edit розгорнута, а інші ці дві панелі згорнуті.

Collapsing and Expanding¶

Трикутник зліва від назви показується у двох станах для панелі -- розгорнуто -- expanded (▼) та згорнуто -- collapsed (►).

  • Клацок LMB на заголовку панелі розгортає або згортає її.
  • Натискання A розгортає/згортає панель під курсором миші.
  • Клацання Ctrl-LMB на заголовку певної панелі згортає усі інші панелі й залишає розгорнутою лише цю.
  • Протягування з утримуваною натисненою LMB по заголовках даватиме розгортання або згортання кількох панелей за раз.

Позиція¶

Ви можете змінювати позицію панелі у межах її регіону, клацнувши та перетягнувши її з утримуваною натисненою LMB за віджет хапання (::::) в її верхньому правому кутку.

Пришпилення¶

Часто бажаним є огляд панелей одночасно з різних вкладок. Це може бути вирішено за допомогою пришпилення -- pinning панелей.

Пришпилена панель залишається видимою, незалежно від того, яка з вкладок вибрана. Ви можете пришпилити -- pin панель за допомогою утримуваної натисненою Shift та клацку ЛКМ на її заголовку або клацку RMB на її заголовку та вибору пункту «Пришпилення» -- Pin в контекстному меню.

На ілюстрації вище показано приклад пришпиленої панелі «Опції Сіті» -- Mesh Options.

Зумування¶

Фактор зумування (присунення-відсунення) цілого регіону може бути змінено клацанням і утримуванням натисненими Ctrl-MMB та рухом миші у межах цього регіону або використання NumpadPlus та NumpadMinus для присування та відсування його вмісту. Натискання Home (Показати Усе -- Show All) скидає зумування екрану/панелі, фокусованих під курсором миші.

Вирівняння¶

Вирівняння панелей у Редакторі Властивостей -- Properties Editor може бути змінено між вертикальним і горизонтальним. Для цього клацніть RMB десь у межах головного регіону Редактора Властивостей -- Properties Editor та вибрати або «Горизонтально» -- Horizontal, або «Вертикально» -- Vertical з контекстного меню. Майте на увазі, що панелі оптимізовані для вертикального вирівняння.
