Режим редагування штрихів¶

Enter Stroke Edit Mode with the Mode select menu in the 3D Views header or toggle the Enable Editing in the Grease Pencil panel (only available in 2D editors such as UV/Image editor, Node editor, etc.), or use D-Tab. In this mode, many common editing tools will operate on Grease Pencil stroke points instead.

These tools let you move and reshape Grease pencil strokes after they have been drawn.

Відкрийте вкладку «Нарисний Олівець» -- Grease Pencil на Полиці Інструментів -- Tool Shelf. Перегляньте інструменти на панелі «Правка Штрихів» -- Edit Strokes, показані тут:

Edit panel with Grease pencil strokes.

Виділення¶

Штрихи нарисного олівця формуються із серії з'єднаних між собою вершинних точок. Щоб робити зміни, спершу слід виділити точки на штрихах, які ви хочете редагувати. ВИ можете виділяти точки тільки на активному шарі. Виділені точки підсвічуються, як показано вище на ілюстрації.

Gợi ý

Задайте для шару товщину (thickness) штрихів як 1, щоб зробити його точки більш видимими.

Використовуйте мишу для виділення точок або одну з кнопок виділення на панелі, як описано тут -- Basic Selection.

Можуть використовуватися різні функції виділення подібні до тих, що доступні при редагуванні сітей:

Виділити Усе -- Select All A
Виділ Рамкою -- Border Select B
Виділ Кругом -- Circle Select C
Виділ Ласо -- Lasso Select Ctrl-LMB
Виділити Пов’язане -- Select Linked L, Ctrl-L
Виділити Більше -- Select More Ctrl-NumpadPlus
Виділити Менше -- Select Less Ctrl-NumpadMinus
Виділити Штрих -- Select Stroke Alt-LMB

Редагування¶

Заголовок¶

Some tools can be accessed through the 3D View header. e.g. Copy/Paste.

Калька -- Onion Skinning
Перемикає функцію Калька -- Onion Skinning.
Маска Виділу -- Selection Mask, Альфа -- Alpha
Дивіться нижче Додаткові опції.

Меню¶

Стоншення/Потовщення -- Shrink/Fatten Alt-S
Регулює значення натиску у виділених точках штриха. Це надає спосіб модифікування товщини штрихів за допомогою руху миші або її коліщатка Wheel.
Видалити Усі Активні Кадри -- Delete All Active Frames D-X
Видаляє усі штрихи в активному кадрі. Ця команда також доступна за допомогою D-X (будь-де), Shift-X (тільки у режимі «Правка Штрихів» -- Edit Strokes) або меню «НарОлівець > Видалення» -- menuselection:GPencil --> Delete. Це полегшує швидке позбавлення від каракулів.
Перемістити на Шар -- Move to Layer M
Може використовуватися для переміщення штрихів між шарами (включаючи й на новий шар).

Панель «Правка Штрихів» -- Edit Strokes¶

Орієнтир

Mode:Edit Stroke Mode
Panel:Tool Shelf ‣ Grease Pencil ‣ Edit Strokes
Меню:GPencil
Копіювання -- Copy Ctrl-C
Копіює виділені штрихи Нарисного Олівця (або фактично, точки і сегменти).
Вставляння -- Paste Ctrl-V
Вставляє з буферу попередньо скопійовані штрихи.
Вставити й Об'єднати -- Paste & Merge
Вставляє попередньо скопійовані штрихи та об'єднує їх з тими, що на активному шарі.
Видалення -- Delete X
Точки -- Points
Видаляє виділені точки, залишаючи проміжок у штриху.
Розчинення -- Dissolve
Видаляє і повторно з'єднує точки, щоб були відсутні проміжки у штриху.
Штрихи -- Strokes
Видаляє увесь штрих, що містить хоча б одну з виділених точок.
Кадр -- Frame
Видаляє кадр при здійсненні анімування нарисів -- Animating Sketches.
Дублювання -- Duplicate Shift-D
Робить копію виділених точок у тій же локації. Використовуйте мишу для пересуву -- Translate дублікатів в іншу позицію. LMB підтверджує їх поміщення у нову позицію. RMB виходить з цієї інтерактивної операції та вилучає зроблені дублікати.
Перемкнути Зацикленість -- Toggle Cyclic
Закриває або відкриває виділений штрих, додаючи сегмент між останньою і першою його точками.
Згин -- Bend Shift-W
Згинає виділений штрих між 3D-курсором і курсором миші.
Дзеркалення -- Mirror Ctrl-M
Mirrors selected strokes along one or more axes.
Зріз -- Shear Shift-Ctrl-Alt-S
Робить зріз форми виділених штрихів вздовж горизонтальної екранної осі.
Сферування -- To Sphere Shift-Alt-S
Переміщує виділені вершини назовні по сферичній формі навколо їх середньої точки.
Впорядкування Штрихів -- Arrange Strokes

Впорядковує виділ штрихів уверх/вниз за порядком рисування на активному шарі.

Bring Forward, Send Backward, Bring to Front, Send to Back

Перемістити у Колір -- Move to Color
Задає активний колір як новий колі для усіх виділених штрихів.
Subdivide
Adds a number of new points evenly between two or more points, to set the number of points to add use Number of Cuts.
Сполучення штрихів
Тип
Join Ctrl-J
Сполучає виділені штрихи.
Join & Copy Shift-Ctrl-J
Сполучає виділені штрих як новий штрих.
Залишити Проміжки -- Leave Gaps
Залишає проміжки між сполученими штрихами замість їх заповнення.
Перевернути Напрям -- Flip Direction
Перевертає старт і кінець штриха між собою.
Показати Напрями -- Show Directions
Показує напрямок нарису штриха за допомогою більшої зеленої точки у стартовій точці та меншої червоної точки у кінцевій точці штриха.
Перепроектувати Штрихи -- Reproject Strokes
Plane
Перепроектовує виділені штрихи з поточної точки зору для отримання усіх точок в одній площині знову. Це може бути корисним для виправлення проблем з випадковим переміщенням 3D-курсора або змінами оглядвікна.
Surface
Project strokes onto geometry, instead of only doing this in a planar (i.e. parallel to viewplane) way (experimental).

Interpolate Panel¶

Орієнтир

Mode:Edit Stroke Mode
Panel:Tool Shelf ‣ Grease Pencil ‣ Interpolate
Меню:GPencil ‣ Interpolate

The following two operators can be used for interpolating between a pair of Grease Pencil frames. To use these operators, there need to be Grease Pencil frames on both sides of the current frame. (Also, note that the current frame cannot be on one of the frames that the interpolation is occurring between).

Interpolate Ctrl-Alt-E
It allows you to interactively pick a new sketch interpolate from the neighboring sketches. This is equivalent to the Breakdown tool for armatures.
Sequence Shift-Ctrl-E
This fills the space between a pair of Grease Pencil frames with interpolated frames. It is equivalent to the Sample tool in the Dope Sheet Editor.
Remove Breakdowns
Can be used to delete generated "breakdown" keyframes (i.e. the small blue points).

Options¶

Інтерполювати Усі Шари -- Interpolate All Layers
Checkbox to interpolate all layers, not only active.
Інтерполювати Виділені Штрихи -- Interpolate Selected Strokes
Checkbox to interpolate only the selected strokes in the original frame.

Sequence Options¶

It is possible to control how the Interpolate Sequence tool blends between the two frames.

Linear
The tool will interpolate between the two frames at a constant rate.
Custom Curve
It is also possible to define a custom curve to get more fine-grained control over the interpolation using a curve widget.
Easing Equations
The Robert Penner easing equations (and associated controls) can also be used to control the interpolation speed/shape, just like for F-Curves or Keyframes (see also for more information).

Панель «Ліпка Штрихів» -- Sculpt Strokes Panel¶

Орієнтир

Mode:Edit Stroke Mode
Panel:Tool Shelf ‣ Grease Pencil ‣ Sculpt Strokes
Меню:GPencil ‣ Sculpt Strokes/Brushes
Hotkey:E-LMB

Різні інструменти для редагування штрихів Нарисного Олівця забезпечуються у формі пензлів, які можна використовувати для «малювання» або «ліплення» вигляду цих штрихів без потреби втомливого порядку дій правки «вибір-коригування-вибір-коригування».

Утримуйте натисненим E-LMB та тягніть з натисненою ЛКМ миші для ліплення.

Пензлі¶

Поточно впроваджені пензлі:

Згладження -- Smooth

Дозволяє вам вибірково послаблювати тремтіння/коливання та вибоїстість для прибирання безладних частин ваших штрихів.

Задіяти Натиск -- Affect Pressure
Використовуйте цю опцію для здійснення згладження по значеннях товщини штриха.
Товщина -- Thickness
Пензель «Товщина» може бути використаний для збільшення (Додання -- Add) або зменшення (Відняття -- Subtract) товщини частин штриха під курсором миші.
Сила -- Strength
Increase/decrease (Ctrl) the alpha value of the stroke. E.g. for creating fading effects.
Захоп -- Grab
Бере точки штриха, які підпадають під кільце пензля після його активації, та дозволяє вам пересувати цей відібраний набір точок.
Уштовх -- Push
Пензель «Уштовх» дуже подібний до пензля «Захоп» у тому, що також дозволяє користувачу пересувати точки штриха. Однак, на відміну від пензля «Захоп», пензель «Уштовх» не обмежений в оперуванні тільки першим набором точок під курсором при його активації. Натомість, при кожному русі пензлі точки, що поточно знаходяться під пензлем, будуть пересуватися, базуючись на величині останнього руху пензля.
Закрут -- Twist
Використовується для закручування частин/обертання точок штриха навколо курсора, створюючи ефект «круговерті». Він корисний для застосування низьких рівнів спотворення у точках штрихів. Радіокнопка «напрямку» -- Direction визначає, куди будуть обертатися точки -- за годинниковою стрілкою -- ЗаГС -- CW або проти годинникової стрілки -- «ПротиГС» -- CCW.
Щипок/Роздув -- Pinch/Inflate

Використовується для пересування точок від або до курсора миші.

Щипок -- Pinch
Пересуває точки до курсора.
Роздув -- Inflate
Штовхає точки від курсора.
Випадково -- Randomize
Randomizes the stroke attributes. e.g. with Position enabled it displaces the points randomly in screen space to create jittered/jagged lines.
Клонування -- Clone

Used to paste the previously copied points (in the copy-paste buffer on the active layer), located at the point where you clicked.

Утримуйте натисненою LMB та з натисненою ЛКМ тягніть для розміщення та регулювання вставлених штрихів. Центр штрихів наслідує рухи пензля/курсора («режим штампу»).

Вжити Спад -- Use Falloff
When the Use Falloff option is enabled, instead of moving all the newly pasted strokes by the same amount, only the points that are currently under the cursor get affected. Thus, in this mode of operation, the brush is closer to a Paste and Push operation instead ("Stamp and Smudge").

Загальні опції¶

Радіус -- Radius Shift-F/Wheel
Розмір пензля. Збільшення/зменшення розміру пензля за допомогою Shift-F, не в активному процесі ліплення, або Wheel -- у ході активного ліплення (наприклад, коли стилус вже натискає по планшету або вже натиснута кнопка миші).
Сила -- Strength Ctrl-F/Shift-Wheel
The Strength of the brush, can be changed by the pressure of the stylus. (In/decrease see Radius).
Вжити Спад -- Use Falloff
Вмикає лінеарний, прямолінійний спад для обчислення вплив пензля на точку. Тобто, точка ближче до середини пензля (тобто точка під курсором) буде більше підпадати під вплив пензля, ніж та, що ближче до його країв.
Direction Ctrl-E-LMB
Радіокнопка для інвертування впливу пензля.
Задіння -- Affect
Вмикає ліплення для позиції, сили (альфа значення) та товщини при активних пензлях «Згладження» -- Smooth та «Випадково» -- Randomize.

Додаткові опції¶

Маска Виділу -- Selection Mask
Використовується для обмеження дії пензля тільки по виділених точках.
Альфа -- Alpha Ctrl-H
Альфа-значення візуалізації виділених вершин. Видимість може перемикатися (схов/знесхов) за допомогою Ctrl-H.
