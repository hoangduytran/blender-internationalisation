Вступ¶

Вмикається «Нарисний Олівець» -- Grease Pencil клацком на кнопках «Рисування» -- Draw, «Лінія» -- Line, «Полі» -- Poly або «Стирання» -- Erase на Полиці Інструментів T. Новий шар буде автоматично доданий для рисування на ньому.

Новий шар може бути доданий на панелі «Шари Нарисного Олівця» -- Grease Pencil Layers у боковині «Властивості» -- Properties. Ця панель може також використовуватися для налаштування вигляду ліній олівця на цьому шарі. Зміни цих устав впливатимуть на усі штрихи на поточному шарі.

Приклад рисування Нарисним Олівцем у Blender'і.

Нариси Нарисного Олівця -- Grease Pencil можуть конвертуватися у редаговану геометрію та використовуватися у процесі анімування.

Рисування¶

Орієнтир

Mode:All Modes
Panel:Tool Shelf ‣ Grease Pencil ‣ Grease Pencil

Полиця Інструментів -- Tool Shelf надає ряд опцій для рисування Нарисним Олівцем -- Grease Pencil, які докладно описуються нижче.

Рисування -- Draw D-LMB
Рисує новий штрих (кілька коротких, з'єднаних ліній). Рисування штриха завершується, коли ви відпускаєте кнопку миші.
Лінія -- Line Ctrl-D-LMB
Рисує нову лінію у режимі гумової смуги. Рисування лінії завершується, коли ви відпускаєте кнопку миші.
Полі -- Poly Ctrl-D-RMB
Рисує з'єднані лінії за клацанням по позиціям, в яких ви хочете додати наступні точки. Лінії будуть автоматично додаватися для з'єднання двох точок. Утримування натисненою LMB і ковзання мишею дозволяє вам розмістити передогляд нової точки/сегмента. Транформування цієї точки блокується по осі X/Y, заданої початковим напрямком руху миші.
Стирання -- Erase D-RMB, Eraser

Стирає сегменти штрихів, що підпадають під радіус стирача-«пензля» (з прямолінійним спадом від центру кільця стирача). Стирання продовжується, допоки не буде відпущена кнопка миші, через поступове зменшення товщини штрихів перед повним їх вилученням. Стирач працює на усіх видимих та редагованих шарах. Якщо увімкнено режим «Стирання» -- Erase, то проведення як RMB, так і LMB буде стирати штрихи. Курсор стирача має вигляд червоного кільця з пунктирним контуром.

Розмір стирача-«пензля» може визначатися або коліщам миші Wheel, або клавішами NumpadPlus та NumpadMinus, при утримуванні натисненою RMB.

Insert Blank Frame D-B
This operator adds a new frame with nothing in it on the current frame. If there is already a frame there, all existing frames are shifted one frame later.
Delete Frame(s) D-X
Remove all active frames if they are not locked.

Доповняльне Рисування -- Additive Drawing¶

With the "Additive Drawing" option enabled the active frame's strokes will be carried over/copied if you start drawing on an empty frame (i.e. one without any keyframe already). This saves the effort of keeping a Dope Sheet open, and having to remember to duplicate the current frame before starting to draw the next pose (or risk managing to draw the perfect pose, but without everything else).

This option makes it easier to animate shots where you're building on a result from a previous frame. Examples of cases where this comes in handy include animating facial expressions (when all outlines are on the same layer), or animating "growing" things (e.g. vines, or concentric circles growing from a central point).

Ghi chú

Навіть без увімкнення цієї опції, її режим є стандартною поведінкою стирача на «порожньому» кадрі. Це полегшує отримання знімків, де ви просто змінюєте частини виразу обличчя або анімуєте ефект «стирача».

Безперервне Рисування -- Continuous Drawing¶

Безперервне Рисування -- Continuous Drawing дозволяє швидке нарисування «Нарисним Олівцем» -- Grease Pencil, коли бажано продовжити рисування кількох штрихів зразу. Тому, вам прийдеться утримування натисненою D лише для першого штриха. Окрім того, стяг «Безперервне Рисування» -- Continuous Drawing також увімкнено, якщо клавіша D відпускається, а кнопка LMB натискається. Стирач для одноразових стирань штрихів (RMB) все ще доступний. Зауважте, що стирання Стирачем -- Eraser після початку рисування доступне як за LMB, так і за RMB.

Use Esc or Return or clicking outside the current viewport (e.g. another region or editor) to exit the mode. Continuous drawing can be disabled using E key in order to get fast access to sculpt mode.

Рисування на Задньоплані -- Draw on Back¶

Нові штрихи переміщуються за вже нарисоване при підтвердженні їх рисування (опускаються вниз стосу).

Розміщення Штриха -- Stroke Placement¶

Панель «Нарисний Олівець» -- Grease Pencil на Полиці Інструментів.

Визначає, як штрихи розміщуються у 3D (або 2D) просторі.

Огляд -- View
Нові штрихи розміщуються в екранному просторі (2D) та блокуються, прив'язуються до огляду.
Курсора -- Cursor

New strokes are drawn in 3D space, with position determined by the 3D cursor and the view rotation at the time of drawing. Cursor is available as an option in the UV/Image Editor but it functions identically to the View option. (3D View only)

Блокування осі
Блокує проекцію певною віссю.
Поверхня -- Surface
New strokes are drawn in 3D space, with their position projected onto the first visible surface. (3D View only)
Штрих -- Stroke
New strokes are drawn in 3D space, with their position projected onto existing visible strokes. Note that strokes created with View are not in 3D space and are not considered for this projection. (3D View only)
Лише Кінцеві точки -- Only Endpoints
Застосовує уставу рисування тільки до кінцевих точок штриха. Частина штриха між кінцевими точками регулюється так, що вона лежить на площині, яка проходить між цими кінцевими точками.

The effect of different drawing settings on Grease Pencil strokes.

Mẹo

Примітки для користувачів планшетами:

  • Товщина штриха у певній точці залежить від натиску, що використовується для рисування тієї частини штриха.
  • «Стиральний» кінець стилуса може використовуватися для стирання штрихів.
Вмикання редагування
Дивіться -- Режим редагування штрихів. При увімкненні у верхньому правому куті редакторів показується перекриття.
