Пензлі¶

Рисувальні пензлі¶

Орієнтир

Mode:Stroke Edit Mode
Panel:Tool Shelf ‣ Grease Pencil ‣ Drawing Brushes
Пензлі
Список Огляди списків передуставлених пензлів. Ви можете перемикатися між пензлями за допомогою звичайних цифрових клавіш від 1 до 0. Вибраний рисувальний пензель -- це пензель у списку, розміщений у такій позиції.
Товщина -- Thickness
Ширина повно натиснутих штрихів у пікселів, стала в оглядвікні, тобто не змінена зумуванням. Ця товщина може бути нижчою, залежно від натиску.
Чутливість -- Sensibility
Adjust the sensitivity of the thickness to the pressure of the pen on the tablet. The pressure can be disabled by clicking the small button to the right.
Сила -- Strength
Similar to sensitivity, but affects the alpha value of the color. This parameter gives the effect of color fading or watercolor.
Випадковість -- Randomness
The properties for Sensitivity and Strength additionally have a randomness factor which can be enabled using the jagged line icon to the right of the number sliders.
Тремтіння -- Jitter
Визначає випадковість тремтіння у штриху.
Кут -- Angle

Визначає кут, коли товщина штриха буде 100%. Будь-яка зміна у напрямку буде давати зміну товщини.

Фактор -- Factor
Defines the effect for drawing angle changes in thickness.

Mẹo

The Angle and Angle Factor parameters allow you to create drawing brushes such as markers that change the thickness depending on the angle of drawing. This can help create a more natural-looking line.

Stroke Quality¶

These settings are per-brush settings that are applied after each stroke is drawn (when converting from 2D/screen space coordinates to 3D/data space coordinates). These are per-brush settings so that you can apply varying properties to different types of brushes. E.g. higher smoothing and/or subdivision for final "beauty", and less smoothing/subdivision for initial "blocking" strokes.

Згладження -- Smooth

Визначає, наскільки застосовується згладження (використовує такий же метод, як і пензель «Згладження» -- "Smooth"). Використовується для усунення зазублених країв та коливання/тремтіння руки.

Повтори згладження -- Iterations
Визначає, скільки разів застосовується згладження. На кожен додатковий етап здійсненого згладження, сила застосованого згладження зменшується на половину, тобто перший раз це буде 100% фактора згладження, далі 50%, потім 25% тощо. Ця устава є найбільш корисною для покращення якості важко підподілених штрихів, де багаторазові повтори згладження можуть допомагати скорочувати артефакти «гранування».
Кроки Підподілу -- Subdivision Steps

Визначає, скільки разів штрих буде підподілений. Кожен раз при підподіленні штриха, додаткові точки штриха додаються між кожною парою вже наявних точок штриха. Головне використання цієї устави -- зробити штрихи на вигляд менш «гранованими» (особливо великі швидко нарисовані штрихи). Штрихи підподіляються перед застосуванням згладження.

Випадковість -- Randomness
Amount of randomness to add to new strokes after subdivision.

Криві Пензлів -- Brush Curves¶

Орієнтир

Mode:Stroke Edit Mode
Panel:Tool Shelf ‣ Grease Pencil ‣ Brush Curves

Ця панель дозволяє вам коригувати параметри, використовувані планшетами для отримання персональних уподобань. Доступні криві, що можуть бути редагованими:

  • Чутливість -- Sensibility
  • Сила -- Strength
  • Тремтіння -- Jitter
