Шари¶

Орієнтир

Mode:All Modes
Panel:Properties region ‣ Grease Pencil Layers

Grease Pencil sketches are organized in layers, much like the image layers in the GIMP or Photoshop®. These layers are not related to any of the other layer systems in Blender.

Головне призначення цих шарів -- збирати нариси, пов'язані між собою певним значущим способоам (наприклад, «примітки блокування», «режисерські коментарі на блокуванні» або «настанови»). З цієї причини усі штрихи на шарі (не тільки ті, що зроблені після певної зміни) визначаються уставами шару щодо кольору, прозорості та товщини штрихів.

Управління шарми здійснюється на панелі «Шари Нарисного Олівця» -- Grease Pencil Layers на правій боковині «Властивості» -- Properties (викликається клавішею N).

Grease Pencil panel.

Дані Нарисного Олівця¶

Використовуйте наступні контролери для додання, вилучення або регулювання позиції шару у списку.

Джерело
Сцена -- Scene
Grease Pencil data attached to the current scene is used, unless the active object already has Grease Pencil data (i.e. old files).
Об'єкт -- Object
Grease Pencil data attached to the active object is used. This is required when using pre 2.73 add-ons.
Нарисний Олівець
Використовується для вибору блоку даних Нарисного Олівця, з якого будуть використовуватися шари. Про контролери блоку даних дивіться Меню блоку даних.
Активний Шар

Це Огляди списків шарів, прикріплених до кожних сцени або об'єкта.

Предок -- Parent (іконка кістки)
Вказує, що для даного шару задано предка -- parent.
Блокування -- Lock (іконка замка)
Блокує здатність поточного шару редагуватися.
Схов -- Hide (іконка ока) H
Ховає поточний шар у регіоні рисування.
Unlock Color (color wheel with arrow icon)
Знімає захист виділених кольорів від подальшого редагування та/або зміни кадрів.
Ізоляція -- Isolate (іконка замка) NumpadAsterix
Обмежує редагування тільки активного шару.
Ізоляція видимості -- Isolate (visible) (іконка ока)
Опція Ізоляції також впливає на видимість.
Спеціальне -- Specials
Дублювати Шар -- Duplicate Layer
Створює копію поточного шару.
Показати Усе -- Show All Alt-H
Робить усі сховані шари видимими.
Сховати Інші -- Hide Others Shift-H
Hides all non-selected layers.
Блокувати/Блокувати Усе -- Lock/Unlock All
Блокує/Розблоковує усі шари. Це може бути корисним для запобігання небажаним правкам.
Об'єднати Вниз -- Merge Down
Об'єднує поточний шар із шаром, що знаходиться під ним у списку.

Ghi chú

Стандартно, більшість операцій відбуваються тільки, коли активний шар підсвічений у списку.

Устави вигляду¶

Ці устави можуть використовуватися для зміни показу активного шару.

Безпрозорість -- Opacity
Міра прозорості шару.
Просвіт -- X-Ray
Робить лінії видимими на переді, коли вони проходять позаду інших об'єктів у сцені.
Показати Точки -- Show Points
Рисує стартові/кінцеві точки сегментів, що утворюють штрих.
Підбарвлення -- Tint
Колір -- Color
Колір підбарвлення шару.
Фактор -- Factor
Величина Кольору Підбарвлення -- Tint Color для шару.
Зміна Товщини -- Thickness Change

Відносна зміна у пікселях для застосування до товщини усіх штрихів на активному шарі (працює як модифікатор).

Застосування зміни товщини (іконка з рукою та випнутої лінії)
Якщо ця кнопка застосування натиснута, то зміна товщини застосовується та значення скидається у нуль.

Анімація¶

Предок -- Parent

A Ідентифікатор даних --- Data ID to select the parent object. The strokes of the layer will follow parent transformations.

Тип

Тип відношення приріднення -- parent relation.

Об'єкт -- Object, Арматура -- Armature, Кістка -- Bone

Заблокувати Кадр -- Lock Frame
Блокує поточний кадр для показу цього шару.
Видалити Кадр -- Delete Frame
Видаляє активний кадр для активного Шару Нарисного Олівця.

Калька -- Onion Skinning¶

Калька -- onion-skinning (дослівно «лушпиння цибулини»), також відоме як привиддя, привижання -- ghosting, допомагає аніматору показувати сусідні кадри як зникні сліди.

Приклад кальки для Нарисного Олівця.

Калька -- Onion Skinning
Стяг для увімкнення кальки.
Always Show (camera icon)

If enabled ghosts are displayed when scrubbing the view, playing back animation and/or in the OpenGL renders. There are two use cases where this is quite handy:

  • For creating a cheap motion-blur effect, especially when the before/after values are also animated.
  • If you've animated a shot with onion skinning enabled, the poses may end up looking odd if the ghosts are not shown (as you may have been accounting for the ghosts when making the compositions).
Use Custom Colors (color wheel icon)
Перемикає використання контролерів «Перед» -- Before та «Після» -- After для зміни кольорів привидів у заданих кадрах.
Перед -- Before / Після -- After
Колір -- Color
Колір штрихів перед/після поточного кадру.
Діапазон Перед/Після -- Before/After Range
Максимальна кількість кадрів з привидами для показу перед/після поточного кадру. 0 означає показ тільки попереднього/наступного нарису, а -1 не показує жодного кадру з привидами перед/після поточного.
