Кольори¶

Орієнтир

Mode:Stroke Edit Mode
Panel:Properties region ‣ Grease Pencil Colors
Палітра

A Меню блоку даних to select a palette, which is a set of colors. Switching palettes will change all strokes color in all layers if the same colors are linked.

Нове -- New +

Якщо існує більш однієї палітри, усі кольори старої палітри будуть перенесені у новостворену палітру.

  • Якщо колір існує у новій палітрі (має таке ж ім'я), то штрих пов'яжеться з новим кольором.
  • Якщо колір не існує у новій палітрі, то новий колір додасться у цю палітру, щоб зберегти штрих.
Кольори

Це Огляди списків кольорів, згрупованих у палітрі, пов'язаній як штрих або заливка кольорами. Якщо колір зі штрихами вилучається, то й всі штрихи цього кольору вилучаються. Будь-яка зміна кольору лінії або заливки, заповнення дадуть зміну будь-якого штриха на будь-якому шарі. що використовує цей колір. Палітра повинна містити принаймні один колір, а тому останній в ній не може бути видалений.

Штрих -- Stroke
Задає колір лінії та максимальну безпрозорість (на яку також впливають устави пензля).
Заповнення -- Fill
Задає колір внутрішнього простору у межах замкненого штриха. Збільшення цієї безпрозорості від нуля робить заповнення, заливку видимим. Заповнення, заливка працює найкраще на опуклих формах, якщо ви не використовуєте увімкнення опції «Високоякісне Заповнення» (дивіться нижче).
Об'ємні Штрихи -- Volumetric Strokes
An alternative drawing technique by drawing strokes as a series of filled screen-aligned disks. Get best results with partial opacity and large stroke widths.
Високоякісне Заповнення -- High Quality Fill
Використовує кращий алгоритм заповнення, що працює краще для угнутих рисунків.
