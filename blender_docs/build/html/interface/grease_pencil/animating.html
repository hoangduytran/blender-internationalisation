Анімування нарисів¶

Ви можете використовувати Нарисний Олівець для створення 2D-анімацій (наприклад, у стилі біглих картинок) та змішувати їх з 3D-об'єктами та композицією.

Sketches are stored on the frame that they were drawn on, as a separate drawing (only on the layer that they exist on). A keyframe is automatically added per layer. Each drawing is visible until the next drawing for that layer is encountered. The only exception to this is the first drawing for a layer, which will also be visible before the frame it was drawn on.

Таким чином, просто зробити олівцевий тест/серію анімованих нарисів:

  1. Перейти на перший відповідний кадр. Нарисувати.
  2. Перейти на наступний відповідний кадр. Нарисувати ще щось.
  3. Keep repeating process, and drawing until satisfied. Voilà! Animated sketches.

Режим «Нарисний Олівець» -- Grease Pencil у редакторі Аркуш Експозицій -- Dope Sheet.

Компонування¶

The Grease pencil layers create a pass inside OpenGL render result. This result can be exported to EXR multilayer and used in composition.
