Загальні скорочення¶

Існує ряд скорочень клавіш, загальних для всієї програми серед багатьох типів.

Миша¶

In Blender the RMB (Right Mouse Button) is generally used for selection and the LMB (Left Mouse Button) initiates or confirms actions.

RMB Для виділення елемента.
Shift-RMB Для додання елементів у виділене.
LMB Для здійснення дії з виділеним.

Відео: Дізнайтеся більше про використання клавіш миші у Blender.

Ghi chú

Існує кілька особливих випадків, коли LMB використовується для виділення. Наприклад, для виділення та вибору файлів в Оглядачі Файлів -- File Browser.

Нависання¶

При нависанні (коли курсор утримується над кнопкою).

Властивості¶

  • Ctrl-C -- Copy the (single) value of the button.

  • Ctrl-V -- Paste the (single) value of the button.

  • Ctrl-Alt-C -- Copy the entire vector or color of the button.

  • Ctrl-Alt-V -- Paste the entire vector or color of the button.

  • RMB -- Відкриває контекстне меню.

  • Backspace -- Зчищає значення (установлює в нуль або очищає текстове поле).

  • Minus -- Починає увід від'ємних значень (множення на -1.0).

  • Ctrl-Wheel -- Змінює значення прирістними кроками.

    Для кнопок спливних опцій меню, це зациклює зміну значення.

  • Return -- Activates menus or toggles the value.

  • Alt -- Hold while editing values to apply the change to all selected items (objects, bones, sequence-strips).

    This can be used for number buttons and toggles.

Анімація¶

  • I -- Вставляє ключкадр -- keyframe.
  • Alt-I -- Зчищає наявний ключкадр.
  • Shift-Alt-I -- Clear all keyframes (removing all F-Curves).
  • Ctrl-D -- Призначає драйвер.
  • Ctrl-Alt-D -- Зчищає наявний драйвер.
  • K -- Додає Набір Ключування -- Keying Set.
  • Alt-K -- Зчищає наявний Набір Ключування.

Скриптування на Python¶

  • Ctrl-C -- При нависанні над будь-якими кнопками -- Кнопки запуску операцій копіює їх команду на Python в обмін-буфер.

    This can be used in the Python Console or in the Text editor when writing scripts.

  • Shift-Ctrl-C -- Over property buttons copies their data path for this property (also available from the context menu).

    Корисно при записуванні драйверів або скриптів.

  • Shift-Ctrl-Alt-C -- Over property buttons copies their full data path for the data-block and property.

    Зауважте, що у більшості випадків -- це найкращий доступ до значень на основі контексту, а не за іменем.

Перетягування¶

  • Ctrl -- При перетягу дає прилип до дискретних кроків.
  • Shift -- Дає точний контроль за значенням.
  • Shift-Ctrl -- Precise snap will move the object with high precision along with the snapping constraint.

Редагування тексту¶

  • Home -- Переходить на початок.
  • End -- Переходить на кінець.
  • Left, Right -- Пересуває курсор на одиничний символ.
  • Ctrl-Left, Ctrl-Right -- Пересуває курсор на ціле слово.
  • Backspace, Delete -- Видаляє символи.
  • Ctrl-Backspace, Ctrl-Delete -- Видаляє слова.
  • Shift -- При утриманні натисненою цієї клавіші та руху курсора в тексті виділяє.
  • Ctrl-A -- Виділяє увесь текст.
  • Ctrl-C -- Копіює виділений текст.
  • Ctrl-X -- Вирізає виділений текст.
  • Ctrl-V -- Вставляє текст у позицію курсора.

Підтвердження і скасування¶

  • Esc, RMB -- Скасовує.
  • Return, LMB -- Confirms.
