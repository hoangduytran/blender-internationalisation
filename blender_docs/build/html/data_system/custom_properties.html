Власні Властивості -- Custom Properties¶

Custom Properties panel.

Custom properties are a way to store your own metadata in Blender's data-blocks which can be used for rigging (where bones and objects can have custom properties driving other properties), and Python scripts, where it's common to define new settings not available in Blender.

Тільки певні дані підтримують власні властивості:

  • Усі типи блоків даних -- data-blocks types.
  • Кістки та позовані кістки.
  • Смужки відеорядника.

To add a custom property, find the Custom Properties panel, found at the bottom of most Properties Editor or Properties region, and hit Add.

Редагування властивостей¶

Інтерфейс Користувача -- User Interface¶

Власні властивості можуть бути редаговані за допомогою панелі, доступної для типів даних, що підтримують їх.

Custom Properties edit pop-up.

Ім'я Властивості -- Property Name
The name of the custom property.
Значення Властивості -- Property Value

This does two things, first it sets the default value of the custom property second, it define the type of property it is. For example, custom properties can either be of type: Integers, Floats, or Booleans. See the table below for a list of examples for each:

Integers:1, 2, 3, 4...
Floats:3.141, 5.0, 6.125,
Booleans:True, False

Ghi chú

Booleans are handled very similar to integers and only work when using Min/Max values that are integers and that are no more than 1 apart.

At this point, the booleans will still look like integers but behave like booleans having one lower, off, value and a higher, on, value.

Мін, мінімум -- Min
Мінімальне значення власної властивості, яке вона може приймати.
Макс, максимум -- Max
Максимальне значення власної властивості, яке вона може приймати.
Вжити М'які Ліміти -- Use Soft Limits

Вмикає ліміти, у межах яких повзунок устави «Значення Властивості» -- Property Value може коригуватися без потреби уведення числового значення з клавіатури.

М'який Мін -- Soft Min
Мінімальне значення для м'якого ліміту.
М'який Макс -- Soft Max
Максимальне значення для м'якого ліміту.
Підказка -- Tooltip
Дозволяє вам записати тут власну Підказку -- Tooltip для вашої властивості.

Доступ до Python¶

Власні властивості можуть бути доступні аналогічно у словниках -- dictionaries, що визначають, які тільки ключі можуть бути рядками, та які тільки значення можуть бути рядками, числами, масивами та вкладеними властивостями.
