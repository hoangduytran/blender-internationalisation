Властивості Сцени -- Scene Properties¶

Сцена -- Scene¶

Reference

Panel:Properties editor ‣ Scene ‣ Scene
Камера
Вживається для вибору, яка камера буде використовуватися як активна камера. Ви можете також задати активну камеру у 3D Огляді -- 3D View за допомогою Ctrl-0.
Фон -- Background

Дозволяє вам використовувати сцену як фон, це типово корисно, коли ви хочете зосередитися на анімуванні передньоплану, наприклад, без перешкоджання елементів фону.

This scene can have its own animation, physics simulations, etc, but you will have to select it from the Scene data-block menu, if you want to edit any of its contents.

Набори можуть самі мати набір фону (вони рекурсивно включені). Тому, ви можете завжди робити доповнення в наявні сцени, використовуючи їх як фон для ново створених сцен, де ви доповнення зробили.

Mẹo

Це може також бути використано у комбінації з пов'язування до сцени -- Linking to a Scene, при якому один з blend-файлів міститиме середовище, яке може повторно використовуватися в багатьох місцях.

Активний Кліп -- Active Clip
Активний відеокліп для рисування оглядвікна та примусів.

Одиниці -- Units¶

Reference

Panel:Properties editor ‣ Scene ‣ Units
Передустави Довжин -- Length Presets
Загальні одиниці виміру для використовування.
Довжина -- Length
Нема -- None
Використовує Одиниці Blender'а -- Blender Units.
Метрична -- Metric, Імперська -- Imperial
Стандартна одиниця вимірювання для довжин.
Кут -- Angle

Стандартна одиниця для вимірювання кутів.

Градуси -- Degrees, Радіани -- Radians

Mẹo

Коли ви використовуєте градуси -- Degrees, то значення у радіанах також показується у підказці.

Масштаб Одиниці -- Unit Scale

Фактор масштабування при конвертуванні між одиницями Blender'а -- Blender Units та Метричною/Імперською -- Metric/Imperial системами виміру.

Mẹo

Зазвичай, ви схочете використовувати передустави довжини -- Length presets для зміни фактора масштабування, оскільки це не вимагає пошуку значень для конвертування.

Відокремити Одиниці -- Separate Units
При показі одиниць у системах «Метрична» -- Metric або «Імперська» -- Imperial вимір буде відображатися одночасно у вигляді кількох одиниць, наприклад, "2.285m" стане показуватися як "2m 28.5cm".

Аудіо -- Audio¶

Reference

Panel:Properties editor ‣ Scene ‣ Audio

Опції для керування глобальними уставами аудіо -- відтворення звуку.

Гучність -- Volume
Гучність для сцени.
Оновити Кеш Анімації -- Update Animation Cache
Оновлює кеш аудіо для анімації. Це корисно, якщо ви починаєте помічати артефакти при відтворенні аудіо.

Модель Відстані -- Distance Model¶

Модель Відстані -- Distance Model
Distance model to use for sound attenuation (how sound volume diminishes over distance). For an exact description of each option see the OpenAL documentation.
Швидкість -- Speed
Швидкість звуку для розрахунків ефекту Доплера -- Doppler effect.
Доплер -- Doppler
Фактор висоти для розрахунку ефекту Доплера.

Формат -- Format¶

These settings, along with the settings found in the Encoding Panel change how sound is exported while rendering.

To control how sounds plays back from within Blender, see the audio settings in the User Preferences.

Канали -- Channels
Sets the audio channel count. Available options are: Mono, Stereo, 4 Channels, 5.1 Surround, 7.1 Surround.
Частота змішання -- Mix Rate
Sets the audio sampling rate.
