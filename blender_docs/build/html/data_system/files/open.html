Відкриття файлу -- Opening Files¶

The File Browser.

Використання¶

Орієнтир -- Reference

Menu:Info Editor ‣ File ‣ Open
Hotkey:Ctrl-O or F1

Текстове поле вгорі показує поточний шлях до каталогу з файлом, а текстове під ним містить ім'я обраного файлу.

Cảnh báo

For Linux and macOS Users

При виході ви не попереджуєтеся про потребу збереження ще незбережених змін у файлі, з яким ви працювали. А тому, переконайтеся, що зберегли свою роботу.

На MS-Windows передбачена опція Збереження і Завантаження -- Save & Load попередження про вихід з програми.

Опції¶

Завантажувати Інтерфейс Користувача -- Load UI

При увімкненні опції Load UI буде завантажена розстава екрану, збережена у відкриваному blend-файлі, та замінить поточну розставу. Інакше, розстава екрану з відкриваного файлу буде проігнорована.

Mẹo

If you want to work on the blend-file using your own defaults, start a fresh Blender, then open the File Browser and turn off the Load UI button, and then open the file.

Довірене Джерело -- Trusted Source
При увімкненні, скрипти Python та драйвери, що можливо включені у відкриваний файл, будуть запускатися автоматично. Вмикайте це тільки, якщо ви створили цей файл самі, або ви довіряєте особі, що дала вам його і ви знаєте, що він не містить жодного зловмисного коду. Про конфігурування стандартних опцій довіри дивіться тут -- Python Security.

Інші опції відкриття файлу¶

З меню «Файл» -- File ви можете також відкривати файли за допомогою наступних засобів:

Відкрити Останнє -- Open Recent
Містить перелік останньо використаних файлів. Клацок на одному з них дасть його завантаження у програму.
Відновити Останню Сесію -- Recover Last Session
Це завантажує файл quit.blend, який Blender автоматично зберігає перед виходом. Ця опція дозволяє вам відновлювати вашу останню сесію роботи, якщо, наприклад, ви випадково закрили Blender.
Відновити Авто Збереження -- Recover Auto Save
Це дозволить вам відкрити автоматично збережений файл для його відновлення.
