Hệ Thống Trợ Giúp -- Help System¶

Blender có nhiều tùy chọn trợ giúp dựa trên web và tích hợp sẵn.

Chú Giải Công Cụ -- Tooltips¶

Chú giải công cụ của nút lựa chọn Bộ Máy Kết Xuất (Renderer) trong trình Biên Soạn Thông Tin (Info Editor).

Khi di chuột qua một nút bấm hoặc một cài đặt, sau một vài lần di, mình sẽ thấy một chú giải công cụ xuất hiện.

Các yếu tố -- Elements¶

Chú giải công cụ theo ngữ cảnh có thể chứa một số yếu tố sau:

Miêu Tả Vắn Tắt -- Short Description
Chi tiết liên quan còn tùy thuộc vào điều khiển nói đến.
Tổ Hợp Phím Đường Tắt -- Shortcut
Bố trí đường tắt bàn phím hoặc chuột liên quan đến công cụ.
Giá Trị -- Value
Giá trị của tính chất.
Thư Viện -- Library
Tập tin nguồn của vật thể đang hoạt động. Xin xem thêm trong mục Linked Libraries.
Đã Bị Giải Hoạt (Tắt) (màu đỏ) -- Disabled (red)
Lý do tại sao giá trị không thể chỉnh sửa được.
Python
Đối với việc Viết tập lệnh -- scripting -- Một lệnh Python được liên kết với điều khiển (thường là một toán tử (operator) hoặc một tính chất (property)).

Truy Cập Hướng Dẫn Sử Dụng Nhạy Cảm Với Ngữ Cảnh Vận Hành -- Context-Sensitive Manual Access¶

Tham Khảo/Tra Cứu -- Reference

Chế độ:Tổng thể -- All modes
Trình đơn:NCP, Hướng dẫn sử dụng trực tuyến, trên mạng -- Online Manual
Phím nóng:Alt-F1

Bạn có thể muốn truy cập trợ giúp đối với một công cụ hoặc một khu vực nào đó ngay từ trong Blender.

Sử dụng phím tắt hoặc trình đơn ngữ cảnh (context menu -- bấm phải chuột) để truy cập các trang trong bản hướng dẫn sử dụng và tra cứu này trong Blender. Thao tác này sẽ mở một trang web liên quan đến nút bên dưới con trỏ, hỗ trợ cả nút công cụ và giá trị.

Ghi chú

Hiện tại chúng tôi chưa hoàn toàn quản lý được 100%, bạn có thể nhìn thấy cảnh báo trong thanh tiêu đề thông tin (info header) nếu một số các công cụ nào đó không có liên kết đến bản hướng dẫn sử dụng.

Có những lúc các nút bấm có thể sẽ liên kết đến các phần chung chung hơn của bản tài liệu hướng dẫn sử dụng này.

Tham Chiếu API của Python -- Python API Reference
Tham chiếu -- Reference -- cho Giao Diện Lập Trình Ứng Dụng Python (API : Application Programming Interface).
Bảng Liệt Kê Operator -- Operator Cheat Sheet
Tạo khối dữ liệu văn bản OperatorList.txt (Bảng Liệt Kê Operator), mà bạn có thể truy cập trong trình Biên Soạn Văn Bản -- Text Editor. Bạn còn có thể sử dụng chức năng Blender Search (Lùng Tìm Blender) để tạo tập tin nữa. Nó liệt kê các Operator (toán tử) Python có sẵn để sử dụng.

Lưu Thông Tin Hệ Thống -- Save System Info¶

Truy cập Help (Trợ Giúp)‣ Save System Info (Lưu Thông Tin Hệ Thống).

Thao tác này trích xuất thông tin hệ thống có thể hữu ích để đưa vào các bản báo cáo lỗi phần mềm, kiểm tra cấu hình hoặc chẩn đoán sự cố.

Bạn sẽ được nhắc để lưu tập tin văn bản system-info.txt.

Tập tin văn bản có chứa các phần sau:

Blender
Phần này cho bạn thấy số phiên bản của Blender, chi tiết về cấu hình xây dựng (biên dịch) và đường dẫn mà bản Blender đang hoạt động sử dụng.
Python
Phiên bản Python bạn đang sử dụng, hiển thị các đường dẫn về ngôn ngữ lập trình Python.
Các Thư Mục -- Directories

Các đường dẫn sử dụng cho các tập lệnh, các tập tin dữ liệu, các giá trị đặt trước và các tập tin tạm thời.

Những thư mục đó được cấu hình bằng cách sử dụng trình Biên Soạn Tùy Chọn Người Dùng -- User Preferences.

OpenGL
Phần này hiển thị phiên bản OpenGL, tên của nhà sản xuất và liệt kê các khả năng của phần cứng và trình điều vận (driver) của bạn.
