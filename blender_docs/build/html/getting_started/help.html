Hệ Thống Trợ Giúp -- Help System¶

Blender має діапазон вбудованих та веб-базованих опцій довідки.

Спливні підказки -- Tooltips¶

Tooltip of the Renderer selector in the Info Editor.

При наведенні курсора миші над кнопкою або уставою після деякого часу з'являється спливна підказка -- tooltip.

Елементи спливної підказки¶

Контекстно-чутлива спливна Підказка може містити деякі з таких елементів:

Короткий опис
Пов'язані деталі залежно від цього контролера.
Скорочення
Скорочення для виклику команди цього засобу за допомогою клавіатури та/або миші.
Значення
Значення властивості цього контролера.
Library
Source file of the active object. See also Пов'язані Бібліотеки -- Linked Libraries.
Disabled (red)
The reason why the value is not editable.
Python
Для скриптування scripting -- команда на мові Python, асоційована з цим контролером (зазвичай оператора або властивість).

Context-Sensitive Manual Access¶

Орієнтир

Mode:All modes
Menu:RMB, Online Manual
Hotkey:Alt-F1

Ви можете захотіти отримати доступ до довідки у цьому посібнику для певного засобу або області із самого Blender'а.

Використовуйте вищенаведені скорочення клавіш або контекстне меню для відвідування відповідної сторінки у цьому посібнику, яка буде викликатися прямо з Blender'а та відкриватися у вашому стандартному веб-оглядачі. Це відкриває веб-сторінку, відповідну кнопці чи уставі під курсором миші.

Ghi chú

Ми поточно не маємо 100% готових довідок для всіх елементів інтерфейсу, тому ви можете про це бачити повідомлення у заголовку Редактора Інфо, якщо певний засіб не має посилання на цей довідник.

Іноді, при відсутності точної довідки, такий доступ може мати посилання на більш загальні розділи цієї документації.

Меню «Довідка» -- Help Menu¶

Меню «Довідка» -- Help знаходиться у заголовку Редактора Інфо -- Info Editor.

Help menu.

Веб-посилання¶

The first options of this menu provide direct links to Blender-related websites: The same links can also be found in the Екран Заставки -- Splash Screen.

Посібник -- Manual
Це посилання на офіційний посібник Official Blender Manual, який ви зараз читаєте.
Огляд Випуску -- Release Log
The release notes on the Web for the changes made for the current Blender version.
Веб-сайт Blender'а -- Blender Website
Домашня сторінка blender.org .
Сховище Blender -- Blender Store
The Blender Store, where you can buy Training DVD's, books, t-shirts and other products.
Спільнота Розробників -- Developer Community
Сторінка blender.org Get Involved . Це запускова сторінка про розробку програми Blender'а, простеження вад, латки та скрипти, навчання і тренування, розробку документації та пошук функціональності.
Спільнота Користувачів -- User Community
Списки багатьох різних майданчиків підтримки.
Повідомити про Ваду -- Report a Bug
Сторінка простежника вад -- Blender Bug Tracker (необхідна реєстрація).

Веб-оглядач та з'єднання з Інтернет

Деякі форми Довідки запускають ваш стандартний веб-оглядач та здійснюють доступ до веб-серверів Blender Foundation. Тому, для цього, ви повинні мати налаштований стандартний веб-оглядач на вашій операційній системі та мати з'єднання до Інтернету.

Посилання щодо скриптування¶

Довідка з Python API -- Python API Reference
Довідка з прикладного програмного інтерфейсу мови Python -- Python application programming interface (API) Reference.
Шпаргалка про Оператор -- Operator Cheat Sheet
Creates the OperatorList.txt text data-block, which you can access in the Text Editor. You can also use Blender Search to generate the file. It lists the available Python operators.

Зберегти Системну Інформацію -- Save System Info¶

Доступ через «Довідка > Зберегти Системну Інформацію» -- Help ‣ Save System Info.

Це витягує системну інформацію, яка може бути корисною для включення у звіт про вади, інспектування конфігурації або діагностування проблем.

Вам буде запропоновано зберегти її як текстовий файл system-info.txt.

Цей текстовий файл містить розділи:

Blender
Цей розділ показує вам цю версію Blender'а, деталі про конфігурацію її будови та шлях, по якому запускається Blender.
Python
Версія про Python, який ви використовуєте, з показом шляхів до Python.
Directories

Paths used for scripts, data files, presets and temporary files.

Ці теки конфігуруються у редакторі Уподобань Користувача -- User Preferences.

OpenGL
Цей розділ показує версію OpenGL, назву виробника, список здатностей вашого устаткування та драйвер.
