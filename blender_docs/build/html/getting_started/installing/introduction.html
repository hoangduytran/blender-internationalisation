Вступ¶

Blender доступний для звантаження для Linux, macOS чи MS-Windows.

Мінімальні вимоги¶

Перевірте, чи ваша система відповідає мінімальним або рекомендованим вимогам.

Завжди перевіряйте, чи оновлені графічні драйвери і чи підтримується OpenGL.

Підтримка для іншого устаткування, як графічні планшети та 3D-миші, розкриваються пізніше у розділі Підтримуване Устаткування -- Supported Hardware.

Звантаження Blender'а¶

Blender Foundation розповсюджує Blender у три різні способи, з яких ви можете вибирати потрібний комплект.

The options comprise binary packages for all the supported platforms and the source code. Within the binary packages, you can choose from a stable release or a daily build. The first has the benefit of being more reliable, the latter provides the newest features, as they are developed. Blender is released approximately every three months. You can keep up to date with the latest changes through the release notes.

Останній Стабільний Випуск -- Latest Stable Release
Це бінарний дистрибутив найостаннішої версії Blender'а. Він вважається стабільним і без регресій.
Щоденні Побудови -- Daily Builds
Це бінарний дистрибутив Blender'а, що оновлюється щоденно для включення найновіших змін з розробки. Ці версії не є ретельно тестовані, як стабільний випуск, і можуть «падати», хоча вони офіційні та загалом не високо експериментальні.
Побудова з Джерела -- Build from Source

Джерело Blender'а є доступним для орієнтування та встановлення з наступними перевагами:

  • Blender is always up to date.
  • It allows access to any version or branch where a feature is being developed.
  • It can be freely customized.

Ghi chú

Це дається для повноти картини, але не очікується, що звичайні користувачі повинні компілювати свої власні побудови Blender'а.

Встановлення Blender'а¶

Процедура для встановлення бінарної, останнього стабільного випуску або щоденної побудови, є однаковою. Для встановлення здійсніть наступні кроки залежно від вашої операційної системи:

  • Встановлення на MS-Windows
  • Встановлення на macOS
  • Встановлення на Linux
