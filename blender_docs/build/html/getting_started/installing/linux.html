Встановлення на Linux¶

Перевірте мінімальні вимоги й де взяти Blender, якщо ви цього ще не зробили.

Specific Packages for Distributions¶

Деякі дистрибуції Linux мають свої репозиторії специфічного пакету для Blender.

Встановлення Blender'а через рідні механізми розповсюдження гарантує узгодженість з іншими пакетами на вашій системі та може надавати інші функції (дані менеджером пакетів), такі як список пакунків, повідомлення про оновлення та автоматичне конфігурування меню. Однак, майте на увазі, однак, що такий пакет може бути застарілим у порівнянні з найостаннішим офіційним випуском або не включати деякі функції Blender'а. Наприклад, деякі дистрибуції не будуть Blender з підтримкою CUDA через причини ліцензування.

Якщо є конкретний пакет для вашої дистрибуції, ви можете вибрати найбільш бажаний та зручний для вас, інакше, немає нічого поганого у використанні офіційного бінарника з blender.org.

Звантаження з blender.org¶

Download the Linux version for your architecture and uncompress the file to the desired location (e.g. ~/software or /usr/local).

Blender може бути тепер запущено подвійним клацком на його виконуваному файлі.

For easy access, you can configure your system by adding a menu entry or shortcut for Blender and associate and open blend-files with Blender when opening from the File browser. These settings typically belong to the Window Manager (KDE, Gnome, Unity).

Running from the Terminal¶

Для виконання Blender'а з терміналу без потреби переходу у каталог з виконуваним файлом до дайте шлях до витягненої теки у середовищну змінну PATH.

Додайте наступну команду у ~/.bashrc чи ~/.profile, що вказуватиме на каталог з бінарником Blender'а:

export PATH=/path/to/blender/directory:$PATH

Якщо ви використовуєте щоденні побудови та оновлюєте Blender часто, то можете зробити посилання або завжди перейменовувати вашу теку на 'blender' та вживати це ім'я для змінної середовища PATH і збереження оновленим менеджера вікон.

Запобігання конфлікту Alt+Миша¶

Багато Менеджерів Вікон стандартно передбачають Alt-LMB для переміщення вікон, яке є скороченням, яке використовує Blender для імітації три-кнопкової миші. Ви можете або вимкнути цю опцію «Уподобання Користувача > Увід > Емулювати 3-кнопкову Мишу» -- User Preferences ‣ Input ‣ Emulate 3 Button Mouse або змінити устави Менеджера Вікон, щоб він замість «Alt» використовував клавішу Meta (також звану Super або Windows key):

  • KDE: System Settings ‣ Window Behavior ‣ Window Behavior ‣ Window Actions, Switch 'Alt' for 'Meta' key.

  • Unity/Gnome: Enter the following in a command line (effective at next login):

    gsettings set org.gnome.desktop.wm.preferences mouse-button-modifier '<Super>'
