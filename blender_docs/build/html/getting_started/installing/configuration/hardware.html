Пристрої Уводу -- Input Devices¶

Blender підтримує різні типи пристроїв уводу:

  • Клавіатура (рекомендовано: клавіатура з цифровою частиною справа, Англійська розкладка працює найкраще)
  • Миша (рекомендовано: 3-кнопкова миша з коліщам прокрутки)
  • NDOF Пристрій (відомий також як 3D-Миша)
  • Графічний планшет

Миша-маніпулятор¶

Емуляція кнопок миші¶

Якщо у вас немає 3-кнопкової миші, то вам необхідно увімкнути їх емуляцію, вибравши відповідну опцію в Уподобаннях Користувача -- User Preferences.

Таблиця нижче показу використовувані комбінації:

3-кнопкова Миша LMB MMB RMB
2-кнопкова Миша LMB Alt-LMB RMB

Клавіатура¶

Емуляція цифрової частини справа¶

Якщо ви не маєте справа на вашій клавіатурі цифрової частини -- Numpad, то ви можете емулювати її клавіші (використовує цифрові клавіші на звичайній клавіатурі натомість, однак, вилучає швидкий доступ до видимості шарів).

Non-English Keyboards¶

If you use a keyboard with a non-English keyboard layout, you still may benefit from switching your computer to the UK or US layout as long as you work with Blender.

Ghi chú

Ви можете також змінювати стандартну карту клавіш або стандартні скорочення в Уподобаннях Користувача -- User Preferences, однак, цей посібник припускає, що ви будете використовувати стандартну карту клавіш для Blender'а.

Графічний планшет¶

Графічні планшети можуть використовуватися для забезпечення за допомогою пера-стилуса для керування курсором миші. Це може забезпечити більш знайомий спосіб для митців, хто використовує подібні інструменти для малювання і рисування, а також надати додаткові контролери, такі як чутливість натиску.

Ghi chú

Якщо ви використовуєте графічний планшет замість миші і чутливість натиску не працює правильно, спробуйте навести вказівник миші на вікно Blender'а та від'єднати і приєднати ваш графічний планшет. Це має допомогти.

3D-миша¶

3D-миша або NDOF пристрій -- це устаткування, яке ви можете використовувати для навігації по сцені у Blender'і. Поточно підтримуються пристрої, вироблені тільки 3Dconnexion. Ці пристрої дозволяють вам рухатися по сцені, а також режими Ходи/Польоту -- Walk/Fly modes.
