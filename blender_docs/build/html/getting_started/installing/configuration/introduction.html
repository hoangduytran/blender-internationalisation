Giới Thiệu -- Introduction¶

Dưới đây là một số tùy chọn mà bạn có thể muốn đặt ban đầu. Danh sách đầy đủ và lời giải thích các tùy chọn của người dùng được ghi lại trong phần User Preferences.

Ngôn Ngữ -- Language¶

Vào File ‣ User Preferences ‣ System, bật Phông Quốc Tế -- International Fonts để chọn Ngôn Ngữ -- Language và chọn những gì cần dịch, từ Giao diện -- Interface (Chữ hiện trên trình đơn và các hộp đối thoại), Chú Giải Công Cụ -- Tooltips (Dòng tóm tắt chú giải công năng của trình đơn và hộp đối thoại) đến Dữ liệu mới -- New Data (Tên, chỉ danh của các dữ liệu, vật thể v.v..).

Xin xem mục Internationalization để biết thêm chi tiết.

Đầu Vào -- Input¶

Nếu bạn có một bàn phím thu nhỏ mà không có bảng phím số riêng biệt thì xin hãy bật File (Tập Tin) ‣ User Preferences (Tùy Chọn Người Dùng) ‣ Input (Đầu Vào) ‣ Emulate Numpad (Mô Phỏng Bàn Phím Số) lên.

Nếu chuột của bạn không có nút chuột giữa, bạn có thể bật File (*Tập Tin*) ‣ User Preferences (*Tùy Chọn Người Dùng*) ‣ Emulate 3 Button Mouse (*Mô Phỏng Chuột 3 Nút*).

Xin xem thêm Tùy Chọn về Đầu Vào -- Input Preferences để biết thêm chi tiết.

Tập Tin và Đường Dẫn -- File and Paths¶

Tại mục File (*Tập Tin*) ‣ User Preferences (*Tùy Chọn Người Dùng*) ‣ File (*Tập Tin*) bạn có thể thiết lập các tùy chọn như những gì, ví như Trình chỉnh sửa hình ảnh, bên ngoài sử dụng, chẳng hạn như GIMP, Krita hoặc Trình Chơi Hoạt Họa (Animation Player).

Sắp đặt về thư mục tạm thời -- Thư Mục Tạm Thời -- đặt nơi lưu trữ các tập tin, ví như các bản kết xuất tạm thời và các bản tự động lưu của phần mềm.

Mẹo

Các ký tự // ở đầu mỗi đường dẫn trong Blender có nghĩa là thư mục của tập tin blend hiện đang mở, được sử dụng để tham chiếu các đường dẫn tương đối.

Nếu bạn tin tưởng nguồn của các tập tin blend của mình thì bạn có thể bật Tự động thi hành tập lệnh Python -- Auto Run Python Scripts lên. Tùy chọn này là nhằm để bảo vệ bạn khỏi các tập lệnh Python độc hại mà ai đó có thể đưa vào bên trong một tập tin blend. Việc này sẽ không xảy ra một cách bất ngờ và hầu hết người dùng bật tùy chọn này để cho phép tự động chạy các tập lệnh thường được sử dụng trong các quá trình lắp ráp (rigs) cao cấp (chẳng hạn như "Lắt Ráp -- Rigify" để điều khiển bộ xương của một giàn lắp ráp mô hình người).

Xin xem thêm Tùy Chọn về Tập Tin -- File Preferences để biết thêm chi tiết.
