Вступ¶

Ось деякі з уподобань, як можливо ви забажаєте задати на початку. Повний список та пояснення уподобань користувача задокументовано у розділі «Уподобання Користувача» -- User Preferences.

Мова¶

У вкладці «Система» в Уподобаннях Користувача -- File ‣ User Preferences ‣ System увімкніть опцію «Іншомовні Шрифти» -- International Fonts, виберіть потрібну Мову -- Language та увімкніть на вибір «Інтерфейс» -- Interface, «Підказки» -- Tooltips та/або «Нові Дані» -- New Data, що має бути перекладено на обрану вами мову.

Детальніше дивіться Інтернаціоналізація.

Увід¶

If you have a compact keyboard without a separate number pad, enable File ‣ User Preferences ‣ Emulate Numpad.

If you do not have a middle mouse button, you can enable File ‣ User Preferences ‣ Emulate 3 Button Mouse.

Детальніше дивіться Уподобання щодо Уводу -- Input Preferences.

Фал та шляхи¶

У вкладці «Файл» в Уподобаннях Користувача -- File ‣ User Preferences ‣ File ви можете задати опції щодо використання зовнішнього Редактора Зображень -- Image Editor, наприклад, GIMP або Krita, а також щодо Відтворника Анімації -- Animation Player.

Поле устави Тимчасовий каталог установлює, де зберігатимуться такі файли, як тимчасові зображі та автозберігання.

Mẹo

Символи // на початку кожного шляху у Blender'і означають каталог поточно відкритого blend-файлу та що використовуються посилання на відносні шляхи.

Якщо ви довіряєте джерелу ваших blend-файлів, то можете увімкнути опцію «Авто Виконання Скриптів Python» -- Auto Run Python Scripts. Ця опція призначена для захисту вас від зловмисних скриптів на Python, які хтось може включити у свій blend-файл. Це не може статися випадково, а тому більшість користувачів залишають цю опцію увімкненою для автоматичного виконання скриптів, що часто використовуються у просунутих оснастках (таких як "Rigify", що керує скелетом в людській оснастці).

Детальніше дивіться Уподобання щодо файлів -- File Preferences.
