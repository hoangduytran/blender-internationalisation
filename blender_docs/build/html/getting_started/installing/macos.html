Cài đặt trên hệ điều hành macOS -- Installing on macOS¶

Xin hãy kiểm tra trang về Cài đặt Blender -- Installing Blender để tìm hiểu về các yêu cầu tối thiểu và nơi lấy Blender (nếu bạn chưa làm như vậy).

Sau khi tải xuống máy bản Blender cho hệ điều hành macOS, xin hãy giải nén tập tin và kéo blender.app vào thư mục Ứng dụng (Applications folder).

Mẹo

Blender does not use the standard macOS menu system so, you will likely have a redundant menu-bar at the top.

To remove it see this post on Macworld, but beware that it is somewhat complex. As an alternative: simply make Blender full screen by Alt-F11 or by Info ‣ Window ‣ Toggle Window Fullscreen.
