Спільнота Blender¶

Будучи вільно доступним з самого початку, навіть поки закрите джерело значно допомогло спільноті прийняти Blender. Велика, стабільна та активна спільнота користувачів зібралася навколо Blender'а ще з 1998. Ця спільнота показала свою підтримку для Blender'а у 2002, коли вони допомогли зібрати €100000 за сім тижнів, щоб Blender став відкритим джерелом Open Source під загальною публічною ліцензією GNU GPL License.

Незалежні сайти¶

Існують кілька незалежних вебсайтів такі, як форуми, блоки, новини та навчальники, присвячені Blender'у.

Одним з найбільшого форуму спільноти є Blender Artists, де користувачі Blender збираються, щоб продемонструвати свої твори, отримати відзив, попросити та отримати допомогу і, загалом, обговорити Blender.

Отримання підтримки¶

Спільнота Blender є однією з найчудовіших можливостей, а тому окрім цього посібника користувача, існує багато різних способів отримати підтримку від інших користувачів, зокрема через IRC та Stack Exchange.

Є також більш офіційні джерела підтримки, як Certified Trainers та Blender Cloud. Якщо ви думаєте, що виявили проблему щодо Blender'а, то ви можете легко повідомити про помилку.

Більш детально про підтримку можна довідатися на сторінці підтримки.

Розробка¶

Будучи відкрито-джерельною, більшість розробки Blender'а здійснюється виключно волонтерами, добровольцями. Спілкування між розробниками проводиться здебільшого через три платформи:

  • Система developer.blender.org
  • Різні списки розсилки
  • IRC-канал #blendercoders (дивіться нижче)

Якщо ви зацікавилися допомогти у розробці Blender'а, дивіться сторінку Get Involved.

IRC-канали¶

Для обговорення у режимі реального часу є кілька IRC-каналів для Blender у мережі Freenode. Ви можете під'єднатися до них за допомогою вашого улюбленого IRC-клієнта:

  • #blender -- для отримання відповідей від спільноти.
  • #blenderchat -- для загального обговорення та чату з інших тем.
  • #blendercoders -- для розробників з метою обговорення розробки Blender'а.
  • #blenderpython -- для підтримки від розробників у використанні Python API.
  • #gameblender -- для обговорення проблем щодо створення ігор за допомогою ігрового рушія Blender'а.
  • #blenderwiki Для обговорення щодо документації з Blender'а.

Ghi chú

Якщо ви не маєте IRC-клієнт, то ви можете отримати доступ до IRC за допомогою вебчату.

Там також є кілька каналів більш орієнтованих на Blender, що не перелічені тут (наприклад, канали для співрозмовників на певній мові). Ми рекомендуємо вам переглянути ресурси Freenode, щоб знайти їх усіх там.
