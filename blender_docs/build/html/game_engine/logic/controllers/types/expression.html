Контролер «Вираз» -- Expression Controller¶

Цей контролер обчислює написаний користувачем вираз та дає позитивний (TRUE) вивід, коли результат Цього виразу є TRUE та Цей об'єкт знаходиться у даному призначеному Стані. Для усіх інших умов цей контролер дає негативний (FALSE) вивід.

Контролер «Вираз» -- Expression.

Вираз -- Expression¶

Вираз, що записується у цьому полі, може містити змінні, константи та оператори. Вони повинні відповідати правилам, викладеним нижче.

Змінні -- Variables¶

Ви можете використовувати:

  • імена сенсорів,
  • властивості: призначте властивість гри для об'єкта та використовуйте її у виразі контролера.

Вони не повинні містити пробіли.

Операції -- Operations¶

Математичні Операції -- Mathematical Operations¶

Оператори: *, /, +, -

Повертає: число

Приклади: 3 + 2, 35 / 5

Логічні Операції -- Logical Operations¶

  • Оператори порівняння: <, >, >=, <=, ==, !=
  • Булеві оператори: AND, OR, NOT

Повертає: True або False.

Приклади: 3 > 2 (True), 1 AND 0 (False)

Умовне Твердження -- Conditional Statement (if)¶

Використання:

if( expression, pulse_if_expression_is_true, pulse_if_expression_is_false )

Якщо цей контролер обчислює expression як True:

  • якщо pulse_if_expression_is_true є True, то цей контролер посилає позитивний пульс до під'єднаних до нього актуаторів.
  • якщо pulse_if_expression_is_true є False, то цей контролер посилає негативний пульс до під'єднаних до нього актуаторів.

Якщо цей контролер обчислює expression як False:

  • якщо pulse_if_expression_is_false є True, то цей контролер посилає позитивний пульс до під'єднаних до нього актуаторів.
  • якщо pulse_if_expression_is_false є False, то цей контролер посилає негативний пульс до під'єднаних до нього актуаторів.

Приклади¶

Задано, що об'єкт має властивість coins, яка дорівнює 30:

coins > 20

повертає True (цей контролер посилає позитивний пульс до під'єднаних до нього актуаторів).

Задано, що об'єкт має:

  • Сенсор з іменем Key_Inserted, що дорівнює True.
  • Властивість з іменем Fuel, що дорівнює False.
Key_Inserted AND Fuel.

Буде повертати False (цей контролер посилає негативний пульс до під'єднаних до нього актуаторів).

Це так само робиться:

if (Key_Inserted AND Fuel, True, False)

Натомість, ви можете зробити:

if (Key_Inserted AND Fuel, False, True)

для повернення позитивного пульсу, коли Key_Inserted AND Fuel повертає False.

Ви можете також зробити:

if ((Key_Inserted AND Fuel) OR (coins > 20), True, False)

Цей вираз повертає True, звідси, у цьому випадку, цей контролер посилає позитивний пульс до під'єднаних до нього актуаторів.

Частини контролера Expression¶

Вираз для обчислення.

Про решту опцій дивіться опис тут -- standard controller parts.
