Редагування Контролера -- Controller Editing¶

Controller Column with a typical sensor.

Контролери у Blender'і можуть установлюватися та редагуватися у центральному стовпці панелі Логіки. Ця сторінка описує загальні засоби керування стовпця, ті параметри, які є загальними для усіх окремих типів контролерів, а також різні стани для об'єктів у логічній системі, які можуть укладатися та редагуватися.

Ілюстрація тут показує стовпець з типовим одиничним контролером. Вгорі цього стовпця, як і для сенсорів та актуаторів, його заголовок містить меню та кнопки для керування тим, які з усіх контролерів у поточній Логіці Гри будуть показуватися.

Заголовок Стовпця -- Column Heading¶

Controller Column headings.

Заголовок даного стовпця містить контролери для установлення того, які контролери показуються та заданого рівня деталізації у цьому стовпці. Це дуже корисно для ховання непотрібних контролерів, так що необхідні з них будуть видимі та їх легше буде шукати. Обидва ці атрибути можуть керуватися окремо.

Контролери -- Controllers¶

Показати Об'єкти -- Show Objects
Розгортає усі об'єкти.
Сховати Об'єкти -- Hide Objects
Згортає усі об'єкти, залишаючи тільки смуги з їх іменами.
Показати Контролери -- Show Controllers
Розгортає усі Контролери.
Сховати Контролери -- Hide Controllers
Згортає усі Контролери у смуги з їх іменами.

Також можливим є фільтрування того, які контролери будуть видимі, використовуючи такі чотири кнопки заголовка:

Sel -- Вид
Показує усі контролери для виділених об'єктів.
Act -- Акт
Показує тільки ті контролери, що належать активному об'єкту.
Link -- Зв'яз
Показує контролери, які мають пов'язання з яким-небудь актуаторами/сенсорами.

Заголовок Об'єкта -- Object Heading¶

У списку стовпця контролери групуються за об'єктом. Стандартно, у цьому списку показуються контролери для кожного виділеного об'єкта, але це може бути модифіковано за допомогою фільтрів заголовка стовпця.

Вгорі списку кожного показуваного об'єкта показуються два записи:
Кнопка Used States показує, які стани використовуються для цього об'єкта. Детальний опис цієї панелі дається тут -- States.
Ім'я -- Name
Ім'я об'єкта.
Додати Контролер -- Add Controller
При клацку на ньому з'являється меню з доступними типами контролерів. Вибір якогось із записів додає новий контролер для цього об'єкта. Про список доступних типів контролерів дивіться тут -- Controllers.

Стандартні частини Контролера¶

Заголовок контролера є стандартним для кожного контролера.

Меню Тип контролера -- Type
Визначає тип контролера.
Ім'я контролера -- Controller Name
Ім'я контролера. Це може бути задано користувачем. Це використовується для доступу до контролерів за допомогою Python; воно повинно бути унікальним серед імен задіяних об'єктів.
Індекс Стану -- State Index
Задає позначений стан, для якого цей контролер буде оперувати.
Кнопка Уподобання -- Preference Button
При увімкненні цей контролер буде оперувати перед усіма іншими контролерами, для яких не увімкнене уподобання (корисно для скриптів при запуску).
Стяг Активності -- Active Checkbox
При вимкненні даний контролер деактивується, жоден з пульсів не буде послано до під'єднаних з ним актуаторів.
X кнопка
Видаляє цей блок логіки.
