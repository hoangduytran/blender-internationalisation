Актуатор «Сцена» -- Scene Actuator¶

Актуатор Scene управляє сценами у blend-файлі користувача, вони можуть бути використані як рівні або для інтерфейсу користувача та задньоплану.

Актуатор «Сцена» -- Scene.

Властивості -- Properties¶

Даний актуатор має кілька режимів, описаних нижче:

Перезапуск -- Restart
Перезапускає поточну сцену, все у сцені скидається.
Задання Сцени -- Set Scene
Змінює сцену на вказану.
Задання Камери -- Set Camera
Змінює те, яка камера використовується.
Додання Перекривної Сцени -- Add Overlay Scene
Це додає іншу сцену та прорисовує її поверх поточної сцени. Це добре для інтерфейсу: утримування смуги здоров'я, лічильника боєприпасів, покажчика швидкості у перекривній сцені робить їх завжди видимими.
Додання Фонової Сцени -- Add Background Scene
Це протилежне режиму Перекривної сцени, фонова сцена рисується позаду поточної сцени.
Вилучення Сцени -- Remove Scene
Вилучає вказану сцену.
Призупинення Сцени -- Suspend Scene
Призупиняє вказану сцену.
Поновлення Сцени -- Resume Scene
Поновлює вказану призупинену сцени.

Ghi chú

Сцена, яка призупинена, не може поновити сама себе. Вам необхідна активна сцена для поновлення іншої сцени, яка призупинена.
