Актуатор «Примус» -- Constraint Actuator¶

The Constraints Actuator adds a constraint to the location or orientation of an object. This is useful for controlling the physics of the object in-game.

Властивості -- Properties¶

Режим Примусів -- Constraints Mode
Тип примусу для використання. Кожен з цих описується нижче.

Примус Силового Поля -- Force Field Constraint¶

Створює буферну зону силового поля уздовж однієї з осей об'єкта.

Примус Силового Поля -- Force Field Constraint.

Згасання -- Damping
Фактор згасання пружини горизонтального силового поля.
Відстань -- Distance
Висота області горизонтального силового поля.
Rot Fh -- Оберт горизонтального Поля
Змушувати вісь об'єкта гри бути паралельною до нормалі пускового об'єкта.
Напрям -- Direction
Вісь, по якій створюється силове поле (може бути + або -, чи None).
Сила -- Force
Значення сили, яке буде використано.
N -- Нормаль
При увімкненні використовує горизонтальну пружинну силу на схилах.
M/P -- Матеріал/Властивість
Пускач на інший Об'єкт буде або матеріалом -- Material (M) або властивістю -- Property (P).
Властивість -- Property
Властивість -- Property / Матеріал -- Material, що запускає примус Силового Поля (порожньо для усіх Властивостей/Матеріалів).
Per -- Стійкість
Кнопка «Стійкість» -- Persistence Коли увімкнена, то примус силового поля завжди шукає Властивість/Матеріал; коли вимкнута, то вимикає себе, якщо не може знайти задану Властивість/Матеріал.
Час -- Time
Кількість кадрів, упродовж яких примус залишається активним.
RotDamp -- Згасання Обертання
Фактор згасання для обертання.

Примус Орієнтації -- Orientation Constraint¶

Примушує задану вісь у Грі вказувати у заданому напрямку по осі Світу.

Примус Орієнтації -- Orientation Constraint.

Напрям -- Direction
Вісь гри, що буде модифікуватися (X, Y, Z або None).
Згасання -- Damping
Затримка (кадри) відповіді примусу.
Час -- Time
Час (кадри), упродовж якого примус залишається активним.
Напрямок Орієнтира -- Reference Direction
Орієнтувальний напрямок (глобальні координати) для заданої осі гри.
Мін Кут -- Min Angle
Мінімальний кут для модифікації осі.
Макс Кут -- Max Angle
Максимальний кут для модифікації осі.

Примус Відстані -- Distance Constraint¶

Підтримує відстань, на якій Об'єкт Гри повинен бути від поверхні.

Примус Відстані -- Distance Constraint.

Напрям -- Direction
Напрямок осі (X, Y, Z, -X, -Y, -Z чи None).
L -- Локаль
Якщо увімкнено, то використовується локальна вісь (інакше використовується вісь Світу).
N -- Нормаль
Якщо увімкнено, орієнтує вісь Об'єкта Гри по нормалі сіті.
Діапазон -- Range
Максимальна довжина променя, використовувана для перевірки, чи є Матеріал/Властивість на іншому об'єкті гри.
Відстань Сили -- Force Distance
Відстань, що буде підтримуватися між цим об'єктом та об'єктом з Матеріалом/Властивістю, що запускає примус Distance.
Згасання -- Damping
Затримка (кадри) відповіді примусу.
M/P -- Матеріал/Властивість
Пускач на інший Об'єкт буде або матеріалом -- Material (M) або властивістю -- Property (P).
Властивість -- Property
Властивість -- Property / Матеріал -- Material, що запускає примус Силового Поля (порожньо для усіх Властивостей/Матеріалів).
Per -- Стійкість
Кнопка «Стійкість» -- Persistence Коли увімкнена, то примус силового поля завжди шукає Властивість/Матеріал; коли вимкнута, то вимикає себе, якщо не може знайти задану Властивість/Матеріал.
Час -- Time
Кількість кадрів, упродовж яких примус залишається активним.
Rotation Damping -- Згасання Обертання
Фактор згасання для обертання.

Примус Локації -- Location Constraint¶

Лімітує позицію Об'єкта Гри у межах одного напрямку Осі Світу. Для лімітування переміщення у межах площі або об'єму використовуйте два або три таких примуси.

Примус Локації -- Location Constraint.

Ліміт -- Limit
Вісь, по якій застосовуються ліміти (LocX, LocY, LocZ чи None).
Мін, мінімум -- Min
Мінімальний ліміт по вказаній осі (Одиниці Blender).
Макс, максимум -- Max
Максимальний ліміт по вказаній осі (Одиниці Blender).
Згасання -- Damping
Затримка (кадри) примусу.
