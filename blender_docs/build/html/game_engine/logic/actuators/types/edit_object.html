Актуатор «Правка Об'єкта» -- Edit Object Actuator¶

The Edit Object Actuator allows the user to edit settings of objects in-game. In example edits the object's mesh, adds objects, or destroys them. It can also change the mesh of an object (and soon also recreate the collision mesh).

Властивості -- Properties¶

Правка Об'єкта -- Edit Object
Як об'єкт модифікується, кожен з методів описується нижче.

Динаміка -- Dynamics¶

Надає меню Операції Динаміки -- Dynamic Operations для задання опцій динаміки для об'єкта.

Правка Об'єкта -- Edit Object: Динаміка -- Dynamics.

Задання Маси -- Set Mass
Дозволяє користувачу задати масу поточного об'єкта для Фізики.
Вимкнення Твердого Тіла -- Disable Rigid Body
Вимикає стан Твердого Тіла для об'єкта -- вимикає зіткнення.
Увімкнення Твердого Тіла -- Enable Rigid Body
Вмикає стан Твердого Тіла для об'єкта -- вмикає зіткнення.
Призупинення Динаміки -- Suspend Dynamics
Призупиняє динаміку об'єкта (скорість об'єкта).
Відновлення Динаміки -- Restore Dynamics
Відновлює динаміку об'єкта (скорість об'єкта).

Стеження За -- Track To¶

Makes the object "look at" another object, in 2D or 3D. The Y axis is considered the front of the object.

Правка Об'єкта: Стеження За -- Track To.

Об'єкт -- Object
Об'єкт для відстеження.
Час -- Time
Кількість кадрів, за яку він повернеться у напрямку цільового об'єкта.
Кнопка 3D (перемикач).
Вмикає 2D (X, Y) або 3D (X, Y, Z) простеження.

Заміна Сіті -- Replace Mesh¶

Замінює сіть іншою. Може замінюватися сіть та/або її фізика, разом або незалежно.

Правка Об'єкта: Заміна Сіті -- Replace Mesh.

Сіть -- Mesh
Ім'я сіті для заміни поточної сіті.
Кнопка Gfx
Замінює видиму сіть.
Кнопка Phys
Замінює фізику сіті (не складені форми).

Закінчення Об'єкта -- End Object¶

Руйнує поточний об'єкт (зауважте, властивості налагодження будуть показувати помилку Zombie Object у консолі).

Правка Об'єкта: Закінчення Об'єкта -- End Object.

Додання Об'єкта -- Add Object¶

Додає об'єкт у центр поточного об'єкта. Об'єкт, що додається, потребує бути на іншому, схованому, шарі.

Правка Об'єкта: Додання Об'єкта -- Add Object.

Об'єкт -- Object
Ім'я об'єкта, який додається.
Час -- Time
Час (у кадрах), упродовж якого об'єкт залишається живим, перед тим, як зникне. Нуль примушує його залишатися живим завжди.
Лінеарна Скорість -- Linear Velocity
Лінеарна Скорість працює як і в актуаторі руху, але на новому створеному об'єкті, замість власне на цьому об'єкті. Корисно для стріляння об'єктами, створення їх з початковою швидкістю.
Кутова Скорість -- Angular Velocity
Кутова скорість працює як і в актуаторі руху, але на новому створеному об'єкті, замість власне на цьому об'єкті.
