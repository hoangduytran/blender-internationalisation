Актуатор «Гра» -- Game Actuator¶

Актуатор Game обробляє гру цілком та таким чином дозволяє користувачу здійснювати специфічні функції гри, такі як перезапуск, вихід та завантаження.

Актуатор «Гра» -- Game.

Властивості -- Properties¶

Гра -- Game
Завантаження -- Load bge.logic.globalDict
Завантажує bge.logic.globalDict з .bgeconf.
Збереження -- Save bge.logic.globalDict
Зберігає bge.logic.globalDict у .bgeconf.
Вихід з Гри -- Quit Game
Коли такий актуатор активується відтворювач Blender'а виходить зі стану виконання.
Перезапуск Гри -- Restart Game
Після активації такого актуатора відтворювач Blender'а перезапускає гру (перезавантажує з файлу).
Запуск Гри З Файлу -- Start Game From File

При активації такого актуатора відтворювач Blender'а запускає blend-файл зі вказаного шляху.

Файл -- File
Шлях до blend-файл для завантаження.

Ghi chú

Якщо ви використовуєте сенсор Keyboard як гачок для Esc та у разі збою актуатора Quit Game через помилку у файлі Python, таку гру буде неможливо закрити. Дані можна відновити з quit.blend File ‣ Recover Last Session
