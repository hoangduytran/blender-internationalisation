Актуатор «Фільтр 2D» -- Filter 2D Actuator¶

The Filter 2D Actuator adds image filters, that apply on final render of objects. There are several 2D filters each listed below. Most are self-explanatory, however, some are special and will be described in detail later.

  • Власний Фільтр -- Custom Filter
  • Інвертування -- Invert
  • Сепія -- Sepia
  • Шкала Сірого -- Gray Scale
  • Прюїтт -- Prewitt
  • Собель -- Sobel
  • Лапласіан -- Laplacian
  • Ерозія -- Erosion
  • Розширення -- Dilation
  • Різкішання -- Sharpen
  • Розмив -- Blur
  • Розмив Рухом -- Motion Blur
  • Вилучення Фільтра -- Remove Filter
  • Вимкнення Фільтра -- Disable Filter
  • Увімкнення Фільтра -- Enable Filter

Редагування актуатора об'єкта.

Властивості -- Properties¶

Тип Фільтра 2D -- Filter 2D Type
Selects the type of 2D filter to use. All 2D filters are rendered with fragment shaders so your hardware must support fragment shaders. Several of the filters are called "built-in filters", these are: Blur, Sharpen, Dilation, Erosion, Laplacian, Sobel, Prewitt, Gray Scale, Sepia, and Invert. There are however some filters that work differently from the ones above and are described separately.
Номер Проходу -- Pass Number
Номер проходу для використання на ньому фільтра.

Деталі спеціальних фільтрів описуються нижче.

Розмив Рухом -- Motion Blur¶

Motion Blur є 2D Filter, що потребує попередньої інформації рендерингу для продукування ефекту руху на об'єктах. Нижче ви можете бачити фільтр Motion Blur у вікні Blender'а разом з його цеглинами логіки:

Фільтр 2D: Розмив Рухом -- Motion Blur.

Ви можете увімкнути фільтр Motion Blur за допомогою контролера Python:

from bge import render
render.enableMotionBlur(0.85)

Та вимкнути його:

from bge import render
render.disableMotionBlur()

Ghi chú

Ваше графічне устаткування (відеокарта) та драйвер OpenGL повинні підтримувати буфер накопичення (функція glAccum).

Фільтри Увімкнення/Вимкнення -- Enable/Disable Filters¶

Є два фільтри, які можуть використовуватися для увімкнення -- Enable або вимкнення -- Disable інших фільтрів.

Для увімкнення/вимкнення фільтра на певному проході:

  1. Створіть відповідні сенсор(и) та контролер(и).
  2. Створіть актуатор 2D Filter.
  3. Select either Enable Filter or Disable Filter depending on what you want to do.
  4. Задайте номер проходу, на якому ви хочете вимкнути фільтр.

Фільтри Вилучення -- Removing Filters¶

Тип Remove Filter використовується для вилучення інших фільтрів 2D.

Для вилучення фільтра на певному проході:

  1. Створіть відповідні сенсор(и) та контролер(и).
  2. Створіть актуатор 2D Filter.
  3. Виберіть Remove Filter.
  4. Задайте номер проходу, з якого ви хочете вилучити фільтр.

Власні Фільтри -- Custom Filters¶

Власні фільтри дають вам можливість визначати ваш користувацький фільтр 2D за допомогою GLSL. Його використання подібне, як і для вбудованих фільтрів, але ви повинні вибрати Custom Filter в актуаторі 2D Filter, далі написати програму шейдера у Редакторі Тексту і потім вказати ім'я скрипту цього шейдера у полі Script на актуаторі.

Фільтри 2D: Власний Фільтр -- Custom Filter.

Приклад блакитної Сепії:

uniform sampler2D bgl_RenderedTexture;
void main(void)
{
  vec4 texcolor = texture2D(bgl_RenderedTexture, gl_TexCoord[0].st);
  float gray = dot(texcolor.rgb, vec3(0.299, 0.587, 0.114));
  gl_FragColor = vec4(gray * vec3(0.8, 1.0, 1.2), texcolor.a);
}
