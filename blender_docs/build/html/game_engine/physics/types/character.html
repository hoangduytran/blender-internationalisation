Фізика «Персонаж» -- Character Physics¶

Тип фізики «Персонаж» використовується для керованих гравцем персонажів, для яких інші типи фізики часто дають неочікувані результати (відскакування від стін, ковзання тощо) і для яких проста кінематика пропонує набагато більшу точність.

Властивості -- Properties¶

Висота Кроку -- Step Height
Максимальна висота кроків, яку персонаж може взяти.
Сила Стрибання -- Jump Force
Уверх спрямована скорість, застосовувана до персонажа при стрибанні.
Макс Швидкість Падіння -- Fall Speed Max
Максимальна швидкість, з якою персонаж буде падати.
Макс Стрибків -- Max Jumps
Максимальна кількість стрибків, яку персонаж може зробити, перед тим, як він потрапить на землю, основу.

Ghi chú

Проходження перешкоди (наприклад, крок піднімання) управляється такими факторами (у порядку важливості):

  • Скорість об'єкта персонажа
  • Форма та припуск меж зіткнення (персонаж та перешкода)
  • Параметр Step Height
  • Провідний схил перешкоди

Швидкість падіння персонажа управляється наступними факторами (у порядку важливості):

  • Параметр Fall Speed Max
  • Параметр Step Height (більше Step Height -> більша швидкість падіння)
