Фізика «Статично» -- Static Physics¶

Об'єкти з типом фізики Static беруть участь в симуляції, впливаючи на інші об'єкти, але без впливу останніх на них. Мається на увазі, що вони не реагують на фізику, у тому числі на гравітацію і зіткнення, і таким чином вони не реагують на зміни локації, обертання або деформацій.

It will, however, give collision reactions. Objects will bounce off of Static Objects, and rotational inertia will transfer to objects capable of rotating (that is, rigid body objects will spin in response, though Dynamic Objects will not).

Зауважте, що ніщо з цього не заважає вам трансформувати об'єкти із Static за допомогою цеглин логіки -- Logic Bricks або коду Python. Візуально об'єкти будуть коректно переміщатися, а представлення фізики у рушії буде оновлюватися.

Іншим важливим зауваженням є те, що стандартно для Collision Bounds формою є Triangle Mesh, що означає вищі вимоги до обчислення, але дає також більшу детальність. Це, у свою чергу, означає, що опція "Radius" стандартно не впливає на них.

Детальніше дивіться тут -- physics page.

Опції¶

Ghi chú

Доступ до bpy

Note that, most of these properties are accessible through the non-BGE scripting API via bpy.data.objects["ObjectName"].game, which is of type bpy.types.GameObjectSetting. This is useful, for example, to set a range of objects to have gradated values via a for-loop.

Актор -- Actor
Вмикає виявлення сенсорами Near та Radar.
Привид -- Ghost
Вимикає зіткнення повністю, подібно до No collision.
Невидимо -- Invisible
Робить об'єкт не показуваним, так само, як вимкнення для об'єкта здатності рендеритися (вимкнення іконки з камерою в Outliner).
Радіус -- Radius
Дивіться тут -- rigid body.
Анізотропне Тертя -- Anisotropic Friction
Ізотропне тертя є ідентичним в усіх кутах. Анізотропія є залежною від напрямку. Тут ви можете варіювати коефіцієнтами для трьох осей індивідуально або повністю вимкнути тертя. Властивості Python: obj.game.use_anisotropic_friction (boolean) та obj.game.friction_coefficients (3-елементний масив).

Межі Зіткнення -- Collision Bounds¶

Тип Static відрізняється від інших тим, що стандартом для нього є форма меж Triangle Mesh, замість простої Sphere. Дивіться детальніше тут -- rigid body.
