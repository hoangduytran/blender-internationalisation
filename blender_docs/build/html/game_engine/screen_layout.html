Розстава Екрану «Логіка Гри» -- Game Logic Screen Layout¶

Розроблення, конструювання, налагодження та виконання гри використовує широкий діапазон функцій Blender'а. Для допомоги з цим процесом Blender включає пропоновану розставу екрану для укладання ігор BGE. Це включає багато вже відомих панелей, а також нову панель Редактора Логіки -- Logic Editor (4), присвячену виключно BGE.

Ілюстрація нижче демонструє цю стандартну розставу екрану «Логіка Гри» -- Game Logic разом з відповідними опціями для укладу/налагодження/виконання гри (вони повинні бути укладені у показаному порядку).

Розстава Екрану «Логіка Гри» -- Game Logic.

1) Game Logic

Вибрана зі списку розстав екрану для різних застосувань. Це включає багато вже відомих панелей (Information, 3D View, Properties), а також нову панель редактора Logic Editor, що присвячена виключно для BGE.

2) Blender Game

Обрано з меню вибірника рушіїв рендерингу. Це визначає, що увесь вивід буде виводитися реально-часовим рендерером, зображувачем Blender Game Engine. Це також відкриває різні інші опції меню, такі як опції меню «Гра» -- Game у Редакторі Інфо (дивіться нижче) та діапазон властивостей для рендерера BGE у Редакторі Властивостей (дивіться нижче).

Game options.

3) Game

Це меню надає різні опції для умов виконання рушія Game Engine. Зауважте, що це меню доступне у Редакторі Інфо тільки, коли рушій рендера установлено як Blender Game.

Запустити Рушій Ігор -- Start Game Engine
Виконує гру у рушії Game Engine (P чи Shift-P, коли вказівник миші знаходиться над редактором 3D View).
Показати Властивості Налагодження -- Show Debug Properties
Показує властивості, позначені для налагодження під час виконання гри.
Show frame rate and profile
Show frame rate and profiling information while game runs.
Показати Фізику -- Show Physics
Показує візуалізацію меж та взаємодій фізики.
Попередження про Небажаність -- Deprecation Warnings
Виводить попередження, коли використовуються небажані, несхвалювані функції в Python API.
Записати Анімацію -- Record Animation
Записує анімацію гри у функціональні криві F-Curves.
Авто Старт -- Auto Start
Автоматично запускає гру під час завантаження.

4) Logic Editor panel

Редактор Логіки -- Logic Editor -- це місце, де укладаються логіка, властивості та стани -- logic, properties and states для керування поведінкою об'єктів у грі. (Показ панелі редактора Logic Editor може бути також увімкнено вибором пункту Logic Editor у меню вибірника редакторів Display Editor, натисненням Shift-F2 або Ctrl-Right).

5) Properties

Два значення одного і того ж слова

Зауважте, що слово «властивість» -- "Property" має два різних використання у термінології Blender'а -- перше у широкому сенсі для панелі Properties лівого регіону Редактора Логіки, про описується тут, та друге -- як термін, використовуваний для певних змінних логіки рушія Game Engine, що також називаються "properties".

Панель Properties лівого регіону Редактора Логіки вибирається, як і звичайно, з меню його заголовка. Однак, зауважте, що деякі секції цієї панелі змінюється при зміні рушія рендера (2) з Blender Render на Blender Game.

Дивіться окремі підрозділи про деталі вмісту панелей вкладки Physics у Редакторі Properties.
