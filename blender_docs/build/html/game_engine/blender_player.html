Окремий Відтворювач -- Standalone Player¶

Окремий відтворювач дозволяє грі з Blender'а бути запущеною без потреби завантаження системи Blender'а. Це дає можливість розповсюджувати ігри серед інших користувачів без потреби детального знання ними Blender'а (а також без можливості неавторизованої модифікації). Зауважте, що Save as Runtime ігрового рушія -- це додаток, який повинен бути попередньо завантажений перед використанням.

Наступна процедура дасть окрему версію робочої гри.

  1. Увімкнути стяг -- File ‣ User Preferences ‣ Add-ons ‣ Game Engine ‣ Save As Game Engine Runtime. (Ви можете також клацнути на кнопці Save User Settings, що дасть автоматичне завантаження цього додатка разом з кожним запуском Blender'а).
  2. Вибір команди File ‣ Export ‣ Save As Game Engine Runtime (задати відповідний каталог/ім'я файлу) та підтвердити кнопкою Save as Game Engine Runtime.

Ця гра далі може бути виконана шляхом запуску відповідного файлу .exe. Зауважте, що всі відповідні бібліотеки автоматично завантажуються з цим додатком.

Якщо ви зацікавлені у ліцензуванні вашої гри, то прочитайте тут -- Licensing для обговорення пов'язаних з цим питань.

Mẹo

Експортування...

If the game is to be exported to other computers, make a new empty directory for the game runtime and all its ancillary libraries, etc. Then make sure the whole directory is transferred to the target computer.
