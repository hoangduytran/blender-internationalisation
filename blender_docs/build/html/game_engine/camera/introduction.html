Вступ¶

Камера ігрового рушія Game Engine багато у чому схожа до Камери у системі звичайного рендерингу Blender'а та створюється, налаштовується та маніпулюється подібними способами. Однак, через її використання як реально-часового пристрою, камера Рушія Ігор має ряд додаткових функціональних можливостей -- вона може використовуватися не тільки як статична камера, але також як рухомий пристрій з його стандартними характеристиками (тобто, з його власними програмованими рухами) або вона може відстежувати інший об'єкт у грі. Більше того, будь-який ігровий об'єкт може використовуватися як камера; огляд береться тоді з точки початку об'єкта. Нарешті, для нього може бути задано спеціальні здатності, як Стерео бачення, Купольна візуалізація тощо, які мають особливе значення для ігрової технології.

Коли ви запускаєте Рушій Ігор, то початковий огляд камери береться з найостаннішого перед цим 3D Огляду. Це може бути виділений об'єкт камери або стандартна камера (дивіться нижче). Звідси, для запуску гри з певної камери, ви повинні виділити камеру та натиснути Numpad0 перед переходом у Game Engine.

Mẹo

Avoiding Camera Distortions

Завжди присувайте огляд, допоки об'єкт камери не заповнить все оглядвікно.

Стандартна Камера -- Default Camera¶

Огляд стандартної камери береться з найостаннішого 3D Огляду на відстані, еквівалентній до глядача. Це означає, що якщо активним є звичайний 3D Огляд, то сцена не змінюється при запуску Рушія Ігор.

Об'єкт Камери -- Camera Object¶

Об'єкт Камери у Game Engine має таку ж структуру, як і звичайна камера у Blender -- дивіться детальніше про укладання, маніпулювання та виділення камери тут -- Camera. Наступні підрозділи показують деякі спеціальні можливості, доступні для камер в BGE.

Приріднення Камери до Об'єкта -- Parent Camera to Object¶

Камера буде слідувати за об'єктом. Спершу виділіть камеру, а потім виділіть об'єкт. Далі натисніть Ctrl-P ‣ Make Parent.

Зауважте, що якщо ваш об'єкт має будь-які обертання, то камера також бути мати ці обертання. Для запобігання цьому використовуйте Parent to Vertex.

Приріднення до Вершини -- Parent to Vertex¶

Найлегший спосіб домогтися цього -- виділити ваш об'єкт та натиснути Tab для переходу у режим Edit Mode. Далі вибрати вершину та натиснути Tab для повернення у режим Object Mode.

Наступне, зняти виділення з будь-яких об'єктів, виділити камеру та, утримуючи натисненою Shift, виділити цей об'єкт. Натиснути Tab для переходу у Edit Mode, натиснути Ctrl-P та вибрати «Вершина» -- Make vertex parent.

Тепер камера буде слідувати за об'єктом та наслідуватиме його обертання, коли він обертається.
