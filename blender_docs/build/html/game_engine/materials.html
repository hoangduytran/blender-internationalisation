Матеріали для Гри -- Game Materials¶

Устави Гри -- Game Settings¶

Game Settings panel.

This panel contains properties that control how the object surfaces that use the material are rendered in real-time by the Blender Game Engine.

Відкид Тилу -- Backface Culling
Hide the back-faces of objects rendered with this material. If "Off", both sides of the surface are visible (at the expense of lower rendering speed). Note that this setting is applied per material and not per face; e.g. if the material is applied to a cube, only the back and front faces of the cube are visible, and not both sides of each face.
Невидимо -- Invisible
Ховає усі грані об'єктів, рендерених з цим матеріалом.
Текст -- Text
Використовує матеріал як об'єкт тексту -- Text object в рушії Game Engine.
Змішання Альфа -- Alpha Blend

Керує тим, як використовується альфа канал для створення прозорої текстури в рендереному зображенні.

Сортування Альфа -- Alpha Sort
Впорядковує послідовність, за якою прозорі об'єкти прорисовуються один поверх іншого, так що ті, що спереду отримують більше світла, ніж ті, що позаду.
Змішання Альфа -- Alpha Blend
Використовує значення альфа, присутні у растровому зображенні, що подається з гнізда Image.
Відсік Альфа -- Alpha Clip
Використовує альфа канал як просту маску.
Додання -- Add
Рендерить грань прозорою та додає колір грані.
Безпрозоро -- Opaque (стандарт)
Усі значення альфа ігноруються; сцена є повністю не-прозорою.

Орієнтація Грані -- Face Orientation¶

Забезпечує опції щодо орієнтації (тобто, трансформування обертання) граней, до яких цей матеріал застосовується.

Тінь -- Shadow
Грані використовуються для тіні.
Плакат -- Billboard
Плакат з примусом по осі Z.
Ореол -- Halo
Screen-aligned billboard.
Нормаль -- Normal (стандарт)
Без трансформування.

Матеріал: Фізика -- Material Physics¶

Панель Physics у вкладці Material.

This panel contains physical properties that control how the object surfaces that use the material are rendered in real-time by the Blender Game Engine.

Устави фізики є видимими, коли для рендерингу використовується рушій Game Engine, та обробляються рушієм фізики ігрового рушія -- Game Engine's physics engine.

Тертя -- Friction
Коефіцієнт кулонівського тертя всередині області відстані фізики.
Еластичність -- Elasticity

Еластичність зіткнень визначає, скільки кінетичної енергії зберігається після зіткнення. Значення 1 буде дасть зіткнення, при яких об'єкти відскакують один від одного, а кінетична енергія після цього значення є такою ж, як і перед ним. Значення 0 даватиме зіткнення, при яких об'єкти прилипають одне до одного після зіткнення, оскільки уся енергія буде конвертована у тепло (або інші форми енергії, які Blender не моделює).

На макроскопічному рівні (набагато більшому, ніж атомні частинки) еластичність зі значенням 1 ніколи не видима, оскільки принаймні якась енергія конвертується у тепло, звук тощо. Еластичне (еластичність=високо) зіткнення відбувається, коли стикаються два металеві м'ячі. Нееластичне (еластичність=низько) зіткнення видиме, коли два напів-надуті пляжні м'ячі стикаються.

Силове Поле -- Force Field

Керують уставами силового поля.

Сила -- Force
Сила пружини вгору, коли всередині області відстані фізики.
Відстань -- Distance
Відстань області фізики.
Згасання -- Damping
Згасання сили пружини, коли всередині області відстані фізики.
Вирівняти за Нормаллю -- Align to Normal
Вирівнює динамічні ігрові об'єкти уздовж нормалі поверхні, коли всередині області відстані фізики.
