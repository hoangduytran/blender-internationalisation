Устави Матеріалу -- Material Settings¶

Орієнтир -- Reference

Panel:Material ‣ Settings

Устави Матеріалу -- Material Settings.

Поверхня -- Surface¶

Багатовибірка за Значністю -- Multiple Importance Sample

Стандартно, об'єкти з емітувальними матеріалами використовують обидва методи відбору як прямого, такі побічного освітлення, але у деяких випадках, можна зменшити загальний шум, вимкнувши відбір прямого освітлення для деяких матеріалів. Це може бути зроблено, вимкненням опції Multiple Importance Sample. Це особливо корисно для великих об'єктів, що емітують мало світла у порівнянні з іншими джерелами світла.

Ця опція матиме вплив тільки, якщо матеріал містить вузол Emission; в іншому випадку, вона буде автоматично вимкнена.

Прозорі Тіні -- Transparent Shadows
Використовує прозорі тіні, якщо матеріал містить вузол Transparent BSDF, вимкнення ж цієї опції пришвидшить рендеринг, але не дасть точних тіней.

Об'єм -- Volume¶

Подібні устави об'єму як і для Світу -- World settings, але для цього матеріалу.

Зміщення -- Зміщення¶

Ghi chú

Ці опції доступні тільки, якщо набір експериментальних функцій -- Experimental Feature Set увімкнено.

Метод Зміщення -- Displacement Method

Method used to perform Displacement on materials.

Displacement Only
Вершини сіті будуть зміщені перед рендерингом, модифікуючи фактичну сіть. Це дає найкращу якість результатів, якщо сіть дрібно підподілена. Як наслідок, цей метод також є найбільш затратним щодо пам'яті.
Bump only
When executing the surface shader, a modified surface normal is used instead of the true normal. This is a less memory intensive alternative to actual displacement, but only an approximation. Surface silhouettes will not be accurate and there will be no self-shadowing of the displacement.
Displacement and Bump
Обоє -- Both -- методи можуть комбінуватися для зроблення зміщення на більш грубій сіті та використання розкладки вигинів для дрібних деталей.

Устави для Оглядвікна -- Viewport Settings¶

Колір в Оглядвікні -- Viewport Color¶

Колір -- Color
Diffuse color of the object in the 3D viewport.
Альфа -- Alpha
Transparency of the object in the 3D viewport.

Блікування в Оглядвікні -- Viewport Specular¶

Колір -- Color
Specular reflection color of the object in the 3D viewport.
Жорсткість -- Hardness
Roughness control for the object in the 3D viewport.

Альфа в Оглядвікні -- Viewport Alpha¶

Режим Змішання -- Blend Mode

Blend modes -- режими змішання для прозорих граней.

Безпрозоро -- Opaque
Рендерить колір текстурованої грані як колір.
Додання -- Add
Рендерить прозорість та додає колір грані.
Відсік Альфа -- Alpha Clip
Використовує значення альфа зображення, відсікаючи ті, що не змішуються (двійкове альфа).
Змішання Альфа -- Alpha Blend
Рендерить полігон прозорим, залежно від каналу альфа текстури.
Сортування Альфа -- Alpha Sort
Сортує грані для коректного прорису альфа (повільно, використовуйте Alpha Clip натомість, якщо можливо).
Загладжування Альфа -- Alpha Anti-Aliasing
Використовує альфа текстури для додання маски загладжування, вимагає багато-вибірковий показ з OpenGL.

Індекс Проходу -- Pass Index¶

Індекс Проходу -- Pass Index
Порядковий номер для Індексу Матеріалу -- Material Index render pass. Це може використовуватися для додання маски у матеріал, а потім зчитування її за допомогою вузла ID Mask Node у компонівнику.
