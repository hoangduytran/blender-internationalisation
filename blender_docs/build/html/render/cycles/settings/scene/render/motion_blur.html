Розмив Рухом -- Motion Blur¶

Орієнтир -- Reference

Panel:Render ‣ Motion Blur

Анімації у Blender'і стандартно зображуються, рендеряться як відеоряд, послідовність ідеально статичних зображень. Хоча це чудово для ефектів стоп-кадру та часо-плину, це є нереалістичним, оскільки швидко рухомі об'єкти мають виглядати розмитими у напрямку руху, як на кадрах фільму, так і на фотографіях, отриманих за допомогою камер з реального світлу.

Motion blur example.

Ghi chú

При наявності частинок чи інших систем фізики у сцені переконайтеся, що запекли їх перед рендерингом, інакше ви можете не отримати коректний або послідовний рух.

Опції¶

Motion Blur settings.

Позиція -- Position

Керує, в який момент відкривається затвор, відносно кадру.

  • Кінець на Кадрі -- End on Frame
  • Центр на Кадрі -- Center on Frame
  • Старт на Кадрі -- Start on Frame
Швидкість Затвора -- Shutter Speed
Час між кадрами, упродовж якого обчислюється розмив рухом. Час затвора 1.0 дає розмиття по довжині 1-го кадру, 2.0 -- по довжині двох кадрів, від попереднього до наступного.
Крива Затвора -- Shutter Curve
Крива, що визначає, як затвор відкривається та закривається.
Тип Затвора -- Shutter Type

Імітує датчики цифрових камер на структурах CMOS шляхом рендерингу ефекту провертання затвора за допомогою прогляду порядково (scanline).

  • Зверху-Вниз -- Top Bottom: Рендерить провертання затвора зверху вниз по зображенню.
Тривалість Провертання Затвора -- Rolling Shutter Duration
Керує балансом між чистим ефектом провертання затвора та чистим ефектом розмиву рухом. При значенні нуль провертання затвора буде відсутнє зовсім, при значенні один -- воно буде присутнім повністю.

Cảnh báo

An object modifier setup that changes mesh topology over time can not render deformation motion blur correctly. Deformation blur should be disabled for such objects.

Поширеними прикладами цього є анімовані модифікатори Boolean, Deformation перед Edge Split, Remesh, Skin або Decimate.

Xem thêm

Кожен об'єкт має власні устави для керування розмивом руху. Ці устави можна знайти у вкладці Object редактора Properties. Дивіться детальніше тут -- object setting.
