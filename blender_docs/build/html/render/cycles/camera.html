Камера -- Camera¶

Об'єктив -- Lens¶

Орієнтир -- Reference

Panel:Camera ‣ Lens

Тип¶

Перспектива -- Perspective¶

Фокусна Відстань/Поле Зору -- Focal Length/Field of View
Визначає поле зору за властивістю об'єктива або за кутом, як це обрано у меню «Одиниця Об'єктива» -- Lens Unit.

Gợi ý

Наїзд-Від'їзд Зумуванням -- Dolly Zoom

При переміщенні камери у напрямку об'єкта властивість Focal Length може зменшуватися для продукування ефекту камери «Наїзд-Від'їзд Зумуванням» -- Dolly Zoom, або цей ефект навпаки, рух від об'єкта та збільшення фокусної відстані.

This video демонструє ефект камери Dolly Zoom.

Ортогонально -- Orthographic¶

Масштаб -- Scale
Визначає розмір об'єктів, проектованих на зображення.

Панорамно -- Panoramic¶

Cycles supports Equirectangular and Fisheye panoramic cameras. Note that these cannot be displayed with OpenGL rendering in the viewport; they will only work for rendering.

Рівнопрямокутно -- Equirectangular¶

Render a panoramic view of the scenes from the camera location and use an equirectangular projection, always rendering the full 360° over the X axis and 180° over the Y axis.

This projection is compatible with the environment texture as used for world shaders, so it can be used to render an environment map. To match the default mapping, set the camera object rotation to (90, 0, -90) or pointing along the positive X axis. This corresponds to looking at the center of the image using the default environment texture mapping.

Мінімум/Максимум Широти/Довготи -- Minimum/Maximum Latitude/Longitude
Лімітує вертикальний та горизонтальний кути поля зору.
«Риб'яче око» -- Fisheye¶

Об'єктиви «риб'яче око» -- це типово ширококутні об'єктиви із сильним спотворенням, корисні для створення панорамних зображень, наприклад, для проекції купола або як мистецького ефекту.

Об'єктиви типу «Риб'яче око Рівноплощинно» -- Fisheye Equisolid найкраще відповідатимуть об'єктивам реальних камер при отриманні цього ефекту. Цей тип забезпечує відповідні фокусну відстань та кут поля зору об'єктива, а також враховує розміри датчика, сенсора.

The Fisheye Equidistant lens does not correspond to any real lens model; it will give a circular fisheye that does not take any sensor information into account but rather uses the whole sensor. This is a good lens for full-dome projections.

Об'єктив -- Lens
Lens focal length in millimeters.
Поле Зору -- Field of View
Кут поля зору, що наближається до 360 та більше, з метою захоплення середовища цілком.
Дзеркальна Куля -- Mirror Ball¶

Render is if taking a photo of a reflective mirror ball. This can be useful in rare cases to compare with a similar photo taken to capture an environment.

Відсікання -- Clipping¶

Старт Відсіку -- Clip Start та Кінець Відсіку -- Clip End
Це інтервал, в якому об'єкти є безпосередньо видимими. Будь-які об'єкти поза цим діапазоном все ще впливають на зображення опосередковано, оскільки подальші відскоки світла не відсічені.

При увімкненні опції «Ліміти» -- Limits на панелі «Показ» -- Display стають видимі ці межі відсікання у вигляді двох жовтих точок на лінії зору камери.

Mẹo

Змінення значень відсікання може мати серйозний вплив на продуктивність зобража, рендера. Важливо завжди задавати ці значення Start та End на безпечних відстанях, щоб вони обидва не були надто екстремальними або надто малими, з метою отримання оптимальної тривалості зображування.

Глибина Різкості -- Depth of Field¶

Орієнтир -- Reference

Panel:Camera ‣ Depth of Field

Фокус -- Focus¶

Об'єкт -- Object
Задає об'єкт, що буде використаний для фокальна точка камери, що змушує камеру зосереджуватися, фокусуватися на початку локальних координат цього вказаного об'єкта.
Відстань -- Distance
Коли об'єкт у полі Focus не використовується, то камеру можна змусити фокусуватися на певній області у 3D просторі на заданій відстані від неї. Використовуючи опцію Показу Limits, ви у змозі наглядно бачити цю відстань у 3D просторі.

Оглядвікно -- Viewport¶

Висока Якість -- High Quality
Enables the High Quality viewport depth of field, giving a more accurate representation of depth of field. This allows the viewport depth of field to be closely represented to that of the render and render preview depth of field.
Д-число -- F-Stop
Viewport depth of field aperture measured in F-Stops. Smaller numbers will cause more blur in the viewport, OpenGL renders, and Sequencer.
Пелюстки -- Blades
The number of polygonal sides to give blurred objects in the viewport. The minimum number of blades needed to enable the bokeh effect is 3 (triangle). (Only available with High Quality).

Апертура -- Aperture¶

Тип Апертури -- Aperture Type
Використовує Д-число -- F-Stop або Радіус -- Radius для задання апертури для рендера та передогляду рендера. Д-число -- F-Stop -- це фокусна пропорція, а Радіус -- Radius -- це радіус фокальної точки.
Розмір/Число -- Size/Number
Для радіусу апертури розмір або число для Д-числа, що використовується для рендера та передогляду рендера. Використання F-Stop з низьким Number або Radius з великим Size даватиме сильний розмив, що також дозволяє використовувати ефект боке.
Пелюстки -- Blades
Total number of polygonal blades used to alter the shape of the blurred objects in the render, and render preview. As with the viewport, the minimum amount of blades to enable the bokeh effect is 3, resulting in a triangular-shaped blur.
Обертання -- Rotation
Обертає сторони багатобічника навколо вказувальної на камеру осі за або проти годинникової стрілки.
Пропорція -- Ratio
Змінює величину спотворення для імітування ефекту анаморфного боке. Значення 1.0 дає показ без спотворення, тоді як значення нижче 1.0 буде продукувати горизонтальне спотворення, а вище 1.0 -- вертикальне спотворення.
