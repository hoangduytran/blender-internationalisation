Вузол «Уздовж Штриха UV» -- UV Along Stroke Node¶

Вузол «Уздовж Штриха UV» -- UV Along Stroke Node.

Увідний вузол UV Along Stroke розкладає текстури уздовж довжини штриха, роблячи можливим імітування нарисів олівцем, малювальним пензлем та іншими художніми засобами.

Для Blender Render еквівалентні опції можна знайти на панелі «Розкладка» -- Properties editor ‣ Texture ‣ Mapping.

Ghi chú

Ці карти UV стають доступні тільки у процесі рендерингу Freestyle. Звідси, вузол UV Along Stroke не може бути замінено звичайним увідним вузлом UV Map, який приймає наявну карту UV, вже визначену як частина даних сіті.

Уводи -- Inputs¶

Цей вузол не має уводів.

Властивості -- Properties¶

Вжити Кінчики -- Use Tips
Дозволяє використати нижні чверті зображення текстури для головного та хвостового кінчиків штриха, тоді як верхня її половина буде застосована для тіла штриха.

Виводи -- Outputs¶

UV
Карти UV, визначені уздовж штрихів.

Приклад¶

Наступні екранознімки показують типове дерево вузлів відтінювача, що розкладає квіткове зображення текстури уздовж штрихів. Увідний вузол UV Along Stroke отримує карти UV, визначені Freestyle уздовж генерованих штрихів та подає ї на увідний канал Vector вузла Image Texture. Зображення текстури вибирається на вузлі Image Texture та його колір подається на канал Alpha вузла Line Style Output. Оскільки Alpha Factor задано як одиниця, зображення текстури замінює базову альфа прозорість активного стилю ліній (показано на панелі Freestyle Line Style). З іншого блоку, режим змішання Mix обрано на вузлі Line Style Output з Color Factor як нуль, так що градієнтний колір лінії, вказаний в активному стилі ліній, застосовується уздовж штрихів.

.blend.

Слід зазначити, що зображення текстури FS_floral_brush.png, показане на цьому екранознімку, є прикладом намальованих пензлем зображень Freestyle за допомогою кінчиків. Зокрема, верхня половина зображення використовується як безшовна горизонтальна плитка для тіла штриха, а частини нижньої половини є кінчиками (торцями штриха) на обох кінцях штриха.
