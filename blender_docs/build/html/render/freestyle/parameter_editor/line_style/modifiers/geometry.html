Геометрія -- Geometry¶

Ці модифікатори не мають жодних устав змішування та впливу, оскільки вони завжди повністю застосовуються до геометрії штриха (як це роблять модифікатори об'єкта). Вони приймають результатні двовимірні штрихи з набору ліній Freestyle та зміщують або деформують їх різними способами.

Як і для інших стеків, стосів модифікаторів у Blender'і, вони застосовуються зверху вниз.

2D Зсув -- 2D Offset¶

Модифікатор 2D Offset додає деякі двовимірні зсуви у стрижневу геометрію штриха. Він має два набори незалежних опцій/ефектів:

Старт -- Start та Кінець -- End
Ці дві опції додають задану величину зсуву для стартової (або кінцевої) точки штриха уздовж (2D) нормалі у цих точках. Ефект змішується по всьому штриху, якщо ви, наприклад, задасте для Start значення 50, то початок штриха зсунеться на 50 пікселів уздовж його нормалі, середина штриха -- на 25 пікселів уздовж її нормалі, а кінець штриха не буде пересунуто.
X та Y
Ці дві устави просто додають сталий горизонтальний та/або вертикальний зсув для всього штриха.

2D Трансформації -- 2D Transform¶

Модифікатор 2D Transform застосовує двовимірне масштабування та/або обертання для стрижневої геометрії штриха. Масштабування застосовується перед обертанням.

Центр (опорна точка) цих 2D трансформацій може бути:

Центр Штриха -- Stroke Center
Серединна точка штриха.
Старт Штриха -- Stroke Start
Початкова точка штриха.
Кінець Штриха -- Stroke End
Кінцева точка штриха.
Параметр Точки Штриха -- Stroke Point Parameter
Фактор Stroke Point Parameter керує тим, де уздовж штриха є опорна точка (стартова точка, якщо задано 0.0; кінцева точка, якщо задано 1.0).
Абсолютна 2D Точка -- Absolute 2D Point

Устави «Опорточка X» -- Pivot X та «Опорточка Y» -- Pivot Y дозволяють вам визначати позицію опорної точки у фінальному рендері (з нижнього лівого кутка).

Quan trọng

Поточно, ви повинні враховувати реальний розмір рендера, тобто роздільність та відсоток роздільності.

Масштаб X -- Scale X та Масштаб Y -- Scale Y
Фактори масштабування по відповідних осях.
Кут Обертання -- Rotation Angle
Кут обертання.

Приклад модифікатора 2D Transform File:Toycar_Three_Contours.zip.

Розтяг Стрижня -- Backbone Stretcher¶

Модифікатор Backbone Stretcher розтягує (додає деяку довжину) початок і кінець штриха.

Довжина Стрижня -- Backbone Length
Довжина для додання до кінців штриха.

Крива Безьє -- Bézier Curve¶

Модифікатор Bézier Curve замінює штрих приблизно подібною кривою Bézier.

Помилка -- Error
Максимальна відстань, дозволена між новою кривою Безьє та оригінальним штрихом.

Демонстрація модифікатора Bézier Curve від T.K. File:toycar_bezier.zip.

Креслення -- Blueprint¶

Модифікатор Blueprint продукує штрихи, подібні як при кресленні, використовуючи кругові, еліптичні або квадратні контури. Креслення або синька передбачає такі лінії, які рисуються на початку рисування від руки для охоплення силуету об'єктів за допомогою простих форм, таких як кола, еліпси та квадрати.

Форма -- Shape
Які базові форми використовувати для цього креслення: «Кола» -- Circles, «Еліпси» -- Ellipses чи «Квадрати» -- Squares.
Обведення -- Rounds
Скільки генерується обведень, ніби ручка кілька раз рисує один і той же штрих (тобто, скільки разів повторити цей процес).
Випадковий Радіус -- Random Radius та Випадковий Центр -- Random Center
Для форм Circles та Ellipses. Додає деяку випадковість для кожного обведення у відповідному аспекті. Використання більше ніж одного обведення без жодної випадковості буде безглуздим, оскільки вони просто точно накладатимуться одне на одне.
Довжина Стрижня -- Backbone Length та Випадковий Стрижень -- Random Backbone
Для форм Squares. Перше додає деяку додаткову довжину для кожного краю генерованих квадратів (також визначається другим параметром). Друге додає деяку випадковість для квадратів.

Зауважте, що устава Min 2D Length з параметрів Strokes є досить зручною тут для запобігання генеруванню шуму малими штрихами...

Напрямні Лінії -- Guiding Lines¶

Модифікатор Guiding Lines замінює штрих прямою лінією, що з'єднує обидва його кінці.

Зсув -- Offset
Зсув стартової та кінцевої точок уздовж оригінального штриха перед генеруванням нової прямої лінії.

Це модифікатор продукуватиме розумні результати, коли штрихи є достатньо короткими, тому що коротші штрихи більш подібні на прямі лінії. Тому, рекомендується використовувати цей модифікатор разом з однією з опцій розділення (за 2D кутом чи за 2D довжиною) з панелі Strokes.

Приклад модифікатора Guiding Lines від T.K. File:Toycar_Guiding_Line.zip.

Шум Перліна 1D -- Perlin Noise 1D¶

Модифікатор Perlin Noise 1D додає одновимірний шум за алгоритмом Перліна у штрих. Криволінійна абсциса (значення між 0 та 1, визначене позицією точки відносно першої та останньої точок штрах) використовується як увід для функції шуму для генерування шумних зміщень.

Це означає, що цей модифікатор даватиме ідентичний результат для двох штрихів з однаковими довжиною та інтервалом відбору.

Частота -- Frequency
Наскільки щільним є шум (вид фактора масштабування уздовж штриха).
Амплітуда -- Amplitude
Наскільки шум спотворює штрих у напрямку кута Angle.
Базис -- Seed
Базис генератора випадковості, рандомізації (однаковий базис для штриха завжди буде давати однаковий результат).
Октави -- Octaves
«Рівень деталізації» шуму.
Кут -- Angle
В якому напрямку застосовується шум (0.0 -- повністю горизонтально).

Шум Перліна 2D -- Perlin Noise 2D¶

Модифікатор Perlin Noise 2D додає двовимірний шум за алгоритмом Перліна у штрих. Цей модифікатор генерує шумні зміщення, використовуючи 2D координати вершин штриха як увід генератора шуму.

Його устави такі ж, як для модифікатора Perlin Noise 1D.

Полігонізація -- Polygonization¶

Модифікатор Polygonization спрощує штрихи, наскільки це можливо (іншими словами, він трансформує згладжені штрихи у зазублені полілінії).

Помилка -- Error
Максимальна відстань, дозволена між новим спрощеним штрихом та оригінальним штрихом (чим більше це значення, тим більш зазубленими/приблизними є результатні полілінії).

Відбір -- Sampling¶

Модифікатор Sampling змінює визначення, точність штриха для наступних під ним модифікаторів.

Відбір -- Sampling
Чим менше це значення, тим більш точними є штрихи. Будьте обережні: надто малі значення вимагають величезної кількості часу та пам'яті під час рендерингу!

Спрощення -- Simplification¶

Модифікатор Simplification об'єднує вершини штриха, що знаходяться близько одна до одної, як і модифікатор Decimate для сітей.

Допуск -- Tolerance
Вимір того, як близько точки мають бути одна до одної, щоб бути об'єднаними. Вищий допуск означає, що більше вершин об'єднаються.

Синусоїдальне Зміщення -- Sinus Displacement¶

Модифікатор Sinus Displacement додає синусоїдальне зміщення у штрих.

Довжина хвилі -- Wavelength
Наскільки широкими є хвилі уздовж штриха.
Амплітуда -- Amplitude
Наскільки високими є хвилі уздовж штриха.
Фаза -- Phase
Дозволяє «зсування» («пересування») хвиль уздовж штраха.

Приклад модифікатора Sinus Displacement від T.K. File:Toycar_Sinus.zip.

Просторовий Шум -- Spatial Noise¶

Модифікатор Spatial Noise додає деякий просторовий шум для штриха. Просторові шумні зміщення додаються у напрямку нормалі (тобто, напрямку перпендикулярному до дотичної лінії) та обчислюються у кожній вершині штриха.

Амплітуда -- Amplitude
Наскільки шум спотворює штрих.
Масштаб -- Scale
Наскільки широким є шум уздовж штриха.
Октави -- Octaves
Рівень деталізації шуму.
Згладження -- Smooth
При увімкненні застосовує деяке згладження по генерованому шуму.
Справжня Випадковість -- Pure Random
При вимкненні, наступне значення генерованої випадковості залежить від попереднього; інакше, вони є повністю незалежними. Вимкнення цієї устави дає більш «послідовний» шум уздовж штриха.

Вилучення Кінчиків -- Tip Remover¶

Модифікатор Tip Remover вилучає шматки штриха на його початку та на кінці.

Довжина Кінчиків -- Tip Length
Довжина штриха для вилучення на обох його кінцях.
