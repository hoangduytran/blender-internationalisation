Властивості -- Properties¶

Існує доступних кілька модифікаторів для властивостей вершин штрихів (тобто кольору ліній, альфа прозорості та товщини). Як і для інших стеків, стосів модифікаторів у Blender'і, вони застосовуються зверху вниз.

Загальні опції¶

Мікс -- Mix
Вивід цих модифікаторів може змішуватися з базовою властивістю за допомогою звичайних методів (як наприклад для компонівного вузла «Мікс» -- Mix compositing node).
Вплив -- Influence
Наскільки результат такого модифікатора впливає на поточну властивість.

Розкладання -- Mapping¶

Розкладання між визначеним діапазоном та діапазоном уводу модифікатора. тобто діапазоном значень складки.

Колір -- Color¶

Колірний Градієнт -- Color Ramp
Градієнт кольорів -- color ramp, що розкладає властивість на колір штриха.

Альфа -- Alpha¶

Розкладання -- Mapping

Прямолінійна, лінеарна прогресія (від 0.0 до 1.0) або власне розкладання по кривій -- curve.

Ghi chú

Зауважте, що лінеарний не-інвертований варіант є еквівалентом «відсутності змін», оскільки оригінальні значення матеріалів вже знаходяться у діапазоні (0.0 до 1.0). Це стосується: Crease Angle, 3D Curvature, Material, Noise, Tangent.

Інвертування -- Invert
Інвертує розкладку Mapping.

Товщина -- Thickness¶

Мін Товщина -- Min Thickness та Макс Товщина -- Max Thickness
Мінімум та максимум призначеної товщини.
Розкладання -- Mapping
Прямолінійна, лінеарна прогресія (від Min Thickness до Max Thickness) або власне розкладання по кривій (на тому ж діапазоні).
Інвертування -- Invert
Інвертує розкладку Mapping.

Типи¶

Уздовж Штриха -- Along Stroke¶

Модифікатор Along Stroke змінює базову властивість за допомогою нової на основі заданого діапазону, розкладеного уздовж довжини кожного штриха. Іншими словами, він застосовує градієнт уздовж кожного штриха.

Каліграфія -- Calligraphy¶

Модифікатор Calligraphy (тільки для Thickness) імітує деякі широкі та плоскі ручки для каліграфії. Він генерує різну товщину на основі орієнтації штриха.

Орієнтація -- Orientation
Кут (орієнтація) віртуального інструмента рисування відносно вертикальної осі картинки. Наприклад, кут 0.0 імітує ручку, вирівняну по вертикальній осі. Звідси, товстіше штрихи будуть по вертикалі, тобто напрямок штриха вирівнюється за цим кутом, а найтонше вони будуть по горизонталі, тобто напрямок штриха є перпендикулярним до цього кута.

Приклад модифікатора Calligraphy від T.K. File:Toycar_Calligraphy.zip.

Кут Складки -- Crease Angle¶

Модифікатор на основі кута складки -- Crease Angle (кут між двома суміжними гранями). Якщо сегмент штриха не лежить на складці (тобто ребро не має природи Кута Складк), то його властивості не зачіпаються цим модифікатором.

Модифікатор для Alpha.

Мін Кут -- Min Angle та Макс Кут -- Max Angle
Діапазон увідних значень для розкладання. Значення поза діапазоном кута складки будуть затиснути цими значеннями Мінімального та Максимального кутів та їх відповідними значеннями властивості.

Приклад модифікатора Crease Angle від T.K. File:Render_freestyle_modifier_crease_angle.blend.

Кривина 3D -- Curvature 3D¶

Приклад модифікатора Curvature 3D від T.K. File:Render_freestyle_modifier_curvature_3d.blend.

A modifier based on radial curvatures of the underlying 3D surface. The curvature of a 2D curve at a point is a measure of how quickly the curve turns at the point. The quicker the turn is, the larger the curvature is at the point. The curvature is zero if the curve is a straight line. Radial curvatures are those computed for a 2D curve that appears at the cross section between the 3D surface and a plane defined by the view point (camera location) and the normal direction of the surface at the point.

Для обчислення радіальних кривих (і, отже, щоб цей модифікатор мав якийсь ефект) має бути увімкнена опція Face Smoothness та для об'єкта повинно бути увімкнено Smooth Shading.

Модифікатор для Alpha.

Мін Кривина -- Min Curvature та Макс Кривина -- Max Curvature
The limits of the mapping. If the current point of the stroke is at Min Curvature or less from the target, it will take the start point of the mapping, and conversely, if it is at Max Curvature or more from the target, it will take the end-point value of the mapping.

Відстань від Камери/Об'єкта -- Distance from Camera/Object¶

Модифікатори Distance from Camera та Distance from Object змінюють базову властивість новою на основі заданого діапазону, використовуючи відстань до активної камери або до заданого об'єкта як параметр.

Модифікатор Distance from Object для Alpha.

Ціль -- Target
Об'єкт, до якої вимірюється відстань (тільки для Distance from Object).
Мін Діапазону -- Range Min та Макс Діапазону -- Range Max
Ліміти розкладання «відстані від камери» на «властивість у розкладці». Якщо поточна точка штриха знаходиться у Range Min або менше від активної камери чи об'єкта, то вона прийме значення стартової точки розкладки, і навпаки, якщо вона знаходиться у Max Curvature або більше від камери/об'єкта, то вона прийме значення кінцевої точки розкладки. Ці значення задаються у поточних одиницях сцени, а не у пікселях!
Заповнити Діапазон Виділенням -- Fill Range by Selection
Задає мінімальне/максимальне значення діапазону з відстаней між поточно виділеними вершинами сіті та камерою або ціллю.

Матеріал -- Material¶

Модифікатор для Color.

Модифікатор Material змінює базову властивість новою на основі заданого діапазону, розкладеного на поточний матеріал під штрихом.

You can use various properties of the materials, among which many are mono-component (i.e. give B&W results). In this case for the color modifier, an optional color ramp can be used to map these gray-scale values to colored ones.

З іншого боку, для властивостей матеріалів, які є багато-компонентними (тобто дають RGB результати), середнє значення буде використовуватися для модифікаторів альфа та товщини.

Якщо використовується опція Split by Material у вкладці Strokes, то результат не буде розмитим між матеріалами уздовж штрихів.

Приклад модифікатора Material від T.K. File:Lilies_Color_Material.zip.

Шум -- Noise¶

Модифікатор Noise використовує генератор псевдо-випадкових чисел для варіативного розподілення властивості уздовж штриха.

Модифікатор для Thickness.

Амплітуда -- Amplitude
Максимальне значення шуму. Вища амплітуда означає менше прозорості (більше суцільності) штриха.
Період -- Period
Період шуму. Це означає, як швидко значення властивості може змінюватися. Вище значення означає більш плавне змінення кольору уздовж штриха.
Базис -- Seed
Базис, використовуваний генератором псевдо-випадкових чисел.
Асиметрично -- Asymmetric
Тільки для Thickness -- дозволяє товщині буде розподіленою нерівномірно у кожній точці. Внутрішньо, штрих представляється як стрижень з товщиною у праву та ліву сторони. Усі інші відтінювачі товщини гарантують, що значення товщини вліво та управо є рівними. Для відтінювача Noise, однак, значущий (та добрий на вигляд) результат може бути створено призначенням різних значень для кожної сторони стрижня.

Ефект, генерований за допомогою модифікатора Noise для товщини Thickness з використанням асиметричної -- Asymmetric товщини.

Дотично -- Tangent¶

Цей модифікатор базується на впливі на напрямок проходження штриха, що обчислюється у вершинах штриха.
