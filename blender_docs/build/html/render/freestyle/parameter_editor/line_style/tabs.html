Вкладки -- Tabs¶

Штрих -- Stroke¶

Штрихи -- це фінально рендерені лінії. Тим не менш, ви можете підправляти їх, наприклад, вилучаючи довші/коротші з них за вказаний поріг, зчіплювати лінії в єдиний штрих або розбивати штрих на кілька різних на основі кутів, задавати пунктирний узор тощо.

Зчіплювання -- Chaining¶

Зчіплювання -- Chaining.

Стандартно всі отримані лінії з набору ліній зчіплюються між собою. Є два базових методи зчіплювання:

Просто -- Plain
Стандартний метод зчіплювання; він створює прості ланцюги.
Нарисно -- Sketchy
Цей варіант зчіплювання дозволяє генерувати ланцюги вирізнених ребер за допомогою нарисних багатьох штрихів. Базово, він генерує штрихи з обведеннями Round замість одиничних. Це тільки дійсно корисно, якщо ви використовуєте деякі випадково керовані модифікаторів у стилі ліній!
Обведення -- Rounds
Визначає кількість обводів у нарисних штрихах.

Зчіплювання також може бути вимкнено для рендерингу кожної лінії окремо, що може бути корисним для стилів ліній, які залежать від точного представлення набору ліній.

Розділення -- Splitting¶

Розділення -- Splitting.

Ви можете розділяти ланцюги ліній Freestyle на основі одного з таких критеріїв:

Мін -- Min, Макс 2D Кут -- Max 2D Angle
Розділяє ланцюги вирізнених країв, коли вони утворюють 2D кут більше (чи менше) мінімального (чи максимального) порогу.
2D Довжина -- 2D Length
Розділяє ланцюги, коли вони є довшими, ніж задане значення.
Рубіж Матеріалу -- Material Boundary
Розділяє ланцюги вирізнених країв, якщо вони перетинаються на рубежі між двома матеріалами.
Розділення за Рискою, Пропуском -- Split Dash, Gap

Розділяє ланцюги, використовуючи заданий взірець пунктиру на основі довжин рисок і пропусків ("D" означає "dash" -- риска, "G" означає "gap" -- пропуск; дивіться також Dashed Line).

Р1, П1, Р2, П2, Р3, П3 -- D1, G1, D2, G2, D3, G3

Сортування -- Sorting¶

Сортування -- Sorting.

Ви можете сортувати порядок ваших штрихів, дозволяючи лініям накладатися у заданому порядку.

Ключ Сортування -- Sort key

Ключ сортування використовується для визначення порядку накладання ліній.

Відстань від Камери -- Distance from camera
Лінії, що ближче до камери, будуть накладатися поверх дальших від неї ліній.
2D Довжина -- 2D Length
Довші лінії накладаються поверх коротших ліній.
Проектоване X, Y -- Projected X, Y
Сортує за проектованим по X або Y значенням у системі координат зображення.
Тип Інтеграції -- Integration Type

Використовується у тандемі з Sort Key для визначення діапазону сортування. Оскільки відстань лінії від камери може варіюватися залежно від вершин, то дана опція обчислює ключ сортування для лінії залежно від значень відстаней, обчислених до окремих вершин. Обчислене значення для лінії може бути:

Середнє -- Mean
Середнє від значень, отриманих для вершин.
Мін, мінімум -- Min
Мінімальне зі значень, отриманих для вершин.
Макс, максимум -- Max
Максимальне зі значень, отримане для вершин.
Перше -- First
Значення, отримане для першої вершини.
Останнє -- Last
Значення, отримане для останньої вершини.
Порядок Сортування -- Sort Order
Для обраного типу сортування ви можете обрати стандартний прямий -- Default або зворотній -- "Reverse" порядок сортування.

Виділення -- Selection¶

Виділення -- Selection.

Ви можете також вибирати для рендерингу тільки виділені ланцюги.

Мін -- Min, Макс 2D Довжина -- Max 2D Length
Ланцюги довше та/або коротше, ніж 2D Length.
Кількість Ланцюгів -- Chain Count
Дозволяє виділення перших N ланцюгів.

Торці -- Caps¶

Торці кінців лінії.

Ви можете обирати між трьома типами торців ліній:

Упритул -- Butt
Плоский торець, точно у точці, де закінчується лінія.
Round -- Округлено
Півколо, центроване у кінцевій точці лінії.
Квадрат -- Square
Квадрат, центрований на кінцевій точці лінії (звідси, як і для округлості, прорис кінця лінії дещо розширеним у порівнянні з обчисленим значенням).

Приклад торців лінії.

Пунктирна Лінія -- Dashed Line¶

Інтерфейс Dashes Line.

By enabling the Dashed Line checkbox, you can specify three pairs of dash and gap lengths. Dash values define the lengths of dash strokes, while gap values specify intervals between two dashes.

Якщо пропуск визначено нульовим, то відповідна йому риска ігнорується, навіть якщо вона має не-нульове значення.

Риски обробляються як окремі штрихи, що означає, що ви можете застосовувати до них торці ліній, а також модифікатори кольору, альфа та товщини.

Колір -- Color¶

Інтерфейс Color для стилю ліній.

У цій вкладці ви керуєте кольором ваших штрихів.

Базовий Колір -- Base Color
Базовий колір для цього стилю ліній.

Альфа -- Alpha¶

Інтерфейс Alpha для стилю ліній.

У цій вкладці ви керуєте альфа (прозорістю) ваших штрихів.

Базова Прозорість -- Base Transparency
Базове альфа для цього стилю ліній.

Товщина -- Thickness¶

У цій вкладці ви керуєте товщиною ваших штрихів.

Базова Товщина
Базова товщина для цього стилю ліній.
Позиція Товщини -- Thickness Position

Керує позицією товщини штриха відносно оригінальної (стрижневої) геометрії штриха. Є чотири варіанти вибору:

Центр -- Center
Товщина рівномірно розділяється на ліву і праву сторони геометрії штриха.
Всередині -- Inside
Штрихи рисуються у межах границі об'єкта.
Зовні -- Outside
Штрихи рисуються поза границею об'єкта.
Відносно -- Relative
Це дозволяє вам вказати відносну позицію у числовій кнопці «Пропорція Товщини» -- Thickness Ratio за допомогою числа між 0.0 (всередині) та 1.0 (зовні).

Опції позиції товщини застосовуються тільки для штрихів з типом країв Silhouette та Border, оскільки вони є єдиними типами країв, визначеними у термінах рубежу, границі об'єкта. Штрих інших типів країв завжди рисуються з використанням опції Center.

Геометрія -- Geometry¶

Інтерфейс Geometry для стилю ліній.

У цій вкладці ви керуєте геометрією ваших штрихів. Вона містить тільки опцію для додання модифікаторів.

Текстура -- Texture¶

Інтерфейс Texture для стилю ліній.

Вжити Вузли/Текстури -- Use Nodes/Textures
Blender Render використовує панелі Mapping та Influence для текстур. У Cycles текстури визначаються за допомогою вузлів -- nodes відтінювача.
Інтервал Уздовж Штриха -- Spacing Along Stroke
Дозволяє задати «темп» текстур, розкладених уздовж довжини штрихів.
Перейти до Властивостей Текстур Стилю Ліній -- Go to LineStyle Textures
Кнопка "Go to LineStyle Textures" є ярликом для переходу до устав текстури в іншій вкладці.
