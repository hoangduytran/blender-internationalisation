Відтворювач Анімації -- Animation Player¶

Меню «Редактор Інфо > Зображ > Відтворити Зображену Анімацію» -- Info Editor ‣ Render ‣ Play Rendered Animation запустить відтворення рендереної анімації у новому вікні.

Ви можете також перетягувати файли зображень або фільмів у запущений відтворювач анімації. Це буде далі перезапускати відтворювач з новими даними.

Також може використовуватися зовнішній відтворювач замість вбудованого у Blender'і. Для цього задайте його в Уподобаннях Користувача -- User Preferences.

Скорочення¶

Наведена нижче таблиця показує доступні скорочення для відтворювача анімації.

Відтворення -- Playback

Дія -- Action Скорочення
Старт/Пауза: Spacebar
Запустити відтворення (з паузи): Return
Вихід: Esc

Часолінія -- Timeline

Дія -- Action Скорочення
Прокрутити у часі: LMB
Крок назад на один кадр: Left
Крок вперед на один кадр: Right
Крок назад на 10 кадрів: Down
Крок вперед на 10 кадрів: Up
Перехід по кадрах вручну: NumpadPeriod

Опції Відтворення -- Playback Options

Дія -- Action Скорочення
Відтворення назад: Shift-Down
Відтворення вперед Shift-Up
Уповільнення відтворення: Minus
Прискорення відтворення: Plus
Перемикання зациклювання: Numpad0
Перемикання пропуску кадрів: A
Перемикання пінг-понг: P

Показ -- Display

Дія -- Action Скорочення
Перемикання Курсора Часу (Індикатор): I
Перевертання прорису по осі X: F
Перевертання прорису по осі Y: Shift-F
Утримування для показу номерів кадрів: Shift
Присування -- Zoom in: Ctrl-Plus
Відсування -- Zoom out: Ctrl-Minus
