Командний Рядок -- Command Line¶

У деяких ситуаціях ми хочемо збільшити швидкість рендерингу, мати доступ до Blender'а віддалено для зображування чогось або будувати скрипти, що використовують командний рядок -- command line.

Однією з переваг використання командного рядка є те, що нам не потрібен графічний показ (наприклад, не потрібно для Х сервера на Linux), та, як наслідок, ми можемо рендерити через віддалену оболонку (зазвичай, SSH).

Дивіться Command Line Arguments про повний список аргументів (наприклад, для визначення того, яку сцену рендерити, номер кінцевого кадру тощо), або просто запустіть:

blender --help

Ghi chú

Аргументи виконуються у порядку, в якому вони задані!

Наступна команда не буде працювати, оскільки вивід та розширення задані після того, як дана Blender'у команда запустити рендеринг:

blender -b file.blend -a -x 1 -o //render

Наступна команда буде поводитися, як очікується:

blender -b file.blend -x 1 -o //render -a

Завжди розміщуйте -f або -a як останній аргумент.

Платформи¶

Як фактично виконати Blender з командного рядка, залежить від платформи й того, де ви інсталювали Blender. Ось базові інструкції для різних платформ.

Linux¶

Відкрийте термінал, далі перейдіть у каталог, де інстальовано Blender, та запустіть Blender так:

cd <blender installation directory>
./blender

Якщо ви інсталювали Blender у ваш PATH (зазвичай, коли Blender інстальовано через розподільний пакет), ви можете просто запустити:

blender

macOS¶

Відкрийте застосунок терміналу, перейдіть у каталог, де інстальовано Blender, та запустіть виконуваний файл у межах пакеті застосунків, за допомогою таких команд:

cd /Applications/Blender
./blender.app/Contents/MacOS/blender

Якщо вам необхідно робити це часто, то ви можете створити псевдонім -- alias, щоб просто уводити blender у терміналі. Для цього ви можете запустити таку команду у терміналі (з відповідним шляхом).

echo "alias blender=/Applications/Blender/blender.app/Contents/MacOS/blender" >> ~/.bash_profile

Якщо ви далі відкриєте новий термінал, то вже працюватиме наступна команда:

blender

MS-Windows¶

Відкрийте командний рядок -- Command Prompt, перейдіть у каталог, де інстальовано Blender, і далі запустіть Blender:

cd c:\<blender installation directory>
blender

Ви можете також додати теку Blender'а у ваш системний PATH, щоб вам не треба було уводити команду з cd кожен раз.

Приклади¶

Одиничне Зображення -- Single Image¶

blender -b file.blend -f 10
-b
Рендерить у фоні (без Інтерфейсу Користувача).
file.blend
Шлях до blend-файлу для рендерингу.
-f 10
Рендерити тільки 10-й кадр.
blender -b file.blend -o /project/renders/frame_##### -F EXR -f -2
-o /project/renders/frame_#####
Шлях до місця для збереження рендереного зображення, що використовує п'ять підставних нулів для номера кадру.
-F EXR
Замістити формат зображення, визначений у цьому blend-файлі, та зберегти у зображення формату OpenEXR.
-f -2
Рендерити тільки другий останній кадр.

Cảnh báo

Аргументи чутливі до регістру! -F та -f -- це не те ж саме.

Анімація -- Animation¶

blender -b file.blend -a
-a
Рендерити всієї анімації, використовуючи усі устави, збережені у цьому blend-файлі.
blender -b file.blend -E BLENDER_RENDER -s 10 -e 500 -t 2 -a
-E BLENDER_RENDER
Використати рушій "Blender Render". Для отримання списку, доступних рушії рендерингу, запустіть blender -E help.
-s 10 -e 500
Задати стартовий кадр як 10 та кінцевий кадр як 500.
-t 2
Використати тільки два потоки.
