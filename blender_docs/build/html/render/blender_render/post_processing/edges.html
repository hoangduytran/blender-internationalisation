Зображування, рендеринг Контуру -- Edge Rendering¶

Сцена з мультиковими -- Toon матеріалами.

Мультикові -- toon -- відтінювачі можуть давати зображ, рендер з виглядом, подібним у коміксах або манга, впливаючи на відтінки кольорів. Цей ефект не є ідеальним, оскільки реальні комікси і манга зазвичай мають контури, зроблені тушшю, китайським чорнилом. Blender можу додавати цю функцію як операцію після оброблення.

Опції¶

Устави для мультикового контуру.

Контур -- Edge
Це змушує Blender шукати краї, ребра на зображі, рендері та додавати до них «контур».

Перед повтором зображування, рендерингу необхідно задати деякі параметри:

Поріг -- Threshold
Поріг кута між гранями для рисування контурів, від 0 до 255. Значення 10 дасть просто контур об'єкта на фоні, а вищі значення почнуть додавати контури на краях поверхонь, починаючи з гостріших кутів. При максимальній інтенсивності контур буде навіть злегка висвітлювати геометрію ліній підподілених ребер в областях недосконалого згладжування.
Колір RGB -- Color RGB
Колір рендерених контурів.

Приклади¶

Повторно зображена, рендерена сцена із заданим мультиковим контуром.

Можна відділити шар контуру за допомогою однойменного шару зобража -- render layer, що спеціально для цього призначений. Значення альфа каналу буде 0 там, де немає контуру, та 1, де контур наявний. Відділивши контур на окремий шар, ви можете далі розмити його, змінити колір, маскувати тощо. Ілюстрація вище показує, як це робити. У цьому прикладі шар зобража з Контуром створено так, що він має тільки шари Sky та Edge. Для інших шарів зобража шар Edge вимкнено, а тому вони дають звичайне зображення. На панелі «Після Оброблення» увімкнено опцію Edge та задано «Поріг» як 10 та чорний колір для контура. Далі, дані з цього шару подаються на вузол «Розмив» -- Blur. Використовуючи вузол «Альфа Над» -- Alpha Over, далі, куб компонується поверх розмитого контура. Результат дає свого роду ефект м'якої тіні. Зауважте, що на вузлі «Альфа Над» увімкнено опцію «Передмноження Альфа» -- Premultiply, оскільки зображення з Контуром вже має альфа зі значенням 1.0.
