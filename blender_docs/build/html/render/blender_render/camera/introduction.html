Вступ¶

Камера -- Camera -- це об'єкт, забезпечує значення зображування, рендерингу зображень з Blender'а. Вона визначає, яка частина сцени є видимою на зображеному, рендереному зображенні. Стандартно, сцена містить одну камеру. Однак, сцена може містити більше, ніж одну камеру, але тільки одна з них буде використовуватися одночасно. Тому, вам треба буде тільки додати нову камеру, якщо схочете зробити стики, переходи між оглядами з них. Дивіться детальніше тут -- Animating Cameras.

Зміна активної камери¶

Орієнтир -- Reference

Mode:Object Mode
Hotkey:Ctrl-Numpad0

Активна камера (зліва).

Активна -- active камера -- це камера, що поточно використовується для зображування, рендерингу та огляду сцени через неї, Numpad0.

Виберіть камеру, яку ви хочете зробити активною та натисніть Ctrl-Numpad0 (зробивши так, ви також перемикнете огляд на огляд камери). Для рендерингу, зображування кожна сцена **повинна* мати активну камеру.

Активна камера може також бути задана у вкладці «Сцена» -- Scene Редактора Властивостей -- Properties Editor.

Камера із суцільно забарвленим трикутником вгорі -- це активна камера. Вказівники ліміту та імли камери рисуються темнішим, якщо камера не є активною камерою для поточної сцени.

Ghi chú

Активна камера, також як і шари сцени, може бути специфічною для даного огляду або глобальною (блокованою) для усієї сцени. Дивіться детальніше тут -- Local Camera.

Границя Зобража -- Render Border¶

Орієнтир -- Reference

Mode:All modes
Menu:View ‣ Render Border
Hotkey:Ctrl-B

Перемикач Границя Зобража -- Render Border.

Коли активний огляд камери, ви можете визначити підрегіон для зобража, рендера, нарисувавши прямокутник у межах кадру камери. Ваші зображі, рендери тепер будуть обмежені частиною сцени, видимою у межах такого прямокутника. Це може бути дуже корисним для скорочення часу на зображування з метою швидких передоглядів потрібної ділянки.

Цей зображ у межах може вимикатися та вмикатися за допомогою опції «Границя» -- Border на панелі «Розміри» -- Dimensions у вкладці «Зображ» -- Render Редактора Властивостей.

Ghi chú

Коли Границя Зобража є активною, то Вибірний Розмив Рухом -- Sampled Motion Blur стає доступним у 3D Огляді -- 3D View.
