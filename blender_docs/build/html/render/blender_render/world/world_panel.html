Панель «Світ» -- World Panel¶

Світ -- World
Для Світу Меню блоку даних.
Рахунок Текстур -- Texture Count
Показує рахунок текстур у стеку текстур світу.

Ghi chú

Фонове, задньопланове зображення у 3D

Для використання зображення як фонового, задньопланового у 3D View, наприклад, як довідкового орієнтира при здійсненні моделювання, дивіться тут -- using a Background Image.

Передогляд -- Preview¶

Показує огляд всередині сфери, на якому фонові текстури розкладаються.

Світ (Фон, задньоплан)¶

Панель «Світ» -- World.

Небо -- Sky¶

Як інтерпретувати кольори нижче залежить від того, який обрано вид неба Sky.

Жодне Не увімкнено
Якщо жодна з цих трьох кнопок не увімкнена, то ваш фон буде простим плоским кольором (використовується тільки колір горизонту).
Паперове Небо -- Paper Sky
If this option is added, the gradient keeps its characteristics, but it is clipped in the image (it stays on a horizontal plane (parallel to XY plane): on any camera angle, the horizon is always at the middle of the image).
Змішане Небо -- Blend Sky
Фоновий колір змішується у градієнті від кольору горизонту до кольору зеніту. Якщо тільки ця кнопка увімкнена, то цей градієнт буде йти знизу уверх на рендереному зображенні незалежно від орієнтації камери.
Реальне Небо -- Real Sky
Якщо ця опція додається, то продукований градієнт має два переходи, від надиру (такий же колір як у зеніту) до горизонту та далі до зеніту; це змішання також залежить від орієнтації камери, що робить його більш реалістичним. Колір горизонту є точним на горизонті (на площині XY), а колір зеніту використовується для точок, вертикальних вище та нижче камери.

При використання для сонцевої лампи Sun Lamp стають доступними опції Небо й Атмосфера -- Sky & Atmosphere у вкладці даних об'єкта Lamp.

Текстури -- Textures¶

Розкладання -- Mapping¶

Instead of a color, or blend of two colors, Blender can use a 2D image which it maps to a very large Box or sphere which encompasses the entire scene, or which it maps to a virtual space around the scene.

Текстури для Світу або Світові текстури доступні у вкладці Texture (просто виберіть вкладку World, а потім вкладку Texture). Вони використовуються майже так само, як Матеріалові текстури, за винятком кількох відмінностей. Ці текстури можуть бути розкладені відповідно до:

Меню вибору Texture Coordinates.

Координати Текстури -- Texture Coordinates
Огляд -- View
The default orientation, aligned with the coordinates of the final render.
Глобально -- Global
Використовує глобальні координати.
Кутова Карта -- Angular Map
Used to wrap a standard hemisphere angular map around the scene in a dome. This can be used for image-based lighting with Ambient Occlusion set to sky color. You will generally need a high dynamic range image (HDRI) angular map. (It will look like a weird spherical image).
Сфера -- Sphere
Сферичне розкладання, подібне до того, яке є для матеріалів.
Труба -- Tube
Обгортає прямокутну текстуру навколо циліндра, подібно до того, яке є для матеріалів.
Об'єкт -- Object
Позиція текстури відносно локального текстурного простору вказаного об'єкта.

Вплив -- Influence¶

Панель «Вплив» -- Influence для світових текстур у вкладці Texture.

Світова текстура впливає тільки на колір, але чотирма різними способами:

Змішання -- Blend
Робить проглядання кольору Горизонту, де текстура є не-нульовою.
Горизонт -- Horizon
Впливає на колір горизонту.
Зеніт Угорі -- Zenith Up
Впливає на колір зеніту над головою.
Зеніт Внизу -- Zenith Down
Впливає на колір зеніту під ногами.

Якщо ви розчаровані тим, що ваша камера виглядає, ніби несе текстуру замість того, щоб обертатися у текстурі, то ви повинні перевірити стяг Real Sky на панелі World.
