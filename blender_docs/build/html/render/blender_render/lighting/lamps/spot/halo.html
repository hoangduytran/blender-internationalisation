Об'ємні ефекти Прожектора -- Spot Volumetric Effects¶

Опції ефекту «Ореол» -- Halo для лампи типу «Прожектор» -- Spot.

Освітлювачі типу Spot можуть також продукувати «об'ємні» ефекти. Дивіться детальніше тут -- Volumetric Light.

Ореол -- Halo
Стяг «Ореол» -- Halo вмикає об'ємний ефект для лампи Spot. Ця кнопка повинна бути активною, щоб цей об'ємний ефект був видимий. Зауважте, що при використанні режиму буферованих тіней, ви отримуєте додаткові опції, описані тут -- Spot Buffered Shadows.
Інтенсивність -- Intensity
Повзунок Intensity керує тим, наскільки інтенсивним/щільним є об'ємний ефект, що генерується цим джерелом освітлення. Чим нижче значення Intensity, тим менш видимим є цей об'ємний ефект, тоді як вищі значення Intensity дають більш помітний та щільніший ефект.
Крок -- Step
Це поле може мати значення між (0 до 12). Воно використовується для визначення, чи лампа Spot буде давати відкидання об'ємних тіней та яку якість ці об'ємні тіні будуть мати. Якщо значення Step задано як 0, то не буде генеруватися жодної об'ємної тіні. На відміну від інших контролерів, при збільшенні значення Step якість об'ємних тіней зменшується (але вимагає менше часу на зображ, рендер), і навпаки.

Mẹo

Значення устави Step

Значення 8 для Halo Step є зазвичай добрим компромісом між швидкістю та точністю.

Blender імітує об'ємне освітлення тільки для ламп типу Spot, при використанні його внутрішнього зображувача, рендерера. Це може призводити до деяких дивних результатів для певних комбінацій устав Energy для освітлювача та Intensity для ореолу. Наприклад, наявність нульового або дуже низького значення Energy для Spot та дуже високого значення Intensity для ореолу може давати темний/чорний ореол, чого не трапляється у реальному світі. Просто пам'ятайте про цю можливість, коли використовуєте ореоли за допомогою внутрішнього зображувача.

Ghi chú

Ефект ореолу може бути значно покращено за допомогою використання буферованих тіней: коли значення Step для ореолу не є нульовим, то вони можуть утворювати «об'ємні тіні». Дивіться детальніше тут -- Buffered Shadows.
