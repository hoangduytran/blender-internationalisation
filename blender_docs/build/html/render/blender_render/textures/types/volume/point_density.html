Текстура «Щільність Точок» -- Point Density Texture¶

Щільність точок рендерить задану хмару точок (вершини об'єкта або систему частинок) як 3D об'єм, використовуючи визначений користувачем радіус для цих точок. Внутрішньо, ця система використовує структуру даних BVH (bounding volume hierarchy -- ієрархія межових об'ємів) для швидкого прогляду діапазону.

Стандартно ці рендерені точки є сферичними з різними опціями спаду згладження, а також простими опціями Турбулентності для зміщення результату з шумом, що додає детальність. При використанні Point Density із системою частинок додаткова інформація, як скорість частинок, вік та швидкість, може бути візуалізована за допомогою градієнта кольорів/альфа.

Масштаб -- Scale
Множник для привнесення швидкості частинок у межі прийнятного діапазону.
Джерела Кольору Вершин -- Vertex Color Sources
Колір Вершин -- Vertex Color

Використовує шар кольору вершин для забарвлення текстури Point Density.

Ghi chú

Кольори вершин визначаються по кутках граней. Одинична вершина може мати стільки різних кольорів, скільки є граней, частиною яких вона є, Фактичний колір текстури Щільність Точок усереднюється з усіх вершинних кутків.

Вагомість Вершини -- Vertex Weight
Використовує вагомості з групи вершин як значення інтенсивності. Додатковий градієнт кольорів може використовуватися для конвертування інтенсивності у кольори RGB.
Нормалі Вершин -- Vertex Normals
Використовує нормалі вершин з об'єктного простору як значення RGB.
