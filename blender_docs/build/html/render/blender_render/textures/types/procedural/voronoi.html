Вороний -- Voronoi¶

Текстура Voronoi використовується для генерування дуже переконливого вигляду ефекту металу, особливо «кованого». Органічні відтінювачі (наприклад, шкури, вени у шкірі).

Voronoi Texture panels.

Опції¶

Метрика Відстані -- Distance Metric

Ця процедурна текстура має сім варіантів Метрики Відстані. Вони визначають алгоритм для знаходження відстані між комірками текстури. Цими варіантами є:

  • Мінковський -- Minkowsky
  • Мінковський -- Minkowsky 4
  • Мінковський -- Minkowsky 1/2
  • Чебишев -- Chebychev
  • Манхеттен -- Manhattan
  • Квадрат Відстані -- Distance Squared
  • Справжня Відстань -- Actual Distance

Устава Minkowsky має визначуване користувачем значення (кнопка Exponent), яке визначає експоненту Мінковського e для функції відстані:

(xe + ye + ze)1/e

Це значення продукує метрику відстані Manhattan, значення, менше ніж це, продукує зірки (як 0.5, воно дає Minkowsky 1/2), і вищі значення продукують квадратні комірки (як 4.0, воно дає Minkowsky 4, як 10.0 дає Chebychev). Таким чином, усі устави відстані базово це одне і теж -- варіація Minkowsky.

Ви можете отримувати нерівної форми округлені комірки за допомогою опцій Actual Distance / Distance Squared.

Minkowsky Exponent: 0.5 (Minkowsky 1/2).

Minkowsky Exponent: 1 (Manhattan).

Minkowsky Exponent: 2 (Actual Distance).

Minkowsky Exponent: 4 (Minkowsky 4).

Minkowsky Exponent: 10 (Chebychev).

Distance Squared (більш контрастно, ніж Actual Distance).
Вагомості Вирізнення -- Feature Weights
Ці чотири повзунки на панелі Voronoi представляють значення чотирьох констант Ворлі -- Worley, які використовуються для розрахунку відстаней між кожними комірками у текстурі на основі метрики відстані. Коригування цих значень може давати цікаві ефекти на кінцевий результат...
Забарвлення -- Coloring
Чотири устави (Інтенсивність -- Intensity, Позиція -- Position, Позиція та Обвід -- Position and Outline, та Позиція, Обвід та Інтенсивність -- Position, Outline, and Intensity), що можуть використовувати чотири різних базиси шуму як методи для розрахунку кольору та інтенсивності виводу текстури. Це дає текстуру Вороний, яку ви створюєте за допомогою «повзунків Ворлі» зовсім з іншим виглядом та, яка є еквівалентом устави базису шуму для інших текстур.
