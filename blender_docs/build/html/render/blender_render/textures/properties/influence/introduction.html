Вступ¶

Текстури можуть впливати не тільки на колір матеріалу, вони також можуть впливати на багато інших властивостей матеріалу. Різні аспекти матеріалу, на які може впливати текстура, скеровуються на панелі Influence.

Ghi chú

Ці опції текстури є для типів матеріалів Surface та Wire, а також у деяких випадках вони доступні для типів матеріалу Volume та Halo.

Матеріали типу «Поверхня» -- Surface та «Дріт» -- Wire¶

Панель «Вплив» -- Influence у вкладці «Текстура» -- Texture для типу матеріалу «Поверхня» -- Surface.

Розсіяння -- Diffuse¶

Інтенсивність -- Intensity
Величина впливу текстури на розсіяну, дифузну відбиваність.
Колір -- Color
Величина впливу текстури на базовий колір або значення RGB матеріалу.
Альфа -- Alpha
Вплив на безпрозорість матеріалу. Також використовується Z Transparency для світла та якщо комбінуються одночасно кілька каналів.
Півпрозорість -- Translucency
Вплив на півпрозорість.

Блікування -- Specular¶

Інтенсивність -- Intensity
Величина впливу текстури на блікову відбиваність.
Колір -- Color
Впливає на колір блікування -- Specular, колір відбивів, створюваних лампами на глянцевому матеріалі.
Жорсткість -- Hardness
Вплив на жорсткість блікування. Значення 1 для DVar є еквівалентом значення 130 для Hardness of 130, DVar як 0.5 еквівалент Hardness як 65.

Відтінення -- Shading¶

Оточення -- Ambient
Вплив на величину Освітлення Оточення, яке матеріал приймає.
Емітування -- Emit
Впливає на величину світла, емітованого матеріалом.
Дзеркалення -- Mirror
Influences the mirror color. This works with environment maps and ray-traced reflection.
Дзеркалення Променя -- Ray Mirror
Influences the strength of ray-traced mirror reflection.

Геометрія -- Geometry¶

Нормаль -- Normal
Тут визначається таке явище, яке відоме, зазвичай, як розкладання вигинів -- bump mapping, це змінює напрямки нормалей поверхні. Це використовується для підробки, імітування вад або нерівностей поверхні або для створення рельєфу, через розкладку вигинів.
Жолоблення -- Warp
Warp дозволяє текстурам впливати/спотворювати текстурні координати наступного каналу текстури. Це спотворення залишається активним для всіх наступних під ним каналів, допоки не буде задане новий стан Warp. Задання фактора цієї устави як нуль скасовує цей ефект.
Зміщення -- Displace
Впливає на зміщення вершин за допомогою Карт Зміщення -- Displacement Maps.

Інші контролери¶

Розкладка Вигинів -- Bump Mapping

Устави для розкладання вигинів.

Метод -- Method
Найкраща Якість -- Best Quality, Стандарт -- Default, Сумісно -- Compatible, Оригінал -- Original
Простір -- Space
Простір Текстури -- Texture Space, Простір Об'єкта -- Object Space, Простір Огляду -- View Space

Матеріалу типу «Об'єм» -- Volume¶

Панель «Вплив» -- Influence у вкладці «Текстура» -- Texture для типу матеріалу «Об'єм» -- Volume.

Спеціальні опції текстур для матеріалів типу «Об'єм» -- Volume.

Щільність -- Density
Вплив текстури на щільність об'єму.
Емісія -- Emission
Вплив текстури на емісію світла об'ємом.
Розсіювання -- Scattering
Величина впливу текстури на розсіювання світла об'ємом.
Відбиття -- Reflection
Величина впливу текстури на яскравість розсіяного світла.
Колір Емісії -- Emission Color
Величина впливу текстури на колір емісії.
Пропускання -- Transmission
Величина впливу текстури на результатний колір світла, яке було розсіяне/поглинуте об'ємом.
Колір Відбиття -- Reflection Color
Величина впливу текстури на колір розсіяного світла.

Матеріали типу «Ореол» -- Halo¶

Панель «Вплив» -- Influence у вкладці «Текстура» -- Texture для типу матеріалу «Ореол» -- Halo.

Спеціальні опції текстур для матеріалів типу «Ореол» -- Halo.

Розмір -- Size
Величина впливу текстури на дзеркалення променя.
Жорсткість -- Hardness
Величина впливу текстури на жорсткість відбиття.
Додання -- Add
Величина впливу текстури на півпрозорість.
