Режими Змішання Текстури -- Texture Blending Modes¶

Режими Змішання -- Blending Modes -- це різні методи керування тим, як текстура впливає на властивості матеріалу. У той час як режим змішання визначає здійснювану конкретну операцію, фактор змішання керує величиною, загально «силою» цієї операції. Для текстур такий фактор змішання задається за допомогою повзунків на панелі «Вплив» -- Influence.

Змішання -- Blend
Здійснювана операція змішання. Про кожен з типів змішання дивіться детальніше «Режими Змішання Кольорів» -- Color Blend Modes.
RGB в Інтенсивність -- RGB to Intensity
При увімкненні цієї опції текстура RGB (яка стандартно впливає на кольори) використовується як текстура інтенсивності (впливатиме на значення).
Колір Змішання -- Blend Color
Якщо текстура розкладається на Колір, то який колір буде змішуватися, відповідно, з інтенсивністю текстури?
Негативно -- Negative
Ефект розвертання навпаки впливу текстури. Зазвичай, білий означає наявність, чорний означає відсутність, Negative розвертає це навпаки.
Трафарет -- Stencil
Активна текстура використовується як маска для усіх наступних текстур. Це корисно для напівпрозорих текстур та «карт бруду». Чорний задає піксель як «нетекстурованоздатний». Режим Stencil працює подібно до маскового шару у 2D програмі. Ефект текстури трафарету не може бути заміщено, тільки розширено. Вам потрібно подавати на увід карту інтенсивності.
Цільове Значення -- Destination Value

Значення (не для RGB), за допомогою якого текстура Інтенсивності змішується з поточним значенням. Два приклади:

  • Значення Emit зазвичай є 0. При розкладанні текстури на Emit ви отримаєте максимальний ефект, оскільки DVar стандартно є 1. Якщо ви задасте DVar як 0, то текстура не здійснюватиме жодного впливу.
  • Якщо ви хочете мати прозорий матеріал та використовуєте розкладання текстури на Alpha, то нічого не відбудеться зі стандартними значеннями устав, оскільки значення Alpha на у вкладці Material дорівнює 1. Тому ви повинні задати DVar як 0 для отримання прозорого матеріалу (а також, звичайно, увімкнути Z Transparency). Це поширена проблема для початківців. Або роблять це навпаки: задають Alpha як 0 та залишаються Dvar як 1. Звичайно, текстура використовується тоді інвертовано.
