Вузли Комбінування/Розділяння -- Combine/Separate Nodes¶

Усі ці вузли роблять по суті однакову річ:

  • Розділяння -- Separate: Розділяє зображення на його компонівні колірні канали.
  • Комбінування -- Combine: Пере/комбіновує зображення з окремих його компонівних колірних каналів.

Дані вузли можуть використовуватися для маніпулювання кожним з колірних каналів окремо. Кожен тип цих вузлів відрізняється від застосовуваного колірного простору -- color space.

У контексті компонівних і текстурових вузлів кожен з цих вузлів підтримує Альфа-канал. У контексті текстурових вузлів доступний для них тільки колірний простір RGB. У контексті відтінювальних, шейдерних вузлів внутрішній зображувач Blender'а (Blender internal) додає колірний простір HSV, а зображувач Cycles у контексті відтінювальних вузлів пропонує додаткову пару вузлів для комбінування/розділяння вектора (XYZ).

The Combine nodes could also be used to input single color values. For RGBA and HSVA color spaces it is recommended to use the Вузол RGB -- RGB Node. Some common operation could easier executed with the Вузли типу «Колір» -- Color Nodes.

Вузли Розділяння/Комбінування HSVA -- Separate/Combine HSVA Nodes¶

Вузол «Комбінування RGBA» -- Combine RGBA Node.

Вузол «Розділяння RGBA» -- Separate RGBA Node.

Input/Output¶

Зображення -- Image
Стандартний увід/вивід зображення.
  • R (Red - Червоний)
  • G (Green -- Зелений)
  • B (Blue -- Синій)
  • A (Alpha - Альфа)

Властивості¶

Цей вузол наразі немає властивостей.

Приклади¶

An example of blurring the alpha channel.

У першому прикладі ми беремо Альфа-канал і розмиваємо його, а потім комбінує знову з усіма кольорами. При застосуванні до сцени краї об'єктів будуть розмиті, а не жорсткі. Це майже як загладжування (anti-aliasing), але у тривимірному сенсі. Використовуйте цей уклад вузлів при доданні комп'ютерно-графічних елементів у відзняте для вилучення будь-яких жорстких країв. Анімування цього ефекту у більш широкому масштабі може дати появу об'єкта при вході чи виході з «фази» подібно до ефекту «виходу з фази» при подорожі у часі.

Приклади¶

An example of the scaling the Luminance channel.

This example has a Math (Multiply) node increasing the luminance channel (Y) of the image to make it brighter.

Mẹo

If running these channels through a Color Ramp node to adjust value, use the Cardinal scale for accurate representation. Using the Exponential scale on the luminance channel gives a high-contrast effect.
