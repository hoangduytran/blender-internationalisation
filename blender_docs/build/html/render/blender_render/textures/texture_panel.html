Панель «Текстура» -- Texture¶

У Редакторі Властивостей -- Properties Editor виберіть вкладку «Текстура» -- Texture; це покаже панель «Текстура» -- Texture.

Панель «Текстура» -- Texture.

Контекст Текстури -- Texture Context

Ці радіокнопки вибирають тип даних текстури, тобто, вид текстури, що редагується.

Світ -- World
Фон Світу -- World Background.
Матеріал/Лампа -- Material/Lamp
Матеріаловий тип, що описується у цьому підрозділі. Лампові текстури -- Текстури ламп -- Lamps Textures, що описуються у підрозділі про освітлення.
Пензель -- Brush
Пензлеві текстури, що застосовуються на етапі Малювання і Ліплення -- Painting & Sculpting.

Стек Текстур -- Textures Stack¶

Активна Текстура -- Active Texture
Гнізда текстур показуються через огляд списком -- Огляди списків. Порядок у цьому стеку визначає, як текстури накладаються у рендереному зображенні. Стяг вмикає/вимикає вибрану текстуру.

Блок даних Текстури -- Texture Data-Block¶

Текстура -- Texture
Текстурове Меню блоку даних для вибраного гнізда текстури.

Тип Текстури -- Texture Type¶

Тип Текстури -- Texture Type
Дає змогу обрати тип текстури, що використовується для поточного блоку даних текстури. Ці типи описуються детально тут -- in this section.
