Запікання Зобража -- Render Baking¶

Запікання -- baking, у загальному, -- це акт перед-обчислення чогось, щоб пришвидшити деякі інші процеси, що йдуть далі по конвеєру. Зображування, рендеринг з нуля забирає багато часу, залежно від обраних вами опцій. Тому, Blender дозволяє вам «запекти» деякі частини зобража, рендера наперед для обраних об'єктів. Звідси, коли ви далі натиснете команду Render, то вся сцена буде зображуватися, рендеритися набагато швидше, оскільки кольори запечених даних для цих об'єктів не треба бути повністю обчислювати.

Запікання зобража створює 2D растрові зображення зображеної, рендереної поверхні сітьових об'єктів. Ці зображення можуть бути пере-розкладені на об'єкт за допомогою координат UV цього об'єкта. Запікання здійснюється для кожної окремої сіті і може бути зроблене, якщо така сіть була попередньо розгорнута по UV. Хоча це забирає час для укладання і самого процесу, однак, це далі заощаджує час на зображування, рендеринг. Якщо ви зображуєте, рендерите тривалу анімацію, то час, витрачений на запікання, може бути набагато меншим, ніж час, витрачений на зображування, рендеринг кожного кадру тривалої анімації.

Запікання Зобража вживається для таких ресурсо-затратних випадків, таких як «Загорода Оточення» -- Ambient Occlusion (AO) або м'які тіні для площинних -- area освітлювачів. Якщо ви запекли Загороду Оточення (AO) для головних об'єктів, вам не треба бути вмикати її для повного зображування, що заощаджує час для нього.

Use Full Render or Textures to create an image texture; baked procedural textures can be used as a starting point for further texture painting. Use Normals to make a low resolution mesh look like a high resolution mesh. To do that, UV unwrap a high resolution, finely sculpted mesh and bake its normals. Save that normal map, and Mapping (texture settings) the UV of a similarly unwrapped low resolution mesh. The low resolution mesh will look just like the high resolution, but will have much fewer faces/polygons.

Переваги

  • Можливість значно скоротити час на зображування, рендеринг.
  • Легше здійснення малювання текстур.
  • Скорочений рахунок, кількість полігонів, з яких складається сіть.
  • Повторні зображі, рендери здійснюється швидше, багато заощаджуючи час для них.

Недоліки

  • Об'єкт повинен бути розгорнутий по UV.
  • Якщо запікаються тіні, освітлювачі та об'єкт не можуть рухатися один відносно одного.
  • Запікання великих текстур (наприклад, 4096×4096) може бути сильно ресурсно-затратним, особливо щодо оперативної пам'яті, і звідси може здійснюватися так же повільно, як і повне зображування, рендеринг.
  • На розгортання, запікання, збереження файлів та застосування текстур на канал має бути витрачено людський (робочий) час.

Опції¶

Режим Запікання -- Bake Mode¶

Повний Зображ -- Full Render¶

Запікає усі матеріали, текстури та освітлення, за винятком бліковості (specularity) та підповерхневого розсіювання (SSS).

Загорода Оточення -- Ambient Occlusion¶

Запікає загороду оточення, як вона визначена на панелях вкладки «Світ» -- World. Ігнорує усі освітлювачі у сцені.

Загорода Оточення -- Ambient Occlusion.

Нормалізовано -- Normalized
Вмикає нормалізацію, використовуючи устави матеріалу.

Тінь -- Shadow¶

Запікає тіні та освітлення.

Нормалі -- Normals¶

Bakes tangent and camera-space normals (among many others) to an RGB image.

Нормалі -- Normals.

Простір Нормалей -- Normal Space

Нормалі можуть бути запечені у різних просторах:

Простір Камери -- Camera space
Стандартний метод.
Простір Світу -- World space
Нормалі у координатах світу, залежно від трансформацій та деформацій об'єкта.
Простір Об'єкта -- Object space
Нормалі у координатах об'єкта, незалежно від трансформацій об'єкта, але залежно від його деформацій.
Простір, дотичний до об'єкта -- Tangent space
Нормалі у координатах дотичного простору, незалежно від трансформацій та деформацій об'єкта. Це є новий стандарт, та є правильним вибором у більшості випадків, оскільки далі така карта нормалей може використовуватися також для анімованих об'єктів.

Такі ж простори можуть бути вибрані також для матеріалів серед опцій зображеннєвої текстури для устави «Карта Нормалей» -- Normal Map. Для коректних результатів ця устава для текстури повинна відповідати уставі, що використовується для запікання.

Текстури -- Textures¶

Запікає тільки кольори матеріалів та текстур без відтінення.

Зміщення -- Зміщення¶

Подібно до запікання карт нормалей, карти зміщення можуть також бути запечені з високо роздільного об'єкта та розкладені на низько роздільний об'єкт, використовуючи опцію «Виділене на Активне» -- Selected to Active.

Зміщення -- Зміщення.

Нормалізовано -- Normalized
Нормалізує відстань.

When using this in conjunction with a Subdivision Surface and Displacement modifiers within Blender, it is necessary to temporarily add a heavy Subdivision Surface Modifier to the 'low-res' model before baking. This means that if you then use a Displacement Modifier on top of the Subdivision Surface, the displacement will be correct, since it is stored as a relative difference to the subdivided geometry, rather than the original base mesh (which can get distorted significantly by a Subdivision Surface). The higher the render subdivision level while baking, the more accurate the displacements will be. This technique may also be useful when saving the displacement map out for use in external renderers.

Емісія -- Emission¶

Запікає колір емітування -- Emit або сяяння -- Glow матеріалу.

Альфа -- Alpha¶

Запікає значення альфа або, іншими словами, прозорість матеріалу.

Колір Дзеркалення -- Mirror Color та Інтенсивність Дзеркалення -- Mirror Intensity¶

Запікають значення кольору або інтенсивності дзеркалення -- Mirror, відповідно.

Колір Бліків -- Specular Color та Інтенсивність Бліків -- Specular Intensity¶

Запікають значення кольору або інтенсивності бліків -- Specular, відповідно.

Full render.

Додаткові опції¶

Зчищення -- Clear
Якщо увімкнено, то зчищає зображення в обраний фоновий колір (стандартним є чорний) перед запіканням зобража.
Припуск -- Margin
Baked result is extended this many pixels beyond the border of each UV "island", to soften seams in the texture.
Розділення -- Split
Фіксовано -- Fixed
Розділяє передбачувано чотирибічники (0, 1, 2) (0, 2, 3).
Fixed Alternate
Розділяє передбачувано чотирибічники (0, 1, 2) (0, 2, 3).
Автоматично -- Automatic
Розділяє чотирибічники так, щоб у результаті запікання було найменше спотворення.
Виділене на Активне -- Select to Active

Вмикає інформацію з інших об'єктів для запікання на активному об'єкті.

Відстань -- Distance
Controls how far a point on another object can be away from the point on the active object. Only needed for Selected to Active. A typical use case is to make a detailed, high-poly object, and then bake its normals onto an object with a low polygon count. The resulting normal map can then be applied to make the low-poly object look more detailed.
Відхил -- Bias
Bias towards further away from the object (in Blender units).

Ghi chú

Сіть повинна бути видима для зобража, рендера

Якщо сіть не є видимою для звичайного рендера, зобража, наприклад, через те, що така властивість вимкнута для зображування, рендерингу у Структураторі -- Outliner або увімкнено для неї уставу DupliVerts, то вона не може бути запечена.

Робхід¶

  1. In a 3D View editor, select a mesh and enter UV/Face Select mode.
  2. Unwrap the mesh object.
  3. У редакторі UV/Зображень -- UV/Image Editor створіть нове або відкрийте наявне зображення. Якщо для 3D Огляду -- 3D View задано режим показу «Текстуровано» -- Textured, то ви повинні тепер побачити це зображення, розкладене на вашу сіть. Переконайтеся, що усі її грані виділені.
  4. In the Bake panel at the bottom of the Render menu, bake your desired type of image (Full Render, etc.).
  5. Після завершення зображування/запікання Blender замінить задане зображення Запеченим зображенням.
  6. Збережіть запечене зображення.
  7. Apply the image to the mesh as a UV texture. For displacement and normal maps, refer to Bump and Normal Maps. For full and texture bakes, refer to Textures.
  8. Покращіть це зображення за допомогою процесу, описаного нижче, за допомогою Малювання Текстур -- Texture Paint або скориставшись зовнішнім редактором зображень.
