Зображування, рендеринг Ореолу -- Halo Rendering¶

Активування зображування, рендерингу Ореолу.

Ореолові матеріали передбачають зображування, рендеринг кожної точки об'єкта як сяючої точки або маленьких хмар світла. Хоча, ці точки не є реальними освітлювачами, оскільки не відкидають світло у сцену, як лампа. Вони називаються «ореоли» -- Halos, оскільки ви можете бачити їх, але вони не мають жодної речовини.

Ореоли рендеряться, зображуються за допомогою відтінювачів вершин, а не відтінювачів граней.

Цей матеріал є корисним для імітування спеціальних ефектів, як частинкові ефекти або відблиски в об'єктиві.

Опції¶

Панелі ефекту «Ореол» -- Halo.

Для вмикання Ореолів клацніть на кнопці «Ореол» -- Halo на панелі «Матеріал» -- Material.

As you will see in the 3D View, the mesh faces are no longer rendered. Instead just the vertex is rendered, since that is where each halo will originate. Halos can be hard to find in a crowded scene, so name it well for easy location in the Outliner.

У Редакторі Властивостей -- Properties Editor на місці, де зазвичай знаходяться панелі Diffuse, Specular та Shading, ми тепер побачимо панелі, що стосуються характеристик Halo:

Панель «Ореол» -- Halo¶

Альфа -- Alpha
Прозорість.
Колір Розсіяння -- Diffuse Color
Колір власне ореолу.
Базис -- Seed
If non-zero, randomizes the ring dimension and line location. To use, give any (integer) number to start the random number generator.
Розмір -- Size
Задає розмір ореолу.
Жорсткість -- Hardness
Задає жорсткість ореолу. Подібно до жорсткості блікування.
Додання -- Add

Визначає, наскільки кольори ореолу «додаються до», а не змішуються з, кольорів об'єкта позаду та разом з іншими ореолами. При збільшення цієї устави буде здаватися, що ореол освітлює об'єкти, що переміщуються позаду та крізь нього.

Ефект устави «Додання» -- Add.

Текстура -- Texture
Задає для ореолу текстуру. Стандартно, текстури накладаються на об'єкти по Об'єктним -- Object координатам та відображають ореоли, впливаючи на їх колір, у цілому, на основі кольору вершин, з яких походить ореол. Вмикання цієї функції дає те, що текстура здійснюватиме ефект у межах ореолу, і звідси він матиме варіювання кольорів або прозорості; це буде накладати текстуру повністю на кожний ореол. Ця техніка виявляється дуже корисною, коли ви хочете створити, як приклад, реалістичний ефект дощу за допомогою систем частинок.
Нормаль Вершини -- Vertex Normal
Використовує нормаль вершини для визначення розміру ореолу.
Екстремум Альфа -- Extreme Alpha
Посилює альфа.
Відтінено -- Shaded

Дозволяє ореолу приймати світла та тіні від зовнішніх об'єктів.

При увімкненні цієї опції на ореол буде впливати локальне світло; лампи робитимуть його яскравішим та впливати на його колір розсіяння, дифузії та інтенсивність.

М'якість -- Soft
Пом'якшує краї ореолу на перетинах з іншою геометрією.

Крім того, кілька інших спеціальних ефектів на базі ореолу доступні. Для вмикання деяких або усіх цих ефектів задайте кількість точок/кілець або установіть колір для кожного ефекту індивідуально:

Кільця -- Rings
Додає кругові кільця навколо ореолу.
Лінії -- Lines
Додає лінії від центру ореолу.
Зіркові верхівки -- Star tips
Задає для ореолу зіркоподібну форму.

You cannot use color ramps. Lines, Rings and an assortment of special effects are available with the relevant toggle buttons, which include Flare, Rings, Lines, Star, Texture, Extreme Alpha, and Shaded. Halo variations shows the result of applying a halo material to a single vertex mesh.

Halo variations.

Панель «Відблиск» -- Flare¶

Стяг у назві цієї панелі вмикає додаткове зображування, рендеринг ефекту відблиску в об'єктиві для ореолу.

Розмір -- Size
Задає фактор, на який відблиск є більшим, ніж ореол.
Посилення -- Boost
Задає додаткову силу для відблиску.
Базис -- Seed
Визначає зсув у таблиці базису чисел для відблиску.
Підвідблиски -- Subflares
Задає кількість підвідблисків.
Підрозмір -- Subsize
Задає розміри для підвідблисків, точок та кілець.

Відблиски в Об'єктиві -- Lens Flares¶

Наші очі навчені вірити, що зображення є реальним, якщо на ньому показуються артефакти, які є результатом механічного процесу фотографування. Розмив Рухом -- Motion blur, Глибина Різкості -- Depth of Field та відблиски в об'єктиві -- lens flares є лише трьома прикладами цих артефактів. Перші два обговорюються окремо у главі про зображування, рендеринг; третій може продукуватися за допомогою спеціальних ореолів. Імітований відблиск в об'єктиві повідомляє глядачу, що зображення було створено за допомогою камери, яка змушує глядача думати, що вона реальна, справжня.

We create lens flares in Blender from a mesh object using first the Halo button and then the Flare options in the Shaders Panel of the material settings. Try turning on Rings and Lines, but keep the colors for these settings fairly subtle. Play with the Subflares number and Flare Seed settings until you arrive at something that is pleasing to the eye. You might need to play with Boost: for a stronger effect Fig. Lens flare. settings.

Зауважте, що цей засіб не імітує фізику фотонів, що проходять крізь лінзу об'єктива; він просто приємний для ока.

Відблиск в об'єктив за допомогою Blender'а виглядає гарно у русі та зникає, коли інший об'єкт загороджує сіть, на основі якої він утворюється.

Lens flare.

Текстурування ореолу¶

Стандартно, текстури накладаються на об'єкти по Об'єктним -- Object координатам та відображають ореоли, впливаючи на їх колір, у цілому, на основі кольору вершин, з яких походить ореол. Щоб текстура здійснювала ефект у межах ореолу, і звідси він мав варіювання кольорів або прозорості, увімкніть опцію Texture; це буде накладати текстуру повністю на кожний ореол. Ця техніка виявляється дуже корисною, коли ви хочете створити, як приклад, реалістичний ефект дощу за допомогою систем частинок.

Іншою опцією є «Відтінено» -- Shaded. При увімкненні цієї опції, на ореол буде впливати локальне світло; лампи робитимуть його яскравішим та впливати на його колір розсіяння, дифузії та інтенсивність.

Приклади¶

Dot Matrix Display¶

Let us use a halo material to create a dot matrix display:

  1. Для початку додайте ґратку -- grid з розмірами 32×16. Далі додайте камеру та скоригуйте вашу сцену так, щоб ви мали гарний вигляд плакату.
  2. Use a 2D image program to create some red text on a black background, using a simple and bold font (you can just save the picture below on your hard drive...). Зображеннєва текстура матриці точок. shows an image 512 pixels wide by 64 pixels high, with some black space at both sides.

Зображеннєва текстура матриці точок.

  • Додайте матеріал для свого плаката та задайте для нього тип зобража як Halo. Установіть Halo: Size як 0.06, запустіть зображ сцени, та після нього ви повинні побачити ґратку білих точок.
  • Увімкніть кнопку Texture на панелі Halo, перейдіть у вкладку Texture, додайте текстуру та задайте її тип як Image or Movie. Після укладу зображеннєвої текстури запустіть зображ знову й ви повинні побачити, що деякі точки ґратки підбарвилися червоним.
  • Поверніться у вкладку Material та скоригуйте значення параметра Size близько 0.5, знову запустіть зображ, рендер; текст повинен тепер відцентруватися на плакаті.
  • Для вилучення білих точок задайте колір матеріалу як темно-червоний та запустіть зображ. Ви повинні зараз мати тільки червоні точки, але плакат все ще є надто темним. Для виправлення цього перемкніться у Edit Mode для плаката та здублюйте всі його вершини за допомогою Shift-D (будьте обережні -- не пересувайте їх, тільки дублюйте!). Далі скоригуйте яскравість за допомогою значення Add на панелі Halo.

Dot matrix display.

Ви можете тепер анімувати текстуру, щоб вона рухалася по плакату, використавши значення Offset X на панелі Mapping у вкладці Texture. (Ви можете використати вищу роздільність для цієї ґратки, але для цього ви маєте скоригувати розмір ореолів, звузивши їх, інакше вони будуть перекриватися. Ілюстрація Dot matrix display.).

Material Indices

Ореолові матеріали працюють тільки, коли ефект ореолу застосовується на матеріалі з першим індексом проходу. Будь-які матеріал(и) з другим та наступним індексами, якщо на них є ефект ореолу, не будуть зображуватися, рендеритися.
