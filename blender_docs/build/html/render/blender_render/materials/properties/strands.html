Пасма -- Strands¶

Панель «Пасмо» -- Strand у вкладці «Матеріал» -- Material є спеціально призначеною для зображування, рендерингу частинок Волосся -- Hair. Існує два доступних різних методи зображування пасма:

Полігональні пасма -- Polygon strands
Це стандартний (старий) метод. Пасма зображуються, рендеряться як плоскі полігони. Кількість полігонів залежить від устави «Кроки» -- Steps на панелі «Зображ» -- Render у вкладці «Частинка» -- Particle Редактора Властивостей -- Properties Editor.
Примітив Пасма -- Strand Primitive
You activate Strand Primitive with the button Strand render in the Render panel of the particle system. The hair curves are not stored as polygons; only the key points are stored, which are then converted to polygons on-the-fly. A second difference is the way transparency works. Rather than rendering using the existing system, all strand segments in a part are sorted front to back and rendered in that order.

Примітиви Пасма -- Strand Primitives

  • Є більш заощадливі щодо пам'яті та швидші, дають змогу зображувати, рендерити велику кількість частинок хутра та трави. Для гарної продуктивності значення устави Render: Steps повинне бути доволі низьким (наприклад, значення 2 має бути досить достатнім навіть для хутра), оскільки результат у будь-якому разі буде згладженою кривою. Вам необхідно від 1 до 2 кроків зображу Render: Steps, менш ніж кроків показу Display: Steps у 3D Огляді -- 3D View. Також, використання більше частин зображу допомагає скоротити використання пам'яті.
  • Мають відстань скорочення бачення (на панелі Render під Child Simplification) для нащадків, емітованих з граней.
  • Можуть щезати у напрямку верхівки без додаткової текстури.
  • Are not ray-traced. So they are not visible through ray-transparent materials or in a ray mirror (you can use Environment Mapping for that).
  • Мають проблеми з формою, якщо рендеряться, зображуються з більшою шириною.
  • Cannot carry a UV texture along the strand.

Полігональні пасма -- Polygon strands

  • Добре працюють з більшою шириною, а тому ви можете вживати їх, як альтернативу плакатним частинкам, оскільки пасма можуть мати анімовану форму.
  • Can be textured with a UV texture along the strands.
  • Є видимими при промінь-простеженні.

Відтінення пасм -- Strands Shading¶

Strands panel.

Strands are rendered with the material of the underlying face/vertex, including shading with a UV texture. Since you can assign more than one material to each face, each particle system may have its own material and the material of the underlying face can be different from the material of the strands.

Additionally strands can be shaded along the strand (from root to tip) with a mono-dimensional texture; only polygon strands can carry a two-dimensional UV texture.

Опції для відтінення пасма знаходяться на панелі «Пасмо» -- Strand у вкладці «Матеріал» -- Material.

Корінь -- Root
Ширина волосинки на корені.
Верхівка -- Tip

Ширина волосинки на верхівці.

Мінімум -- Minimum
Це мінімальна товщина (у пікселях) для пасм. Пасма, що мають розмір, нижчий за значення цієї устави, не зображуються, рендеряться меншими, а щезають в альфа прозорість (ну, зникання працює тільки для примітивів пасма). Це дає набагато кращий результат зображування, рендерингу для тонкого волосся.
Одиниці Blender -- Blender Units
Зазвичай пасма є досить тонкими; їх товщина задається в екранних пікселях. Якщо ви використаєте Одиниці Blender (BU), то можете задавати значення ширини на корені до 2 Одиниць, а значення на верхівці -- до 1 Одиниці. Ви повинні зважати на загальний розмір об'єкта, оскільки найменший можливий розмір -- 0.001 Одиниці Blender. А тому, якщо ви використовуєте 1 Одиницю як 1 метр, то найменший можливий розмір може бути 1 мм (надмірна товщина для тонкого волосся).
Дотичне Відтінення -- Tangent Shading
Обчислює світло ніби пасма є дуже тонкими і круглими. Це дає яскравіший та блискучіший вигляд волосся. Вимкнення стяга "Tangent Shading" буде все ще давати гарний зображ, рендер, але пасма здаватимуться більш жорсткими, ніби вони зроблені з металу або дерева.
Форма -- Shape

Цей повзунок дозволяє вам керувати інтерполяцією уздовж пасма. Стандарт (0.0) -- це лінеарна, прямолінійна інтерполяція між коренем -- Root та верхівкою -- Tip. Від'ємні значення даватимуть більш вузько загострені пасма, а додатні -- більш товсто округлі пасма.

Різні значення устави Shape.

Зліва направо: 0 (для першого корінь та верхівка рівні), 0, -0.4, -0.9, 0.4, 0.9.
Ширина Зникання -- Width Fade
Щезання уздовж ширини пасма. Це працює тільки для Примітивів Пасма. 0.0 -- зникання немає взагалі, 1.0 -- прямолінійне щезання.
Карта UV -- UV Map
You can texture polygon strands with a UV texture. Fill in the name of the UV map (not the texture) here. You also have to load the texture in the Texture tab and Material tab (Mapping: UV; you may use every Influence setting you like -- especially the alpha value; see Fig. Різні значення устави Shape.).
Розсіяння Поверхні -- Surface Diffuse

Обчислює нормаль пасма, беручи до уваги нормаль поверхні. Це значно полегшує забарвлення та освітлення волосся, особливо для Примітивів Пасма. По суті, волосся реагує подібно до звичайних поверхонь та не показуються перебільшені сильні та великі блікові підсвітки.

Відстань -- Distance
Відстань в Одиницях Blender, понад яку підмішується нормаль поверхні (якщо ви хочете вжити Surface Diffuse тільки для трави/хутра можуть бути більші відстані).

Текстурування уздовж пасма¶

Пасма можуть бути текстуровані уздовж, тобто від кореня до верхівки. Для цього ви повинні задати значення «Пасмо/Частинка» -- Strand/Particle для опції «Координати» -- Coordinates на панелі «Розкладка» -- Mapping у вкладці «Текстура» -- Texture.

Набагато важливішою є устава, показана на Ілюстрації Зникання пасма в альфа прозорість., яка показує, як зробити зникання в альфа прозорість у напрямку верхівки пасма, що отримати гарне, розпливчасте на вигляд волосся. Зазвичай, ви будете використовувати лінеарне, прямолінійне змішування текстури для цього.

Зникання пасма в альфа прозорість.

Результат зображування, рендерингу.

Ви можете, звичайно, задати й будь-який бажаний для вас інший атрибут, особливо, колір. Будьте обережні з бліковістю; волосинки тяжіють бути надто блискучими.

Спрощення зображу, рендеру пасма¶

Якщо ви використовуєте Примітиви Пасма (увімкнено стяг Strand Render у вкладці «Частинка» -- Particle) та маєте активованим режим «Нащадки: Інтерпольовано» -- Children: Interpolated, то на панелі Render з'явиться опція «Спрощення Нащадків» -- Child Simplification. Зображ Пасма має опції для вилучення нащадків пасм у ході того, як грані об'єкта стають меншими.

Спрощення Нащадків при Зображі Пасма.

Орієнтувальний Розмір -- Reference Size
Це приблизний розмір об'єкта на екрані (у пікселях), після якого починається спрощення.
Частота -- Rate
Наскільки швидко вилучаються пасма.
Перехід -- Transition
Період переходу для зникання пасм при їх вилученні.
Оглядвікно -- Viewport

Це вилучає пасма на гранях, що знаходяться поза оглядвікном.

Частота -- Rate
Керує тим, як швидко ці пасма вилучаються.
