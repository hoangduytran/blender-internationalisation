Опції -- Options¶

Панель «Опції» -- Options у вкладці «Матеріал» -- Material.

Ця панель надає ряд опцій керування тим, як об'єкти, що використовують цей матеріал, будуть виглядати на зображеному, рендереному зображенні. Усі контролери стандартно вимкнені, якщо не вказано інше.

Простежуваність -- Traceable
Include this material and the geometry that uses it in ray-tracing calculations. See Transparency for details of ray tracing.
Повний Перебір -- Full Oversampling
Змушує цей матеріал зображуватися, рендеритися з повним відтіненням та текстуруванням для усіх вибірок загладжування -- anti-aliasing.
Небо -- Sky
Дає зображування, рендеринг цього матеріалу з нульовим альфа, але з фоном неба sky background (лише порядково).
Вжити Імлу -- Use Mist
Використовує імлу -- mist на цьому матеріалі (дивіться детальніше підрозділ про устави Світу сцени).
Інвертувати глибину Z -- Invert Z depth
Render material's faces with an inverted Z-buffer (scanline only).
Зсув Z -- Z Offset
Надає граням штучний зсув Z для прозорості Z.
Група Світла -- Light Group
Лімітує освітлення лампами у цій групі світла -- light group.
Виключно -- Exclusive
Використовує виключно цю групу світла -- light group. Ці лампи будуть виключені з іншого освітлення сцени.
Локально -- Local
При пов'язуванні групи світла з іншого файлу використовує замість неї локальну (у цьому файлі) групу світла -- light group з таким же ім'ям.
Текстури Граней -- Face Textures
Замінює базовий колір об'єкта кольором зображеннєвих текстур, розкладених по карті UV -- UV map.
Альфа Текстур Граней -- Face Textures Alpha
Замінює базову альфа прозорість об'єкта альфа прозорістю зображеннєвих текстур, розкладених по карті UV -- UV map.
Малювання Кольорами Вершин -- Vertex Color Paint
Замінює базовий колір об'єкта кольорами заданими малюванням вершин -- vertex paint (множиться на колір призначених «текстур на грані»).
Світло Кольорів Вершин -- Vertex Color Light
Додає кольори, задані малюванням вершин -- vertex paint, як додаткове освітлення. (Це може використовуватися для продукування добрих ефектів накалювання).
Колір Об'єкта -- Object Color
Модулює результат за допомогою об'єктного кольору. Дивіться детальніше тут -- Object Display panel.
Проекція UV -- UV Project
Використовується для гарантування коректності інтерполяції UV для проекцій камери (вживається разом з модифікатором «Проекція UV» -- UV Project Modifier).
Індекс Проходу -- Pass Index
Індексний номер для проходу зобража «Індекс Матеріалу» -- Material Index.
