Тіні -- Shadows¶

With this panel you can control how objects using this material cast and receive shadows. The shadows that appear in a scene are affected by a combination of the layout of objects, the shape of the objects, the materials of the objects, and the lighting. In Blender, the Display Mode (single-texture, multi-texture, or GLSL) also affects the appearance of shadows. See Shadows for a more complete description of this subject.

Mẹo

Тіні у текстурованому режимі 3D

Щоб бачити тіні у текстурованому режимі 3D, ви повинні перемкнутися на режим GLSL перед початком розробки будь-яких матеріалів. У режимі «Багатотекстурно» -- Multitexture тіні будуть показуватися тільки на зображеному, рендереному зображенні: жодна з них не буде видима на передоглядному зображенні.

Shadow panel.

Панель «Тінь» -- Shadow у вкладці «Матеріал» -- Material Редактора Властивостей -- Properties Editor (дивіться Ілюстрацію Shadow panel.) впливами, які матеріал може мати на тіні, що показуються у сцені. Різні властивості звідси описуються нижче.

Сцена з вимкненням усіх тіней.

Опції¶

The following properties can be set for each individual material with which objects in the scene are shaded. The effects are illustrated with rendered images for a simple scene (Fig. Сцена з вимкненням усіх тіней.) consisting of two "posts", one with a red (totally non-transparent) material; one green (partially transparent) material, set up on a light blue plane to receive the shadows. The illustrations were all taken in Blender Renderer, with Multitexture mode.

Shadow-Receiving Object Material¶

Наступні опції впливають на матеріал, що приймає тіні:

Приймання -- Receive
Дозволяє цьому матеріалу приймати тіні з повною інтенсивністю (Ілюстрація Площина з увімкненою опцією приймання тіней -- Receive.).
Приймати Прозоре -- Receive Transparent
Дозволяє цьому матеріалу приймати тіні, чия інтенсивність модифікується прозорістю та кольором об'єкта, що відкидає тіні (Ілюстрація Площина з увімкненою опціями приймання тіней -- Receive та Receive Transparency.).

Площина з увімкненою опцією приймання тіней -- Receive.

Площина з увімкненою опціями приймання тіней -- Receive та Receive Transparency.

Shadow-Casting Object Material¶

Наступні опції впливають на матеріал, що відкидає тіні:

Лише Відкидання -- Cast Only
Призводить до того, об'єкти з цим матеріалом, відкидають лише саму тінь, а самі не показуються на зображах, рендерах. (Ілюстрація Стовпи з увімкненою для них опцією Cast Only.).
Альфа Відкидання -- Casting Alpha
Задає Альфа прозорість відкидання тіней. Використовується для буферування нерегулярних та глибоких тіней.
Лише Тіні -- Shadows Only

Зображує, рендерить тіні ніби зі значенням альфа для матеріалів, роблячи матеріали прозорими, за винятком областей, де вони приймають тіні від інших об'єктів, а також, при цьому, матеріал зберігає власну прозорість (Ілюстрація Стовпи з увімкненою для них опцією Shadows Only.). Зверніть увагу на слабкий образ частково-прозорого стовпа.

Тип Тіні -- Shadow Type для опції «Лише Тіні» -- Shadow Only

Задає тип тіней, що використовуються при увімкненні опції Shadows Only.

  • Тінь і Відстань -- Shadow and Distance
  • Лише Тінь -- Shadow Only
  • Тінь і Відтінення -- Shadow and Shading

Стовпи з увімкненою для них опцією Cast Only.

Опції Буферованої Тіні -- Buffered Shadow¶

Окрім опцій для тіней, описаних вище, є додаткові властивості матеріалу, які керують функціями буферованої або буферної тіні. Про цей тип тіні дивіться детальніше тут -- Spot Buffered Shadows.

Відкидати Буферні Тіні -- Cast Buffer Shadows
Дає відкидання тіней для ламп з увімкненою для них буферної тіні.
Відхил Буфера -- Buffer Bias
Фактор множення для буферних тіней (0 = ігнорування).
Авто Відхил Променя -- Auto Ray Bias
Prevent ray-traced shadow errors on surfaces with smooth-shaded normals.
Відхил Променя -- Ray Bias
Shadow ray-tracing bias value to prevent terminator artifacts on shadow boundary.
Відкидання Приблизно -- Cast Approximate
Дозволяє цьому матеріалу відкидати тіні, коли використовується приблизна -- Approximate загорода оточення -- Ambient Occlusion.
