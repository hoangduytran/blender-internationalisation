Призначення Матеріалу -- Assigning a Material¶

Materials available in the currently open blend-file can be investigated by clicking on the Materials icon in the Properties editor Header. In this section we look at how to assign or remove a material to/from the Active Object in Blender, either by:

  • Створення нового матеріалу,
  • повторного використання наявного матеріалу, або
  • видалення матеріалу.

Ми також дамо поради про практичне використання матеріалів.

Створення нового матеріалу¶

Кожен раз при створенні, доданні у сцену нового Об'єкта, він не немає жодного пов'язаного з ним матеріалу. Ви можете створити новий матеріал для об'єкта шляхом:

  • Виділення об'єкта.
  • Відкрити Редактор Властивостей -- Properties Editor (якщо його ще ніде не відкрито).
  • Click on the Materials button in the Properties editor header.

Далі, у цій вкладці з'явиться панель відтінення. Вона містить наступні елементи:

Додання нового матеріалу.

  • Context: The currently selected scene and object.
  • Object Material Slots: this panel shows the "slots" for the material (or materials) that this object data contains.
  • Active Material: Initially empty, asking for "New".

Для додання нового матеріалу клацніть на кнопці «+» або «Нове» у блоці «Активний Матеріал». Ця дія має ряд ефектів:

Панель Матеріалів з новим уводом.

  1. Відкриває новий матеріал у блоці «Активний Матеріал».
  2. Привносить додаткові кнопки тут же на панелі.
  3. Додає новий матеріалу у список Доступних Матеріалів.
  4. Додає новий матеріал у список «Гнізда Матеріалів Об'єкта» для активного об'єкта (або його даних об'єкта -- дивіться нижче).
  5. Привносить у вкладку нижче передогляд -- preview для цього нового матеріалу.
  6. Надає вам ряд панелей, що дозволяють задати властивості -- properties для нового матеріалу.

Нові кнопки на панелі для матеріалу¶

Детальніше, додаткові кнопки, що з'являються на панелі Material для нового Активного Матеріалу, є наступними:

Активний Матеріал
Огляд списком наявних у даному blend-файлі матеріалів.
Матеріал -- Material
Меню блоку матеріалових даних -- Data-Block Menu для виділеного гнізда з матеріалом.

Mẹo

Іменування матеріалів

Дуже хорошою ідеєю є надання вашим матеріалам зрозумілих імен, щоб ви могли простежувати їх, особливо, коли вони пов'язані одночасно з багатьма об'єктами. Намагайтеся робити імена для них, описовими відповідно до матеріалу, а не його функції (наприклад, "Yellow Painted" («пофарбований жовтим»), а не "Kitchen Table Color" («колір кухонного стола»)).

Вузол -- Node
Вмикає/вимикає вживання вузлів для побудови матеріалу.
Дані -- Data
Визначає, чи буде матеріал пов'язаний до Об'єкта -- Object, чи до даних об'єкта -- Data.
Тип матеріалу -- Material type
Ці чотири кнопки дозволяють визначати те, як об'єкт буде зображуватися, рендеритися.

Повторне використання наявних матеріалів¶

Blender побудований так, щоб давати вам змогу повторно використовувати будь-що, включаючи устави матеріалів, між багатьма об'єктами. Замість створення дублікатів матеріалів ви можете просто повторно використати наявний вже матеріал. Існують кілька способів це робити за допомогою меню блоку даних матеріалів:

Одиничний Об'єкт -- При виділенні одного об'єкта клацніть на кнопці з іконкою сфери, що зліва від імені Матеріалу. З'явиться спливне меню, що показує усі матеріали, доступні у поточному blend-файлі. Для призначення одного з них просто клацніть ЛКМ на ньому.

Вибір з наявних матеріалів.

Mẹo

Пошук матеріалів

Поле пошуку внизу списку матеріалів дозволяє вам шукати за їх іменами у цьому списку. Наприклад, при уведенні слова "wood" усі наявні матеріали відфільтруються так, що залишаться показуватися тільки матеріали, що містять це слово "wood".

Багато Об'єктів -- У 3D Огляді -- 3D View при виділенні кількох об'єктів натиснення Ctrl-L дає змогу швидко пов'язати до усіх виділених об'єктів матеріал (або інші властивості, аспекти) активного об'єкта -- active object. Дуже це корисно, коли для вас потрібно задати для великої кількості об'єктів однаковий матеріал; просто виділіть їх усіх та виділіть останнім об'єкт, який має бажаний матеріал, та скористайтеся спливним меню за Ctrl-L для пов'язування їх з матеріалом на цьому останньому об'єкті. (Дивіться детальніше поради щодо пов'язування даних при створенні).

Видалення матеріалу¶

Для видалення матеріалу виберіть потрібний матеріал та клацніть на кнопці з «X» в його блоці даних.

Хоча, матеріал, начебто, зникає негайно, дія команди «Видалення» може залежати від того, як матеріал використовується деінде.

Якщо матеріал, пов'язаний з цим Об'єктом, також використовується іншими об'єктами, то цей матеріал буде вилучено з властивостей цього об'єкта (та залишиться у властивостях усіх інших пов'язаних з ним об'єктів).

Якщо увімкнена кнопка (F) -- "Fake User" -- «Псевдо Користувач» у блоці матеріалу, то такий матеріал буде залишено у blend-файлі, при збереженні останнього, навіть якщо матеріал не має жодного користувача для нього.

Тільки, якщо матеріал немає жодних (0) «реальних» користувачів та для нього не увімкнено властивість «Псевдо Користувач» -- «F», то після закриття програми цей матеріал буде остаточно видалений. Зауважте, що він все ще залишатиметься видимим у списку матеріалів, допоки його blend-файл не буде закрито, та зникне після перевідкриття його blend-файлу.

Одночасно кілька матеріалів¶

Зазвичай, різні кольори та візерунки на об'єкті досягаються шляхом додання текстур у матеріали. Однак, у деяких випадках ви можете отримувати одночасно кілька кольорів на об'єкті шляхом призначення різних матеріалів на окремі грані об'єкта.

Додання нового матеріалу.

To apply several materials to different faces of the same object, you use the Material Slots options in the Materials header panel.

Панель «Матеріал» у Режимі Правки -- Edit Mode для об'єкта.

Хід роботи для призначення другого матеріалу на деякі грані одного і того ж об'єкту шляхом покриття базового матеріалу є наступним:

  1. У Режимі Об'єкта -- Object Mode призначте базовий матеріал для об'єкта повністю (як розказано у параграфі doc:Assigning a material </render/blender_render/materials/assigning_a_material>).
  2. Створіть/виберіть другий матеріал (увесь об'єкт забарвиться цим новим матеріалом).
  3. In the Active Material box, re-select the base material.
  4. У 3D Огляді перемкніться для об'єкта у Режим Правки -- Edit Mode та увімкніть у ньому режим виділу граней -- Face Select (над блоком Активного Матеріалу з'являться три кнопки «Признач» -- Assign / «Виділ» -- Select / «Зневиділ» -- Deselect).
  5. Виділіть грань/грані, що мають бути забарвлені другим матеріалом.
  6. In the Object Material Slots box, click the Plus to create a new slot, and while this is still active, click on the second material in the Available Materials list.
  7. Клацніть на кнопці «Признач» -- Assign і цей другий матеріал з'явиться на виділених гранях об'єкта.

Ви можете також зробити цей новий матеріал копією наявного матеріалу, додавши такий блок даних:

Виділіть об'єкт, виберіть матеріал, клацніть RMB та скопіюйте дані в обмін-буфер. Після перейменування матеріалу клацніть "Link: Data" для пов'язування з наявним матеріалом. Продовжуйте призначати на грані при потребі. Примітка: Якщо ви зміните матеріал на оригінальному об'єкті, то колір на новому об'єкті зміниться також.
