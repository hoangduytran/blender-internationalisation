Вступ про Вузли -- Introduction to Nodes¶

Окрім створення матеріалів за допомогою використання описаних устав на панелях для них Blender дозволяє створювати матеріал з'єднуванням властивостей базових матеріалів через набір вузлів. Кожен вузол здійснює певну операцію з матеріалом, змінюючи його вигляд при застосуванні до сіті, та результат цієї операції подає на наступний вузол. У такий спосіб можна домогтися створення матеріалу з дуже складним виглядом.

Ви повинні вже бути знайомі із загальними концепціями щодо матеріалів та з тим, як створювати матеріали/текстури, використовуючи меню і панелі з уставами для них. Ви повинні також мати загальне розуміння систем координат текстури, доступних у Blender'і (наприклад, Генеровані, UV тощо). Також, багато з аспектів щодо вузлів будуть тут пропущені, оскільки у подальших підрозділах ви побачите їх функції більш розширено. Кожен підрозділ вибудовується на основі попереднього.

Для початку, система вузлів не призводить до того, що меню та панелі щодо матеріалів стають застарілими. Багато функцій та устав матеріалів все ще доступні тільки через панелі для матеріалів (наприклад, Ray Mirror). Однак, з появою вузлів стало можливим створення більш складних та фантастичних матеріалів, оскільки у нас з'явилося більше засобів керування.

Just in case you are not (yet) familiar with the concepts: when you create a system of nodes, you are describing a data-processing pipeline of sorts, where data "flows from" nodes which describe various sources, "flows through" nodes which represent various processing and filtering stages, and finally "flows into" nodes which represent outputs or destinations. You can connect the nodes to one another in many different ways, and you can adjust "properties" or parameters, that control the behavior of each node. This gives you a tremendous amount of creative control. And, it will very quickly become intuitive.

Сказавши все це, давайте почнімо зі звичайного матеріалу.

Тут ми маємо стандартний матеріал, який доданий нами для сіті куба. Ми можемо, як і раніше, додати колір та інші устави для цього матеріалу і він буде мати, звичайно, гарний вигляд. Але давайте скажімо собі, що ми не отримуємо те, що шукаємо? Що, якщо ми хочемо керувати створенням більш щільно та додати більше складності? Ось тут на допомогу приходять вузли.

Зроблення карти, розкладки вузлів досягається через її розробку у Редакторі Вузлів -- Node Editor. Цей підрозділ охоплює такі аспекти:

  • Увімкнення режиму Матеріалових Вузлів -- Material Nodes.
  • Редактор Вузлів -- Node Editor, його базові контролери та робота з вузлами.
  • Конкретні типи вузлів, доступні для матеріалів.

Доступ до Редактора Вузлів -- Node Editor¶

Спершу, перейдемо у Редактор Вузлів -- Node editor та переконаймося, що увімкнений режим Матеріалових вузлів (або ще режим вузлів Відтінювача -- Shader) -- натиснута кнопка з іконкою сфери, а не увімкнений будь-який інший (вузлів компонування чи текстурування) режим.

Вмикання серед устав матеріалу режиму створення матеріалів на основі вузлів¶

Давайте візьмімо цей базовий матеріал та увімкнемо кнопку «Вжити Вузли» -- Use Nodes справа від блоку даних з іменем матеріалу на панелі «Матеріал» -- Material у Редакторі Властивостей -- Properties Editor або увімкнемо однойменний стяг «Вжити Вузли» у заголовку Редактора Вузлів -- Node Editor. Ви побачите, що панелі устав для матеріалу змінилися.

Кнопка перемикання режиму використання матеріалових вузлів «Вжити Вузли».

Те, що ви щойно зробили, повідомляє Blender'у розпочати робити цей матеріал за допомогою дерева вузлів. Більшість панелей, які ми зазвичай знаходимо у вкладці для матеріалів, тепер зникли.

Розстава екрану Compositing screen, в яку ви можете перемикнутися з Ctrl-Left, якщо знаходитися у стандартній (Default) розставі, містить вікно з Редактором Вузлів -- Node Editor у верхній половині екрану. Після увімкнення вживання матеріалових вузлів вищезгаданою кнопкою у Редакторі Вузлів -- Node Editor автоматично додадуться вузол «Матеріал» -- Material та вузол «Вивід» -- Output.

Стандартні вузли.

Також мати доступ до Редактора Вузлів можна і на стандартному (Default) екрані, розділивши 3D Огляд -- 3D View на два вікна та в одному з них змінивши тип вікна на Node Editor.

Важливо відзначити, що ви можете тут додати новий матеріал (який можете редагувати і змінювати, як і будь-який інший матеріал на панелях для матеріалів), додати вже створений матеріал або прилучити матеріал з іншого blend-файлу, а також використати інший матеріал для створення дерева вузлів.

Перший матеріал додано в уклад вузлів.

Тут, ми додали новий матеріал "Material.001" у Редактор Вузлів -- Node editor і після цього ми отримали доступ до властивостей цього матеріалу також у вкладці «Матеріал» -- Material Редактора Властивостей -- Properties Editor.
