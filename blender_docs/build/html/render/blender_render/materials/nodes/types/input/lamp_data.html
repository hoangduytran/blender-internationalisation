Вузол «Дані Лампи» -- Lamp Data Node¶

Вузол «Дані Лампи» -- Lamp Data node.

Вузол «Дані Лампи» використовується для отримання інформації з вказаного об'єкта типу «лампа».

Уводи -- Inputs¶

Цей вузол не має уводів.

Властивості -- Properties¶

Об'єкт Лампи -- Lamp Object
Дає змогу вказати об'єкт лампи з наявних у проекті.

Виводи -- Outputs¶

Текстури світла та текстури тіні впливають на виводи «Колір» -- Color та «Тінь» -- Shadow, відповідно.

Колір -- Color
Колір лампи, помножений на енергію лампи.
Вектор Світла -- Light Vector
Одиничний вектор у напрямку від лампи до відтіненої точки.
Відстань -- Distance
Відстань від відтіненої точки до лампи.
Тінь -- Shadow
Колір тіні, який інші об'єкти відкидають на відтінену точку.
Фактор Видимості -- Visibility Factor
Пропорція спаду освітлення у відтіненій точці.

Ghi chú

Переносність у різні сцени

Якщо одночасно кілька матеріалів використовують вузол «Дані Лампи» для подачі даних з одного і того ж об'єкта лампи, то рекомендується включати цей вузол «Дані Лампи» у групу вузлів, груповий вузол. Інакше, після імпортування сітьових об'єктів в іншу сцену, можливо доведеться модифікувати всі матеріали.
