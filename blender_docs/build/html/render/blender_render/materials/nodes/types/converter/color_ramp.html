Вузол «Колірний Градієнт» -- Color Ramp Node¶

Вузол «Колірний Градієнт» -- Color Ramp Node.

Вузол «Колірний Градієнт» -- Color Ramp Node використовується для розкладання значень кольорів за допомогою використання градієнта.

Уводи -- Inputs¶

Фактор -- Factor
Увід «Фактор» -- Factor використовується як індекс, вказівник для колірного градієнта.

Властивості¶

Колірний Градієнт -- Color Ramp
Про віджет цього контролера дивіться Віджет колірного градієнта.

Виводи -- Outputs¶

Зображення -- Image
Стандартний вивід зображення.
Альфа -- Alpha
Стандартний вивід альфа.

Приклади¶

Створення Альфа-Маски¶

Потужна, але часто ігнорована, особливість застосування вузла «Колірний Градієнт» -- створення Альфа-Маски або маски, що накладається поверх іншого зображення, і, подібно до реальної маски, дозволяє задньо-плану проглядатися крізь маску. Карта вузлів, наведена нижче, демонструє як використовувати вузол «Color Ramp» для цього:

Використання вузла «Колірний Градієнт» -- Color Ramp для створення альфа-маски.

У карті вузлів вище, чорно-біле зображення вихору, в якому відсутній альфа канал, подається у вузол «Колірний Градієнт» -- Color Ramp на роз'єм «Фактор» -- Factor. (Технічно, ми повинні конвертувати це зображення у значення за допомогою вузла RGB to BW, але, це працює добре і так, оскільки ми використали чорно-біле зображення як увід.)

Ми задали на лівому кінці спектру вузла «Колірний Градієнт» -- Color Ramp повністю прозорий колір, а на правому його кінці -- повністю червоний колір. Як видно на вузлі «Оглядач» -- Viewer, вузол «Колірний Градієнт» -- Color Ramp викладає повністю прозору маску на тих ділянках, де зображення чорне. Чорний дорівнює значенню нуль, а тому вузол «Колірний Градієнт» використовує колір у лівому кінці спектру для установлення чорних ділянок прозорими. На ділянках, де зображення біле (1.00), вузол «Колірний Градієнт» -- Color Ramp установлює червоний безпрозорий колір.

Ми перевіряємо, що вивідне зображення маски є дійсно прозорим, накладаючи його поверх іншого зображення.

Забарвлення Зображення¶

Реальна потужність вузла «Колірний Градієнт» -- Color Ramp полягає у тому, що у колірний спектр можуть бути додані одночасно кілька, більше ніж два, кольорів. Цей приклад компонувальної карти бере нудне ЧБ зображення та робить його полум'яним вихором!

У цьому прикладі ми розклали відтінки сірого з увідного зображення на три кольори, синій, жовтий та червоний, усі повністю безпрозорі (Альфа -- 1.00). Там, де зображення чорне, «Колірний Градієнт» підставляє синій, поточно вибраний для цього колір. Там, де присутній який відтінок сірого, «Колірний Градієнт» вибирає відповідний колір зі спектру (синюватий, жовтий, до червонуватого). Де зображення є повністю біле, «Color Ramp» підмінює червоним.
