Вузол «Матеріал» -- Material Node¶

Вузол «Матеріал» -- Material node.

Вузол «Матеріал» використовується власне для відтінення, забарвлення матеріалу.

Уводи -- Inputs¶

Колір -- Color
Колір розсіяння або текстури.
Блікування -- Specular
Колір блікування або текстури, що відбивається, коли точка зору отримує перпендикуляр до джерела світла, освітлення від якого відбивається від поверхні.
Інтенсивність Розсіяння -- Diffuse Intensity
Величина дифузного розсіювання. Може також задаватися текстурою.
Нормаль -- Normal
Стандартний увід нормалі поверхні. Вживається для розкладання відповідно до нормалей.

Ghi chú

Заміщення даних щодо нормалей

Дані з увідного роз'єму «Нормаль» не змішуються з даними щодо нормалей геометрії, а повністю їх заміщають.

Властивості -- Properties¶

Матеріал -- Material
Вживається для знаходження та вибору матеріалу.
Розсіяння -- Diffuse
Активує/дезактивує розсіяне відтінення.
Блікування -- Specular
Активує/дезактивує блікове відтінення.
Інвертувати Нормаль -- Invert Normal
При увімкненні інвертує увід даних на матеріалів щодо нормалей. (який є комбінацією даних щодо 3D нормалі, заданих 3D об'єктом, плюс даних, заданих уводом «Нормаль» на цьому вузлі).

Виводи -- Outputs¶

Колір -- Color
Відтінене значення кольору.
Альфа -- Alpha
Значення альфа прозорості.
Нормаль -- Normal
Напрямок нормалі.

Приклади¶

Використання вузла «Матеріал» для задання бліковості¶

Вузол «Матеріал» використовується для бліковості.

Щоб вузол «Матеріал» дійсно генерував колір, принаймні має бути заданий базовий увід кольору та, при потребі, колір бліковості. Колір бліковості -- це колір відблиску від інтенсивного світла.

Наприклад, розглянемо ілюстровану тут карту вузлів. Базовий колір, темно-синій, під'єднується з вузла генерування кольору «RGB» на увідний роз'єм Color. Колір блікування, жовтий, задається на уводі «Блік» -- Spec. При освітленні на основі нормалі Normal плоскої поверхні цей матеріал буде продукувати глибокий синій колір та, оскільки точка зору наближається до місця, перпендикулярного до світла, стає видимий жовтий колір бліку.

Ghi chú

Вмикання бліку

Щоб бачити бліковість, має бути також увімкнений стяг «Блікування» -- Specular, що знаходиться під стягом «Колір» -- Color.
