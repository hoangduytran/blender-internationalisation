Вузол «Геометрія» -- Geometry Node¶

Вузол «Геометрія» -- Geometry node.

Вузол «Геометрія» використовується для визначення того, як світло відбивається від поверхні. Цей вузол вживається для зміни відклику матеріалу по нормалі на умови освітлення.

Використовуйте цей вузол для подання даних на векторний увід «Нормаль» -- Normal вузла «Матеріал» -- Material, щоб бачити, як матеріал буде виглядати (наприклад, блиск або відбите світло) під різними умовами освітлення. Можливі ваші варіанти:

Уводи -- Inputs¶

Цей вузол не має уводів.

Властивості -- Properties¶

Карта UV -- UV Map
Дає змогу вибрати карту UV для подачі її даних у вузлову мережу.
Шар Кольору -- Color Layer
Дозволяє вибрати дані кольорів вершин (зроблені з допомогою Малювання Вершин -- Vertex Paint чи Малювання Вагомостей -- Weight Paint) для подання їх у мережу вузлів.

Виводи -- Outputs¶

Глобально -- Global
Глобальна позиція поверхні.
Локально -- Local
Локальна позиція поверхні.
Огляд -- View
Позиція поверхні в огляді.
Оригінальні Координати -- Original Coordinates
Використання Оригінальних Координат сіті.
UV
Використання координат UV для сіті, з карти вибраної внизу цього вузла.
Нормаль -- Normal
Нормаль поверхні; на плоскій площині з одним освітлювачем над нею та відбиттям світла поверхнею управо.
Колір Вершин -- Vertex Color
Дозволяє вивести значення групи кольорів вершин, з шару кольору, вибраного внизу цього вузла.
Альфа Вершин -- Vertex Alpha
Дозволяє вивести значення альфа-прозорості вершин.
Спереду/Позаду -- Front/Back
Дає змогу вивести значення освітленості з врахуванням передньої або задньої сторін поверхні відносно камери.

Ghi chú

Ці устави вузла «Геометрія» є аналогами устав з панелі «Розкладка» -- Mapping для текстур -- Textures, хоча деякі з устав, як «Напруженість» -- Stress або «Дотично» -- Tangent тут відсутні. Зазвичай, ви буде використовувати цей вузол для подачі увідних даних на вузол «Текстура» -- Texture Node.

Geometry Node Example using a UV Image¶

Setup to render a UV-mapped image texture.

Наприклад: для зображування, рендерингу розкладеного на поверхні матеріалу по UV зображення ви зазначаєте карту UV та подаєте дані з виводу UV вузла «Геометрія» -- Geometry на увід «Вектор» -- Vector вузла «Текстура» -- Texture. Далі ви подаєте дані з виводу «Колір» -- Color вузла «Текстура» -- Texture на увід «Колір» -- Color вузла «Матеріал» -- Material.
