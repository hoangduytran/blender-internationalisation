Мітки -- Markers¶

Мітки використовуються для позначення кадрів з ключовими моментами або значущими подіями у межах анімації. Наприклад, це можуть бути моменти, в якому починається анімація персонажа, камера змінює свою позицію або відкриваються двері. Мітки можуть мати значущі імена, що полегшує орієнтування по них. Вони доступні у багатьох редакторах Blender'а.

Ghi chú

На відміну від ключкадрів, мітки завжди розміщуються у цілому кадрі, ви не можете задати мітку у кадрі 2.5.

Мітки можуть створювати та редагуватися у наступних редакторах:

  • Редактор Графів -- Graph Editor
  • Аркуш Експозицій -- Dope Sheet
  • Редактор НЛА -- NLA Editor
  • Редактор Відеорядів -- Video Sequence Editor
  • Часолінія -- Timeline

Ghi chú

Мітка, створена в одному з цих редакторів, буде також показуватися у всіх інших редакторах, які підтримують мітки.

Типи¶

Next to the standard markers pose markers are another type of markers, which are specific to armatures and shape keys. They are used to denote poses in the Action Editor mode and Shape Keys Editor of Dope Sheet.

Візуалізація¶

Стандарт¶

Мітки: маленькі, але корисні.

У більшості редакторів мітки візуалізуються однаково: як маленькі трикутники зверху часолінії, та забарвлені білим, якщо вони поточно не виділені, або жовтим, якщо поточно виділені.

Якщо вони мають задане ім'я, то воно показується справа від трикутника, і забарвлюється білим, коли мітка виділена.

Відеорядник -- Sequencer¶

Мітки у Відеоряднику або Редакторі Відеорядів.

Редактор Відеорядів Video Sequence Editor додає вертикальну пунктирну лінію для кожної мітки (забарвлена сірим, коли мітка невиділена, та білим, коли вона виділена).

3D-Огляд -- 3D View¶

Мітка у 3D-Огляді.

3D-Огляд -- 3D View не дозволяє створювати, редагувати або вилучати мітки, він тільки показує їх імена в зоні інформації про об'єкт Object Info, що знаходиться у лівому нижньому куті цього редактора, коли курсор часу знаходиться на кадрі, який позначає мітка (дивіться ілюстрацію «Мітка у 3D-Огляді»).

Мітки Поз -- Pose Markers¶

Мітки поз у Редакторі Дій -- Action Editor.

Мітки поз показуються діаманто-подібною іконкою у редакторі Аркуш Експозицій -- Dope Sheet. У Редакторі НЛА -- NLA Editor мітки поз показуються як червоні пунктирні лінії.

Додання Мітки¶

Орієнтир

Режим:Усі режими
Меню:«Мітка > Додання Мітки» -- Marker ‣ Add Marker
Скорочення:M або Ctrl-Alt-M у редакторі Відеорядів -- VSE Editor

Найпростіший спосіб додати мітку -- це перемістити курсор часу у кадр, в якому ви хочете, щоб вона показувалася, та натиснути M.

Gợi ý

Мітки можуть також додаватися під час відтворення.

Мітки Поз -- Pose Markers¶

Якщо опція «Показати Мітки Поз» Show Pose Markers увімкнена, то додаватимуться мітка пози та нова поза у Бібліотеці Поз -- Pose Library.

Виділення¶

Орієнтир

Режим:Усі режими
Скорочення:RMB

Клацайте RMB на трикутнику мітки для її виділення. Використовуйте :kbd:`Shift-RMB для виділення кількох міток одночасно.

У Редакторі Графів -- Graph Editor, Аркуші Експозицій -- Dope Sheet, Редакторі НЛА -- NLA Editor та Редакторі Відеорядів -- Video Sequence Editor ви можете також виділяти усі мітки за допомогою Ctrl-A та застосовувати такі засоби виділення для них як Border Select тощо (та як звичайно, LMB для виділення, RMB для зневиділення). Ці відповідні опції знаходяться у меню «Виділ» -- Select цих редакторів.

У редакторі Часолінія -- Timeline ви можете виділяти усі мітки за допомогою A та виділу рамкою B.

Редагування¶

Дублювання Мітки¶

Орієнтир

Режим:Усі режими
Меню:«Мітка > Дублювання Мітки» -- Marker ‣ Duplicate Marker
Скорочення:Shift-D

Ви можете дублювати виділені мітки, натиснувши Shift-D. Після операції дублювання дублікати автоматично поміщаються у режим захопу-пересуву, а отже ви можете зразу їх перемістити у бажану локацію.

Ghi chú

Зауважте, що на відміну від інших операцій дублювання у Blender'і, імена дубльованих міток взагалі не змінюються (жоден числовий лічильник типу .001 не додається).

Дублювання Мітки у Сцену¶

Орієнтир

Режим:Усі режими
Меню:«Мітка > Дублювання Мітки у Сцену» -- Marker ‣ Duplicate Marker to Scene...

Дублює виділені мітки в іншу сцену.

Видалення Міток¶

Орієнтир

Режим:Усі режими
Меню:«Мітка > Видалення Мітки» -- Marker ‣ Delete Marker
Скорочення:X

Для видалення виділених міток просто натисніть X та підтвердьте спливне повідомлення клацком LMB.

Перейменування Мітки¶

Орієнтир

Режим:Усі режими
Меню:«Мітка > Перейменування Мітки» -- Marker ‣ Rename Marker
Скорочення:Ctrl-M

Наявність десятків міток, розкиданих по часолінії вашої сцени, не допоможе вам сильно, допоки ви не знатимете, для чого вони там стоять. Ви можете перейменувати мітку, виділивши її, натиснувши Ctrl-M, увівши ім'я та клацнувши по кнопці OK.

Захоп/Пересув Мітки¶

Орієнтир

Режим:Усі режими
Меню:«Захоп/Пересув Мітки» -- Marker ‣ Grab/Move Marker
Скорочення:G

Після виділення однієї чи кількох міток, наведіть вказівник миші над смугою з мітками, натисніть G, порухайте їх у потрібну позицію та підтвердьте цей пересув за допомогою LMB або Return (та як звичайно, скасування пересуву здійснюється клацком RMB або натисканням Esc). Або перетягуйте їх за допомогою RMB.

Стандартно, ви захоплюєте і пересуваєте мітки з кроком в один кадр, але якщо ви при цьому утримуватимете натисненою Ctrl, то мітки пересуватимуться на відповідну кількість кадрів з кроком в одну секунду (згідно заданої частоти кадрів у сцені -- FPS).

Показ Міток Поз¶

Орієнтир

Режим:Редактор Дій -- Action Editor та Редактор Ключів Форм -- Shape Keys Editor
Меню:«Мітка > Показ Міток Поз» -- Marker ‣ Show Pose Markers

Only pose markers are shown and editable in Action editor or Shape Keys editor by enabling the Marker ‣ Show Pose Markers checkbox.

Зроблення Міток Локальними¶

Орієнтир

Режим:Усі режими
Меню:«Мітка > Зроблення Міток Локальними» -- Marker ‣ Make Markers Local

It is possible to convert standard markers into pose markers with Marker ‣ Make Markers Local. Note that the original marker will be gone. If you want to keep it, make a duplicate before you convert.

Перехід до Наступної/Попередньої Мітки¶

Орієнтир

Режим:Усі режими
Меню:«Стрибання до Наступної/Попередньої Мітки» -- Marker ‣ Jump to Next/Previous Marker

Moves the time cursor to the next/previous marker relative to the current time cursor position.

Прив'язка Камери до Мітки¶

Орієнтир

Редактор:Редактор Часолінії -- Timeline Editor
Меню:«Огляд > Прив'язка Камери до Міток» -- View ‣ Bind Camera to Markers
Скорочення:Ctrl-B

Перемикання між камерами може бути зроблено за допомогою оператора Часолінії Timeline -- «Прив'язка Камери до Міток» -- Bind Camera to Markers при виділенні одночасно потрібних камери та мітки.

Трикутник над камерою ставатиме затіненим, коли вона буде активною.

Робпотік¶

Спершу на Часолінії -- Timeline додайте набір міток, що будуть використовуватися для перемикання камер. Натискайте M для додання мітки, потім Ctrl-M для її перейменування, дубльовані мітки повинні мати однакове ім'я.

  1. У 3D-Огляді -- 3D View виділіть Камеру -- Camera, на огляд через яку буде перемкнуто у кадрі з міткою.
  2. У редакторі Часолінія -- Timeline виділіть Мітку(и), у кадрі якого(х) буде перемкнуто на огляд через обрану Камеру.
  3. Наведіть вказівник миші над Часолінією та натисніть Ctrl-B для прив'язки обраної Камери до виділеної Мітки.
