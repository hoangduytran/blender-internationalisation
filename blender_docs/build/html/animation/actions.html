Дії -- Actions¶

При анімуванні об'єктів та властивостей у Blender'і «дії» (actions) містять дані анімування. Як і все інше у Blender'і, Дії є блоками даних (data-blocks).

Дії.

Тому, коли ви анімуєте об'єкт, змінюючи його локація за допомогою ключкадрів, така анімація збережеться всередині Дії.

Кожна властивість має канал, в який записуються відповідні дані, наприклад, зміна локації по осі X Cube.location.x запишеться у канал «X Location». Анімовані властивості X location та Y location можуть бути спільними одночасно для кількох об'єктів, якщо ці об'єкти мають анімаційні значення для цих властивостей X location і Y location.

Редактор Графів. Кожен Канал має функційну криву F-Curve, представлену лініями між ключкадрами.

Дії -- Actions
Записують та містять дані анімації.
Групи -- Groups
Є групами каналів.
Канали -- Channels
Записують властивості.
Ф-Криві -- F-Curves

Ключкадри -- Keyframes
Keyframes -- Ключкадри -- використовуються для задання значень властивостей, пов'язаних з певним моментом у часі.

Робота з Діями¶

Меню блоку даних «Дія» -- Action.

Коли ви перший раз анімуєте об'єкт, додаючи ключкадри, Blender створює «дію» Action, в яку записує дані анімування.

Actions can be managed with the Action data-block menu in the Dope Sheet Action Editor header, or the Properties region of the NLA Editor.

Якщо ви зробили кілька дій для одного і того ж об'єкта, клацайте на кнопці F у блоку даних дії редактора Аркуш Експозицій для увімкнення властивості «Псевдо Користувач» -- Fake User для інших потрібних дій, які поточно не зв'язані з об'єктом, щоб Blender зберіг ці незв'язані дії у файлі при закритті програми.

При редагуванні на об'єкті може використовуватися одночасно тільки одна «Дія» Action. Для змішування кількох дій разом використовується Редактор Нелінійної Анімації NLA Editor.

Запікання Дії -- Bake Action¶

Орієнтир

Редактор:«3D Огляд» -- 3D View
Режим:Режими Об'єкта та Пози -- Object and Pose Modes
Панель:«Полиця Інструментів > Анімація > Анімація > Дія: Запікання Дії» -- Tool Shelf ‣ Animation ‣ Animation ‣ Action: Bake Action
Меню:«3D Огляд > Об'єкт/Поза > Анімація > Запікання Дії» -- 3D View ‣ Object/Pose ‣ Animation ‣ Bake Action...

Інструмент «Запікання Дії» -- Bake Action перетворює інтерпольовані кадри в індивідуальні ключові кадри. Це може бути корисним для додання відхилення у зациклену дію, як наприклад «цикл ходи» -- walk cycle. Це також може бути корисним для ключкадрових анімацій, створених за допомогою драйверів або примусів.
