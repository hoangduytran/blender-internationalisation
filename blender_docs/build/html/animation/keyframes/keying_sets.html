Набори Ключування -- Keying Sets¶

Активний ідентифікатор даних «Набори Ключування» -- Keying Sets в Часолінії.

Набори Ключування -- Keying Sets -- це колекція властивостей. Вони використовуються для запису кількох властивостей одночасно.

Тепер, коли ви натиснете I в 3D Огляді -- 3D View, Blender додаватиме ключкадри для усіх властивостей в активний набір ключування.

Існують деякі вбудовані Набори Ключування, а також, власні користувацькі Набори Ключування, що називаються «Абсолютними Наборами Ключування» -- Absolute Keying Sets.

Для вибору і використання Набору Ключування установіть «Активний Набір Ключування» -- Active Keying Set у заголовку Часолінії -- Timeline Header, чи на панелі «Набори Ключування» -- Keying Sets, або натисніть Ctrl-Alt-Shift-I у 3D Огляді -- 3D View.

Панель «Набори Ключування» -- Keying Sets¶

Орієнтир

Редактор:Редактор Властивостей -- Properties editor
Панель:«Сцена > Набори Ключування» -- Scene ‣ Keying Sets

Ця панель використовується для додання, вибору та управління Абсолютними Наборами Ключування -- Absolute Keying Sets.

Панель «Набори Ключування» -- Keying Sets.

Активний Набір Ключування

Огляд Списком -- List View Наборів Ключування в активній Сцені.

Додання -- Add +
Додає порожній Набір Ключування.

Властивості¶

Опис -- Description
Короткий опис набору ключування.
Експорт у Файл -- Export to File
Експортує Набір Ключування у скрипт Python File.py. Для повторного додання набору ключування у сцену з файлу File.py відкрийте та виконайте файл File.py у Редакторі Тексту -- Text Editor.
Устави Ключкадрування -- Keyframing Settings

Ці опції керують усіма властивостями у даному Наборі Ключування. Зауважте, що такі самі устави в Уподобаннях Користувача -- User Preferences, при їх увімкненні, перезапишуть ці устави.

Лише Необхідне -- Only Needed
При увімкненні вставлятися будуть лише ключкадри там, де вони потрібні на відповідних Ф-Кривих -- F-Curves.
Видиме Ключування -- Visual Keying
При увімкненні вставлятимуться ключкадри на видимих трансформуваннях.
XYZ у RGB -- XYZ to RGB
При увімкненні, для нових Ф-Кривих для набору властивостей задаються кольори у RGB, наприклад, Location XYZ.

Панель «Активний Набір Ключування» -- Active Keying Set¶

Орієнтир

Редактор:Редактор Властивостей -- Properties editor
Панель:«Сцена > Активний Набір Ключування» -- Scene ‣ Active Keying Set

Ця панель використовується для додання властивостей в активний Набір Ключування.

Панель «Активний Набір Ключування» -- Active Keying Set.

Регіон каналів Редактора Графів Graph Editor з іменованими групами.

Шляхи -- Paths для Активного Набору Ключування

Колекція шляхів в Огляді Списком -- List View, кожен з яких має «Шлях Даних» -- Data Path до властивості, що призначений для додання її в активний Набір Ключування.

Додання -- Add +
Додає порожній шлях.

Властивості¶

Ціль -- Target
ID-Блок -- ID-Block
Задає Тип ID -- ID Type та Шлях Даних ID Об'єкта -- Object IDs Data Path для властивості.
Шлях Даних -- Data Path
Задає решту Шляху Даних для властивості.
Ціль Масиву -- Array Target
Вмикає використання «Всі Елементи» -- All Items зі Шляху Даних -- Data Path або при вимкненні дозволяє вибрати індекс масиву для певної властивості.
Групування Ф-Кривих -- F-Curve Grouping

Визначає, яка група буде додавати канали.

Ім'я Набору Ключування -- Keying Set Name, Нема -- None, Іменована Група -- Named Group

Устави Ключкадрування -- Keyframing Settings

Дані опції визначають окремі властивості у Наборі Ключування.

Лише Необхідне -- Only Needed
При увімкненні вставлятися будуть лише ключкадри там, де вони потрібні на відповідних Ф-Кривих -- F-Curves.
Видиме Ключування -- Visual Keying
При увімкненні вставлятимуться ключкадри на видимих трансформуваннях.
XYZ у RGB -- XYZ to RGB
При увімкненні, для нових Ф-Кривих для набору властивостей задаються кольори у RGB, наприклад, Location XYZ.

Додавання Властивостей¶

Орієнтир

Меню:«Контекстне меню > Додати Усе/Одиничне у Набір Ключування» -- Context menu ‣ Add All/Single to Keying Set
Скорочення:K

Деякі способи додання властивостей у набори ключування.

Клацайте RMB на потрібній властивості в інтерфейсі користувача -- User Interface, потім у контекстному меню виберіть пункт «Додати Одиничне у Набір Ключування» -- Add Single to Keying Set або «Додати Усе у Набір Ключування» -- Add All to Keying Set. Це додасть цю або ці властивості в активний набір ключування або у новий набір ключування, якщо жодного у сцені поки що не існує.

Наводьте вказівник миші над потрібними властивостями і далі натискаєте K для додання усіх властивостей у набір ключування Add All to Keying Set.
