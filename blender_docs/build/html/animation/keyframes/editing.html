Редагування¶

Вставка Ключкадрів¶

Орієнтир

Режим:«Режим Об'єкта» -- Object Mode
Панель:«Полиця Інструментів > Анімація > Анімація > Ключкадри: Вставка» -- Tool Shelf ‣ Animation ‣ Animation ‣ Keyframes: Insert
Меню:«Об'єкт > Анімація > Вставка Ключкадра» -- Object ‣ Animation ‣ Insert Keyframe...
Скорочення:I

Існує кілька методів додання нових анімаційних ключів. А саме:

  • У 3D-Огляді -- 3D View натискання клавіші I викликає меню, в якому вибирається для яких властивостей додати ключкадр.
  • Наведення курсора миші над потрібною властивістю і натискання I або клацок RMB з курсором миші над потрібною властивістю, що викликає контекстне меню, і вибір команди «Вставити Ключкадр» -- Insert Keyframe з цього меню.

Авто Ключкадр¶

Авто Ключкадр на Часолінії.

Опція «Авто Ключкадр» вмикається червоною кнопкою запису у заголовку Часолінії Timeline. Авто Ключкадр додає ключкадри автоматично у кадр, де знаходиться курсор часу, якщо значення для обраного типу властивостей трансформацій змінюється.

Детальніше дивіться тут - Timeline Keyframe Control.

Видалення Ключкадрів¶

Орієнтир

Режим:«Режим Об'єкта» -- Object Mode
Панель:«Полиця Інструментів > Анімація > Анімація > Ключкадри: Вилучити» Tool Shelf ‣ Animation ‣ Animation ‣ Keyframes: Remove
Меню:«Об'єкт > Анімація > Видалення Ключкадрів...» --Object ‣ Animation ‣ Delete Keyframes...
Скорочення:Alt-I

Існує кілька методів вилучення ключкадрів:

  • У 3D Огляді натискання Alt-I вилучає ключі анімації у поточному кадрі для виділених об'єктів.
  • При наведенні вказівника миші над значенням, натиснення Alt-I.
  • Клацок RMB над потрібним значенням та вибір «Видалити Ключкадр» Delete Keyframe з меню.

Зчищення Ключкадрів¶

Орієнтир

Режим:«Режим Об'єкта» -- Object Mode
Меню:«Об'єкт > Анімація > Зчищення Ключкадрів...» -- Object ‣ Animation ‣ Clear Keyframes...

Вилучає усі ключкадри з виділеного об'єкта.

Редагування Ключкадрів¶

Ключкадри можуть редагуватися у двох редакторах. Для цього використовуємо Редактор Графів -- Graph Editor або Аркуш Експозицій -- Dopesheet.

Приклади¶

Ключкадрова Анімація¶

Цей приклад показує вам, як анімувати локацію, оберт та масштаб куба.

  1. Спершу, у редакторі Часолінія -- Timeline або іншому з редакторів анімації установлюємо курсор часу на кадр 1.
  2. З виділеним кубом Cube у режимі об'єкта -- Object Mode наводимо вказівник миші над вікном 3D Огляду -- 3D View і натискаємо I.
  3. З контекстного меню Insert Keyframe Menu, що з'явиться, вибираємо пункт LocRotScale. Це дасть запис значень локації, обертання та масштабу для куба Cube на кадрі 1.
  4. Установимо курсор часу у кадр 100.
  5. Використаємо інструменти «Захоп/Пересув» -- Grab/Move G, «Обертання» -- Rotate R, «Масштаб» -- Scale S для трансформування цього куба.
  6. З наведеним вказівником миші над 3D Оглядом натиснемо I. З контекстного меню Insert Keyframe Menu, що з'явиться, вибираємо пункт LocRotScale.

Для перевірки даної анімації натиснемо Alt-A для вмикання її відтворення.

Ця анімація у кадрах 1, 50 та 100.
