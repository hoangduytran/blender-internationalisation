Панель «Ключі Форм» -- Shape Keys¶

Орієнтир

Редактор:Редактор Властивостей - Properties editor
Режим:Усі режими
Панель:«Редактор Властивостей > Дані Об'єкта > Ключі Форм» -- Properties editor ‣ Object Data ‣ Shape Keys

Панель «Ключі Форм» -- Shape Keys.

Устави¶

Індекс Активного Ключка Форми -- Active Shape Key Index

Детальніше тут -- Огляди списків.

Значення -- Value
Поточне Значення даного Ключа Форми (0.0 до 1.0).
Приглушення -- Mute (іконка з оком)
Це візуально вимикає показ дії ключа форми у 3D-Огляді -- 3D View.
Спеціальне -- Specials
Перенести Ключ Форми -- Transfer Shape Key
Переносить активний Ключ Форми Shape Key з іншого об'єкта. Виділіть два об'єкти і активний Ключ Форми скопіюється на активний об'єкт.
Сполучити як Форми -- Join as Shapes
Переносить Поточну Форму Current Shape з іншого об'єкта. Виділіть два об'єкти і Форма Shape скопіюється на активний об'єкт.
Дзеркалити Ключ Форми -- Mirror Shape Key
Якщо ваша сіть гарна і симетрична, в Режимі Об'єкта Object Mode, ви можете віддзеркалити ключі форми по осі X. Це не працює допоки, вершини сіті не є ідеально симетричними. Використовуйте для цього «Сіть > Симетризація» -- Mesh ‣ Symmetrize у Режимі Правки -- Edit Mode.
Дзеркалити Ключ Форми (Топологія) -- Mirror Shape Key (Topology)
Працює так само, як і «Дзеркалити Ключ Форми» Mirror Shape Key, хоча ця команда визначає дзеркальні вершини на основі топології сіті. Вершини сіті не мають бути ідеально симетричними для цього.
Нова Форма З Міксу -- New Shape From Mix
Додає новий ключ форми з поточно деформованої форми об'єкта.
Видалити Усі Форми -- Delete All Shapes
Видаляє усіх ключів форм.
Відносно -- Relative
Вмикає тип даних ключів форм як «Відносно» Relative, при вимкненні -- «Абсолютно» Absolute.
Показати Лише (іконка шпильки)
Показу форму активного ключа форм без інтерполяції у 3D-Огляді -- 3D View. Допоки об'єкт знаходиться у Режимі Правки Edit Mode опція Show Only автоматично увімкнена, якщо її не вимикає устава «Режим Правки» Edit Mode, про яку мова нижче.
Режим Правки -- Edit Mode
Модифікує ключ форми при знаходженні об'єкта у Режимі Правки Edit Mode.

Відносні Ключі Форм -- Relative Shape Keys¶

Відносні ключі форм деформують поверхню на основі виділеного ключа форм. Стандартно, усі відносні ключі форм деформують відносно першого ключа форм, що зветься «База» Basis.

Опції Відносних Ключів Форм.

Зчистити Вагомості -- Clear Weights X
Установлює усі значення у нуль.
Значення -- Value
Значення активного ключа форм.
Діапазон -- Range
Мінімальне Min та максимальне Max значення діапазону значень активного ключа форм.
Змішання -- Blend
Група Вершин -- Vertex Group
Обмежує деформування активним ключем форм заданою групою вершин.
Відносно -- Relative
Дозволяє вибрати ключ форми, відносно якого буде здійснюватися деформування.

Абсолютні Ключі Форм -- Absolute Shape Keys¶

Абсолютні ключі форм деформують на основі попереднього та наступного (у списку) ключів форм. Вони в основному використовуються для деформування об'єктів у різні форми в часі.

Опції Абсолютних Ключів Форм.

Скинути Часування -- Reset Timing (іконка з годинником)
Скидає часування, розкладку у часі для абсолютних ключів форм.
Інтерполяція -- Interpolation

Визначає тип інтерполяції між ключами форм.

Лінеарно -- Linear, Кардинально -- Cardinal, Катмул-Ром -- Catmull-Rom, Б-Сплайн -- B-Spline

Різні типи інтерполяції.

Червона лінія представляє інтерпольовані значення між ключами (позначені чорними точками).
Оцінка Часу -- Evaluation Time
Використовується для керування впливом ключа форми.

Приклади¶

Скидання Часування¶

Наприклад, якщо ви маєте такі ключі форм у такому порядку -- Basis, Key_1, Key_2.

Скидання Часування Reset Timing зациклить ці ключі форм та установить ключкадри форми з кроком +0.1:

  • Basis 0.1
  • Key_1 0.2
  • Key_2 0.3

Оцінка Часу -- Evaluation Time покаже це як кадр 100:

  • Basis 10.0
  • Key_1 20.0
  • Key_2 30.0

Оцінка Часу -- Evaluation Time¶

Наприклад, якщо ви маєте такі ключі форм у такому порядку -- Basis, Key_1, Key_2, і ви скинули часування:

  • Basis 10.0
  • Key_1 20.0
  • Key_2 30.0
