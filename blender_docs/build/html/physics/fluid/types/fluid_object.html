Об'єкт Рідини -- Fluid Object¶

Reference

Panel:Physics ‣ Fluid
Type:Fluid

Устави об'єкта рідини.

Усі регіони такого об'єкта, що є всередині габаритної коробки домену, будуть використані як фактична рідина в імітації. Якщо ви помістите більше, ніж один об'єкт рідини всередині домену, то вони повинні поточно не перетинатися. Також переконайтеся, що нормалі поверхні вказують назовні. На відміну від об'єктів домену для об'єктів рідини використовується фактична геометрія сіті.

Опції¶

Тип Ініціалізації Об'єму -- Volume Initialization Type
See Volume Initialization Type.
Експорт Анімованої Сіті -- Animated Mesh/Export
See Animated Mesh/Export.
Початкова Скорість -- Initial Velocity
Швидкість рідини на початку імітації, у метрах за секунду.

Mẹo

Напрямок Нормалей Поверхні має велику різницю!

Blender uses the orientation of the Surface Normals to determine what is "inside of" the Fluid object and what is "outside". You want all of the normals to face outside (in Edit Mode, use Ctrl-N or press Spacebar and choose Edit ‣ Normals ‣ Calculate Outside). If the normals face the wrong way, you will be rewarded with a "gigantic flood of water", because Blender will think that the volume of the object is outside of its mesh! This applies regardless of the Volume initialization type setting.
