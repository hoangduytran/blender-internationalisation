Перешкода Рідини -- Fluid Obstacle¶

Reference

Panel:Physics ‣ Fluid
Type:Obstacle

This object will be used as an obstacle in the simulation. As with a fluid object, obstacle objects currently should not intersect. As for fluid objects, the actual mesh geometry is used for obstacles. For objects with a volume, make sure that the normals of the obstacle are calculated correctly, and radiating properly (use the Flip Normal button, in Edit Mode, Mesh Tools panel, in the Tool Shelf), particularly when using a spinned container. Applying a Subdivision Surface Modifier before baking the simulation could also be a good idea if the mesh is not animated.

Опції¶

Тип Ініціалізації Об'єму -- Volume Initialization Type
See Volume Initialization Type.
Тип границі

Determines the stickiness of the obstacle surface, called "Surface Adhesion". Surface Adhesion depends in the real world on the fluid and the graininess or friction/adhesion/absorption qualities of the surface.

Без Сковзання -- No Slip
Спричиняє прилипання рідини до перешкоди (нульова скорість).
Вільне Сковзання -- Free Slip
Дозволяє пересування по перешкоді (лише нульова скорість по нормалі).
Часткове Сковзання -- Partial Slip
Суміш обох типів, з 0 здебільшого буде не сковзання, а 1 ідентично вільному сковзанню.

Зауважте, що якщо сіть пересувається, то це буде оброблено як «без сковзання» автоматично.

Приклад різних типів границі для падіння краплі по похилій стіні. Зліва направо: без сковзання, часткове сковзання 0.3, часткове сковзання 0.7 та вільне сковзання.

Експорт Анімованої Сіті -- Animated Mesh/Export
See Animated Mesh/Export.
Величина часткового сковзання -- Part Slip Amount
Величина змішування між без сковзання та вільним сковзанням, описаними вище.
Фактор Впливу -- Impact Factor
Величина корекції об'єму рідини для підсилення/втрати від впливу рухомих об'єктів. Якщо цей об'єкт не пересувається, то ця устава немає жодного ефекту. Однак, якщо він пересувається та рідина стикається з ним, від'ємні значення забирають об'єм від Домену, а додатні значення додають об'єм до Домену. Діапазон від -2.0 до 10.0.
