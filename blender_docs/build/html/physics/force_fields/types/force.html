Сила -- Force¶

Reference

Panel:Physics ‣ Force Fields
Type:Сила -- Force

Силове поле «Сила» -- Force.

Поле Force є найпростішим із силових полів. Воно дає постійне силування у напрямку до (додатна сила) або від (від'ємна сила) центра об'єкта. Ньютонівські частинки притягаються до поля з негативною силою та відштовхуються від поля з позитивною силою.

Інтерфейс користувача для силового поля Force.

Для частинок типу «Рій» -- Boids Particles це поле з додатною силою може використовуватися як «ціль» -- Goal, а поле з негативною силою може бути використане як «переслідувач, хижак» -- Predator. Чи буде рій -- Boids прагнути до цілі або летіти від переслідувача, залежить від устав фізики -- Physics для рою.
