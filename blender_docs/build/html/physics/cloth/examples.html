Приклади¶

Щоб розпочати створювати імітацію тканини, спершу вам необхідна, звичайно ж, сама тканина. Тому, давайте видалімо стандартний куб та додамо площину. Для того, щоб мати добру гнучкість тканини, вам необхідно підподілити площину кілька разів, добре буде біля восьми разів. Отже, натисніть Tab для переходу у Режим Правки -- Edit Mode, далі натисніть W «Спеціальне > Підподіл» -- Specials ‣ Subdivide multi та задайте кількість розрізів як 8.

Тепер, ми зробимо імітацію поведінки тканини у вкладці «Фізика» -- Physics Редактора Властивостей. Клацніть на кнопці «Тканина» -- Cloth для призначення тканинних властивостей площині. Далі, нижче з'являться багато панелей з цими властивостями, більшість з яких на даний момент нам ще не потрібні.

Для анімації базового укладу імітації тканини цього достатньо, але якщо ви натиснете Alt-A, то ваша тканина просто впаде дуже видовищно. Про те, як задавати закріплення та звисання тканини, обговорюється окремо у підрозділах про пришпилення та зіштовхування.

Використання тканинної імітації для формування/ліплення сіті¶

Ви можете «Застосувати» -- Apply модифікатор «Тканина» -- Cloth у будь-який момент для зафіксування сіті об'єкта у певному кадрі. Далі, ви можете знову призначити властивість «тканина» для цього ж об'єкта, задавши стартовий та кінцевий кадри для іншої подальшої імітації.

Іншим прикладом є створення анімації колихання прапора. Визначте прапор як просту ґратчасту форму та пришпильте один з її країв до флагштока. Відтворіть анімацію цієї імітації тривалістю близько 50 кадрів і прапор впаде в його позицію «спокою». Застосуйте модифікатор «Тканина» -- Cloth. Якщо ви хочете, що прапор розвівався або іншим чином рухався у сцені, визначте інший діапазон кадрів з огляду камери для імітації.

Згладження тканини¶

Now, if you followed this from the previous section, your cloth is probably looking a little blocky. In order to make it look nice and smooth like the picture you need to apply a Smooth and/or Subdivision Surface Modifier in the Modifiers tab. Then, in the Tool Shelf, find the Edit panel and Press Smooth.

Тепер, якщо ви натиснете Alt-A, то тканинна поверхня буде виглядати набагато краще, ви не думаєте?

Cloth on Armature¶

Тканина деформована арматурою, а також враховано додатковий об'єкт зіткнення: Regression blend-file.

Cloth with Animated Vertex Groups¶

Тканина з анімованими пришпиленими вершинами: Regression blend-file. НЕ ПІДТРИМУЄТЬСЯ: При починанні для цілі зі значення 0 та збільшенні його, але при відсутності пришпилення вершини це не працює (наприклад, від goal = 0 до goal = 0.5).

Тканина з Динамічним Малюванням¶

Cloth with Dynamic Paint using animated vertex groups: Regression blend-file. UNSUPPORTED: Starting with a goal of 0 and increasing it, but still having the vertex not pinned will not work (e.g. from goal = 0 to goal = 0.5) because the necessary "goal springs" cannot be generated on-the-fly.
