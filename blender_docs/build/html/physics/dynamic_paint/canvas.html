Полотно -- Canvas¶

Reference

Panel:Physics ‣ Dynamic Paint
Type:Полотно -- Canvas

The Canvas type makes object receive paint from Dynamic Paint brushes.

Головна панель полотна.

Поверхня Малювання -- Paint Surface

Список -- list поверхонь Динамічного Малювання. Ці поверхні є базово шарами малювання, що працюють незалежно один від одного. Ви можете визначати окремі устави для них та запікати їх окремо.

Показати Передогляд -- Show Preview (іконка з оком)
Цей перемикач вмикає видимість передогляду результатів типу/формату поверхні у 3D Огляді -- 3D View, якщо це для нього передбачено.
Активне -- Is Active
Цей стяг перемикає активність поверхні. Якщо він вимкнений, то не робляться щодо неї жодні розрахунки або передогляди.

Нижче ви можете задати тип поверхні та коригувати устави якості та часування.

Формат -- Format

Кожна поверхня має певний формат і тип. Формат визначає, як дані будуть зберігатися та виводитися.

Вершина -- Vertex
Динамічне Малювання оперує безпосередньо даними вершин сіті. Результати зберігаються у кеш точок та можуть показуватися в оглядвікні. Однак, використання рівня вершин також вимагає високо підподіленої сіті для роботи.
Відеоряди Зображень -- Image Sequences
Динамічне Малювання генерує як вивід файли обгорнутих за UV зображень з визначеною роздільністю.
Якість -- Quality: Роздільність -- Resolution
Для відеорядів зображень -- з устав якості ви можете коригувати розміри вивідних зображень.
Загладжування -- Anti-aliasing
Загладжування -- Anti-aliasing згладжує краї малювання, використовуючи 5-кратний метод багато-відбору.
Кадри -- Frames: Старт -- Start, Кінець -- End
Визначає стартовий та кінцевий кадри обробки поверхні.
Під-кроки -- Sub-steps
Під-кроки -- це додаткові вибірки між кадрами, зазвичай потрібні, коли пензель дуже швидкий.

Advanced¶

Reference

Type:Полотно -- Canvas
Panel:Physics ‣ Dynamic Paint Advanced

Панель просунутих устав полотна.

From Advanced panel you can adjust surface type and related settings.

Тип Поверхні -- Surface Type¶

Кожна поверхня має «тип», що визначає, яка використовується поверхня.

Painting -- Малювання¶

Поверхня «Малювання» -- Paint.

Paint is the basic surface type that outputs color and wetness values. In case of vertex surfaces, results are outputted as vertex colors.

A wetmap is a black-and-white output that visualizes paint wetness. White being maximum wetness, black being completely dry. It is usually used as mask for rendering. Some "paint effects" affect wet paint only.

Висихання -- Dry

Повністю вимкнене висихання є корисним для невизначеного розтікання малювання.

Висихання Кольору -- Color Dry
Може використовуватися для визначення рівня вологості, коли кольори малювання починають зсуватися у «фон» поверхні. Нижчі значення можуть бути корисними для запобігання розтікання малювання, яке починає ставати прозорим при висиханні, а вищі значення дають кращі результати у цілому.

Зміщення -- Displace¶

Поверхня типу «Зміщення» -- Displace.

Цей тип поверхні виводить глибину перетинання від об'єктів пензля.

Mẹo

Якщо вивід зміщення здається надто грубим, то зазвичай допомагає це покращити додання модифікатора «Згладження» -- Smooth після (нижче) модифікатора «Динамічне Малювання» -- Dynamic Paint у стеку модифікаторів.

Хвилі -- Waves¶

Поверхня типу «Хвилі» -- Waves.

Цей тип поверхні продукує імітований рух хвиль. Як і зміщення, поверхня типу «хвилі» також використовує глибину перетинання пензля для визначення сили пензля.

Ви можете вжити наступні устави для коригування цього руху:

Відкриті Краї -- Open Borders
Дозволяє хвилям проходити через «краї» сіті, замість відбиття від них.
Масштаб часу -- Timescale
Безпосередньо коригує швидкість імітації без впливу на її результат. Нижчі значення роблять протікання імітації повільніше, і навпаки.
Швидкість -- Speed
Визначає те, як швидко хвилі розходитимуться по поверхні. Ця устава також відповідає розміру імітації. Половина цієї швидкості дорівнює подвійній величині поверхні.
Згасання -- Damping
Скорочує силу хвилі з часом. Базово коригує, наскільки швидко хвиля зникає.
Пружність -- Spring
Коригує силу, що штовхає воду назад до «нульового рівня».

Mẹo

У деяких випадках рух хвилі стає дуже нестабільним навколо пензля. Зазвичай цьому зараджує скорочення швидкості хвилі, «фактора хвилі» пензля або навіть роздільності сіті/поверхні.

Вагомість -- Weight¶

Поверхня типу «Вагомість» -- Weight.

Це спеціальний тип поверхні, що доступний тільки для формату «Вершина». Він виводить вагомості груп вершин, що можуть використовуватися іншими модифікаторами та засобами Blender'а.

Mẹo

Зазвичай, переважно використовується джерело пензля «Близькість» для поверхонь цього типу «Вагомість», щоб мати плавний спад між значеннями вагомості.

Загальні опції¶

Для більшості типів поверхонь є кілька загальних устав. Більшість типів мають устави щодо Dissolve та Brush:

Розчинення -- Dissolve
Використовується для зроблення плавного повернення поверхні в її оригінальний стан упродовж визначеного періоду часу.
Група Пензля -- Brush Group
Використовується для визначення певної групи об'єктів, з якої повинні бути вибрані об'єкти пензлів.
Масштаб Впливу -- Influence Scale, Масштаб Радіусу -- Radius Scale
Для підправлення устав пензля окремо для кожної поверхні.

Output¶

Reference

Type:Полотно -- Canvas
Panel:Physics ‣ Dynamic Paint Output

Canvas Output panel.

На панелі Output ви можете визначати, як виводяться результати поверхні.

Вершина -- Vertex¶

Для поверхонь формату Vertex ви можете вибрати шар даних сіті (колір / вагомість, залежно від типу поверхні) для генерування результатів. Ви можете вживати іконки "+"/"-" для додання/вилучення шарів даних із заданим ім'ям. Якщо шар із заданим іменем не знайдено, то ім'я його показується забарвлене червоним.

Відеоряд Зображень -- Image Sequence¶

For Image Sequence surfaces, you can define used UV maps and output file saving directory, filenames and image format.

Ефекти¶

Reference

Type:Полотно -- Canvas
Panel:Physics ‣ Dynamic Paint Effects

Effects panel.

Це спеціальна функція для поверхні типу "Paint". Вона генерує анімовані рухання на поверхні полотна.

Ефекти
Розтікання -- Spread
Малювання повільно розтікається на сусідні точки та у підсумку заповнює усі з'єднані ділянки.
Стікання -- Drip
Paint moves in specific direction specified by Blender force fields, gravity and velocity with user-defined influences.
Усихання -- Shrink
Намальована ділянка повільно всихає, допоки не зникає повністю.

Для ефектів розтікання та стікання, задіюється тільки «вологість малювання», а тому при висиханні малювання, їх рух стає повільнішим, допоки не зупиняється.

Cache¶

Reference

Type:Полотно -- Canvas
Panel:Physics ‣ Dynamic Paint Cache

Панель кешу полотна.

Ця панель поточно видима тільки для поверхонь формату Vertex. Ви можете використовувати її для коригування та запікання кешу точок.
