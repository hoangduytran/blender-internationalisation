Режим Правки Частинок -- Particle Edit Mode¶

Using Particle Edit Mode you can edit the keyed points (keyframes) and paths of Baked Hair, Particle, Cloth, and Soft Body simulations. (You can also edit and style hair before baking).

Оскільки, робота у режимі «Правка Частинок» -- Particle Edit Mode є досить простою та дуже подібною до роботи з вершинами у 3D Огляді -- 3D View, то ми покажемо, як укласти систему частинок та далі дамо довідку про різні її функції.

Використання¶

Mẹo

Тільки кадри, запечені у пам'ять, можуть редагуватися!

Якщо ви не можете редагувати частинки, то перевірте, чи ви запекли їх систему у дисковий кеш -- Disk Cache.

Уклад системи частинок¶

  1. Створіть систему частинок типу «Волосся» -- Hair.
  2. Задайте для неї початкову скорість у напрямку нормалі -- Normal.
  3. Створіть імітацію.
  4. Увімкніть стяг у назві панелі «Динаміка Волосся» -- Hair Dynamics.

Редагування пасм волосся у режимі «Правка Частинок» -- Particle Edit Mode.

Уклад для імітацій інших типів Частинок -- Particle, Тканини - Cloth та М'якого Тіла -- Soft Body¶

  1. Use Emitter particles, or a cloth/soft body simulation.
  2. Створіть імітацію, уклавши об'єкти та або емітери, задайте для них діапазон часу (вживайте малий діапазон, якщо ви тільки починаєте та експериментуєте), укладіть інші устави імітацій, скористайтеся Alt-A для їх здійснення.

Запікання імітації¶

Після того, як ви задоволені загальним виглядом імітації, запечіть -- bake імітацію у Режимі Об'єкта -- Object Mode. Імітація повинна бути запеченою, щоб можна було її редагувати.

Правка або редагування імітації¶

Перемкніться у режимі «Правка Частинок» -- Particle Edit за допомогою вибірника режимів Mode select menu у заголовку 3D Огляду -- 3D View для редагування, правлення шляхів частинок/ключкадрів. Вам можливо знадобиться натиснути T у межах 3D Огляду, щоб бачити панелі режиму Particle Edit на Полиці Інструментів. Пересуньте курсор часу на часолінії на кадр, в якому ви хочете редагувати імітацію та скористайтеся засобами режиму правки частинок Particle Edit для її правлення. Працюйте повільно, переглядаючи ваші зміни за допомогою Alt-A, та часто зберігайте вашу роботу, щоб ви змогли повернутися до попередньої версії, якщо щось трапиться або коли ви не задоволені останніми зробленими змінами.

Mẹo

Щоб мати змогу чітко бачити, з чим ви працюєте:

  1. Open the Options panel in the Tool Shelf.
  2. Виберіть режим показу точок -- Point select mode (дивіться нижче) у заголовку 3D Огляду. Це стане показувати ключові точки уздовж шляху частинки.

Виділення -- Selecting¶

  • Одиничне: RMB.
  • Усе: A.
  • Пов'язане: Перемістіть мишу на ключовою точкою та натисніть L.
  • Виділ рамкою: B.
  • Перше/останнє: «Спеціальне > Виділити Перше чи Корені / Виділити Останнє чи Верхівки» -- Specials ‣ Select First / Select Last.

Ви можете також використовувати меню «Виділ» -- Select.

Mẹo

Виділення -- Selections

Виділення надзвичайно корисні для модифікування тільки бажаних вам частинок. Наведіть вказівник миші над певним шляхом та натисніть L для виділення цього шляху, наведіть вказівник миші над іншим шляхом та знову натисніть L для додання у виділ і цього іншого шляху. Для вилучення шляху з виділу утримуйте натисненою Shift та натисніть L. Для зневиділення, зняття виділення з усього натисніть A.

Метод для виділення окремих точок такий же, як і у режимі правки вершин -- Edit Mode. RMB для одиничного виділення, Shift-RMB для додання/вилучення точки у/із виділу.

Виділити Випадково -- Select Random¶

Randomly selects particles.

Percent
Percent of particles to randomly select.
Random Seed
Seed value to use for the selection.
Action
Select random can be either used to select or deselect particles.
Type
Selects either hair or points. Here theses terms can be confusing because hair/point does not refer to the particle type but the path/points of the hair/particle.

Режими Виділення -- Selection Modes¶

Режими Виділення -- Selection Modes.

Шлях -- Path
Жодна з ключових точок не видима, ви можете виділяти/зневиділяти шляхи частинок тільки цілком.
Точка -- Point
Ви бачите усі ключові точки.
Верхівка -- Tip
Ви можете бачити та редагувати (включно і пензлями) тільки верхівки частинок, тобто останні їх ключові точки.

Пензель -- Brush¶

Орієнтир -- Reference

Mode:Режим Правки Частинок -- Particle Edit Mode
Panel:Tool Shelf ‣ Tools ‣ Brush

За допомогою цих кнопок ви можете обрати потрібний тип утиліти «чесання».

Нема -- None
Без спеціального інструменту, тільки редагує ключові точки як «звичайні» вершини.
Гребінець -- Comb
Рухає ключові точки подібно до «пропорційної правки».
Згладження -- Smooth
Робить паралельними візуально суміжні сегменти.
Додання -- Add

Додає нові частинки.

Рахунок -- Count
Кількість нових частинок за крок.
Інтерполювання -- Interpolate
Інтерполює форму нових волосинок з наявних.
Кроки -- Steps
Amount of brush steps.
Ключі -- Keys
Визначає, зі скількома ключами робити нові частинки.
Довжина -- Length
Масштабує сегменти, так що робить волосинки довшими з режимом пензля «Подовження» -- Grow або коротшими -- з режимом «Укорочення» -- Shrink.
Розпушення -- Puff

Обертає волосинку навколо її першої ключової точки (кореня). Таким чином, робить розпушення з режимом пензля «Додання» -- Add або загладження -- з режимом пензля «Відняття» -- Sub.

Об'єм Розпушення -- Puff Volume
Apply puff to unselected end points, (helps maintain hair volume when puffing root).
Підріз -- Cut
Масштабує сегменти, допоки остання ключова точка не досягає пензля.
Вагомість -- Weight
This is especially useful for soft body animations, because the weight defines the soft body Goal. A keypoint with a weight of 1 will not move at all, a keypoint with a weight of 0 subjects fully to soft body animation. This value is scaled by the GMin to GMax range of soft body goals...

Загальні опції¶

Нижче типів пензлів з'являються їх устави:

Радіус -- Radius F
Set the radius of the brush.
Сила -- Strength Shift-F
Задає силу ефекту пензля (окрім пензля Add).
Додання/Відняття -- Add/Sub Подовження/Укорочення -- Grow/Shrink
Задає режим пензля для додання його ефекту або відняття.

Редагування -- Editing¶

Cảnh báo

Остерігайтеся касувань!

Використання команди касування Undo у режимі правки частинок Particle Edit Mode може давати дивні результати. Пам'ятайте, що слід часто зберігатися!

Пересування ключових точок або частинок¶

  • Для пересуву виділених ключових точок натисніть G або скористайтеся одним з інших методів пересуву вершин.
  • Для пересуву кореня частинки ви повинні вимкнути опцію Keep > Root на Полиці Інструментів -- Tool Shelf.
  • Ви можете робити багато операцій, як і з вершинами, включаючи масштабування, обертання та вилучення (повністю частинок або лише одиничних їх ключів).
  • Ви не можете дублювати або видавлювати ключі або частинки, але ви можете підподіляти частинки, що додає нові ключові точки за допомогою «Спеціальне > Підподілення» -- Specials ‣ Subdivide.
  • Альтернативно, ви можете повторно задавати кількість ключових точок для частинки за допомогою «Спеціальне > Переключування» -- Specials ‣ Rekey.

Наскільки плавно показуватиметься волосся та шляхи частинок залежить від устави Path Steps на Полиці Інструментів -- Tool Shelf. Нижчі значення продукують грубішу інтерполяцію між точками, тоді як вищі -- плавніший шлях.

Дзеркалення -- Mirror¶

Орієнтир -- Reference

Mode:Режим Правки Частинок -- Particle Edit Mode
Menu:Particle ‣ Mirror

If you want to create an X axis symmetrical haircut you have to do following steps:

  1. Виділити усі частинки за допомогою A.
  2. Дзеркалити частинки за допомогою Ctrl-M або меню Particle ‣ Mirror.
  3. Turn on X Mirror in the Brush panel.

It may happen that after mirroring two particles occupy nearly the same place. Since this would be a waste of memory and render time, you can Remove doubles either from the Specials W or the Particle menu.

Уніфікувати Довжину -- Unify Length¶

Орієнтир -- Reference

Mode:Режим Правки Частинок -- Particle Edit Mode
Menu:Particle ‣ Unify Length, Specials ‣ Unify Length

Цей інструмент використовується для зроблення усіх виділених частинок однакової довжини шляхом знаходження їх усередненого значення їх довжин.

Показ/Схов -- Show/Hide¶

Орієнтир -- Reference

Mode:Режим Правки Частинок -- Particle Edit Mode
Menu:Particle ‣ Show/Hide

Ховання та показ схованих частинок працює подібно до аналогічних операцій для вершин у 3D Огляді. Виділіть одну чи кілька ключових точок частинки, які ви хочете сховати та натисніть H. Частинка фактично не зникає, тільки ключові точки.

Сховані частинки (тобто, частинки, чиї ключові точки сховані) не реагують на різні пензлі. Але:

Якщо ви вжили «Дзеркалення» -- Mirror, то навіть частинки зі схованими ключовими точками можуть бути пересунуті, якщо їх віддзеркалення були пересунуті.

To unhide all hidden particles press Alt-H.
