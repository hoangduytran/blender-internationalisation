Редактор Властивостей -- Properties Editor¶

Верхня частина Редактора Властивостей -- Properties Editor.

Редактор Властивостей -- Properties Editor використовується для редагування даних та властивостей для Активної Сцени -- Active Scene та Активного Об'єкта -- Active Object.

Вкладки -- Tabs¶

Редактор Властивостей має різні вкладки, які можна вибирати за допомогою ряду іконок в його заголовку. Ці вкладки описуються в призначених для них розділах цього посібника, посилання на які наведено нижче.

Сцена -- Scene / Зображ -- Render¶

Ці вкладки використовуються для додання функції та зміни властивостей для Активної Сцени.

Вкладки сцени/зображування.

  • Зображ -- Render та Устави -- Settings: Blender Internal, Cycles
  • Шари Зобража -- Render Layers
  • Сцена -- Scene
  • Світ -- World: Blender Internal, Cycles

Об'єкт -- Object та Дані Об'єкта -- Object Data¶

Ці вкладки використовуються для додання функцій та зміни властивостей для Активного Об'єкта (та інших активних елементів, матеріалу, кривої тощо).

Вкладки даних об'єкта.

The Object Data tabs shown depend on what type of object was selected last (the Active Object).

  • Об'єкт -- Object
  • Примуси -- Constraints
  • Модифікатори -- Modifiers
  • Сіть -- Mesh
  • Крива -- Curve
  • Поверхня -- Surface
  • Метакуля -- Metaball
  • Текст -- Text
  • Порожняк -- Empty
  • Арматура -- Armature
  • Кістки -- Bones
  • Примуси Кісток -- Bone Constraints
  • Решітка -- Lattice
  • Гучномовець -- Speaker
  • Камера -- Camera: Blender Internal, Cycles
  • Лампа -- Lamp: Blender Internal, Cycles
  • Матеріал -- Material: Blender Internal, Cycles
  • Текстура -- Texture: Blender Internal, Cycles
  • Частинки -- Particles
  • Фізика -- Physics

Головний огляд -- Main View¶

Редактор Властивостей з обраною вкладкою «Сіть» -- Mesh.

У верхній частині кожної вкладки список іконок пояснює вміст, в якому будуть редагуватися властивості. У прикладі вище, сіть у формі куба -- Cube пов'язана з об'єктом «Куб» -- Cube, який у свою чергу пов'язаний зі сценою -- Scene.

Перемикання іконки зі шпилькою з лівої сторони дає вказівку Blender'у показувати або, при пришпиленні, тільки властивості даного блоку даних, або властивості згідно контексту поточно виділеного елемента.
