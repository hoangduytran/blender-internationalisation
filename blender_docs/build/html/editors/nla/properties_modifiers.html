Властивості й Модифікатори -- Properties & Modifiers¶

Властивості -- Properties¶

Властивості смужок можуть бути доступні у регіоні «Властивості» -- Properties редактора НЛА.

Дані Анімації -- Animation Data¶

Reference

Panel:Properties region ‣ Animations ‣ Animation Data

Animation Data panel.

Дія -- Action
A Data-Block menu that allows you to edit actions shown in the action track. See also the Action Editor's Action.
Екстраполяція Дії -- Action Extrapolation

Дія для заповнення проміжків після смужки.

Утримання -- Hold
Впливає на обидва боки смужки.
Утримання Вперед -- Hold Forward
Впливає на регіон після дії, тільки.
Ніщо -- Nothing
Ні те, ні інше.
Змішання Дії -- Action Blending

Впливає на поведінку, коли дві стежки одночасно мають криву, що впливає на однакову їх властивість.

Заміна -- Replace
Спричиняє, що верхня смужка має перевагу відповідно до параметрів Увід/Вивід Змішання -- Blend In/Out (дивіться наступну опцію, нижче).
Множення -- Multiply, Відняття -- Subtract, Додання -- Add
The weighted result of the strip is multiplied, subtracted, or added with the accumulated results.
Вплив Дії -- Action Influence
Amount the active Action contributes to the result of the NLA stack.

Активна Стежка -- Active Track¶

Reference

Panel:Properties region ‣ Animations ‣ Active Track
Ім'я -- Name
Ім'я стежки, якій поточно належить смужка.

Активна Смужка -- Active Strip¶

Active Strip panel.

Опції для власне смужки.

Ім'я -- Name
Перейменовує смужки.
Тип -- Type
Буде легше сказати «Кліп Дії» -- "Action Clip", «Перехід» -- "Transition" або «Мета» -- "Meta" відповідно до трьох типів смужок.
Протяги Смужки -- Strip Extents
Границі власне смужки. Зауважте, це буде розтяг тривалості Дії, це не спричинить збільшення або зменшення ключкадрів з Дій для відтворення (дивіться нижче про цю опцію).
Екстраполяція -- Extrapolation
Дивіться вище про «Екстраполяція Дії» -- Action Extrapolation.
Змішання -- Blending
Дивіться вище про «Змішання Дії» -- Action Blending.
Авто Увід/Вивід Змішання -- Auto Blend In/Out
Створює градієнт, що починається з перекриття смужок. Перша смужка матиме повний контроль на початку, який лінеарно, прямолінійно через градієнт значень перенесеться у повний контроль другої смужки на кінці часового періоду їх перекриття.
Увід Змішання -- Blend In
Задає кадр, що представляє момент, коли ця смужка буде мати повний вплив.
Вивід Змішання -- Blend Out
Задає останній кадр повного впливу цієї смужки.
Приглушено -- Muted
Приглушує одиничну смужку (подібно до приглушення стежки, вище). Спричиняє появу контуру навколо смужки.
Зворотно -- Reversed
Спричиняє, що ця смужка буде відтворюватися повністю назад.

Кліп Дії -- Action Clip¶

Reference

Panel:Properties region ‣ Animations ‣ Active Clip

Action Clip panel.

Це представляє «дані об'єкта» для смужки. Схоче на значення трансформацій для об'єкта.

Дія -- Action
Орієнтир на Дію, що вміщена у смужці. Може бути зміненим заміною поточного значення смужки з іншої Дії.
Протяги Дії -- Action Extents

Скільки дія буде використовуватися.

Примітка: Якщо ви виберете значення, що вище або нижче фактичного рахунку ключкадрів для Дії, то буде задіяна Екстраполяція F-Кривої. Яка може бути змінена у Редакторі Графів -- Graph Editor в «Канал > Режим Екстраполяції» -- Channel ‣ Extrapolation Mode.

Синх Довжину -- Sync Length
Спричиняє скидання значень устав «Стартовий Кадр» та «Кінцевий Кадр» -- "Start" and "End" Frames, що описані вище, у значення першого та останнього ключкадрованих кадрів Дії.
«Зараз» Синхронізувати Довжину Дії -- Sync Action Length "Now"
Спричиняє скидання значень устав «Стартовий Кадр» та «Кінцевий Кадр» -- "Start" and "End" Frames, що описані вище, у значення першого та останнього ключкадрованих кадрів Дії.
Устави Відтворення -- Playback Settings
Масштаб -- Scale
Розтягує смужку, ще один спосіб збільшення розміру -- «Протяги Смужки: Кінцевий Кадр» -- Strip Extents: End Frame, дивіться вище.
Повтор -- Repeat
Також розширює смужку, але зациклюючи її від першого ключкадру вперед.

Оцінення -- Evaluation¶

Reference

Panel:Properties region ‣ Animations ‣ Evaluation

Evaluation panel.

Це визначає міру впливу, який має смужка, та впродовж якого часу.

Анімований Вплив -- Animated Influence

Вмикає зміну міри впливу цієї смужки подібно до ключкадрованого значення. Якщо вплив не анімований, то смужки будуть втрачати силу прямолінійно під час перекриття.

Ці можуть бути значені у редакторах Аркуш Експозицій -- Dope Sheet або Редактор Графів -- Graph Editor під «Керівними Кривими НЛА -- NLA Control Curves та виглядають як група каналів. Вони з'являються перед усіма групами/F-Кривими для даного каналу.

Анімована Смужка
Те ж саме, що й «Анімований Вплив» -- Animated Influence, але з «Часом Смужки» -- Strip Time.
Час Циклу Смужки -- Cyclic Strip Time
Цикл анімованого часу у межах старту і кінця дії.

Модифікатори -- Modifiers¶

Reference

Panel:Properties region ‣ Modifiers ‣ Modifiers

Like its counterparts in graph and video editing, Modifiers can stack different combinations of effects for strips.

See F-Curve Modifiers.
