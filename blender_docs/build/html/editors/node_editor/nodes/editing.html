Editing¶

Трансформації -- Transform¶

Орієнтир

Mode:All Modes
Menu:Node ‣ Translate, Rotate, Resize
Hotkey:G, R, S

Move a single node by clicking and dragging it around. A node can be clicked almost anywhere to start dragging. Multiple nodes can be translated after pressing G.

Загалом, рекомендується впорядковувати ваші вузли у межах огляду так, щоб потоки даних проходили зліва направо та зверху вниз.

Розмір вузла можна змінити, перетягуючи його лівий або правий краї.

З'єднання Роз'ємів -- Connecting Sockets¶

Інтерактивно¶

LMB-клацніть на роз'ємі та тягніть мишу. Ви побачите лінію, що виходить з роз'єму: Це називається «зв'язок» або «з'єднання» -- link.

Продовжуючи перетягування, з'єднайте це зв'язок з увідним роз'ємом іншого вузла, а потім відпустіть LMB.

У той час, як з вивідного -- output роз'єму можуть виходити одночасно кілька з'єднань, однак, тільки одне з'єднання може бути під'єднане на увідний -- input роз'єм.

Для зміни розташування вивідних з'єднань вузла, щоб не додавати нове, утримування натисненою Ctrl при перетягу з вивідного роз'єму. Це працює як для одиничного, так і для кількох одночасно вивідних з'єднань.

Вузли, що не мають жодних з'єднань можуть бути вставлені в інше наявне з'єднання. Просто пересуньте вузол над з'єднанням та відпустіть кнопку миші, коли з'єднання забарвиться оранжевим.

Зробити Зв'язки -- Make Links F
Виділіть кілька вузлів з відкритими роз'ємами, а потім скористайтеся командою «Зробити Зв'язки» -- Make Links для створення з'єднань між ними. Використайте «Зробити Зв'язки» -- Make Links знову, якщо є і інші вузли, що можуть бути з'єднані.
Зробити та Замінити Зв'язки -- Make and Replace Links Shift-F
Make and Replace Links works similarly to Make Links, but it will replace existing links if any exist.

Роз'єднання Роз'ємів -- Disconnecting Sockets¶

Інтерактивно¶

Потягніть з'єднання з увідного роз'єму та відпустіть його, не з'єднуючи його більше ні з чим.

Розрізати Зв'язки -- Cut Links¶

Орієнтир

Mode:All Modes
Menu:Node ‣ Cut Links
Hotkey:Ctrl-LMB

Для розриву з'єднання між роз'ємами Ctrl-LMB-клацніть на порожньому місці біля з'єднання, яке ви хочете роз'єднати, та перетягніть мишу: Ви побачите маленьку іконку різака, що з'явиться у місці вказівника миші. Перетягніть мишу навхрест з'єднанню та відпустіть LMB.

Detach Links Alt-D, Alt-LMB-drag
Використовуйте команду «Від'єднати Зв'язки» -- Detach Links, щоб одночасно розрізати усі з'єднання, приєднані до виділених вузлів.

Дублювання -- Duplicate¶

Орієнтир

Mode:All Modes
Menu:Node ‣ Duplicate
Hotkey:Shift-D

Клацніть LMB або RMB на бажаному вузлів, далі натисніть Shift-D та рухайте мишу убік, щоб бачити дублікати виділеного вузла, що з'явиться під вказівником миші.

Ghi chú

When you duplicate a node, the new node will be positioned exactly on top of the node that was duplicated. If you leave it there (and it is quite easy to do so), you can not easily tell that there are two nodes there! When in doubt, grab a node and move it slightly to see if something is hidden underneath.

Видалення -- Delete¶

Видалення -- Delete X, Delete
Видаляє виділені вузол(ли).
Видалити з Перез'єднанням -- Delete with Reconnect Ctrl-X
Видаляє вузол(ли) без втрати з'єднань.

Приглушення -- Mute¶

Орієнтир

Mode:All Modes
Menu:Node ‣ Toggle Node Mute
Hotkey:M

Приглушення вузла вилучає участь цього вузла у вузловому дереві та робить усі з'єднання, що проходять через цей вузол, такими, які не вносять у дерево зміни. З'єднання будуть показуватися червоним, що вказуватиме їх проходження через приглушений вузол.

Показ/Схов -- Show/Hide¶

Hide H
Collapses the node so only the node header is visible. This can also be toggled by clicking the triangle at the top left of the node header.
Toggle Node Preview Shift-H
Shows/Hides a preview region on the node that displays the frame after that node's operation has been applied. This can also be toggled by clicking the material ball icon in the node header.
Toggle Hidden Node Sockets Ctrl-H
Collapses/Expands any input or output sockets that have no other nodes connected to them.
Перемкнути Опції Вузла -- Toggle Node Options
Shows/Hides all node properties.
Collapse and Hide Unused Sockets
Applies both the Toggle Hidden Node Sockets and Toggle Node Options operations.

Шари -- Layers¶

Ghi chú

The tools are only used in the Compositor.

Read Render Layers Ctrl-R
Read all the current scene's render layers from cache, as needed. This can be used to save RAM while rendering because the render layers do not have to be saved in RAM. This can also be used to recover some information from a failed render. For this to work, Save Buffers must be enabled.
Read Full Sample Layers Shift-R
Reads all the render layers using full sample only used in Blender Internal.
