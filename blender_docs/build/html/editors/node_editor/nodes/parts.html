Частини Вузла -- Node Parts¶

Усі вузли у Blender'і базуються на подібній побудові. Це стосується будь-якого типа вузла -- any type of node. Частини такої побудови включають Титр -- Title, Роз'єми -- Sockets, Передогляд -- Preview та інше.

Титр -- Title¶

Титр -- Title показує ім'я/тип вузла. Він може бути заміщений зміненим значенням устави «Позначка» -- Label на панелі «Вузол» -- Node в регіоні Властивості -- Properties Region N. З лівої сторони від титру знаходиться перемикач згортання, що може використовуватися для згортання вузла, аналогічним скороченням для цього є H.

Вигляд згорнутого вузла.

Роз'єми -- Sockets¶

Роз'єми -- Sockets уводять та виводять значення в та із вузла. Вони мають вигляд маленьких забарвлених кружків з обох боків вузла. Невикористані, невжиті роз'єми можуть бути сховані за допомогою Ctrl-H. Є дві функції роз'ємів; «увід» -- inputs та «вивід» -- outputs.

Кожен роз'єм забарвлений колір-кодовано, залежно від типу даних, які через нього подаються.

Color (yellow)
Вказує, що потребується інформація про колір для уводу або буде виводитися з вузла. Це може включати або ні канал альфа-прозорості.
Numeric (gray)
Indicates numeric value's information. It can either be a single numerical value or a so-called "value map". (You can think of a value map as a gray-scale map where the different amount of bright/dark reflects the value for each point). If a single value is used as an input for a "value map" socket, all points of the map are set to this same value. Common use: Alpha maps and value options for a node.
Vector (blue)
Вказує на інформацію про вектор, координату або нормаль.
Shader (green)
Used for shaders in Cycles.

Уводи -- Inputs¶

The Inputs are located on bottom left side of the node, and provide the data the node needs to perform its function. Each input socket, except for the green shader input, when disconnected, has a default value which can be edited via a color, numeric, or vector interface input. In the screenshot of the node above, the second color option is set by a color interface input.

Виводи -- Outputs¶

Виводи або вивідні роз'єми -- Outputs розміщуються з правої сторони вузла та з'єднання з них сполучаються далі з наступними у вузловому дереві вузлами.

Властивості -- Properties¶

Багато з вузлів мають устави, які можуть впливати на спосіб, яким вузол обробляє дані між уводами і виводами. Устави вузла розміщуються нижче виводів чи вивідних роз'ємів та над уводами чи увідними роз'ємами.

Приклад устав або контролерів властивостей на вузлі «Ключ Колірності» -- Chroma Key.

Передогляд -- Preview¶

На деяких з вузлів це показує передогляд зображення, яке генерується вузлом на основі вивідних даних для певного каналу. Зазвичай, він показує дані забарвлення, кольорів.

The preview can be toggled using the icon on the very top right-hand corner of the node, next to the title.

Вигляд вузла без передогляду.
