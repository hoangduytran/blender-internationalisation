Вступ¶

Оглядач Файлів -- File Browser.

Використання¶

Оглядач Файлів -- File Browser використовується для всіх операцій, що здійснюються з файлами. Він має багато можливостей використання, хоча часто використовується для збереження або завантаження, відкриття.

Сюди входять, зокрема:

  • Opening and saving blend-files.
  • Importing/exporting other file formats.
  • Picking new locations for existing file paths (images, video, fonts...).
  • Перегляд всередині інших blend-файлів при використанні Пов'язаних Бібліотек -- Linked Libraries.

Ви можете також зберігати відкритим Оглядач Файлів, як і будь-який інший тип редактора, для проходження по файловій системі. У цьому випадку кнопки підтвердження/скасування будуть відсутні.

Головна мета цього -- можливість перетягу медіа-файлів:

  • Зображень у Редактор Відеорядів -- Video Sequence Editor (для задання фону або застосування, як матеріалових текстур).
  • Медіа-файлів у Редактор Відеорядів -- Video Sequence Editor.

З іншого боку, якщо Оглядач Файлів відкритий для дій з файлами (відкриття, збереження, імпорт тощо), то він з'являється максимізованим та очікує на завершення операції перед поверненням у попередню розставу екрану.

Заголовок -- Header¶

Кнопки з іконками навігації -- Navigation icon buttons

Засоби для навігації по файлах.

Стрілка Вліво -- Left Arrow Backspace
Перехід у попередню теку.
Стрілка Управо -- Right Arrow Shift-Backspace
Перед у наступну теку.
Стрілка Уверх -- Up Arrow P
Перехід у предківський каталог.
Зациклені по колу Стрілки -- Cycle Arrows R, NumpadPeriod
Оновлює поточну теку.
Створити Каталог -- Create Directory
Створює нову теку всередині поточної та запитує увести для неї ім'я, скорочення I.
Рекурсія -- Recursion

Кількість рівнів тек для показу одночасно.

  • Нема -- None (тільки поточна тека)
  • Blend File (всередині blend-файлів)
  • Один Рівень -- One Level
  • Два Рівні -- Two Levels
  • Три Рівні -- Three Levels
Тип показу -- Display type

Керує тим, як будуть показуватися файли у списку.

  • Короткий список -- Short list
  • Докладний список -- Detailed list
  • Мініатюри -- Thumbnails (показує передогляди -- previews)
Розмір Показу -- Display Size

Розмір мініатюр або ширина стовпців.

Крихітно -- Tiny, Дрібно -- Small, Нормально -- Normal, Велико -- Large

Сортування -- Sorting

Сортує файли за одним з наступних методів:

  • Алфавітно -- Alphabetically
  • За типом файлів -- By file type
  • За датою останньої правки -- By date of last edit
  • За розміром файлів -- By file size
Показати Сховане -- Show Hidden
Показує сховані файли (що починаються з .), скорочення H.
Фільтрування показу файлів -- File filtering
Тип Файлу -- File Type

Фільтрує файли за типом.

  • Теки -- Folders
  • blend-файли -- blend-files
  • Резерви blend-файлів
  • Файли Зображень -- Image files
  • Файли Фільмів -- Movie files
  • Файли Скриптів -- Script files
  • Файли Шрифтів -- Font files
  • Файли Звуків -- Sound files
  • Файли Текстів -- Text files
Тип Блоку Даних -- Data-Block Type
Фільтрування за типом блоку даних всередині blend-файлів.
Search field
Фільтрує файли за іменами.

Регіон Файлів -- File Region¶

Шлях Файлу -- File Path
Текстове поле для поточного шляху. Натискання Tab буде давати авто-завершення уводу наявного шляху. Якщо ви уведете не наявний шлях каталогу, то будете запрошені до створення такого нового каталогу.
Ім'я Файлу -- File Name
Текстове поле для уводу, редагування імені файлу та його розширення. Якщо фон забарвлюється у червоний, то файл з таким іменем вже існує у цій теці. Натискання Tab буде авто-завершувати наявні імена у поточному каталогу.
Приріст Імені файлу -- Increment Filename +, -
Додає/збільшує або вилучає/зменшує хвостовий номер в або із вашому імені файлу (використовується для зроблення версій файлу).
Підтвердження -- Confirm

The main button to Open Directory/File or Save (As) Return or double-click with LMB on the entry confirms with that file or data-block.

  • Shift-LMB -- Відкриває файл зовнішньо (задається у «Файл» -- File).
  • Alt-LMB -- Відкриває каталог зовнішньо (використовується файловий менеджер операційної системи).
Скасування -- Cancel
Cancels the Open or Save file selection and closes the File Browser Esc or by using the Back to Previous in the Info editor header.

Полиця Інструментів -- Tool Shelf¶

Регіон зліва показує різні шляхи знаходження файлів та деякі інші опції. Клацання LMB тут на одному із записів дасть перехід Оглядача Файлів у відповідну теку.

Система -- System¶

Панель «Система» містить список дисків, що доступні для переходу в них для знаходження там файлів.

Системні Закладки -- System Bookmarks¶

Закладки, що є загальними для певної операційної системи.

Закладки -- Bookmarks¶

A List View of shortcuts to folders, that you want to be able to access often without having to navigate to them in the File Browser.

Додання -- Add +
Ця кнопка додає поточну теку у такий список.

Останнє -- Recent¶

Це список тек, в які останньо здійснювалося входження. Ви можете керувати тим, скільки тек повинно показуватися у цьому списку за допомогою числової кнопки «Останні Файли» -- Recent Files на вкладці «Файл» -- File в Уподобаннях Користувача -- User Preferences.

Головний Регіон -- Main Region¶

Навігація -- Navigation¶

Входження у Каталог -- Entering a Directory
Одиничний клацок LMB на записі каталогу дає входження у нього.
Предківський Каталог -- Parent Directory Backspace, P
Дає перехід на один рівень в ієрархії каталогів уверх, у предківський каталог.

Клавіші зі стрілками -- Arrow Keys¶

Навігація по каталогам також можлива за допомогою клавіш зі стрілками у поєднанні з утримуваною натисненою Alt:

  • Перейти у Предківський каталог -- Go to Parent Alt-Up
  • Previous Directory Alt-Left
  • Наступний Каталог -- Next Directory Alt-Right

Вкидання файлу у Blender -- File Drop¶

You now can simply drag & drop files from your local file explorer into the Blender File Browser. This will relocate the File Browser to the directory of the dropped file and the file will be selected.

Виділення -- Selection¶

Виділ -- Select
Застосовуються як клацок LMB, так і RMB.
(Зне)виділити Усе -- (De)select All A
Перемикає виділення усіх файлів.
Натягування рамки -- Dragging
Натягування з утримуваною натисненою LMB рамки розпочинає виділ рамкою -- border selection.

Клавіші зі стрілками -- Arrow Keys¶

Також можливо виділяти/зневиділяти файли, «ходячи» по них за допомогою клавіш зі стрілками:

  • Просте натискання клавіш зі стрілками дасть в обраному напрямку дасть виділення наступного файлу та зневиділення всіх інших.
  • Утримування натисненою Shift при цьому буде додавати у виділ невиділені файли або знімати виділення з поточно виділених файлів (у послідовності має бути мінімум два файли).
  • Holding down Shift-Ctrl further selects/deselects all files in between.

Якщо жоден з файлів не виділено, то клавіша зі стрілкою дає перехід до першого або останнього файлу у цьому каталозі залежно від напрямку руху.

If you select a directory and hit Return, you will go into that directory (and highlighting 'parent' entry will bring you up one level).

Управління файлами -- File Management¶

Видалити Файли -- Delete Files Delete, X
Видаляє поточно виділені файли.
Перейменування -- Rename CTRL-LMB
Може використовуватися на записі файлу або каталогу для його перейменування.
