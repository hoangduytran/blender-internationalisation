Редактор Часолінія -- Timeline Editor¶

Редактор «Часолінія» -- Timeline, що ідентифікується іконкою з годинником, не є настільки редактором у розумінні цього слова, але більше використовується для перегляду інформації та керування відтворенням анімації.

Часолінія -- Timeline.

The Timeline is one of an animator's most useful tools as it can give a broad overview of a scene's animation. The Timeline can communicate the current time frame, either in frames or in seconds, where the keyframes are of the active object, the start and end frames of your animation, markers, etc.

Часолінія -- Timeline має Контролери Відтворника -- Player Controls для відтворення, зупинення анімації та пропуску частин анімації сцени.

Вона також має деякі знаряддя для Ключкадрів -- Keyframes, Наборів Ключування - Keying Sets та Міток -- Markers.

Головний огляд -- Main View¶

Головний регіон Часолінії -- Timeline показує кадри анімації з часом.

Timeline main view.

Регулювання головного огляду¶

Часолінія -- Timeline може бути панорамована перетягом вліво або вправо утримуваною натисненою MMB.

Ви можете зумувати (присувати/відсувати) Часолінію -- Timeline за допомогою Ctrl-MMB, коліща миші Wheel або клавіш NumpadMinus і NumpadPlus.

Курсор Часу -- Time Cursor¶

Курсор Часу -- Time Cursor -- це зелена лінія, що використовується для установлення та показу поточного часу кадру.

Курсор Часу -- Time Cursor.

Курсор Часу -- Time Cursor може бути установлено або пересунуто у нову позицію клацком або перетягом з утримуваною натисненою LMB по Часолінії.

Поточний кадр або секунда може показуватися біля Курсора Часу -- Time Cursor, що визначається уставами меню «Огляд» -- View.

Курсор Часу -- Time Cursor може пересуватися кроками по 1-му кадру за допомогою натискання Left чи Right, або кроками по 10 кадрів натисканням Shift-Up чи Shift-Down.

Діапазон Відтворення/Зображування, рендерингу -- Playback/Rendering Range¶

Стандартно, Діапазон Відтворення/Зображування, рендерингу -- Playback/Rendering Range (Кадр Старту -- Frame Start 1 до Кадр Кінця -- Frame End 200) підсвічується світлішим відтінком сірого кольору. Стартовий і кінцевий кадри можуть бути установлені у місці Курсора Часу -- Time Cursor натисканням S та E, відповідно. Діапазон Відтворення -- Playback Range може бути установлений також шляхом натиснення P далі нарисуванням рамки.

Ключкадри -- Keyframes¶

Для активного та виділених об'єктів ключкадри показуються жовтою лінією. Для Арматур -- Armatures прорисовуються ключкадри об'єктів та ключкадри поз кісток.

Може бути увімкнено «Лише Виділені Канали» -- Only Selected Channels. «Часолінія > Огляд > Лише Виділені Канали» -- Timeline ‣ View ‣ Only Selected Channels. Для Арматур -- Armatures це буде прорисовувати ключкадри об'єктів та ключкадри для активної та виділених поз кісток.

Markers Menu¶

Markers are used to denote frames with key points or significant events within an animation. Like with most animation editors, markers are shown at the bottom of the editor.

Markers in animation editor.

For descriptions of the different marker tools see Editing Markers.

Контролер Діапазону -- Range Control¶

Вжити Діапазон Передогляду -- Use Preview Range (іконка з годинником)
Це альтернативний діапазон, що використовується для передогляду анімацій. Це працює для відтворення в інтерфейсі користувача (КІ -- UI), це не працює для зображування, рендерингу анімації. Дивіться Діапазон Передогляду -- Preview Range.
Зблокувати Курсора Часу з Діапазоном Відтворення -- Lock Time Cursor to Playback Range (іконка із замком)
Це лімітує пересування Курсора Часу -- Time Cursor межами Діапазону Відтворення -- Playback Range.

Синхронізація Відтворення -- Synchronize Playback¶

3D View red FPS.

60:54.75

When you play an animation, the FPS is displayed at the top left of the 3D View. If the scene is detailed and playback is slower than the set Frame Rate (see Панель «Розміри» -- Dimensions Panel), these options are used to synchronize the playback.

Без Синхронізації -- No Sync
Без синхронізації, відтворюється кожен кадр.
Пропуск Кадрів -- Frame Dropping
Пропускає кадри, якщо відтворення є надто повільним. Це вмикає опцію «Пропуск Кадрів» -- Frame Dropping з меню «Відтворення» -- Playback Menu.
AV-синхронізація -- AV-sync
(Audio/Video Synchronization). Sync to audio clock, dropping frames if playback is slow. This enables AV-sync and Frame Dropping from the Playback Menu.

Контролер Ключкадрів -- Keyframe Control¶

Авто Ключкадр -- Auto Keyframe

Авто вставка Ключкадрів на Часолінії.

The record button (red dot) enables something called Auto Keyframe: It will add and/or replace existing keyframes for the active object when you transform it in the 3D View.

Наприклад, при увімкненні, спершу установіть Курсор Часу -- Time Cursor у бажаний кадр, далі пересуньте об'єкт у 3D Огляді або задайте нове значення для його властивості у КІ.

Коли ви це зробите (задасте нове значення для властивостей), то Blender додасть ключкадри у поточному кадрі для властивостей, які були трансформовані. Інші випадки використання -- запис шляху ходьби/польоту -- Fly/Walk Mode або запис навігації в огляді камери -- Lock Camera to View.

Авто Набір Ключування (червона іконка запису)
При увімкненні, «Авто Ключкадр» -- Auto Keyframe буде вставляти нові ключкадри для властивостей в активний Набір Ключування -- Keying Set.
Нашаровано -- Layered (іконка з двома ключами)
Adds a new NLA Track and strip for every loop/pass made over the animation to allow non-destructive tweaking.

Ghi chú

Зауважте, що «Авто Ключкадр» -- Auto Keyframe працює тільки для властивостей трансформування (об'єктів та кісткок) у 3D Оглядах (тобто, ви не можете використовувати його для анімування кольорів матеріалу у Редакторі Властивостей -- Properties editor...).

Тип Ключкадра -- Keyframe Type
Типи Ключкадрів при вставлянні.
Активний Набір Ключування -- Active Keying Set

Набори Ключування на Часолінії.

Keying Sets are a set of keyframe channels in one. They are made so the user can record multiple properties at the same time. With a keying set selected, when you insert a keyframe, Blender will add keyframes for the properties in the active Keying Set. There are some built-in keying sets, LocRotScale, and also custom keying sets. Custom keying sets can be defined in the panels Properties ‣ Scene ‣ Keying Sets + Active Keying Set.

Вставити Ключкадри (іконка з ключем)
Вставляє ключкадри на поточному кадрі для властивостей в активному Наборі Ключування -- Keying Set.
Видалити Ключкадри -- Delete Keyframes (іконка із закресленим ключем)
Видаляє ключкадри на поточному кадрі для властивостей в активному Наборі Ключування -- Keying Set.
