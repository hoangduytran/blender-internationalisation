Chọn Lựa -- Selecting¶

Selection tools are available in the Select Menu in the header, and the shortcuts listed below:

Trình Đơn -- Menu¶

Lựa Chọn theo Đường Khoanh Ranh Giới -- Border Select
Use the box lasso to select UV coordinates B. See Lựa Chọn theo Đường Khoanh Ranh Giới -- Border Select.
Khoanh Vùng Chọn cái Được Ghim Xuống -- Border Select Pinned
Use the box lasso to select only pinned UV coordinates Shift-B.
Lựa Chọn bằng Vòng Tròn -- Circle Select
See Lựa Chọn bằng Vòng Tròn -- Circle Select.
Select/Deselect All
Selects or de-selects all UV coordinates A.
Đảo Nghịch -- Inverse
Inverts the current selection Ctrl-I.
Select Pinned
Selects all pinned UVs Shift-P. See Pinning.
Chọn cái Kết Nối -- Select Linked
This operator selects all UVs that are connected to currently selected UVs Ctrl-L. This works similarly to the tools in 3D View.
More/Less Ctrl-NumpadPlus, Ctrl-NumpadMinus
Expands/Contracts the selection to/from the adjacent elements of the selection type.
Select Split Y
Cuts apart the selected UVs from the map. Only those UVs which belong to fully selected faces remain selected. As the name implies, this is particularly useful to unlink faces and move them elsewhere. The hotkey is analogous to the mesh Split tool.

Tiêu Đề -- Header¶

Sync Selection
Turning on the Sync Selection button causes selection of components in the 3D View to sync with their corresponding elements in the UV/Image editor. If off only the selected faces are displayed in the UV/Image editor. These two modes have very different results when transforming components in the UV/Image editor.

Chế Độ Chọn Lựa -- Selection Modes¶

Select Modes dependent on the Sync Selection.

Sync Selection Off¶

Điểm Đỉnh -- Vertex
Select individual vertices.
Cạnh -- Edge
Select edges.
Bề Mặt -- Face
Select faces.
Hải Đảo -- Island
Select contiguous groups of faces.
Chế Độ Lựa Chọn Dính Kết -- Sticky Selection Mode

This selector lets you enable automatic additional selection.

Cùng Điểm Đỉnh -- Shared Vertex
Selects UVs that share a mesh vertex, even if they are in different UV locations.
Cùng Vị Trí -- Shared Location
Selects UVs that are in the same UV location and share a mesh vertex.
Tắt -- Disabled
Disables Sticky Selection. When you move a UV in this mode, each face owns its own UVs, allowing them to be separated.

Sync Selection On¶

When selecting UVs or Edges, it behave like Shared Vertex mode above. When selecting Faces, it behaves as in Disabled Stick Selection above.

  • Điểm Đỉnh -- Vertex
  • Cạnh -- Edge
  • Bề Mặt -- Face
