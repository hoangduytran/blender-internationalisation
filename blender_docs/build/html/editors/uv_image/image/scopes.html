Аналіз або Охвати -- Scopes¶

Гістограма -- Histogram¶

This mode displays a graph showing the distribution of color information in the pixels of the currently displayed image. The X axis represents values of pixel, from 0 to 1 (or 0 to 255), while the Y axis represents the number of pixels in that tonal range. A predominantly dark image would have most of its information toward the left side of the graph.

Використовуйте цей режим для балансування тонального діапазону зображення. Добре збалансоване зображення повинно мати гарний плавний розподіл колірних значень.

Світність -- Luma
Показує світимість зображення.
RGB
Показує канали RGB, накладені один поверх одного.
R/G/B/A
Залежно від обраного вами каналу показує аналіз відповідного до нього зображення.
Показувати лінії -- Show line
Displays lines rather than filled shapes.

Безпрозорість Хвилеформи -- Waveform Opacity
Безпрозорість точок хвилеформи.
Режим Хвилеформи -- Waveform Mode

ToDo.

RGB канали показуються пліч-о-пліч.

Показує RGB канали накладеними, як «повноколірну» хвилеформу. Це корисно для градуювання кольорів.

Відбір Лінією -- Sample Line¶

Охват «Відбір Лінією» -- Sample Line подібний на Гістограму -- Histogram , але дозволяє вам отримувати відібрані дані із заданої лінії.

Відбір Лінією -- Sample Line
Використовується через рисування мишкою лінії, що вживається для зчитування даних відбору з неї.
