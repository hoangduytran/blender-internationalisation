Вступ¶

Заголовок -- Header¶

Меню «Зображення» -- Image Menu¶

Нове Зображення -- New Image
Створює нове Генероване -- Generated зображення.
Відкрити Зображення -- Open Image
Завантажує зображення з файлу.
Читати Шари Зображу -- Read Render-Layers
Read all the current scene's render layers from cache, as needed. This can be used to save RAM while rendering because the render layers do not have to be saved in RAM. This can also be used to recover some information from a fail render. For this to work, Save Buffers must be enabled.
Зберегти Усі Зображення -- Save All Images
ToDo.
Замінити Зображення -- Replace Images
Заміняє поточне зображення, при цьому зберігаючи посилання на UV карти, з обраного файлу.
Перезавантажити Зображення -- Reload Image
Перезавантажує зображення із зовнішнього файлу.
Зберегти Зображення -- Save Image
Зберігає зображення, якщо воно вже було з файлу, Alt-S.
Зберегти Зображення Як -- Save As Image
Зберігає (рендерене, зображене) зображення в окремий файл, F3, або дає можливість зберегти поточне збережене вже зображення під іншим іменем файлу.
Зберегти Копію -- Save a Copy
Використання команди «Зберегти Копію» -- Save as Copy збереже файл-копію з указаним іменем, але утримає відкритим оригінал у Редакторі UV/Зображень -- UV/Image editor.
Редагувати Зовні -- Edit Externally
Засіб Blender'а «Редагувати Зовні» -- Edit Externally відкриє зовнішній редактор зображень, що вказаний в Уподобаннях Користувача -- User Preferences, та завантажить це зображення у нього для редагування.
Інвертування -- Invert
Інвертувати Кольори Зображення -- Invert Image Colors
Інвертує кольори зображення.
Інвертувати Канал -- Invert Channel
Червоний -- Red, Зелений -- Green, Синій -- Blue, Альфа -- Alpha
Упакування -- Pack
Упакувати Зображення -- Pack Image
ToDo <2.60.
Упакувати як PNG -- Pack As PNG
Упаковує зображення всередині blend-файлу. Дивіться детальніше тут -- Упакування та розпакування даних.

Quan trọng

Рендерені, зображені зображення повинні бути збережені зовнішньо або упаковані.

Контролери -- Controls¶

Зображення -- Image

Меню блоку даних -- Data-block menu використовується для вибору зображень. Коли зображення було завантажене або створене у Редакторі UV/Зображень -- UV/Image editor, то панель «Зображення» -- Image з'являється у регіоні Властивості -- Properties region. Детальніше дивіться тут -- Устави Зображення -- Image Settings.

  • Результат Зображу -- Render Result
  • Вузол «Оглядач» -- Viewer Node
Пришпилити Зображення -- Pin Image
Показує поточне зображення, незалежно від виділеного об'єкта.

Багато-Шар -- Multi-Layer¶

Коли рендерене, зображене зображення показується у Редакторі UV/Зображень -- UV/Image Editor, то стають доступними нові меню.

Гніздо -- Slot
Ви можете зберегти послідовні зображі, рендери у буфер рендеру, зображу, вибравши нове гніздо -- slot перед рендерингом, зображуванням. Якщо зображення було рендерене, зображене у гнізді, то воно може переглядатися після вибору цього гнізда. Порожні гнізда показуються, як пусті сітки у Редакторі UV/Зображень -- UV/Image editor. Використовуйте J та Alt-J для циклічного переходу вперед і назад по збережених зображах, рендерах.
Шар Зображу -- Render Layer
Якщо ви використовуєте Шари Зображу -- Render Layers, то скористайтеся цим меню для вибору того, який шар показувати.
Прохід Зображу -- Render Pass
Якщо ви використовуєте Проходи Зображу -- Render Passes, скористайтеся цим меню для вибору того, який прохід показувати.

Канали -- Channels¶

Прорис Каналів -- Draw Channels

Ці радіокнопки визначають, які канали зображення показуються.

RGBA
Замінює прозорі пікселі шаховим фоном, що позначає альфа канал.
RGB
Рисує кольорове зображення без альфа каналу.
Альфа -- Alpha
Displays the Alpha channel a grayscale image. White areas are opaque, black areas have an alpha of 0.
Z-Буфер -- Z-Buffer
Показує глибину від камери від Старту Відсіку -- Clip Start до Кінця Відсіку -- Clip End, як визначено в уставах камери -- Camera settings.
Червоний -- Red, Зелений -- Green, Синій -- Blue
Single Color Channel visualized as a grayscale image.

Головний огляд -- Main View¶

Коли протягувати мишею з LMB по зображенню, то інформація про колір під вказівником миші показується у нижній частині головного огляду, а також показуються дані про позицію вказівника та колірні значення у колірному просторі color space RGBA, HSV та Luminance.
