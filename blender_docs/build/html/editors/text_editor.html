Редактор Тексту -- Text Editor¶

Blender має Редактор Тексту -- Text Editor серед своїх типів редакторів, що доступний з меню «Тип Редактора» -- Editor type або за скороченням Shift-F11.

Заголовок -- Header¶

The newly opened Text editor is gray and empty, with a very simple header (Fig. Заголовок Редактора Тексту.).

Заголовок Редактора Тексту.

Заголовок Редактора Тексту із завантаженим у ньому текстом.

Тип Редактора -- Editor type
Стандартна кнопка вибору редактора.
Меню -- Menus
Меню -- Menus редакторів.
Текст -- Text
A Data-block menu to select a text or to create a new one. After that the header will change (Fig. Заголовок Редактора Тексту із завантаженим у ньому текстом.).
Показ -- Show

Наступні три кнопки перемикають опції показу.

Номери рядків -- Line numbers, завертання тексту по словах -- word-wrap text, підсвітка синтаксису -- syntax highlighting

Run Script / Script Node Update
Виконує текст як скрипт Python, скорочення Alt-P. Дивіться Скрипт і Шаблони -- Script and Templates.
Реєстрування -- Register
Registers the current text data-block as a module on loading (the text name must end with '.py'). Read more about the registration of Python modules in API documentation.
Позначка -- Label
Ця Позначка показує, чи текст було збережено внутрішньо чи зовнішньо, а також чи є незбережені зміни у зовнішньому файлі.

Меню -- Menus¶

Огляд -- View
Кінець Файлу -- Bottom of File
Пересуває огляд та курсор у кінець тексту.
Початок Файлу -- Top of File
Пересуває огляд та курсор на початок тексту.
Текст -- Text
Створити Блок Тексту -- Create Text Block
Створює новий внутрішній текст.
Відкрити Блок Тексту -- Open Text Block
Дає змогу завантажити зовнішній текст за допомогою Оглядача Файлів -- File Browser, скорочення Alt-O.
Перезавантаження -- Reload
Перевідкриває (перезавантажує) поточний буфер (усі не-збережені модифікації втрачаються), скорочення Alt-R.
Збереження -- Save
Зберігає вже відкритий файл, скорочення Alt-S.
Зберегти Як -- Save As
Зберігає незбережений текст у новому текстовому файлі через Оглядач Файлів -- File Browser, скорочення Shift-Ctrl-Alt-S.
Зробити Внутрішнім -- Make Internal
Зберігає текст всередині цього blend-файлу.
Запуск Скрипту -- Run Script
Виконує текст як скрипт Python, скорочення Alt-P. Дивіться Скрипт і Шаблони -- Script and Templates.
Правка -- Edit
Вирізання -- Cut Ctrl-X
Вирізає позначений текст у текстовий буфер обміну.
Копіювання -- Copy Ctrl-C
Копіює позначений текст у текстовий буфер обміну.
Вставляння -- Paste Ctrl-V
Вставляє текст з буферу обміну у локацію курсора у Редакторі Тексту.
Дублювати Рядок -- Duplicate Line Ctrl-D
Дублює поточний рядок.
Пересунути рядок(ки) уверх -- Move line(s) up
Обмінює місцями поточний рядок з тим, що вище.
Пересунути рядок(ки) вниз -- Move line(s) down
Обмінює місцями поточний рядок з тим, що нижче.
Виділ -- Select
Виділити Рядок -- Select Line, Виділити Усе -- Select All.
Стрибання -- Jump
Показує спливний діалог «Стрибання» -- Jump, що дозволяє вам вибрати номер рядка, на який ви хочете перестрибнути.
Знаходження -- Find...
Показує панель «Знаходження» -- Find у Регіоні Властивості -- Properties Region.
Авто Завершення Тексту -- Text Auto Complete Ctrl-Spacebar
Показує вибірний список команд Python та відповідників слів, що вже використовувалися у тексті.
Текст у 3D Об'єкт -- Text To 3D Object
Один Об'єкт -- One Object, Один Об'єкт на Рядок -- One Object per line.
Формат -- Format
Відступ -- Indent
Додає відступ зліва до виділеного, скорочення Tab.
Зневідступ -- Unindent
Видаляє відступ зліва у виділенні, скорочення Shift-Tab.
Коментування -- Comment
Перетворює виділені рядки у коментар Python.
Розкоментування -- Uncomment
Розкоментовує виділені рядки.
Конвертувати Пропуски -- Convert Whitespace
Конвертує пропуски відступів у символи табуляції або пробілів.
Шаблон -- Template

Дивіться Скрипт і Шаблони -- Script and Templates.

Python, OpenShading Language

Скрипт і Шаблони -- Script and Templates¶

The most notable keystroke is Alt-P which makes the content of the buffer being parsed by the internal Python interpreter built into Blender. Before going on it is worth noticing that Blender comes with a fully functional Python interpreter built-in, and with a lots of Blender-specific modules, as described in the Написання скриптів та Розширення Blender'а section.

Редактор Тексту -- Text Editor має відтепер також деякі окремі скрипти на Python, які додають деякі засоби написання, як оглядач класів/функцій/змінних, завершення... Ви можете мати доступ до них через меню «Шаблон» -- Template у заголовку.

Головний огляд -- Main View¶

Уведення з клавіатури продукує текст у текстовому буфері. Як і звичайно, натискання, перетягування та відпускання LMB виділяє текст.

Mẹo

Використання Редактор Тексту

Редактор Тексту зручний також, коли ви хочете спільно використовувати ваші blend-файли з іншими людьми. Редактор Тексту може використовуватися для написання тексту README, що пояснює вміст вашого blend-файлу. Переконайтеся, що зберегли його видимим при збереженні!
