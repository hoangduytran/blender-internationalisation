Засоби виділення¶

Виділення вказуванням -- Point Selection¶

Орієнтир

Mode:All Modes
Hotkey:RMB and Shift-RMB

Найпростіша форма виділення об'єкта полягає у клацку RMB по ньому. Для додання у виділ ще об'єктів використовується Shift-RMB. Якщо об'єкти перекриваються в огляді, то ви можете скористатися Alt-RMB для циклічного переходу між можливими виборами (тільки у Режимі Об'єкта -- Object Mode). Якщо ви хочете додати у виділ цим способом, то використовуйте скорочення Shift-Alt-RMB. Для активування об'єкта, що вже виділений клацніть на ньому Shift-RMB.

Для «зневиділення» -- deselect активного об'єкта, зняття з нього активності, клацніть Shift-RMB один раз по ньому, а для зняття взагалі виділення клацніть Shift-RMB двічі на будь-якому виділеному неактивному об'єкті. Зауважте, що це працює тільки, якщо жоден інший об'єкт не знаходиться під вказівником миші. Інакше, він просто додасться у виділ. Схоже не існує вирішення цієї вади.

Also you can use Ctrl-RMB and Shift-Ctrl-RMB shortcut to select the object by its center point rather then its contents.

Виділ Рамкою -- Border Select¶

Орієнтир

Mode:Object Mode and Edit Mode
Menu:Select ‣ Border Select
Hotkey:B

To activate the tool, press B. With Border Select you draw a rectangle while holding down LMB. Any object that lies even partially within this rectangle becomes selected. If any object that was last active appears in the selection it will become active.

For deselecting objects, use MMB or Border Select again while holding Shift or Alt. To cancel the selection use RMB.

Gợi ý

Виділ Рамкою додає у попереднє виділення, а тому, щоб виділити лише те, що потрапляє всередину прямокутника, спочатку знімайте виділення зі всього у сцені за допомогою A.

Виділ Кругом -- Circle Select¶

Орієнтир

Mode:Object Mode and Edit Mode
Menu:Select ‣ Circle Select
Hotkey:C

Circle Select C is used by moving with dotted circle through objects with LMB. You can select any object by touching of circle area. It is possible to dynamically change the diameter of circle by scrolling Wheel or with NumpadPlus and NumpadMinus as seen in pictures below. Deselection is under the same principle -- MMB. To cancel the selection use RMB or key Esc or Return.

Виділ Ласо -- Lasso Select¶

Орієнтир

Mode:Object Mode and Edit Mode
Hotkey:Ctrl-LMB

Виділ Ласо або виділ арканом виконується рисуванням пунктирної лінії навколо вершин або початку об'єктів у Режимі Об'єкта -- Object Mode. Для його здійснення утримуйте натисненими Ctrl-LMB та просто обрисовуйте точки об'єктів, які ви хочете виділити.

Lasso select adds to the previous selection. For deselection, use Shift-Ctrl-LMB.

(Зне)виділити Усе -- (De)select All¶

Орієнтир

Mode:All Modes
Menu:Select ‣ (De)select All, Select ‣ Inverse
Hotkey:A

Якщо наразі щось є виділеним, то це знімає виділення з усього -- зневиділяє усе. Інакше, це перемикає між станами виділення і невиділення для кожного видимого об'єкта.

Дія -- Action

Виділення -- Select, Зневиділення -- Deselect, Інвертування -- Invert, Перемикання -- Toggle

Інверсія -- Inverse Ctrl-I
Виділяє усі об'єкти, що не були виділені, та одночасно зневиділяє усі об'єкти, що є поточно виділеними.

Випадково -- Random¶

Орієнтир

Mode:Object and Edit Modes
Menu:Select ‣ Random

Випадково виділяє невиділені об'єкти на основі відсоткової ймовірності на поточно активних шарах. Стає доступним на Полиці Інструментів -- Tool Shelf числове поле відсотка виділення на панелі оператора виділу. Важливо зазначити, що цей відсоток представляє ймовірність виділу невиділеного об'єкта, а не відсоток кількості об'єктів, що будуть виділені.

Дзеркалення -- Mirror¶

Орієнтир

Mode:All Modes
Menu:Select ‣ Mirror
Hotkey:Shift-Ctrl-M

Виділяє дзеркальні об'єкти відносно виділеного об'єкта на основі їх імен, наприклад, "sword.L" та "sword.R".

Виділити Усе за Шаром -- Select All by Layer¶

Орієнтир

Mode:Object Mode
Menu:Select ‣ Select All by Layer

Панель Виділення Усе за Шаром.

Ця опція дозволяє виділяти кожен одиничний об'єкт, що належить даному шару. Виділені об'єкти стають видимими.

Відповідність -- Match
Тип відповідності для виділення.
Розшир -- Extend
Вмикає додання об'єктів у поточний виділ, а не заміну поточного виділення.
Шар -- Layer
Шар, на якому знаходяться об'єкти.

Mẹo

Виділення об'єктів

Чим використовувати команду «Виділити Усе за Шаром» -- Select All by Layer, можливо ефективніше буде зробити усі потрібні шари видимими та скористатися командою виділення усього за A. Цей метод також дозволяє зневиділяти об'єкти.

Виділити Усе за Типом -- Select All by Type¶

Орієнтир

Mode:Object Mode
Menu:Select ‣ Select All by Type

За допомогою цього засобу стає можливим виділяти об'єкти певного типу за раз.

Такими типами є Сіть -- Mesh, Крива -- Curve, Поверхня -- Surface, Мета -- Meta, Шрифт -- Font, Арматура -- Armature, Решітка -- Lattice, Порожняк -- Empty, Камера -- Camera, Лампа -- Lamp, Гучномовець -- Speaker.

Виділити Камеру -- Select Camera¶

Орієнтир

Mode:Object Mode
Menu:Select ‣ Select Camera

Виділяє активну камеру, це може використовуватися у складній сцені для легшого знаходження активної камери.

Виділити Більше/Менше -- Select More/Less¶

Орієнтир

Mode:Object Mode
Menu:Select ‣ More/Less
Hotkey:Ctrl-NumpadPlus, Ctrl-NumpadMinus

Збільшує або зменшує виділ на основі ієрархії.

Більше -- More
Expand the selection to the immediate parent and children of the selected objects.
Менше -- Less
Contrast the selection, deselect objects at the boundaries of parent/child relationships.
Предок -- Parent
Deselects the currently selected objects and selects their immediate parents.
Нащадок -- Child
Deselects the currently selected objects and selects their immediate children.
Розшир до Предка -- Extend Parent
Extends the selection to the immediate parents of the currently selected objects.
Розшир до Нащадка -- Extend Child
Extends the selection to the immediate children of the currently selected objects.

Виділити Груповане -- Select Grouped¶

Орієнтир

Mode:Object Mode
Menu:Select ‣ Grouped
Hotkey:Shift-G

Існує два способи організування об'єктів один відносно одного. Першим є приріднення -- parenting, а другим -- просте групування -- grouping. Виділення цих відношень може бути корисним для митця на основі членства елементів до відповідних родин або груп. Select Grouped використовує активний об'єкт, як базу для виділення всіх інших.

Нащадки -- Children
Виділяє усі ієрархічні нащадки активного об'єкта.
Найближчі Нащадки -- Immediate Children
Виділяє усі прямі нащадки активного об'єкта.
Предок -- Parent
Виділяє предка цього об'єкта, якщо він має хоч одного.
Спільнопредкові -- Siblings
Виділяє об'єкти-нащадки, що мають спільного предка, яким є активним об'єкт. Це може також бути використаним для виділення усіх об'єктів рівня «корінь» (об'єкти без жодних предків).
Тип -- Type
Виділяє об'єкти, що мають однаковий з активним об'єктом тип.
Шар -- Layer
Об'єкти, що знаходяться принаймні на одному спільному шарі.
Група -- Group
Objects that are part of a group (rendered green with the default theme) will be selected if they are in one of the groups that the active object is in. If the active object belongs to more than one group, a list will pop up so that you can choose which group to select.
Гачки Об'єкта -- Object Hooks
Кожен гачок, що належить активному об'єкту.
Прохід -- Pass
Виділяє об'єкти, що призначені на однаковий прохід зображу -- render pass.
Колір -- Color
Виділяє об'єкти, що мають однаковий колір об'єкта -- Object Color.
Властивості -- Properties
Виділяє об'єкти, що мають однакові властивості ігрового рушія -- Game Engine Properties.
Набір Ключування -- Keying Set
Виділяє об'єкти, включені в активний набір ключування -- Keying Set.
Тип Лампи -- Lamp Type
Виділяє лампи з однаковим типом.
Індекс Проходу -- Pass Index
Виділяє об'єкти, що мають однаковий індекс проходу.

Виділити Пов’язане -- Select Linked¶

Орієнтир

Mode:Object Mode
Menu:Select ‣ Linked
Hotkey:Shift-L

Виділяє усі об'єкти, які мають блок даних, спільний для них та активного об'єкта. Select Linked використовує активний об'єкт, як базу для виділення усіх інших.

Дані Об'єкта -- Object Data
Виділяє кожен об'єкт, що пов'язаний з однаковим блоком Дані Об'єкта -- Object Data, тобто такий блок даних, що визначає тип (сіть, крива тощо) та побудову (установні елементи, як вершини, керувальні точки, а також те, де вони знаходяться у просторі) для об'єкта.
Матеріал -- Material
Виділяє кожен об'єкт, що пов'язаний з однаковим блоком даних матеріалу.
Текстура -- Texture
Виділяє кожен об'єкт, що пов'язаний з однаковим блоком даних текстури.
Дубль-Група -- DupliGroup
Виділяє усі об'єкти, що використовують однакову Групу -- Group для дублювання.
Система Частинок -- Particle System
Виділяє усі об'єкти, що використовують однакову Систему Частинок -- Particle System.
Бібліотека -- Library
Selects all objects that are in the same Library.
Бібліотека (Дані Об’єкта) -- Library (Object Data)
Виділяє усі об'єкти, що знаходяться в однаковій Бібліотеці -- Library та обмежені блоком даних об'єкта -- object data.

Виділ за Взірцем -- Select Pattern¶

Орієнтир

Mode:Object Mode
Menu:Select ‣ Pattern

Selects all objects whose name matches a given pattern. Supported wild-cards: * matches everything, ? matches any single character, [abc] matches characters in "abc", and [!abc] match any character not in "abc". As an example *house* matches any name that contains "house", while floor* matches any name starting with "floor".

Чутливо Регістру -- Case Sensitive
Визначає, чи відповідність має бути чутлива до регістру символів (заголовні або рядкові літери).
Розшир -- Extend
При увімкненні стяга «Розшир» -- Extend виділ буде розширюватися замість того, щоб генеруватися заново.
