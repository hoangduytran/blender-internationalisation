Дубль-Група -- DupliGroup¶

Орієнтир

Mode:Object Mode
Panel:Object ‣ Duplication ‣ Group

«Дублювання Групи» -- Duplication Group або «Дубль-Група» -- DupliGroup -- дозволяє вам створювати примірник групи для кожного примірника іншого об'єкта. «Дубль-Групи» -- DupliGroups можуть містити анімації, об'єкти з фізичними імітаціями та навіть інші вкладені «Дубль-Групи» -- DupliGroups.

Базове використання¶

Створення групи:
  • Виділіть об'єкти, що мають бути груповані.
  • Create a new group Object ‣ Group ‣ Create New Group.
  • Rename your group in the Properties editor: Object ‣ Groups.
Створення нового Примірника Групи:
  • Виберіть у меню «Додання > Примірник Групи» -- Add ‣ Group Instance
Зміна Примірника Групи наявних об'єктів:
  • In the Properties editor: Object ‣ Duplication, enable Group.
  • Виберіть ім'я вашої новоствореної групи.

At this point, an instance of the group and an empty object will appear. You can duplicate the empty, and the DupliGroup settings will be preserved for each empty. This way, you can get multiple copies of linked data very easily.

Дубль-Група та динамічне пов'язання¶

Дивіться про прилучення і пов'язання -- Appending and Linking, щоб розуміти, як динамічно пов'язувати дані з іншого blend-файлу у поточний файл. Ви можете динамічно пов'язувати групи з одного blend-файлу в іншому. Коли ви це зробите, то пов'язана група не відображається будь-де у вашій сцені, допоки ви не створите об'єкт, що керує місцем, де має відображатися примірник такої групи.

Quan trọng

Прозорість Матеріалу -- Material Transparency не буде показуватися при примірникуванні дубль-груп; це відоме обмеження оглядвікна Blender'а.

Зроблення дубль-групи реальним об'єктом¶

Say you want to make further edits on a DupliGroup instance: Simply select your DupliGroup and press Shift-Ctrl-A to convert the DupliGroup into regular objects that can be transformed and animated normally.

Ghi chú

Зауважте, що якщо Дубль-Група була пов'язана із зовнішнього файлу, то Дані Об’єкта -- Object Data (сіть, матеріали, текстури, трансформації) будуть також все ще пов'язані з оригінальної групи. Однак, різні відношення «предок-нащадок» не переносяться.
