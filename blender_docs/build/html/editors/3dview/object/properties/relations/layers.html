Шари -- Layers¶

Орієнтир

Mode:Object Mode
Panel:Object ‣ Relations
Menu:Object ‣ Move to Layer...
Hotkey:M

3D сцени часто стають швидко більш заплутаними при зростанні їх складності. Інколи митець також потребує точного контролю, де знаходяться окремі об'єкти, і не хоче, щоб при освітленні одного об'єкта світло потрапляло на наближені до нього інші об'єкти. Для цього і через інші причини, про які мова нижче, об'єкти можуть поміщатися на один чи кілька «шарів» -- "layers". Використовуючи шари для об'єктів чи об'єктні шари, ви можете:

  • Вибірково показувати об'єкти з певних шарів у вашому 3D Огляді, вмикаючи показ цих шарів у заголовку 3D Огляду -- 3D View. Це дозволяє вам пришвидшити перерисовування інтерфейсу, зменшувати плутанину у віртуальному світі та допомагає вам покращити ваш хід роботи.
  • Керувати тим, які освітлювачі будуть освітлювати об'єкт -- which lights illuminate an object, визначаючи освітлення тільки об'єктів на шарах, на яких самі освітлювачі знаходяться.
  • Керувати тим, які сили впливатимуть і на які системи частинок -- particle systems, оскільки частинки підпадають під вплив тільки сил і впливів на одному з ними шарі.
  • Керувати, які шари будуть зображуватися, рендеритися (і звідси, які об'єкти), а також які властивості/канали будуть доступні для компонування за допомогою шарів зображу -- render layers.

Арматури можуть також ставати дуже складним з різними типами костей, контролерів, обчислювачів, власних форм тощо. Оскільки, арматури, зазвичай, розміщуються близько одна від одної, то це може швидко призводити до плутанини. Тому, Blender також надає шари тільки для арматур. Шари для арматур або арматурні шари дуже подібні до об'єктних шарів, у тому, що ви можете ділити арматуру (оснастку) на шари та показувати тільки ті її шари, з якими поточно бажаєте працювати.

Робота з шарами¶

3D шари відрізняються від шарів, про які ви можливо знаєте з програм 2D графіки, оскільки вони не мають жодного впливу на порядок прорису та призначені головним чином (за винятком спеціальних функцій, перелічених вище) для організування вашої сцени.

При зображуванні, рендерингу, Blender буде зображувати, рендерити тільки обрані шари. Якщо усі ваші освітлювачі знаходяться на шарі, що не обраний для зображування, то ви не побачите зовсім нічого на вашому зображі, рендері, за винятком об'єктів, освітлених освітленням оточення.

Групи -- Groups та Приріднення -- Parents -- це інші способи логічного групування відносних наборів об'єктів.

Viewing Layers¶

Blender provides twenty layers whose visibility can be toggled with the small unlabeled buttons in the header (see 3D View layer buttons). To select a single layer, click the appropriate button with LMB; to select more than one, use Shift-LMB -- doing this on an already active layer will deselect it.

Кнопки шарів в 3D Огляді -- 3D View.

Для вибору показу шарів за допомогою клавіатури, натискайте від 1 до 0 (на основній частині клавіатури) для шарів від 1 до 10 (верхній ряд кнопок) та від Alt-1 до Alt-0 -- для шарів від 11 до 20 (нижній ряд кнопок). Використовуйте Shift так само разом з цими скороченнями для вибору показу або схову шарів.

You can select or deselect all Scene Layer buttons at once by pressing Backtick.

Locking to the Scene¶

Стандартно, кнопка зблокування, що знаходиться зразу справа від кнопок шарів, увімкнена. Це означає, що зміни на показуваних шарах впливають на всі інші 3D Огляди, зблоковані з цієї сценою. Детальніше дивіться тут -- navigating the 3D View options page.

Знаходження на кількох шарах одночасно -- Multiple Layers¶

Об'єкт може існувати на кількох шарах одночасно. Наприклад, лампа, що освітлює об'єкти лише на їх спільному шарі, може «бути» на шарах 1, 2 та 3. Об'єкт на шарах 3 та 4 може бути освітленим, а об'єкт на шарах 4 та 5 не буде освітлюватися. Існує багато ситуацій, в яких грають роль впливи залежно від шару, особливо для освітлювачів та частинок.

Moving Objects between Layers¶

Вибір шару.

Для переміщення виділених об'єктів на інший шар натисніть M та далі у спливному меню виберіть шар, на який ви хочете її перемістити. Об'єкти можуть також бути переміщені на кілька шарів одночасно за раз. Для переміщення об'єкта на кілька шарів зараз утримуйте натисненою Shift при клацанні.

Вибір у вкладці «Об'єкт» -- Object.

Інший спосіб огляду або зміни шару виділеного об'єкта -- панель «Відношення» -- Relations у вкладці «Об'єкт» -- Object Редактора Властивостей.

Шани на панелі «Відношення» -- Relations у вкладці «Об'єкт» -- Object.

Ви можете використовувати так само кнопки шарів на панелі «Відношення» -- Relations і таким же чином об'єкт може переміщатися на одночасно кілька шарів за допомогою клацку Shift-LMB.
