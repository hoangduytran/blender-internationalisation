Властивості трансформування -- Transform Properties¶

Кожен об'єкт зберігає при собі значення його позиції, орієнтації та масштабу. Це може потребуватися для маніпулювання ним шляхом числового уводу, скидання трансформацій або їх застосування.

Панель «Трансформації» -- Transform Panel¶

Орієнтир

Mode:Object Mode
Panel:Object ‣ Transform
Panel:Properties region ‣ Transform

Панель «Трансформації» -- Transform у регіоні «Властивості» -- Properties region дозволяє вам переглядати та вручну/числовим уводом керувати позицією, обертанням та іншими властивостями об'єкта у Режимі Об'єкта -- Object Mode. У Режимі Правки -- Edit Mode вона в основному дозволяє вам уводити точні координати для вершини або серединної позиції для груп вершин (включаючи ребро/грань). Кожен тип об'єкта має різний набір опцій в його панелі «Трансформації» -- Transform у Режимі Правки -- Edit Mode, про що дивіться відповідні описи у главі про моделювання -- Modeling.

Опції у Режимі Об'єкта¶

Використовуйте цю панель для редагування або перегляду властивостей трансформування об'єкта, таких як позиція, обертання та/або масштабування. Поля цих устав змінюють початок об'єкта та далі впливають на всі його вершини та грані.

Властивості трансформування на панелі «Transform».

Локація -- Location
Локація початку об'єкта у глобальних координатах.
Обертання -- Rotation
Орієнтація об'єкта відносно глобальних осей та його власного початку.
Режим Обертання -- Rotation Mode

Метод для обчислення обертань, додаткова інформація про це може бути знайдена тут -- here.

Ейлер -- Euler
Держаки маніпулятора вирівнюються по осям Ейлера -- Euler, дозволяючи вам бачити дискретні XYZ осі, на яких базується обертання за Ейлером, а також можливе шарнірне зблокування -- gimbal lock.
Вісь-Кут -- Axis Angle
Координати X, Y та Z визначаються точкою відносно початку об'єкта. Ця точка та початок визначають вісь, навколо якої значення W визначає обертання.
Кватерніон -- Quaternion
X, Y, Z та W відповідають компонентам Кватерніона -- Quaternion.
Масштаб -- Scale
The object's relative scale along the local axis (e.g. the Scale X value represents the scale along the local X axis). Each object (cube, sphere, etc.), when created, has a scale of one Blender unit in each local direction. To make the object bigger or smaller, you scale it in the desired axis.
Розміри -- Dimensions
The size of the object's bounding box (aligned with the local axes -- think of a cardboard box just big enough to hold the object).

Блокування властивостей трансформування¶

Перемикач із замком справа від устави при його вмиканні блокує відповідну трансформацію, тобто відповідне трансформаційне значення не може бути змінене жодною інтерактивною операцією трансформування. Однак, змінити це значення все ще можна не-інтерактивними операціями, наприклад, уведенням відповідного числа або за допомогою Python.

Наприклад, якщо ви заблокували властивість «Локація X» -- Location X, то ви не можете використовувати 3D маніпулятор для пересування об'єкта вздовж глобальної осі X. Однак, ви все ще можете пересунути його, увівши потрібне число для значення властивості Location X. Вважайте цю функцію блокування, як жорстке обмеження, доступне для зміни тільки за допомогою панелі.

Для блокування властивості клацніть на ікону і замком справа від трансформаційної кнопки з числом. Ця кнопка з числом є розблокованою, якщо іконка показує відкритий замок, та заблокованою, коли на іконці показується закритий замок.

Дельта Трансформацій -- Delta Transforms¶

Дельта Трансформацій -- Delta Transforms -- це просто трансформації, що застосовуються поверх трансформацій, описаних вище. Вони можуть бути знайдені у «Редактор Властивостей > Об'єкт > Дельта Трансформацій» -- Properties Editor ‣ Object ‣ Delta Transforms.

Використання¶

Дельта Трансформацій особливо корисні в анімаціях. Наприклад, ви можете анімувати об'єкт за допомогою «звичайних» трансформацій, а потім пересунути його за допомогою Дельта Трансформацій.
