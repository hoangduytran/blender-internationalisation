Опорна Точка -- Pivot Point¶

Орієнтир

Mode:Object Mode and Edit Mode
Header:Опорна Точка -- Pivot Point

Режими Опорної Точки.

При обертанні або масштабуванні об'єкта(ів) або групи вершин/ребер/граней ви можливо схочете зсунути опорну точку -- pivot point, щоб легше маніпулювати об'єктами. Використовуючи цей вибірник у заголовку 3D Огляду -- 3D View, ви можете змінювати локацію опорної точки.
