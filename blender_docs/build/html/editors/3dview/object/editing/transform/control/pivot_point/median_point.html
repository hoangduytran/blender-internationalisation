Серединна Точка -- Median Point¶

Орієнтир

Mode:Object Mode and Edit Mode
Header: Pivot Point ‣ Median Point
Hotkey:Ctrl-Comma

The Median Point can be considered to be broadly similar to the concept of Center of Gravity (COG). If we assume that every element (object, face, vertex, etc.) of the selection has the same mass, the median point would sit at the point of equilibrium for the selection (the COG).

У Режимі Об'єкта¶

In Object Mode, Blender only considers the Object origins when determining the median point. This can lead to some counter-intuitive results. In the Fig. Серединні точки у Режимі Об'єкта. below, you can see that the median point is between the Object origins and can be nowhere near the Objects' mesh (geometric center).

Серединні точки у Режимі Об'єкта.

У Режимі Правки¶

У Режимі Правки -- Edit Mode серединна точка визначається на основі частини виділу, що включає більшість виділених елементів. Наприклад, на ілюстрації Серединні точки у Режимі Правки., коли є два куби з однаковою кількістю вершин, то серединна точка буде лежати безпосередньо між цими двома кубами. Однак, якщо ви підподілити один з кубів кілька разів, що означатиме, що він має більше вершин, то ви можете побачити, що медіанна точка зсунулася у регіон з більшістю вершин.
