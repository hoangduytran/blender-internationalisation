Інструменти трансформацій -- Transform Tools¶

Рандомізація Трансформацій -- Randomize Transform¶

Орієнтир

Mode:Object Mode and Edit Mode
Panel:Tool Shelf ‣ Tools ‣ Mesh Tools ‣ Deform: Randomize
Menu:Object ‣ Transform ‣ Randomize Transform, Mesh ‣ Transform ‣ Randomize

Опції рандомізації трансформацій.

Цей засіб дозволяє вам застосувати випадкове трансформування.

Режим Об'єкта -- Object Mode¶

У Режимі Об'єкта -- Object Mode він рандомізує, застосовує випадкові значення пересуву, оберту та масштабу до об'єкта або кількох об'єктів. При його використання до кількох об'єктів кожен з них отримує своє власне базисне значення та отримує різні результати трансформацій від решти.

Опції¶

Базис Випадковості -- Random Seed
Базис випадковості -- це зсув для випадкового трансформування. Різний базис буде давати інший новий результат.
Трансформації Дельта -- Transform Delta
Рандомізує, випадково застосовує значення Дельта Трансформацій -- Delta Transform замість значень звичайних трансформацій.
Рандомізація Локації -- Randomize Location
Рандомізує, випадково застосовує значення пересуву для зміни локації.
Локація -- Location
Максимальні відстані, на які об'єкти можуть рухатися вздовж кожної з осей.
Рандомізація Оберту -- Randomize Rotation
Рандомізує, випадково застосовує значення оберту.
Обертання -- Rotation
Максимальний кут, на який об'єкти можуть бути обернуті навколо кожної осі.
Рандомізація Масштабу -- Randomize Scale
Рандомізує, випадково застосовує значення масштабу.
Рівномірний Масштаб -- Scale Even
Використовує однаковий масштаб по кожній осі.
Масштаб -- Scale
Максимальне значення, на яке може змінитися масштаб по кожній осі.

Режим Правки -- Edit Mode¶

Інструмент «Рандомізація» -- Randomize у Режим Правки -- Edit Mode дозволяє вам зміщувати вершини сіті вздовж їх нормалей.

Опції¶

Величина -- Amount
Відстань зміщення.
Однорідно -- Uniform
Додає випадковий зсув у значення величини.
Нормаль -- Normal
Додає випадковий зсув у зміщення нормалі.
Базис Випадковості -- Random Seed
Базис випадковості -- це зсув для випадкового трансформування. Різний базис буде давати інший новий результат.

Вирівняти Об'єкти -- Align Objects¶

Орієнтир

Mode:Режим Об'єкта -- Object Mode
Menu:Object ‣ Transform ‣ Align Objects

The Align tool is used to align multiple selected objects so they line up on a specified axis.

Опції¶

Висока Якість -- High Quality
Використовує більш точну математику для кращого визначення локацій для об'єктів. У випадку додатного або від'ємного вирівняння габаритних коробок, якщо один чи кілька виділених об'єктів мають будь-які трансформації обертання (або трансформації дельта обертання), рекомендується вмикати опцію «Висока Якість» -- High Quality, щоб їх загальна габаритна коробка обчислювалася з точністю по усіх трьох глобальних осях.
Режим Вирівняння -- Align Mode

Контролер «Режим Вирівняння» -- Align Mode визначає, за якою своєю частиною об'єкти будуть вирівнюватися:

Центри -- Centers
Центри об'єктів.
Додатні Боки -- Positive Sides / Від'ємні Боки -- Negative Sides
Додатні або від'ємні боки (на глобальних осях) їх відповідних габаритних коробок.
Відносно До -- Relative To

Контролер «Відносно До» -- Relative To дозволяє нам вибрати елемент, відносно якого буде здійснюватися вирівняння об'єктів:

Активне -- Active
Активний об'єкт.
Виділ -- Selection
Серединна точка виділення.
3D Курсор -- 3D Cursor
The current position of the 3D Cursor.
Початок Сцени -- Scene Origin
Глобальний початок.
Вирівняння -- Align X, Y, Z
Дає змогу вибрати вісь, за якою будуть вирівняні об'єкти.
