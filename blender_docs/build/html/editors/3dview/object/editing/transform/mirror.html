Дзеркалення -- Mirror¶

Орієнтир

Mode:Object and Edit Modes
Menu:Object/Mesh ‣ Mirror
Hotkey:Ctrl-M

Дзеркалення виділеного об'єкта або сіті створює розвернуту версію виділеного. Позиція віддзеркаленої версії виділення визначається Опорною Точкою -- Pivot Point. Загальне використання дзеркалення -- це моделювання половини об'єкта, дублювання її та використання трансформації дзеркалення для отримання розвернутої версії для завершення моделі.

Ghi chú

Дзеркальні дублікати можуть також створюватися за допомогою модифікатора «Дзеркалення» -- Mirror Modifier.

Mirroring a selection.

Використання¶

Для дзеркалення виділеного вздовж певної глобальної осі натискайте: Ctrl-M, а потім X, Y або Z. Ілюстрація «Дзеркалення виділеного» -- Mirroring a Selection показує результати цієї дії після того, як елемент було спочатку дубльовано.

In Mesh mode, you can mirror the selection on the currently selected Transform Orientations by pressing the appropriate axis key a second time. For example, if the Transform Orientation is set to Normal, pressing: Ctrl-M, followed by X and then X again will mirror the selection along the X axis of the Normal Orientation.

Mirror operator panel.

Ви можете альтернативно утримувати натисненою MMB для інтерактивного дзеркалення об'єкта, рухаючи мишу у напрямку осі дзеркалення.
