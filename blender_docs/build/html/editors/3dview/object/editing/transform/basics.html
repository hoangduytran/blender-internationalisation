Базові Трансформації -- Basic Transformations¶

Захоп/Рух -- Grab/Move¶

Орієнтир

Mode:Object Mode, Edit Mode, and Pose Mode
Panel:Tool Shelf ‣ Tools ‣ Transform ‣ Translate
Menu:Object type ‣ Transform ‣ Grab/Move
Hotkey:G

У Режимі Об'єкта -- Object Mode опція захопу/руху дозволяє вам пересувати (рухати) об'єкти. Пересування означає зміна локації, місця об'єктів. Вона також дозволяє вам пересувати будь-які елементи, з яких складається об'єкт, у межах 3D простору активного 3D Огляду.

Захоп/Рух -- Grab/Move працює подібним чином і в інших редакторах, як Редактор Вузлів -- Node Editor, Редактор Графів -- Graph Editor, Редактор UV/Зображень -- UV/Image Editor, Відеорядник -- Sequencer тощо.

Натискання G активує трансформаційний режим Захопу/Руху -- "Grab/Move". Виділені об'єкт або елемент далі вільно рухаються відповідно до вказівника миші та камери.

Ви можете також рухати об'єкт, клацнувши та утримуючи натисненою RMB на об'єкті і рухаючи мишею. Для підтвердження цієї дії клацайте LMB.

Поки активний Захоп/Рух -- Grab/Move величина зміни координат по X, Y та Z показується у нижньому лівому куті Редактора 3D Огляду -- 3D View editor.

Показ величини зміни пересування.

Mẹo

Рухання об'єкта у Режимі Об'єкта -- Object Mode змінює початок -- origin цього об'єкта. Рухання вершин/ребер/граней об'єкта у Режимі Правки -- Edit Mode не змінює початок цього об'єкта.

Оберт -- Rotate¶

Орієнтир

Mode:Object and Edit Modes
Panel:Tool Shelf ‣ Tools ‣ Transform ‣ Rotate
Menu:Object/Mesh/Curve/Surface ‣ Transform ‣ Rotate
Hotkey:R

Rotation is also known as a spin, twist, orbit, pivot, revolve, or roll and involves changing the orientation of elements (vertices, edge, face, object, etc.) around one or more axes or the Pivot Point.

Кут обертання показується інтерактивно у нижньому лівому кутку Редактора 3D Огляду -- 3D View editor.

Показ величини зміни обертання.

Трекбольне Обертання -- Trackball Rotation¶

A free rotation mode. Press R-R to enable Trackball rotation.

Масштаб -- Scale¶

Орієнтир

Mode:Object and Edit Modes
Panel:Tool Shelf ‣ Tools ‣ Transform ‣ Scale
Menu:Object/Mesh/Curve/Surface ‣ Transform ‣ Scale
Hotkey:S

Масштабування -- scaling означає зміну пропорцій об'єкта. Натискання S даватиме перехід у трансформаційний режим масштабування -- Scale, в якому виділені елементи масштабуються всередину або назовні відповідно до локації вказівника миші. Масштаб елемента збільшується, якщо вказівник миші рухається від Опорної Точки -- Pivot Point, та зменшується, якщо цей вказівник рухається до неї. Якщо вказівник миші перетинає Опорну Точку -- Pivot Point та рухається на протилежній стороні від неї, то масштабування продовжується у від'ємному напрямку та перевертає елемент.

Базові використання масштабування. Зліва направо ця ілюстрація показує: оригінальний Об'єкт, зменшено масштабований Об'єкт, збільшено масштабований Об'єкт та перевернуто масштабований Об'єкт.

Величина масштабування буде показуватися у нижньому лівому кутку Редактор 3D Огляду -- 3D View editor.

Показ величини зміни масштабування.

Загальні опції¶

Існує кілька способів трансформувати елемент, зокрема:

  • Клавіатурне скорочення.
  • Меню у заголовку або панель «Трансформації» -- Transform на Полиці Інструментів -- Tool Shelf.
  • Віджет «Маніпулятор 3D Трансформацій» -- 3D Transform Manipulator.
  • Панель «Трансформації» -- Transform panel в регіоні «Властивості» редакторів -- Properties region або у вкладці «Об'єкт» -- Object Редактора Властивостей.

Підтвердження і Скасування -- Confirm and Cancel¶

LMB click to accept changes. This behavior can be changed globally by activating Release Confirms in the User Preferences, so that a single RMB drag can be used to move and confirm.

Для скасування трансформації вживається натискання RMB або Esc, натомість. Це буде скидати об'єкт або елемент в його початковий, оригінальний стан.

Xem thêm

Використання комбінацій скорочень дає вам більше контролю за вашими трансформаціями. Дивіться про керування трансформаціями тут -- Transform Control.

Робхід¶

Використання клавіатурних скорочень¶

  1. Клацок RMB використовується для виділення елемента, який ви хочете трансформувати.
  2. Одиничне натиснення G, або R, чи S дає перехід у відповідний режим трансформування.
  3. Трансформовані елементи рухаються мишею.
  4. Клацок LMB приймає зміни.
