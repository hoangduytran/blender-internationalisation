Активний Елемент -- Active Element¶

Орієнтир

Mode:Object Mode and Edit Mode
Header: Pivot Point ‣ Active Element
Hotkey:Alt-Period

The active element can be an object, vertex, edge or a face. The active element is the last one to be selected and will be shown in a lighter orange color when in Object Mode and white when in Edit Mode. With Active Element as Pivot set to active, all transformations will occur relative to the active element.

Показ активного елемента у Режимі Об'єкта, де такий активний елемент (куб) підсвічено світло-оранжевим контуром.

Активні елементи, як вершини, ребра та грані, у Режимі Правки показуються білим.

У Режимі Об'єкта¶

When in Object Mode, rotation and scaling happen around the origin of the active object. This is shown by the figure to the below where the active object (the cube) remains in the same location (note its position relative to the 3D cursor) while the other objects rotate and scale in relation to the active element.

Обертання та масштабування відносно куба, як активного елемента.

У Режимі Правки¶

Використання активного елемента як опорної точки у Режимі Правки -- Edit Mode може здатися більш складним, але всі можливі трансформування слідують таким кільком правилам:

  • The pivot point is always at the median of the active element.
  • Трансформації відбуваються шляхом трансформування вершин виділених елемента(ів). Якщо невиділений елемент має одну чи кілька вершин з виділеним елементом, то цей невиділений елемент також буде отримувати деяку міру трансформування.

Давайте розглянемо наступні приклади: у кожному випадку ми побачимо дію цих двох правил.

Single Selection¶

Коли виділяється один єдиний елемент, то він автоматично стає активним. На ілюстрації нижче ви можете побачити, що коли трансформуються його вершини, і як наслідок, будь-який елемент, що має спільними одну або кілька вершин з активним елементом, також буде трансформуватися.

Режим Правки та виділений тільки один елемент.

Давайте розгляньмо кожен випадок:

  • Грані -- face мають свої опорні точки там, де показується точка їх виділу, і це там само, де є середина між їх вершинами.
  • Ребра -- edge мають свої опорні точки посередині між їх вершинами.
  • Одинична Вершина - vertex не має вимірів зовсім, а тому трансформування інших вершин не буде (за винятком пересування -- translation, який не визначається опорною точкою).

Multiple Selection¶

When multiple elements are selected they all transform. The pivot points stay in the same place as what we have described above. In the image below, the selected elements have been rotated.

Режим Правки та виділи кількох.

  • Для Граней трансформування відбувається навколо точки виділу активної грані.
  • Ребра збережуть свою поведінку, обертаючись навколо середини активного ребра.
  • На цей раз трансформація видима і для Вершин: опорна точка знаходиться у місці активної Вершини. Усі інші вершини трансформуються відносно неї.
