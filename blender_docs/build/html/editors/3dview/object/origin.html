Початок Об'єкта -- Object Origin¶

Кожен об'єкт має свою точку початку -- origin. Локація цієї точки визначає, де цей об'єкт буде розміщено у 3D просторі. Коли об'єкт виділяється, то показується маленький круг, що позначає цю його точку початку. Локація точки початку є важливою при пересуванні, обертанні або масштабуванні об'єкта. Дивіться детальніше тут -- Pivot Points.

Колір початку визначається відповідно до умовностей забарвлення станів -- Кольори Стану, а світло-синє забарвлення позначає бібліотечний зв'язок як на джерельному, так і пов'язаному об'єктах. Його розмір може бути змінено у вкладці «Інтерфейс» -- Interface tab в Уподобаннях Користувача -- User Preferences.

Задати Початок -- Set Origin¶

Орієнтир

Mode:Object and Edit Modes
Panel:Tool Shelf ‣ Tools ‣ Edit ‣ Set Origin
Menu:Object ‣ Transform
Hotkey:Shift-Ctrl-Alt-C

Початок та геометрія об'єкта можуть бути переміщені одне відносно одного та відносно 3D Курсора.

Тип -- Type
Геометрія до Початку -- Geometry to Origin
Переміщує модель до початку і таким чином початок об'єкта буде також у центрі цього об'єкта.
Початок до Геометрії -- Origin to Geometry
Переміщує початок у центр об'єкта і таким чином початок об'єкта буде також у центрі об'єкта.
Початок до 3D Курсора -- Origin to 3D Cursor
Переміщує початок моделі у позицію 3D курсора.
Початок до Центру Маси -- Origin to Center of Mass
Переміщає початок у обчислений центр маси моделі (припускається, що сіть має однорідну, рівномірну щільність).
Центр -- Center
Центр як серединна точка -- Серединний Центр -- Median Center або як центр габаритної коробки -- Центр між Межами -- Bounds Center
