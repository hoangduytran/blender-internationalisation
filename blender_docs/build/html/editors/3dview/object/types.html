Типи об'єктів¶

Орієнтир

Mode:Object Mode
Panel:Tool Shelf ‣ Create ‣ Add Primitive
Menu:Add
Hotkey:Shift-A

Нові об'єкти можуть бути створені за допомогою меню «Додання» -- Add заголовку 3D Огляду -- 3D View.

Сіть -- Mesh
Сіті -- Meshes -- це об'єкти, що скомпоновані з полігональних граней -- face, полігональних ребер -- edge та/або полігональних вершин -- vertex, та можуть редагуватися всіляко за допомогою інструментів Blender'а для правки сітей. Дивіться детальніше тут -- Mesh Primitives.
Крива -- Curve
Криві -- Curves -- це математично визначені об'єкти, якими можна маніпулювати за допомогою керувальних держаків (ручок) -- control handles або керувальних точок -- control points (аналог вершин - vertex), для управління їх довжиною і кривиною. Дивіться детальніше тут -- Curves Primitives.
Поверхня -- Surface
Поверхні -- Surfaces -- це латки, клаптики, якими також маніпулюють за допомогою керувальних точок -- control points. Вони корисні для зроблення простих округлих форм та органічних пейзажів. Дивіться детальніше тут -- Surfaces Primitives.
Метакуля -- Metaball
Meta Objects (or Metaballs) are objects formed by a mathematical function (with no control points or vertices) defining the 3D volume in which the object exists. Meta Objects have a liquid-like quality, where when two or more Metaballs are brought together, they merge by smoothly rounding out the connection, appearing as one unified object. See Meta Primitives.
Текст -- Text
Text objects create a two-dimensional representation of a string of characters.
Арматура -- Armature
Арматури -- Armatures використовуються для оснащення rigging 3D моделей, яке дає змогу їх ставити у пози та анімувати.
Решітка -- Lattice
Решітки -- Lattices є не-зображоздатними, не-рендерздатними каркасами, що зазвичай використовуються для отримання засобів додаткового керування іншими об'єктами за допомогою модифікатора «Решітка» -- Lattice Modifier.
Порожняк -- Empty
Порожняки -- Empties -- це нульові об'єкти, що є простими візуальними трансформаційними вузлами, які не рендеряться, відзображуються. Вони корисні для керування позиціями або переміщенням інших об'єктів.
Гучномовець -- Speaker
Speaker brings a source of sound to the scene.
Камера -- Camera
Це віртуальна камера, що використовується для визначення того, що буде показане на рендері, зображі. Дивіться про камери -- Blender Internal та Cycles.
Лампа -- Lamp
Це об'єкти, що використовуються для розміщення джерел освітлення у сцені. Дивіться про лампи -- Blender Internal та Cycles.
Силове Поле -- Force Field
Силові Поля -- Force Fields використовуються в імітаціях фізичних взаємодій. Вони дають імітації зовнішніх сил, що створюють рух, та представлені у 3D Огляді маленькими керувальними об'єктами.
Примірник Групи -- Group Instance
Дозволяє вам здійснювати вибір зі списку наявних груп об'єктів. При виборі цього пункту меню буде створено об'єкт Порожняк -- Empty з примірником вибраної групи (дублювання груп має бути активним). Детальніше дивіться тут -- Дубль-Група -- DupliGroup.

Загальні опції¶

Ви можете змінювати опції об'єкта на Панелі Оператора -- Operator panel зразу ж після його створення:

Тип -- Type
Деякі з об'єктів дозволяють вам змінювати їх тип після створення за допомогою цього вибірника.
Радіус -- Radius / Розмір -- Size
Задає стартовий розмір.
Вирівняти за Оглядом -- Align to View
Стандартно об'єкти при створенні вирівнюються за осями глобального простору. Ця опція вмикає обертання новоствореного об'єкта, щоб він став вирівняним за простором огляду.
Локація -- Location
Об'єкти стандартно при створенні розміщаються у позиції 3D Курсора. Ці значення дозволяють вам поміщати об'єкт в іншу позицію при його створенні.
Обертання -- Rotation
Ці значення дозволяють вам обертати новостворений об'єкт для зміни стандартного його оберту при створенні.
