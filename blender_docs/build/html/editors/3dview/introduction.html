Вступ¶

3D Огляд -- 3D View використовується для взаємодії з 3D сценою для різних цілей, таких як моделювання, анімація, малювання текстур тощо.

Заголовок¶

Заголовок містить різні меню та контролери, що базуються на поточному режимі -- mode.

Заголовок 3D Огляду.

Меню¶

Огляд -- View
Це меню пропонує інструменти для навігації -- navigate у 3D просторі.
Виділ -- Select
Містить інструменти для виділення -- selecting об'єктів.
Додання -- Add
Надає список різних типів об'єктів -- objects types, які можуть бути додані у сцену.
Об'єкт -- Object
Це меню показується у режимі об'єкта -- Object Mode. Воно містить інструменти для маніпуляції об'єктами -- objects. У режимі правки -- Edit Mode воно змінюється на відповідне до типу виділеного об'єкта меню з інструментами правки, редагування -- editing tools.

Контролери¶

Режим -- Mode

The 3D View has several modes used for editing different kinds of data:

  • Режим Об’єкта -- Object Mode
  • Режим Правки -- Edit Mode
  • Режим Ліпки -- Sculpt Mode
  • Малювання Вершин -- Vertex Paint
  • Малювання Вагомостей -- Weight Paint
  • Малювання Текстур -- Texture Paint
  • Правка Частинок -- Particle Editing
  • Режим Пози -- Pose Mode
  • Edit Strokes Mode
Відтінення Оглядвікна -- Viewport Shading
Allows you to change the way objects are displayed in the viewport. Read more about the different shading modes.
Опорна Точка -- Pivot Point

Використовується для зміни орієнтирної точки (або опорної точки -- pivot point), що використовується багатьма інструментами маніпуляції сіттю.

Read more about Pivot Points.

Маніпулятор Трансформацій -- Transform Manipulator

Ці зручні вибірники дозволяють вам обертати або рухати об'єкти, захоплюючи за їх (клацком вашої миші) контролери та рухаючи мишу по певній осі.

Read more about Transform Manipulators.

Шар -- Layer
Віджет Шарів описано тут -- Layers.
Зблокування зі Сценою -- Lock to Scene

Стандартно, кнопка «зблокування», що знаходиться справа від кнопок шарів, увімкнена. Це означає, що у цьому огляді, активні шари й камера є такими для усієї сцени (і такими, що використовуються під час рендерингу, зображування). Звідси, усі 3D Огляди, зблоковані таким чином зі сценою, будуть мати спільними однакові активні шари й камеру. Коли ви змінюєте їх в одному огляді, то всі інші зблоковані будуть негайно відображати ці зміни.

Проте, якщо ви вимкнете цю кнопку «зблокування» у певному огляді, то ви зможете визначати різні активні шари та камеру у цьому огляді. Це може бути корисно, якщо ви не хочете мати ваші робочі області (огляди) у безладі по всій сцені, а мати допоміжний повноцінний огляд (який буде розблокований з цією сценою і для нього, наприклад, усі шари будуть показуватися). Або, щоб мати кілька оглядів з різними активними камерами. Пам'ятайте, що ви можете скористатися Ctrl-Numpad0, щоб зробити з активного об'єкта умовну активну камеру.

Read more about Scenes.

Пропорційна Правка -- Proportional Edit
Детальніше тут -- Proportional Edit.
Прилип -- Snap
Керує інструментами прилипання -- snapping tools, що допомагають при трансформуванні та моделюванні об'єктів.
OpenGL Зображ -- OpenGL Render
Ці кнопки запуску зображування, рендерингу OpenGL-версії 3D Огляду. Дивіться більше тут -- OpenGL Rendering.

Полиця Інструментів -- Tool Shelf¶

The Tool Shelf is a context-sensitive region containing tools depending on the current mode (for example, modeling tools in Edit Mode, brush tools in Sculpt Mode...).

Регіон «Властивості» -- Properties Region¶

Регіон «Властивості» -- Properties Region містить властиовсті активного об'єкта та виділених об'єктів (такі як їх локації), а також властивості самого редактора:
