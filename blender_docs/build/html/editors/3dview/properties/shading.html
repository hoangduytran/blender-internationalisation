Відтінення -- Shading¶

Відтінення Оглядвікна -- Viewport Shading¶

Орієнтир

Header:Viewport Shading

Меню «Відтінення Оглядвікна» -- Viewport Shading.

Відтінення визначає спосіб прорису та освітлення об'єктів у 3D Огляді -- 3D View.

Габаритна Коробка -- Bounding Box
Показує тільки прямокутну коробку, що окреслює розмір та форму об'єкта.
Каркас -- Wireframe
Об'єкти відображаються, як сіть ліній, що представляє ребра, краї граней та поверхонь.
Суцільно -- Solid
Стандартний режим прорису, що використовує суцільно забарвлені поверхні та просте освітлення.
Текстуровано -- Textured
Shows meshes with an image applied using the mesh's active UV map. For Cycles materials, the image is the last one selected in the Node Editor. For other render engines, the UV map's applied face texture will be shown.
Матеріал -- Material
Швидкий приблизний показ застосованого матеріалу.
Зображено -- Rendered
Точне представлення за допомогою вибраного Рушія Зображу -- Render Engine та освітлення з використанням видимих у сцені освітлювачів.

За винятком режиму «Зображено» -- Rendered, всі інші режим відтінення не залежать від джерел освітлення у сцені. Натомість, вони використовують просте стандартне освітлення, що регулюється контролерами Solid OpenGL Lights на вкладці «Система» -- System в редакторі Уподобань Користувача -- User Preferences.

Це відтінення оглядвікна керує виглядом усіх об'єктів у сцені, але може бути заміщене для окремих об'єктів на панелі «Показ» -- Display у вкладці «Об'єкт» -- Object Редактора Властивостей -- Properties Editor.

Клавіатурні скорочення

Перемикає між режимами прорису «Каркас» -- Wireframe та «Суцільно» -- Solid. Z
Перемикає між поточним та режимом прорису «Зображено» -- Rendered. Shift-Z
Перемикає між режимами прорису «Суцільно» -- Solid та «Текстуровано» -- Textured. Alt-Z

Панель «Відтінення» -- Shading Panel¶

Орієнтир

Panel:Properties region ‣ Shading

3D View Shading panel.

Панель «Відтінення» у регіоні «Властивості» -- Properties 3D Огляду забезпечує додаткове керування виглядом об'єктів у 3D Огляді.

Текстуровано Суцільно
Display assigned face textures in the Solid shading mode. (not available in the Cycles Renderer).
Безвідтінково -- Shadeless
Тільки режим «Текстуровано» -- Textured -- прорисовує текстури без відтінення. Найбільш поширено використовується при малюванні текстур.
Перед-матеріал -- Matcap
Перед-матеріали -- "Matcaps" -- це зображення, розкладені відповідно до нормалей поверхні. Вони забезпечують швидкий спосіб визначення видимих властивостей матеріалу для моделювання і ліплення. Оскільки зображування, рендеринг Перед-матеріалу нехтує кодом відтінювача матеріалу, то це дуже швидкий варіант рендерингу, зображування. Обраний Перед-матеріал задається для кожного 3D Огляду. Таким чином, ви можете мати одночасно кілька оглядів з прорисом різними Перед-матеріалами. (Суцільне відтінення оглядвікна тільки).
Відкид Тилу -- Backface Culling
Показує тільки передню сторону граней. Використовуйте це для знаходження неправильно перевернутих граней, особливо при експорті у програми, що використовують тільки односторонній прорис.
Схований Каркас -- Hidden Wire

Show only front-facing wireframes. This is useful for a retopology workflow. (Mesh, Edit Mode only).

Tip: Optimally this could be combined with the X-Ray display setting.

Глибина Різкості -- Depth of Field
Імітує ефект розмиву поза фокусом камери у 3D Огляді. Це доступно тільки в огляді камери. Для керування цим ефектом використовуйте опції на вкладці у редакторі властивостей Properties Tab для активної камери: Фокусна Відстань -- Focal Length, Розмір Датчика -- Sensor Size, Об'єкт Фокусу -- Focus Object чи Відстань Фокусу -- Focus Distance та Д-число Оглядвікна -- Viewport F-stop.
Загорода Оточення -- Ambient Occlusion

Improves the realism of the viewport image by simulating the darkening effect that occurs in crevices and corners. This is done by ray casting in screen space. Typically such effects are rendered at higher quality, but this is a quick real-time preview which can help when modeling or sculpting.

Сила -- Strength
Цей фактор безпосередньо множиться на обчислений колір ефекту, а тому збільшення цього значення дає стильніший ефект.
Відстань -- Distance
Максимум відстані у просторі світу, на які буде обчислюватися цей ефект. Тобто, як далеко поширюється вплив кутків.
Затухання -- Attenuation
Наскільки сильно цей ефект затухає з відстанню. Збільшення цього робить спадання цього ефекту по поверхні з відстанню. Використовуйте це для позбавлення деякий стрічкових артефактів.
Вибірки -- Samples
Кількість вибірок, що використовується для цього ефекту. Низьке значення продукує ефект зернистості, однак фактична кількість використовується через піднесення до квадрата, а тому вживайте високі значення з обережністю.
Колір -- Color
Колір цього ефекту, може бути модифікованим для отримання різних відчуттів, від освітлення оточення до бруду/іржі.
