Панель «Замісник/Часкод» -- Proxy/Timecode Panel¶

Після того, як ви вибрали параметри Замісника/Часкоду -- Proxy/Timecode, вам необхідно скористатися командою «Кліп > Замісник > Перебудувати Замісник та Індекси Часкоду» -- Clip ‣ Proxy ‣ Rebuild Proxy and Timecode indices для генерування замісного кліпу та щоби він став доступним після того, як Blender зробить його.

Замісник -- Proxy¶

Замісник -- proxy -- це маленьке зображення (швидше за завантаження), що відповідає основному зображенню. Коли ви перебудовуєте замісники, Blender обчислює малі зображення (подібні до мініатюр) для великих зображень і це може зайняти деякий час. Після їх обчислення функції редагування, як прокрутка здійснюються швидше, але дають низько-роздільний результат. Переконайтеся, що ви вимкнули замісники перед здійсненням фінального зображування, рендерингу.

Побудувати на Оригіналі -- Build Original
Використовується для визначення того, з якими роздільностями повинні бути вибудувані замісні зображення.
Побудувати на Неспотвореному -- Build Undistorted
Будує зображення з неспотворених оригінальних зображень за розмірами, заданими вище. Це допомагає забезпечити швидше відтворення неспотвореного фільмоматеріалу.
Якість -- Quality
Визначає якість JPEG зображень, що використовуються для замісників.
Власна Тека Замісника -- Proxy Custom Directory
Стандартно, усі генеровані зображення замісника зберігаються у теці з оригінальним фільмоматеріалом -- <path of original footage>/BL_proxy/<clip name>, але ця локація може бути задана і вручну за допомогою цієї опції.
Перебудувати Замісник -- Rebuild Proxy
Повторно генерує зображення замісника для усіх розмірів, заданих вище, та повторно генерує усі часкоди, які можуть використовуватися пізніше.
Часкод -- Timecode
Дивіться Часкод -- Timecode.
Розмір Зображу Замісника -- Proxy Render Size
визначає, яка роздільність зображення замісника використовується для показу. Якщо увімкнено опцію «Зображ Неспотворено» -- Render Undistorted, то використовуються зображення, створенні з неспотворених кадрів. Якщо ще немає генерованих замісників, то розмір зображу задається як «Без замісника повний зображ» -- "No proxy, full render", а «Зображ Неспотворено» -- Render Undistorted увімкнено, то знеспотворення здійснюється автоматично при прорисі кадру.

Часкод -- Timecode¶

Коли ви працюєте з фільмоматеріалом, прямо скопійованим з камери без його перед-обробки, то це може давати купу артефактів, здебільшого через шукання певного кадру у послідовності. Це відбувається тому, що такий фільмоматеріал зазвичай не має правильних значень частоти кадрів в їх заголовках. Тому, Blender при обчислення позиції потрібного кадру у потоці працює неточно та може давати бродячий результат. Існує два можливих способи запобігти цьому:

  1. Preprocess your video with e.g. MEncoder to repair the file header and insert the correct keyframes.
  2. Скористатися опцією «Замісник/Часкод» -- Proxy/Timecode у Blender'і.

Опції¶

Часкод -- Timecode
Timecode to use on the selected movie strip.

Підтримуються наступні часкоди:

  • Без Часкоду -- No TC in use -- не використовується жоден часкод
  • Хід Запису -- Record Run
  • Вільний Хід -- Free Run
  • Вільний Хід (дата запису) -- Free Run (rec date)
  • Хід Запису Без Пропусків -- Record Run No Gaps

Ghi chú

Хід Запису -- Record Run -- це часкод, який зазвичай є найкращим варіантом для використання, але, якщо файл кліпу повністю пошкоджений, то «Хід Запису Без Пропусків» -- Record Run No Gaps буде єдиним шансом отримати прийнятний результат.
