Панель «2D Стабілізація» -- 2D Stabilization Panel¶

There is one extra panel which is available in reconstruction mode -- 2D Stabilization Panel. The purpose of this feature is to smooth out jerky camera handling on existing real-world footage. To activate the 2D stabilizer, you need to set the toggle in the panel, and additionally you need to enable Display Stabilization in the Display panel. Then you'll need to set up some tracking points to detect the image movements.

Панель «2D Стабілізація» використовується для визначення даних, що вживаються для 2D стабілізації знімка. Кілька опцій доступній на цій панелі: ви можете додати список стежок для визначення бічні зсуви зображення та інший список стежок для визначення рухів відхилення та зумування. Базуючись на усередненому внеску цих стежок обчислюється компенсування руху та застосовується у кожному кадрі.

Коли фільмоматеріал включає рухи панорамування та проїзду, стабілізатор тяжіє до виштовхування зображення поза видиму область. Це може бути компенсовано анімуванням параметрів для навмисних, «очікуваних» рухів камери.

Ghi chú

Для активації 2D стабілізатора вам необхідно увімкнути перемикач у заголовку цієї панелі та додатково треба увімкнути опцію «Показати Стабілізоване» на панелі «Показ» -- Display.

Опції¶

2D Stabilization panel.

Анкерний Кадр -- Anchor Frame
Орієнтувальна точка для закріплення стабілізації: інші кадри будуть коригуватися відносно цієї позиції кадрів. Ви можливо схочете вибрати номер кадру, де ваш головний суб'єкт вирізняється оптимальним способом.
Тип Стабілізації -- Stabilization Type
Обертання -- Rotation
In addition to location, stabilizes detected rotation around the rotation pivot point, which is the weighted average of all location tracking points.
Масштаб -- Scale
Компенсує будь-які зміни масштабу відносно центру обертання.
Стежки Для Стабілізації -- Tracks For Stabilization
Локація -- Location
Список стежок, які будуть використовуватися для компенсації стрибків камери, або рухів локації.
Обертання -- Rotation / Масштаб -- Scale
Список стежок, що будуть використовуватися для компенсації відхилів камери або змін масштабу.
Автомасштаб -- Autoscale

Знаходить найменший фактор масштабування, який, коли застосується до фільмоматеріалу, дозволить ліквідувати усі порожні чорні границі біля границь зображення.

Макс, максимум -- Max
Лімітує величину автоматичного масштабування.
Очікувана Позиція X/Y -- Expected Position X/Y
Known relative offset of original shot, will be subtracted, e.g. for panning shots.
Очікуваний Оберт -- Expected Rotation
Обертання присутнє на оригінальному знімку буде компенсоване, наприклад для навмисного відхилу камери.
Очікуване Зумування / Масштаб -- Expected Zoom / Scale
Явний масштаб результатного кадру для компенсування зумування оригінального знімка.
Вплив -- Influence
Величина трансформування, застосована до фільмоматеріалу, може бути скерована. У деяких випадках не варто повністю компенсувати стрибки камери. Величина стабілізації, застосована до фільмоматеріалу, може бути скерована. У деяких випадках ви можливо не схочете повністю компенсувати деякі стрибки камери. Будь ласка, зауважте, що ці параметри «Впливу» -- Influence здійснюють керування тільки рухами компенсації, обчисленими стабілізатором, а не навмисними рухами, доданими через «Очікувані» -- Expected* параметри.
Інтерполювання -- Interpolate

The stabilizer calculates compensation movements with sub-pixel accuracy. Consequently, a resulting image pixel needs to be derived from several adjacent source footage pixels. Unfortunately, any interpolation causes some minor degree of softening and loss of image quality.

Найближче -- Nearest
Без інтерполяції, використовує найближчий сусідній піксель. Без інтерполяції, використовує найближчий сусідній піксель. Ця устава базово зберігає різкість оригінального зображення. Недоліком цього є те, що ми також зберігаємо залишковий рух під розміром одного пікселя, а компенсація рухів здійснюється кроками в 1 піксель, що може бути помітно, як нерегулярні стрибки.
Білінійно -- Bilinear
Проста прямолінійна інтерполяція між суміжними пікселями.
Бікубічно -- Bicubic
Highest quality interpolation, most expensive to calculate.
