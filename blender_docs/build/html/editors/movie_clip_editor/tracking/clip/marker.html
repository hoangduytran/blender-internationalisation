Мітка Простеження -- Tracking Marker¶

Точка -- Point¶

Схема мітки простеження.

The whole marker can be moved with RMB or by dragging the anchor point (black dot) with LMB. Pressing G also translates the whole marker. When pressing G twice the marker will be translated while keeping the anchor in place. Note that the anchor point outside the pattern area is shown as a cross connected with marker position with a dashed line.

S масштабує мітку повністю. Уся область взірця -- pattern area масштабується цілком тільки після натиснення двічі S. Взірець може також бути обернутий після натискання R і ця операція залежить від заданого типу опорної точки, наприклад, навколо центрів взірців або міток цілком навколо серединної точки.

Для відповідності перспективного трансформування мітки на площині окремі кутки -- individual corners повинні редагуватися вручну. Кожен куток мітки може деформуватися індивідуально для визначення форм. Позиції кутків можуть редагуватися перетягом їх мишею. Перетяг з LMB буде змінювати позицію індивідуального кутка.

Ghi chú

Зауважте, що деформування взірця корисне не тільки для планарного / спорідненого простеження. Оскільки, лише пікселі у межах взірця будуть розглядатися, то це може допомогти визначити кращий взірець для простеження навіть для простого відстеження позиції.

Область пошуку -- Search area не може обертатися; це зроблено навмисно. Немає сенсу деформувати область пошуку.

Plane¶

Лівий нижній куток площини має X/Y вісь (X -- червоним, Y -- зеленим) для допомоги у розрізненні орієнтації площини у просторі.

Цілком ймовірно, що куток об'єкта площини потребує ручного коригування. Для того, щоб зробити це, суньте індивідуальні кутки мишею з LMB або за допомогою загальних інструментів трансформацій -- G, R, S.

Регулювання кутків площини буде зберігатися відповідно площини, визначеної стежками при її початковому створенні.
