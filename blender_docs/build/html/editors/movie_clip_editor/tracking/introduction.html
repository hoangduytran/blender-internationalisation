Вступ¶

Motion Tracking is used to track the motion of objects and/or a background scene and, through the compositor, to apply this tracking data to 3D objects (or just one), which have either been created in Blender or imported into the application. Blender's motion tracker supports a couple of very powerful tools for 2D tracking and 3D motion tracking, including camera tracking and object tracking, as well as some special features like the plane track for compositing. Tracks can also be used to move and deform masks for rotoscoping in the Mask Editor, which is available as a special mode in the Movie Clip Editor.

Огляди -- Views¶

У режимі «Простеження» -- Tracking Mode є доступними три різні режими. Ви можете перемикатися між цими оглядами за допомогою контролерів у заголовку. Коли ви обрали огляд, то вся область Редактора Відеокліпів зміниться. Звідси, для одночасного показу огляду кривих або огляду аркуша експозицій, область редактора слід розділити на дві та в одній з них перемкнути на огляд графів, а в іншій -- огляд аркуша експозицій.

Ручне калібрування об'єктива -- Manual Lens Calibration¶

Усі камери записують спотворене відео. Нічого з цим зробити не можна, оскільки така природа роботи оптичних лінз об'єктива. Для точного простеження руху камери необхідні точні значення фокусної відстані та «сили» спотворення.

Поточно, фокусна відстань може бути автоматично отримана тільки з устав камери або з інформації EXIF. Є кілька засобів, що можуть допомогти знайти наближені значення для компенсації спотворення. Передбачені також повністю ручні інструменти, де ви можете використовувати сітку, яка залежить від моделі спотворення, а її деформовані комірки визначають прямі лінії у фільмоматеріалі.

You can also use the Grease pencil for this -- just draw a line which should be straight on the footage using poly line brush and adjust the distortion values to make the Grease pencil match lines on the footage.

To calibrate your camera more accurately, use the Grid calibration tool from OpenCV. OpenCV is using the same distortion model, so it should not be a problem.

Обчислення руху камери та об'єкта -- Camera and Object Motion Solving¶

Blender не тільки підтримує обчислення руху камери, включаючи зйомки на штативі, але і також обчислення руху об'єкта відносно руху камери. Окрім цього, є Простеження Площини -- Plane Track, яке обчислює рух усіх міток однієї площини.

Також планується додати більше засобів у майбутньому, наприклад, більш автоматичне простеження та обчислення, обчислення багатьох камер та примушені обчислення.

Засоби для Орієнтації Сцени та Стабілізації -- Tools for Scene Orientation and Stabilization¶

Після обчислення, вам необхідно зорієнтувати реальну сцену у 3D сцені для більш зручного компонування. Існують знаряддя для визначення основи, підлоги, початку сцени та осей X/Y для здійснення орієнтації сцени.

Sometimes, the video footage includes spurious jumps and tilting movements, like e.g. when using a hand-held camera. Based on some tracked image elements, the 2D Стабілізація -- 2D Stabilization is able to detect and compensate such movements to improve the quality of the final result.
