Selecting & Editing¶

Selecting¶

Доступні звичайні засоби виділення:

  • Перемикач Виділити Усе -- Toggle Select All: A.
  • Border and Circle Select B, C.
  • Виділ Ласо -- Lasso Select Ctrl-Alt-LMB.
  • Select Linked from selection Ctrl-L.
  • Виділити Пов'язане з мишею -- Select Linked with mouse L.
  • Виділити мітку простеження -- Select tracking marker Ctrl-RMB.

Редагування -- Editing¶

Інструменти та панелі, доступні для редагування масок, однакові для обох режимів редактора. Редагування сплайнів здійснюється у спосіб, подібно до редагування кривих Безьє або шляхів у GIMP чи інших редакторах кривих.

Mẹo

Для отримання інтерактивного відзиву про результати редагування маски у Компонівнику -- Редакторі Вузлів з режимом Компонування вузол «Маска» -- Mask може бути під'єднано прямо на вузол «Оглядач» -- Viewer, що далі допомагає зберігати оновлення результату компонування у ході редагування.

Керувальні Точки -- Control Points¶

Додати Вершину та Посунути -- Add Vertex and Slide Ctrl-LMB
Ctrl-LMB is used to place new control points and define handle orientations by a continued mouse drag. A Ctrl-LMB double-click will also close the curve if the last point was selected.
Трансформації -- Transform
Existing control points can be moved with LMB and translated, scaled and rotated with the usual G, S, R shortcuts. The whole spline can be moved by dragging the center dot with LMB.
Перемкнути Зацикленість -- Toggle Cyclic Alt-C
Toggle to create a closed curve or open it again. Close the mask by joining the last control point to the first.
Видалення -- Delete X
Вилучає керувальні точки.

Держаки Кривої -- Curve Handles¶

Кривина Сплайну Посуванням -- Slide Spline Curvature
Клацок LMB на кривій та перетяг мишею держака.
Задати Тип Держака -- Set Handle Type V
Задає тип держака для виділених точок сплайна.
Перерахувати Нормалі -- Recalculate Normals Ctrl-N
Робить нормалі (напрямки держаків) послідовними.
Перемкнути Напрям -- Switch Direction
Перемикає напрямок держаків всередину/назовні.

Розтушовка -- Feather¶

Додати Вершину Розтушовки і Посунути -- Add Feather Vertex and Slide Shift-LMB
Клацок Shift-LMB використовується для визначення контуру розтушовки кривої. Для створення початкової розтушовки посуньте керувальну точку сплайну назовні або всередину для створення та розміщення точок розтушовки. Після цього клацок Shift-LMB буде вставляти нову точку розтушовки та може використовуватися посування мишею для її руху.
Масштаб Розтушовки -- Scale Feather Alt-S
Буде масштабувати розмір розтушовки.

Анімація -- Animation¶

Masks can be animated with the shape keying system. This can be useful when there are not enough good feature points to track in the footage, or the mask is not based on footage. Mask animation timing can be edited from the Dope Sheet's Mask Mode.

Вставити Ключ Форми -- Insert Shape Key I
Вставлятиме ключ форми для активного шару маски у поточному кадрі. Це працює на рівні шару маски, а тому вставляння ключа форми буде ключкадрувати усі сплайни та точки, які на ньому розміщені.
Зчистити Ключ Форми -- Clear Shape Key Alt-I
Буде зчищати ключ форми для активного шару маски у поточному кадрі.
Скинути Анімацію Розтушовки -- Feather Reset Animation
Скидає зсув розтушовки по всіх анімованих кадрах.
Пере-ключувати Точки Виділених Форм -- Re-Key Points of Selected Shapes
Re-interpolate selected points on across the range of keys selected in the Dope Sheet.

Показ/Схов -- Show/Hide¶

  • Сховати Виділене -- Hide Selected H
  • Сховати Невиділене -- Hide Unselected Shift-H
  • Виявлення, показати сховане -- Reveal Alt-H
