Виділення -- Selecting¶

Активна смужка відеоряду показується зі світлим контуром. Кліп цілком може бути виділено клацком RMB посередині смужки.

Меню «Виділ» -- Select дозволяє вам виділяти смужки різними способами.

Смужки Ліворуч -- Strips to the Left
Виділяє усі смужки зліва від поточно виділеної смужки.
Смужки Праворуч -- Strips to the Right
Виділяє усі смужки справа від поточно виділеної смужки.
Виділити Пов’язане за Часом -- Select Linked Time Ctrl-RMB
Виділяє смужку під вказівником, а також усі смужки з однаковим початком/кінцем.
Виділити Навколишні Держаки -- Select Surrounding Handles Alt-RMB

Виділяє смужку під вказівником, а також держаки сусідніх смужок.

Ghi chú

Виділення цим методом рухатиме смужку, яка знаходиться між іншими, без впливу на довжину виділеної смужки.

Виділити Обидва Держаки -- Select Both Handles Alt-RMB

Виділяє держак під вказівником, а також держаки суміжної смужки.

Ghi chú

Виділяйте цим методом, коли ви хочете змінити часування розрізу.

Виділити Пов’язане -- Select Linked L
Виділяє усі смужки, пов'язані з поточно виділеною смужкою.
Виділити Груповане -- Select Grouped Shift-G

Виділяє смужки відповідно до їх відношення з іншими смужками.

Тип -- Type
Виділяє будь-які смужки однакового типу у межах певної категорії, наприклад, якщо ви маєте виділену смужку ефекту перетину, то це виділить усі інші смужки ефектів.
Глобальний Тип -- Global Type
Selects any strips of the same type, e.g. Effect, Image, Movie, etc.
Тип Ефекту -- Effect Type
Виділяє усі смужки типу «Ефект».
Дані -- Data
Виділяє смужки, що мають спільні однакові дані, наприклад, дві смужки зображення спільно базуються на однаковому файлі зображення.
Ефект -- Effect
Виділяє смужки, що мають призначену для них спільну смужку ефекту.
Ефект/Пов'язане -- Effect/Linked
Виділяє смужки ефекту, якщо такі є, пов'язані з поточно виділеною смужкою.
Перекриття -- Overlap
Виділяє будь-які смужки, що трапляються на однаковому кадрі, як і поточна.
Виділити Усе -- Select All A
Виділяє усі завантажені у редактор смужки.
Інверсія Виділу -- Select Inverse Ctrl-I
Інвертує поточний виділ.
Виділ Рамкою -- Border Select B
Запускає процес виділення Рамкою -- Box. Клацайте та натягайте прямокутний аркан навколо потрібного регіону смужок у робочому просторі вашого Відеорядника. Коли ви відпустите кнопку миші, то всі смужки, зачеплені рамкою будуть виділені.
