Смужки «Фільм» та «Зображення» -- Movie & Image Strips¶

Фільм -- Movie¶

Для додання фільму (із або без аудіо) виберіть файл(и) фільму в Оглядачі Файлів, наприклад, у форматі Audio-Video Interleaved (*.avi файл).

Ghi chú

Кліпи можуть бути Величезними

Трихвилинний файл quicktime .mov може займати 140 Мегабайтів. Завантаження його, навіть через високо-швидкісну LAN може зайняти деякий час. Не вважайте. що ваш комп'ютер або Blender заблокувався, якщо у цей час нічого не відбувається.

Зображення -- Image¶

Одиничне Зображення -- Single Image¶

Коли ви додаєте одиничне статичне зображення (*.jpg, *.png, тощо), то Blender створює смужку тривалістю 25 кадрів, з повторним показом цього зображення впродовж цієї тривалості.

Відеоряд Зображень -- Image Sequence¶

У випадку (нумерованих) відеорядів, послідовностей зображень (наприклад, *-0001.jpg, *-0002.jpg, *-0003.jpg тощо будь-якого формату) ви можете визначати вміст смужки на вибір:

Діапазон -- Range
Перейдіть у їх каталог та RMB клацніть та натягніть рамку над діапазоном потрібних імен для підсвічення потрібних файлів зображень. Ви можете прокручувати вниз та продовжувати з RMB додавати у цей виділ.
Пакет -- Batch
Shift-RMB click selected non-related stills for batch processing; each image will be one frame, in sort order, and can be a mix of file types (jpg, png, exr, etc.).
Усе -- All
Натисніть A для виділення/зневиділення усіх файлів у каталозі.

Mẹo

Випадок з різними розмірами

Справа з різними розмірами зображень та різними розмірами виводів є дещо заплутаною. Якщо ви маєте невідповідність між розміром увідного зображення та розміром вивідного рендера, то VSE буде намагатися авто-масштабувати зображення, щоб воно вписалося у вивід. Це може призвести до відсікання частин увідного зображення. Якщо ви не хочете цього, використовуйте устави опцій «Обрізка» -- Crop та/або «Зсув» -- Offset на панелі «Увід Смужки» -- Strip Input для пересування та вибору регіону зображення, що буде подане на вивід. Коли ви використовуєте опції Crop або Offset авто-масштабування вимикається і ви можете вручну змінити масштаб зображення за допомогою додання ефекту «Трансформації» -- Transform.

Додання смужки Зображення¶

Заповнювачі Зображень -- Placeholder Images¶

Image sequences can use placeholder files. This works by enabling Use placeholders checkbox when adding an image strip. The option detects the frame range of opened images using Blender's frame naming scheme (filename + frame number + .extension) and makes an image sequence with all files in between even if they are missing. This allows you to render an image sequence with a few frames missing and still the image strip will have the correct range to account for the missing frames displayed as black. When the missing frames are rendered or placed in the same folder, you can refresh the Sequencer and get the missing frames in the strip. The option is also available when using the Change Data/File operator and allows you to add more images to the range.

Приклад¶

Якщо ви прокрутите робочий простір, то побачите інформацію про канал (у вертикальній локації каналу 0), що дає вам деякі корисні поради про активну смужку. Приклад вище показує смужку кольору з діапазоном від 1 до 25, далі файл mov і потім смужку зображення. Інформація про канал показує зручну інформацію про смужку зображення, чиє ім'я буде скорочене у показі смужки, але розширене в інформації про смужку.
