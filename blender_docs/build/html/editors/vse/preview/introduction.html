Вступ¶

By default, the VSE only displays the strips, however, there are a few ways to preview the result of your sequence. The first is the preview mode, this can be enabled by hitting the texture button ().

Заголовок -- Header¶

Меню «Огляд» -- View Menu¶

Підігнати Передогляд до Вікна -- Fit preview in window Home
Will resize the preview so that it fits in the window.
Зумування -- Zoom 1:1 Numpad1
Will resize the preview to a 1:1 scale (actual size).
Безпечні Області -- Safe Areas
Показує поверх передогляду рамку області титрів.

Контролери -- Controls¶

Several options in the header allow you change the editor to display the sequence in real-time, and in various ways.

Sequencer Display header.

The second button will change the editor to display only the preview, and the third button displays both the Sequencer and the preview.

Режим Показу -- Display Mode

Режим, що показує різні аспекти результату компонування для поточного кадру:

  • Зображення/Відеоряд: Кольори -- Image/Sequence: Colors (те, що ви бачите).
  • Світність -- Luma: Яскравість/контраст.
  • Колірність -- Chroma: Відтінок та насиченість кольору.
  • Гістограма -- Histogram: Рівні червоного, зеленого та синього.
Канал -- Channel

Вибирає канал для показу у передогляді.

Канал 0 -- це результат компонування усіх смужок. Канал 1 -- це поточне кадроване зображення зі смужки на каналі 1 тільки (канал 1 знаходиться внизу стосу каналів). Показ цих режимів є відображенням компонування (канал 0) або кадру зі смужки (від каналу 1 до n).

Перекрив (іконка з привидом)

Опція для вмикання перекриву. Може використовуватися для порівняння поточного кадру з орієнтувальним кадром.

Кадр -- Frame
The slider controls the offset of the reference frame relative to current frame.
Блокування -- Lock (іконка замка)
Дає можливість блокувати орієнтувальний кадр відносно його поточної позиції.
Тип -- Type

Описує спосіб показу орієнтувального кадру.

Прямокутник -- Rectangle
Означає, що прямокутна область орієнтувального кадру буде показана поверх поточного кадру. Ця область може бути визначена через натискання O над передоглядом.
Орієнтир -- Reference
Буде показуватися тільки орієнтувальний кадр у регіоні передогляду.
Поточне -- Current
Only the current frame is displayed in the preview region.

Mẹo

Можливо мати кілька відкритих Редакторів Відеорядів, кожен з різним типом перекриття. Звідси, є можливим мати показ поточного та орієнтувального кадрів у різних просторах редактора.
