Властивості -- Properties¶

Передогляд/Зображ Сцени -- Scene Preview/Render¶

Дозволяє вам керувати тим, як зображення смужок сцени -- Scene Strips показуються у передогляді.

OpenGL Передогляд -- OpenGL Preview

Використовується швидкий OpenGL передогляд (дивіться OpenGL render для деталей про цю тему), інакше використовується повний зображ, який може бути дуже повільним.

Відтінення Передогляду Відеорядника -- Sequencer Preview Shading
Method for rendering OpenGL renders. See the 3D View's Відтінення Оглядвікна -- Viewport Shading options.
Текстуровано Суцільно
Показує текстури навіть у суцільному режимі.

Устави, що використовуються OpenGL Передоглядами:

  • Устава загладжування з активної сцени використовується для усіх сцен.
  • Устава альфа-прозорості береться з опції «Режим Альфа» -- Alpha Mode смужки кожної сцени.

Устави Огляду -- View Settings¶

Показати Переекспоноване -- Show Overexposed
Показує переекспоновані (яскраво білі) ділянки за допомогою взірця зебри. Поріг показу може коригуватися за допомогою цього повзунка.
Розмір Зображу Замісника -- Proxy Render Size
Розмір показу замісників у регіоні передогляду. Використання меншого розміру передогляду збільшує швидкість взаємодії з програмою.

Безпечні Області -- Safe Areas¶

Показує напрямні, що використовуються для розміщення елементів на екрані, які дають переконатися, що найбільш важливі частини відео будуть видимі на різних типах екранів.
