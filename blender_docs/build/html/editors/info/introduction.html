Вступ¶

Редактор Інфо -- Info Editor знаходиться вгорі екрану «Стандарт» -- Default Screen та має наступні компоненти.

Заголовок -- Header¶

Заголовок Редактора Інфо -- Info Editor.

Вибірник Типу Редактора -- Editor Type Selector (забарвлене червоним), Меню -- Menus (синім), Блок даних Екрану -- Screen Data-block (зеленим), Блок даних Сцени -- Scene Data-block (оранжевим), Вибірник Рушія -- Engine Selector (пурпуровим), Інформація про Ресурси -- Resource Information (кольором морської хвилі).

Меню -- Menus¶

Забезпечують доступ до головних опцій Blender'а.

Файл -- File¶

Дивіться детальніше тут -- Файл -- File.

Зображ -- Render¶

Зображ -- Render
Дивіться детальніше тут -- Панель «Зображ» -- Render Panel.
Зображ OpenGL -- OpenGL Render
See Зображ з OpenGL -- OpenGL Render.
Показати/Сховати Огляд Зображу -- Show/Hide Render View F11
Показує (або ховає) редактор, де здійснювався останній зображ.
Відтворити Зображену Анімацію -- Play Rendered Animation Ctrl-F11
Відтворює останню зображену анімацію за допомогою внутрішнього відтворника, плеєра -- Відтворювач Анімації -- Animation Player або зовнішнього відеоплеєра, який визначається у вкладці «Файл» -- File в Уподобаннях Користувача -- User Preferences.

Вікно -- Window¶

Дублювати Вікно -- Duplicate Window Ctrl-Alt-W
Дублює поточне вікно, так що нове створюється з такою ж розставою екрану та розміром. Корисно для одночасного використання кількох моніторів.
Перемкнути Повноекранний режим Вікна -- Toggle Window Fullscreen Alt-F11
Перемикає із або на повний екран.
Екранознімок -- Screenshot, Екранозапис -- Screencast
See Захват Екрану -- Screen Capture.
Перемикнути Системну Консоль -- Toggle System Console
Shows or hides the System Console (on MS-Windows).

Довідка -- Help¶

Дивіться -- Меню «Довідка» -- Help Menu.

Контролери -- Controls¶

Назад у Попереднє -- Back to Previous
Кнопка, що показується коли область максимізована, для повернення до плитки областей.
Екран -- Screen
Цей блок даних -- Data-block menu використовується для вибору або правки Екранів -- Screens (розстав вікон).
Сцена -- Scene
Цей блок даних -- Data-block menu для вибору з кількох сцен -- Scenes. Наявність кількох Сцен дозволяє вам працювати з окремими віртуальними середовищами, з повністю окремими даним або з даними об'єктів та/або сітей, що пов'язані між ними.
Рушій -- Engine
Надає список доступних для вибору рушіїв створення зображень та ігор.
Прогрес Зображу/Запікання -- Render/Baking progress
A progress bar and a cancel button are shown while rendering or baking. Hovering over them shows a time estimate.
Стоп Захвату -- Capture Stop
Кнопка, що показується у ході запису екрану -- screen casting, для зупинки записування.
Звітне Повідомлення -- Report Message
Позначка для оператора для показу результатів або попереджень. Вона зникає через деякий час. Клацок LMB на іконці зліва дає копіювання повного звіту у новий текстовий блок даних, який ви можете відкрити у Редакторі Тексту -- Text Editor.
Іконка Blender'а -- Blender Icon
Клацання на логотипі Blender'а відкриває Екран Заставки -- Splash Screen.
Версія Blender'а -- Blender version
Позначка, що показує цю версію Blender'а.
Інформація про Ресурси -- Resource Information
Сцена -- Scene
Показує інформацію про поточно завантажену сцени залежно від режиму та типу об'єкта. Коли показуються два числа, то перше описує рахунок про виділ, а друге -- загальний рахунок. Це може бути кількість вершин, граней, трикутників або кісток, а також виділених об'єктів та ламп.
Пам’ять -- Memory
Позначка "Mem" показує обчислену використану пам'ять Blender'ом. Це може допомогти для ідентифікації, коли ви досягаєте лімітів вашого устаткування.
Активний Об'єкт -- Active Object
Об'єктний тип поточно виділеного об'єкта.

Звітна Консоль -- Report Console¶

Коли область Редактора Інфо розсунути, то з'являється Звітна Консоль, в якій показується слід виконання скриптових команд. Кожного разу при виконанні оператора, це залишає звіт зі створенням протоколу.
