Захват Екрану -- Screen Capture¶

Екранознімки -- Screenshots¶

Орієнтир

Mode:All modes
Menu:Window ‣ Save Screenshot
Hotkey:Ctrl-F3

Save Screenshot option.

Скорочення Ctrl-F3 робить знімок екрану Blender'а та далі відкриває Оглядач Файлів -- File Browser, дозволяючи визначити для його збереження ім'я та локацію. Тут на ілюстрації формат PNG буде виводом зробленого екранознімку (устави такі ж, як і для збереження зображу). Коли відкривається Оглядач Файлів -- File Browser, то зліва з'являється вкладка «Зберегти Екранознімок» Save Screenshot, де ви можете знайти устави формату та стяг опції «Повний Екран» -- Full Screen.

  • Вмикання цієї Опції збереже все вікно Blender'а (з повними шириною та висотою вікна Blender'а, в якому ви викликали цю команду).
  • Вимикання цієї опції збереже тільки активну область (де розміщений вказівник вашої миші, коли ви викликали цю команду).

Екранозаписи -- Screencasts¶

Орієнтир

Mode:All modes
Menu:Window ‣ Make Screencast
Hotkey:Alt-F3

This is a quick way to make screencasts from within Blender.

Ghi chú

Це обмежене одиничним вікном та не підтримує аудіо.

Для запису навчальників, туторіалів вам можливо слід скористатися більш зручними сторонніми застосунками.

Екранозаписи -- це записи ваших дій у часі, як відеофайл або послідовність файлів зображення. Тип та локація виводу визначається уставами на панелі «Вивід» -- Output panel вкладки «Зображ» -- Render tab Редактора Властивостей -- Properties Editor. Стандартні устави будуть давати екранозапис, що складається із серії зображень у форматі PNG, захоплених кожні 50 мілісекунд та збережені у теці /tmp. Якщо ви хочете записати відео, то задайте на панелі «Вивід» -- Output один з форматів файлів фільму -- Movie File Formats, що підтримується у вашій операційній системі та доступний для вибору на цій панелі. Якщо ви не впевнені, які відеокодеки підтримує ваша система, то вибирайте AVI JPEG.

Ghi chú

You can change the frame rate for a screencast in the User Preferences.

Коли ви стартуєте екранозапис Blender'а, то заголовок Редактора Інфо -- Info Editor зміниться та покаже ваш кнопку для зупинки записування.

Info Header with the Capture Stop button.

Ghi chú

The Only Way to Stop the Screencast

Натискання кнопки зупинки у заголовку Редактора Інфо -- це єдиний спосіб зупинити захват екранозапису. Якщо ви натиснене Esc, то це скорочення буде працювати тільки для операцій, здійснених в Інтерфейсі Користувача -- User Interface Blender'а (воно зупиняє анімації, відтворення тощо), але це не працює для зупинки Екранозаписів -- Screencasts.

«Зображ > панель Розміри > Діапазон Кадрів» -- Render ‣ Dimensions Panel ‣ Frame Range

The frames are stored using a suffix added to their file name, where the suffix is composed of the numbers present in the fields for start and end frames, defined in the Frame Range of the Dimensions panel, Render tab (see Fig. «Зображ > панель Розміри > Діапазон Кадрів» -- Render ‣ Dimensions Panel ‣ Frame Range highlighted in yellow).

Ghi chú

Значення устави «Кінцевий Кадр» -- End Frame із секції «Діапазон Кадрів» -- Frame Range панелі «Розміри» -- Dimensions не буде автоматично зупиняти захват екрану. Його зупинка виключно можлива лише за допомогою кнопки із заголовку.

Захоплені Відео генеруються внутрішньо так само, як і Екранознімки -- Screenshots на основі ширини і висоти Вікна, в якому ви поточно працюєте. Якщо ви виберете захват у Відео файл, то Blender передасть ці кадри через Відео кодек.

Cảnh báo

Деякі з кодеків лімітують ширину/висоту виводу або якість відео:

  • Коли ви зберігаєте Екранозапис -- Screencast у формат Зображення -- Image, то Зображення будуть зберігатися на основі повного Вікна Blender'а з повними шириною та висотою та якістю, визначеною його типом (наприклад, JPG, PNG тощо) і конфігурацією (наприклад, повзунком «якість» -- quality для формату .JPG).
  • Коли ви зберігаєте ваш Екранозапис -- Screencast у Відео форматі, то він буде посланий у кодек. Залежно від обмежень кодека результатне вивідне Відео може бути зменшене у масштабі. Крім того, деякі комбінації ширини та висоти Вікна не можуть бути оброблені певним кодеками. У таких випадках Екранозапис -- Screencast спробує початися, але негайно ж і зупиниться. Для вирішення цього виберіть інший формат Вікна та/або інший кодек.

Розміри Вікна Blender'а -- Blender Window Dimension¶

There is a way to match the Blender Window dimensions with the Output Video File, achieving standard dimensions for the output of the Blender Screencast (e.g. NTSC, HD, Full HD, etc.). You can control the width and height of your Blender Window, starting Blender from a command line. To learn more about starting Blender from a command line, see the page about the Blender Console Window.
