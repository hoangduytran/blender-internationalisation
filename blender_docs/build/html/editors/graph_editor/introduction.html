Вступ¶

Редактор Графів -- Graph Editor -- це головний редактор анімування. Він дозволяє вам модифікувати анімацію за будь-якими властивостями з допомогою F-Кривих -- F-Curves.

The Graph editor has two modes, F-Curve for Actions, and Drivers for Drivers. Both are very similar in function.

Редактор Графів -- Graph Editor.

Огляд Кривої -- Curve View¶

The curve view allows you to view and edit F-Curves. An F-Curve has several key parts:

Curve

The curve defines the value (Y axis) of the property over time (X axis).

See F-Curves

Keyframes

Keyframes are user-defined values on certain frames and are represented by little black squares. These become orange if selected.

See Keyframes for more information.

Handles

Each keyframe has a handle that helps determine the values of the curve between keyframes. These handles are represented by extruding lines with circular ends and can be selected and modified to change the shape of the curve.

See Типи Держаків -- Handle Types for more information.

Навігація -- Navigation¶

Як і в більшості редакторів, ви можете здійснювати:

Панорамування -- Pan
Панорамує огляд вертикально (значення) або горизонтально (час) з допомогою клацку та перетягу (MMB).
Зумування -- Zoom
Присуває та відсуває огляд коліщам миші (Wheel).
Масштабування Огляду -- Scale View
Масштабує огляд вертикально або горизонтально (Ctrl-MMB).

2D Курсор -- 2D Cursor¶

2D Курсор Редактора Графів.

Поточний кадр представляється зеленою вертикальною лінією, що називається «Курсор Часу» -- Time Cursor.

Як і редакторі Часолінія -- Timeline, ви можете змінювати поточний кадр, клацнувши або тягнувши утримуваною натисненою LMB.

Зелена горизонтальна лінія називається «Курсор» -- Cursor. Її показ може бути вимкнено через меню «Огляд» -- View Menu або панель «Огляд» у регіоні «Властивості -- View Properties.

Курсор Часу -- Time Cursor та Курсор -- Cursor разом утворюють 2D Курсор -- 2D Cursor. 2D Курсор -- 2D Cursor здебільшого використовується для засобів редагування.

See Graph Editor's View Properties.

Осі Огляду -- View Axes¶

For Actions the X axis represents time, the Y axis represents the value to set the property.

For Drivers the X axis represents the Driver Value, the Y axis represents the value to set the property.

Залежно від виділених кривих ці значення можуть мати різне розуміння: наприклад, властивості обертання показуються у градусах, властивості локації показуються в Одиницях Blender'а -- Blender Units. Зауважте, що Драйвери -- Drivers використовують радіани для властивостей обертання.

Діапазон Передогляду -- Preview Range¶

Задати Діапазон Перегляду -- Set Preview Range P
Interactively define frame range used for playback. Allow you to define a temporary preview range to use for the Alt-A real-time playback (this is the same thing as the Playback Range option of the Timeline editor header).
Зчистити Діапазон Перегляду -- Clear Preview Range Alt-P
Зчищає заданий діапазон передогляду.
Авто-Задати Діапазон Передогляду -- Auto-Set Preview Range Ctrl-Alt-P
Автоматично задає діапазон передогляду для відтворення дії цілком або виділених смужок НЛА.

Markers Menu¶

Markers are used to denote frames with key points or significant events within an animation. Like with most animation editors, markers are shown at the bottom of the editor.

Markers in animation editor.

For descriptions of the different marker tools see Editing Markers.

Контролери Огляду -- View Controls¶

View controls.

Показати Лише Виділене -- Show Only Selected (іконка вказівника миші)
Включає криві, що стосуються тільки виділених об'єктів та даних.
Показати Сховане -- Show Hidden (іконка з привидом)
Включає криві для об'єктів/костей, що поточно невидимі.
Показати Лише Помилки -- Show Only Errors (іконка з рятівним колом)
Включає тільки криві та драйвери, що вимкнені або мають помилки. Корисно для налагодження.
Фільтр Пошуку -- Search Filter (іконка з лупою) F

Включає тільки криві, що у назві містять ключові слова, задані у полі пошуку.

Кілька Слів -- Multi-Word (іконка az)
Fuzzy/Multi-Word name filtering matches word snippets/partial words, instead of having to match everything. It breaks down the search string based on white-space placement. e.g. "lo ro" will filter all location and rotation, while "lc rt" will not work.
Фільтр за Типом -- Type Filter

Фільтрує криві за типом властивості.

Сортувати Блоки Даних -- Data-block Sort (іконка az)

Блоки даних об'єктів будуть відображатися в алфавітному порядку, щоб було легше знаходити, де вони відбуваються (або також зберігати разом анімацію по об'єктам в НЛА, наприклад).

Якщо ви виявите, що швидкість вашого відтворення страждає, коли це увімкнено (це повинно бути проблемою тільки, коли працюєте з багатьма об'єктами у сцені), то ви можете вимкнути це.

Нормалізація -- Normalize

Нормалізує криві так, щоб точки максимумів та мінімумів дорівнювали 1.0 та -1.0, відповідно.

Авто -- Auto
Автоматично повторно обчислює нормалізацію кривої при кожній її правці. Це корисно для запобігання стрибкам кривої після її налаштування.
