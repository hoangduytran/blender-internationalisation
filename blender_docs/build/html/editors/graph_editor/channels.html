Канали¶

Регіон Каналів -- Channels Region¶

The Channels region.

The channels region is used to select and manage the curves for the Graph editor. This part shows the objects and their animation data hierarchy each as headers. Each level can be expanded/collapsed by the small arrow to the left of its header.

  • Сцени -- Scenes, Об'єкти -- Objects (темно-синій)
  • Дії -- Actions, Ключі Форм -- Shape keys тощо (світло-синій)
  • Групи -- Groups (зелений)
  • Канали -- Channels (сірий)

Контролери -- Controls¶

На записах є перемикачі для керування уставами каналу:

Пришпилення -- Pin (іконка зі шпилькою)
Make the channel always visible regardless of the current selection (Graph editor only).
Схов -- Hide (іконка з оком)
Ховає канал(и)/криву (лише Редактор Графів).
Модифікатори -- Modifiers (іконка з гайковим ключем)
Деактивує F-Модифікатори кривої або всіх кривих у каналі.
Приглушення -- Mute (іконка з гучномовцем)
Деактивує канал/криву.
Lock (padlock icon) Tab

Toggle channel/curve from being editable. Selected channels can be locked by pressing Tab.

Ghi chú

In the Dope Sheet this is also working inside the NLA, but that it does not prevent edition of the underlying F-Curve.

Виділення -- Selecting¶

  • Виділяє канал (текст змінюється з чорного на білий): LMB
  • Виділення/Зневиділення Кількох: -- Multi Select/Deselect: Shift-LMB
  • Перемикач Виділити Усе -- Toggle Select All: A
  • Виділ Рамкою -- Border Select: (LMB натяг) або B (LMB натяг)
  • Зневиділ Рамкою -- Border Deselect: (Shift-LMB натяг) або B (Shift-LMB натяг)
  • Виділяє усі ключкадри у каналі: подвійний клацок LMB на записі каналу.

Редагування -- Editing¶

  • Перейменування -- Rename CTRL-LMB
  • Видалити виділене -- Delete selected: X або Delete
  • Заблокувати виділене -- Lock selected: Tab
  • Зробити видимим тільки виділене -- Make only selected visible: V
  • Увімкнути Приглушення або Блокування виділеного -- Enable Mute or Lock selected: Shift-Ctrl-W
  • Вимкнути Приглушення або Блокування виділеного -- Disable Mute or Lock selected: Alt-W
  • Перемкнути Приглушення або Блокування виділеного -- Toggle Mute or Lock selected: Shift-W

Повзунки -- Sliders¶

Загальний приклад повзунків для редакторів анімації на прикладі режиму «Редактор Дій» -- Action Editor Аркуша Експозицій -- Dope Sheet.

У редактора анімації записи каналів можуть мати ще й стовпець з числовими кнопками або повзунками, що дозволяють тут змінювати значення поточних ключкадрів або додавати нові ключкадри. Дивіться детальніше про повзунки тут -- Меню «Огляд» -- View Menu.

Меню¶

Видалити Канали -- Delete Channels X

Видаляє увесь канал для поточної дії (тобто відв'язує підлежачу блок даних F-Кривої від блоку даних дії).

Cảnh báo

Скорочення X залежне від області: якщо ви використовуєте його у лівій частині списку, то воно видаляє виділені канали, а якщо ви його вживаєте у головній області, то воно буде видаляти виділені ключкадри.

Роз/Групувати Канали -- Un/Group Channels Ctrl-G, Alt-G
Un/Groups the selected channels into a collection that can be renamed by double clicking on the group name. For example, this helps to group channels that relate a part of an armature to keep the editor more organized.
Settings Toggle/Enable/Disable, Shift-W, Shift-Ctrl-W, Alt-W

Enable/disable a channel's setting (selected in the menu that pops up).

Блокування -- Lock, Приглушення -- Mute

Перемкнути Редагованість Каналу -- Toggle Channel Editability Tab
Locks or unlocks a channel for editing.
Режим Екстраполяції -- Extrapolation Mode
Змінює екстраполяцію -- extrapolation між виділеними ключкадрами.
Розгорнути Канали -- Expand Channels, Згорнути Канали -- Collapse Channels NumpadPlus, NumpadMinus
Розгортає або згортає виділені канали.
Переміщення -- Move...
This allows you to move top-level channels up/down Shift-PageUp, Shift-PageDown, or directly to the top/bottom Shift-Ctrl-PageUp, Shift-Ctrl-PageDown.
Відновити Вимкнуті F-Криві -- Revive Disabled F-Curves
Зчищає тег «вимкнуто» з усіх F-Кривих, що пошкоджені F-Криві знову працювали.
