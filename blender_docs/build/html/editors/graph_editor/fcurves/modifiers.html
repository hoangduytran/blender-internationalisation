Модифікатори F-Кривої -- F-Curve Modifiers¶

Орієнтир

Panel:Properties region ‣ Modifiers ‣ Modifiers

Модифікатори F-Кривої подібні до модифікаторів об'єкта у тому, що вони додають не-руйнівні ефекти, що можуть коригуватися у будь-який час та нашаровуватися для створення більш складних ефектів.

Додання Модифікатора -- Adding a Modifier¶

Modifiers panel.

Панель модифікаторів F-Кривої розміщується у регіоні Властивості -- Properties region. Виділіть криву, виділивши одну її точок, або виділивши її у регіоні каналів. Клацніть на кнопці меню «Додати Модифікатор» -- Add Modifier та виберіть модифікатор.

Типи Модифікаторів -- Types of Modifiers¶

Генератор -- Generator¶

Модифікатор «Генератор» створює Факторизовану -- Factorized Polynomial або Розгорнуту -- Expanded Polynomial поліноміальну, багаточленну функцію. Це базові математичні формули, що представляють лінії, параболи та інші більш складні криві, залежно від використовуваних значень.

Доповняльно -- Additive
Ця опція спричиняє, що значення модифікатора для кривої будуть накладатися, а не заміщуватися більш пізніше доданими, як це стандартно передбачено.
Порядок Багаточлена -- Polynomial Order

Визначає порядок багаточлена або найвищий степінь X для цього багаточлена. (число коефіцієнтів: 1).

Зміна Коефіцієнта для багаточлена -- Coefficient for polynomial змінює форму кривої.

Вбудована Функція -- Built-in Function¶

Це додаткові формули, кожна з яких має однакові опції для керування формою. Зверніть до довідника з математики щодо детальної інформації по кожну з цих функцій:

  • Синус -- Sine
  • Косинус -- Cosine
  • Тангенс -- Tangent
  • Квадратний Корінь -- Square Root
  • Натуральний Логарифм -- Natural Logarithm
  • Нормалізований Синус -- Normalized Sine (sin(x)/x)
Амплітуда -- Amplitude
Регулює масштабування по Y.
Фазовий Множник -- Phase Multiplier
Регулює масштабування по X.
Фазовий Зсув -- Phase Offset
Регулює зсув по X.
Зсув Значення -- Value Offset
Регулює зсув по Y.

Оболонка -- Envelope¶

Дозволяє вам регулювати форму кривої у цілому за допомогою керувальних точок.

Орієнтувальне Значення -- Reference Value
Задає значення Y для оболонки, навколо все центрується.
Мін, мінімум -- Min
Нижнє значення відстані від Орієнтувального Значення -- Reference Value для стандартного впливу 1:1.
Макс, максимум -- Max
Верхнє значення відстані від Орієнтувального Значення -- Reference Value для стандартного впливу 1:1.
Додати Точку -- Add Point
Додає керувальну точку. Вона буде створена у поточному кадрі.
Точка -- Point
Кадр -- Frame
Задає номер кадру для керувальної точки.
Мін, мінімум -- Min
Визначає нижню позицію керувальної точки.
Макс, максимум -- Max
Визначає верхню позицію керувальної точки.

Цикли -- Cycles¶

Модифікатори «Цикли» додає циклічний рух у криву, що має дві або більше керувальних точок. Ці опції можуть бути задані для неї перед або після.

Режим Циклу -- Cycle Mode
Повтор Руху -- Repeat Motion
Повторює дані кривої, зберігаючи їх значення у кожному циклі.
Повтор зі Зсувом -- Repeat with Offset
Повторює дані кривої, але зсуває значення першої точки у значення останньої точки кожного циклу.
Повтор Дзеркально -- Repeat Mirrored
Each cycle the curve data is flipped across the X axis.
Цикли Перед -- Before Cycles, Цикли Після -- After Cycles
Задає кількість разів зациклення даних. Значення «0» дає безкінечне зациклення даних.

Шум -- Noise¶

Модифікує криву за допомогою формули шумування. Це корисно для створення тонкої або екстремальної випадковості для анімованих рухів, як тремтіння камери.

Тип Змішання -- Blend Type
Заміна -- Replace
Додає діапазон функції шуму -0.5 до 0.5 на значення кривої.
Додання -- Add
Додає діапазон функції шуму 0 до 1 на значення кривої.
Відняття -- Subtract
Віднімає діапазон функції шуму 0 до 1 від значень кривої.
Множення -- Multiply
Множить діапазон функції шуму 0 до 1 на значення кривої.
Масштаб -- Scale
Регулює у цілому розмір шуму. Значення, дальші від 0, дають менш частий шум.
Сила -- Strength
Регулює масштабування Y функції шуму.
Зсув -- Offset
Зсуває шум у часі.
Фаза -- Phase
Регулює базис випадковості шуму.
Глибина -- Depth
Регулює деталізацію функції шуму.

Ліміти -- Limits¶

Лімітує значення кривої визначеними діапазонами по X та Y.

Мінімум -- Minimum, Максимум -- Maximum X
Обрізає криву у кадрах цих діапазонів та задає мінімальне значення у цих точках.
Мінімум -- Minimum, Максимум -- Maximum Y
Усікає значення кривої цим діапазоном.

Покрокова Інтерполяція -- Stepped Interpolation¶

Надає кривій каскадний вигляд, округлюючи значення вниз у межах певного діапазону кадрів.

Розмір Кроку -- Step Size
Визначає кількість кадрів для утримування кожного значення.
Зсув -- Offset
Орієнтувальна кількість кадрів перед початком утримування значень. Використовується для утримування за взірцями (1-3) або (5-7).
Вжити Стартовий Кадр -- Use Start Frame
Змушує модифікатор діяти тільки після цього «стартового» кадру.
Вжити Кінцевий Кадр -- Use End Frame
Змушує модифікатор діяти тільки до цього «кінцевого» кадру.
