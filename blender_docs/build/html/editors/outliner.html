Структуратор -- Outliner¶

Редактору структури даних Структуратор -- Outliner.

Редактору структури даних Структуратор -- Outliner -- це список організування даних у blend-файлі. тобто даних сцени, а також Уподобань Користувача -- User Preferences.

Використання

  • Огляд даних у сцені.
  • Виділення та зневиділення об'єктів у сцені.
  • Ховання і показ об'єкта у сцені.
  • Вмикання або вимикання можливості виділення (зроблення об'єкта «невиділюваним» у 3D Огляді -- 3D View).
  • Вмикання або вимикання можливості зображування, рендерингу об'єкта.
  • Видалення об'єктів зі сцени.
  • Відв'язування даних (еквівалент натискання кнопки X поруч з іменем блоку даних).
  • Легкий вибір, який шар зображу -- render layer рендерити, зображувати.
  • Легкий вибір, який прохід зображу -- render pass рендерити, зображувати (наприклад, ви можете обрати для рендерингу, зображування тільки прохід з бліками -- Specular).

Ієрархічний огляд у вигляді дерева -- Tree View¶

Each row in the Outliner shows a data-block. You can click the plus sign to the left of a name to expand the current data-block and see what other data-blocks it contains.

Ви можете виділяти блоки даних у Структураторі -- Outliner, але це обов'язково дасть виділення цього блоку даних у сцені. Для виділення блоку даних у сцені ви повинні активувати його.

Виділення та активування¶

Одиничне виділення не вимагає будь-якого попереднього виділення: просто працює прямо з LMB (та/або RMB -- контекстне меню, дивіться нижче) всередині області ім'я/значок.

Коли ви виділяєте об'єкт у списку таким чином, то він виділяється та стає активним об'єктом в усіх інших 3D Оглядах.

Activating a Data-block¶

Для «активування» блоку даних клацайте LMB на іконці блоку даних. Активування блоку даних автоматично перемкне у відповідний для нього режим. Наприклад, активування даних сіті куба виділіть цей куб та перемкне у Режим Правки -- Edit Mode, тоді як активування даних об'єкта для куба виділить цей куб та перемкне у Режим Об'єкта -- Object Mode (дивіться справа).

Selecting a Group of Data-blocks¶

Корисно, коли ви хочете виділити/зневиділити цілий набір блоків даних. Для цього ви повинні підготувати таке виділення:

  • RMB або LMB,
  • Shift-RMB чи Shift-LMB,
  • RMB та перетяг або LMB та перетяг,

all outside the name/icon area. Those pre-selected have their line in a lighter color. You then can (de)select them with an RMB-click on the name/icon area, which brings on a context menu (see below). A to select/deselect all open items.

Виділення групи блоків даних.

Контекстне меню -- Context Menu¶

Показує контекстне меню для блоку даних після клацку RMB на іконці або імені. Залежно від типу перед-виділених блоку(ів) даних вам будуть доступні усі або частина наступних опцій:

Виділення -- Select, Зневиділення -- Deselect
Ця функція ще не імплементована.
Відв'язання -- Unlink
To unlink a data-block from its "owner" (e.g. a material from its mesh).
Зробити Локальним -- Make Local
Створює «локальний» дублікат цього блоку даних.
Зробити Однокористувацьким -- Make Single User
Ця функція ще не імплементована.
Видалення -- Delete
Видаляє виділений блок даних.
Видалити Ієрархію -- Delete Hierarchy
Видаляє об'єкт та усі його нащадкові об'єкти.
Перерозкласти Користувачів -- Remap Users
Remap Users of a data-block to another one (of same type of course) -- means you can e.g. replace all usages of a material or texture by another one.
Додати Псевдо-Користувач -- Add Fake User, Зчистити Псевдо-Користувач -- Clear Fake User
Додає «фіктивного» (псевдо) користувача, так що виділений блок даних завжди буде зберігатися, навіть якщо він немає жодних користувачів. Такий псевдо-користувач може бути вилучений за допомогою «Зчистити Псевдо-Користувач» -- Clear Fake User.
Rename Ctrl-LMB
Renames the selected data-block.
Виділити Пов’язане -- Select Linked
Виділяє пов'язані дані, про які детальніше дивіться тут -- Виділити Пов’язане -- Select Linked.

Ghi chú

Деякі типи блоків даних не мають для них контекстного меню взагалі!

Обмеження рівня «Об'єкт» -- Object-level Restrictions¶

Три наступні перемикачі, у правій частині редактора Структуратор -- Outliner, доступні для об'єктів, модифікаторів та примусів. При утримування натисненою Ctrl ці перемикачі також впливають на усіх їх нащадкові об'єкти.

Видимість -- Visibility (іконка з оком)
Перемикає видимість об'єкта у 3D Огляді -- 3D View. V буде перемикати цю властивість для будь-яких об'єктів, що виділені у Структураторі -- Outliner.
Виділюваність (іконка з вказівником миші)
Це корисно у випадках, якщо ви помістили щось у сцені та не хочете випадково виділити його при роботі з чимось ще. S буде перемикати цю властивість для будь-яких об'єктів, що виділені у Структураторі -- Outliner.
Зображування, рендеринг -- Rendering (іконка з камерою)
Це все ще зберігає об'єкт видимим у сцені, але при вимкненні дає ігнорування його рендерером, рушієм зображу -- renderer. R буде перемикати цю властивість для будь-яких об'єктів, що виділені у Структураторі -- Outliner.

Заголовок -- Header¶

Меню «Огляд» -- View Menu¶

Сортувати Алфавітно -- Sort Alphabetically
Сортує записи за алфавітом.
Показати Стовпці Обмежень -- Show Restriction Columns
Перемикає показ трьох стовпців обмежень рівня «Об'єкт» -- Обмеження рівня «Об'єкт» -- Object-level Restrictions.
Показати Активне -- Show Active
Центрує огляд дерева на виділеному об'єкті, скорочення Period.
Показати/Сховати Один Рівень -- Show/Hide One Level
Розгортає дерево на один рівень вниз, скорочення NumpadPlus, або згортає його, скорочення NumpadMinus.
Показати Ієрархію -- Show Hierarchy
Згортає усі рівні дерева, скорочення Home.

Режим Показу -- Display Mode¶

Це меню вибірника у заголовку дозволяє вам фільтрувати те, що Структуратор -- Outliner повинен показувати. Цей вибірник допомагає звужувати список об'єктів, щоб ви могли знаходити швидко і легко потрібні елементи.

Усі Сцени -- All Scenes
Shows everything the Outliner can display (in all scenes, all layers, etc.).
Поточна Сцена -- Current Scene
Показує усе, але лише у поточній сцені.
Видимі Шари -- Visible Layers
Показує все лише на видимих (поточно увімкнених) шарах у поточній сцені. Використовуйте кнопки шарів -- layer для зроблення видимими об'єктів на шарах у 3D Огляді -- 3D View.
Виділене -- Selected
Показує об'єкт або перелік об'єктів, що поточно виділені у 3D Огляді -- 3D View. Детальніше дивіться тут -- selecting in the 3D View.
Активне -- Active
Показує тільки активний (останньо виділений) об'єкт.
Однакові Типи -- Same Types
Показує список тільки тих об'єктів у поточній сцені, що є однакового типу з виділеними у 3D Огляді.
Групи -- Groups
Показує тільки групи об'єктів -- Groups та їх членів.
Відеоряд -- Sequence
Показує список блоків даних -- data-block, що використовуються Відеорядником -- Sequencer.
Файл Blender'а -- Blender File
Показує усі дані у поточному blend-файлі.
Блоки Даних -- Data-Blocks
Показує кожен блок даних -- data-block разом з будь-якими властивостями, що він можливо має.
Уподобання Користувача -- User Preferences
Показує опції, що можуть бути знайдені у вікні Уподобання Користувача -- User Preferences, разом з деякими іншими уставами.
Одинокі Дані -- Orphan Data
Показує блоки даних -- data-blocks, які не використовуються та/або будуть втрачені, коли файл буде перевідкрито.

Пошук -- Searching¶

Ви можете шукати для огляду блоки даних за допомогою текстового поля Пошуку у заголовку Структуратора -- Outliner. Меню «Пошук» -- Search дозволяє вам перемикати для шукання наступні опції:

  • Лише з Урахуванням Регістру -- Case Sensitive Matches Only
  • Лише Повні Збіги -- Complete Matches Only
