Режим «Нарисний Олівець» -- Grease Pencil¶

This Mode allows you adjust the timing of the Grease pencil tool's animated sketches. It is especially useful for animators blocking out shots, where the ability to re-time blocking is one of the main purposes of the whole exercise.

Перемикання у цей режим доступний за допомогою пункту «Нарисний Олівець» -- Grease Pencil вибірника «Режим» -- Mode у заголовку редактора Аркуш Експозицій -- Dope Sheet.

Регіон Каналів -- Channels Region¶

Нарисний Олівець -- Grease Pencil (світло-синій)
Регіон каналів показує блоки даних Нарисного Олівця, що містять шари. Одночасно використовуються кілька блоків для кожної області (наприклад, для 3D Огляду -- 3D View та Редактор UV/Зображень -- UV/Image editor).
Шари -- Layers (сірий)
This channels contain the keyframes to which the layer are bind.

Заголовок -- Header¶

Лише Активне -- Active Only
Показує тільки блоки Нарисного Олівця, прикріплені до поточної сцени та об'єктів у ній.

Копіювання нарисів -- Copying Sketches¶

It is possible to copy sketches from a layer/layers to other layers using the Copy and Paste buttons in the header. This works in a similar way as the copy/paste tools for keyframes in the Action Editor.

Нариси можуть також бути скопійовані з блоку даних в інший за допомогою цих засобів. Важливо мати на увазі, що ключкадри будуть вставлятися тільки на виділені шари, а тому шари необхідно створити також і для цільових областей.

Головний огляд -- Main View¶

The keyframes can be manipulated like any other data in the Dope Sheet can be. Interpolated keyframes (alias breakdowns) are visualized as smaller light blue points.

Вставлення Ключкадрів -- Insert Keyframe¶

Вставлення ключкадрів I може використовуватися для створення порожніх ключкадрів Нарисного Олівця у певному кадрі. Це буде створювати порожні кадри, якщо опція «Доповняльне Рисування» -- Additive Drawing вимкнена, інакше воно буде копіювати активний кадр на шар та використовувати його.
