Режим «Редактор Дій» -- Action Editor¶

The Action Editor is where you can define and control actions. It enables you to view and edit the F-Curve data-blocks you defined as Actions in the F-Curve Editor. So it takes place somewhere in between the low-level F-Curves, and the high-level NLA editor.

Він дає вам дещо спрощений огляд блоків даних F-кривих (щось схоже на їх прорис без держаків). Цей редактор може давати список усіх блоків даних Дій для об'єкта одночасно.

Кожен блок даних Дії форму верхньо-рівневий канал (дивіться нижче). Зауважте, що об'єкт може мати різні блоки даних F-кривих для Примусів -- Constraint (один на анімований примус) та Поз -- Pose (для арматур, один на анімовану кістку) і звідси дія може мати кілька цих каналів.

Редактор Дій -- Action Editor.

Заголовок -- Header¶

Попередній/Наступний Шар (іконки з трикутниками вниз і уверх)

Перемикає між різними діями, накладеними одна поверх іншої у стосі НЛА, без потреби перемикатися у Редактор НЛА та залишення режиму коригування та повторного входження в іншу смужку.

Клацання на кнопках зі стрілками уверх/вниз дає перехід по стежках НЛА для дій уверх/вниз відносно смужки НЛА, чия дія коригується у Редакторі Дій.

Якщо кілька одночасно дій/смужок знаходяться на одному шарі/стежці, то тільки буде використана одна найближча у поточному кадрі.

Ці оператори враховують устави для огляду/правки дії в ізоляції (тобто Соло -- Solo та Приглушення -- NLA Muting). Це було зроблено, щоб полегшити передогляд різних запасних дій.

  • Якщо перейти від стежки-соло до активної дії, то стос НЛА буде приглушено, щоб ця дія могла редагуватися в ізоляції.
  • Likewise, if the NLA stack is muted when editing the action, the NLA Track below it will be edited with solo enabled.
  • Якщо перемкнутися між стежками НЛА, то статус соло для попередньої стежки буде перенесено на нову стежку.

Ghi chú

These still work when you're not editing the action used by an NLA Strip. If you're just animating a new action normally, it is possible to use the "down arrow" to temporarily jump down to the previous action without losing the new action you're working on, and then use the "up arrow" to get back to it once you're done checking the other action(s).

Дія -- Action

Меню блоку даних дій -- Data-block menu.

Додання -- Add +
When an action is created it is stored in an NLA Action Stash.
Відв'язання -- Unlink X
При клацку Shift-LMB також зчищається Псевдо Користувач -- Fake User та вилучається запасена дія зі стосу НЛА.
Опущення -- Push Down (іконка з подвійною стрілкою вниз)
Додає активну дію на стос НЛА, як вкладну смужку. Це базово те ж саме, як натискання кнопки «Опущення» -- Push Down у Редакторі НЛА -- NLA Editor.
Stash (snowflake icon)
Запасає активну дію на стосі НЛА. тобто додає, як не вкладну стежку таким же чином, ніби ви створюєте нову дію, натомість.

Ghi chú

В обох цих випадках (Опущення -- Push Down і Запас -- Stash), після додання дії на стос НЛА вона очищається/відв'язується від гнізда активної дії (тобто вона не зможе бути редагована більше з редакторів дій/графів -- Action/Graph Editors, допоки ви не увійдете у «режим уточнення» для відповідних смужок пізніше).
