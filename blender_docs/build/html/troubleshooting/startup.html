Запуск -- Startup¶

Blender¶

Існують деякі поширені причини для проблем при використанні Blender'а. Якщо ви не можете знайти рішення проблеми тут, спробуйте попросити у спільноти про допомогу -- community.

Якщо Blender падає при запуску, існує кілька моментів, які слід перевірити:

  • Подивіться, чи ваш комп'ютер відповідає мінімальним вимогам для Blender'а -- minimum requirements.
  • Переконайтеся, що ваша відеокарта та її драйвери підтримують OpenGL версії мінімум 2.1.
  • Впевніться, що ви використовуєте коректну версію Blender'а (32- чи 64-бітну) для вашої архітектури.

Відомі причини перелічені нижче.

Python¶

Якщо ви отримали таку помилку при запуску, як:

Fatal Python error: Py_Initialize: unable to load the file system codec

ви можливо маєте задати вашу змінну середовища PYTHONPATH для системи.

У цьому випадку, вбудований Python у Blender буде намагатися використовувати PYTHONPATH. Якщо версія Python відрізняється від версії, що використовує Blender, це призведе до падіння Blender'а при запуску.

Для вирішення цієї проблеми, або очистьте PYTHONPATH перед запуском Blender'а (може бути зроблено також за допомогою скрипту запуску) або задайте його відповідну сумісну з Python версію.
