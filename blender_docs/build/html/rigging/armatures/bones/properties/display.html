Панель «Показ» -- Display¶

Орієнтир -- Reference

Mode:Object and Pose Mode
Panel:Bone ‣ Display

Панель «Показ» -- Display.

Панель «Показ» -- Display дозволяє вам налаштовувати вигляд ваших кісток, даючи їм форму іншого наявного об'єкта зі сцени.

Схов -- Hide
Ховає виділену кістку у Режимі Об'єкта та Режимі Пози.
Каркас -- Wireframe
При увімкненні, кістка буде показуватися у каркасному режимі, незалежно від режиму відтінення оглядвікна. Корисно для не-перегороджування власних форм кісток у ланцюгах.

Власна Форма -- Custom Shape¶

Blender дозволяє вам задавати для кожної кістки арматури специфічну форму (у Режимі Об'єкта -- Object Mode та у Режимі Пози -- Pose Mode), використовуючи інший об'єкт як «шаблон». Щоб така власна форма була видимою, необхідно також, щоб стяг «Форми» -- Shapes на панелі «Арматура > Показ» -- Armature ‣ Display був увімкнений.

Опції¶

Власна Форма -- Custom Shape
Об'єкт, що визначає власну форму виділеної кістки.
Розмір Кістки -- Bone Size
Option not to use bones length, so that changes in Edit Mode don't resize the custom shape.
Масштаб -- Scale
Avoids having multiple custom shapes at different sizes.
У -- At
Кістка, що визначає трансформацію показу власної форми цієї кістки.

Робхід¶

Для призначення власної форми для кістки ви маєте:

  1. Перемикнутися у Режим Пози -- Pose Mode Ctrl-Tab.
  2. Виділити відповідну кістку, клацнувши на ній RMB.
  3. Перейти на панель Display та у полі опції «Власна Форма» -- Custom Shape вибрати попередньо вже створений у сцені 3D об'єкт; у цьому прикладі ми використали куб та конус. Ви можете додатково задати у полі At іншу кістку.

Арматура з формою, призначеною для кістки. Зверніть увагу на центр об'єкта конуса Cone.

Ghi chú

  • Ці форми ніколи не будуть рендеритися, зображуватися, як і інші кістки, вони видимі тільки у 3D Огляді.
  • Even if any type of object seems to be accepted by the Object field (meshes, curves, even metas...), only meshes really work. All other types just make the bone invisible; nothing is drawn...
  • Центр об'єкта форми буде у корені кістки (дивіться по корінь/верхівку тут bone page).
  • The object properties of the shape are ignored (i.e. if you make a parallelepiped out of a cube by modifying its dimensions in Object Mode, you will still have a cube-shaped bone...).
  • «Повздовжня вісь кістки» -- це Y й об'єкт форми завжди масштабується так, що одна Одиниця Blender розтягується вздовж всієї довжини кістки.
  • If you need to remove the custom shape of the bone, just right-click in the Custom Shape field and select Reset to default value in the pop-up menu.

Отже, підсумовуючи усе це, ви повинні використовувати тільки сіті як об'єкти форми з їх центром у нижньому кінці по осі Y та загальною довжиною по Y в 1.0 BU.
