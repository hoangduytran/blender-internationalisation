Іменування -- Naming¶

Орієнтир -- Reference

Mode:Edit Mode
Panel:Properties region ‣ Item, Bones tab ‣ Bones panel

Ви можете перейменовувати ваші кістки за допомогою поля «імені» -- name на панелі «Елемент» -- Item в регіоні Властивості 3D Огляду або використовуючи поле name у вкладці «Кістка» -- Bone Редактора Властивостей у Режимі Правки -- Edit Mode, для активної кістки.

Blender також надає вам деякі засоби, що використовують переваги іменованих кісток для симетрії зліва/справа, а також що можуть автоматично іменувати кістки арматури. Давайте гляньмо на це детальніше.

Умовності іменування або називання -- Naming Conventions¶

Умовності іменування або називання у Blender'і корисні не тільки для пошуку вами потрібної кістки, але також вказують Blender'у, коли будь-які дві з них є дзеркальними відображеннями.

In case your armature can be mirrored in half (i.e. it is bilaterally symmetrical), it is worthwhile to stick to a left/right naming convention. This will enable you to use some tools that will probably save you time and effort (like the X-Axis Mirror editing tool).

Приклад іменування кісток за умовністю «зліва/справа» на простій оснастці.

  1. Спершу ви повинні дати своїм кісткам значущі базові імена, як "leg" -- «нога», "arm" -- «рука», "finger" -- «палець», "back" -- «спина», "foot" -- «ступня» тощо.

  2. Якщо у вас є кістка, яка має копію з іншої сторони (пара), як рука, то додайте до через наступні роздільники такі префікси або суфікси:

    • Роздільники для «зліва/справа» можуть спереду у другій позиції -- "L_calfbone" або позаду у передостанній позиції -- "calfbone.R"
    • Якщо вибрано однаково для обох з пари нижній або верхній регістр літер "L", "R", "left" або "right",то Blender обробляє відображення коректно. Дивіться нижче про список допустимих роздільників. Виберіть один та дотримуйтеся його максимально при оснащенні: це окупиться.

    Examples of valid separators:

    • (нічого): handLeft --> handRight
    • _ (підкреслення): hand_L --> hand_R
    • . (крапка): hand.l --> hand.r
    • - (тире): hand-l --> hand-r
    • `` `` (пробіл): hand LEFT --> hand RIGHT

    Ghi chú

    Note that all examples above are also valid with the left/right part placed before the name. You can only use the short "L"/ "R" code if you use a separator (e.g. "handL"/ "handR" will not work!).

  3. Перед тим, як Blender обробляє арматуру для дзеркалення або перевертання, то він спершу вилучає числове розширення в імені кісток, наприклад, ".001".

  4. Ви можете копіювати кістку, іменовану "bla.L" та перевернути її ім'я за допомогою «Спеціальне > Перевернути Імена» -- Specials ‣ Flip Left-Right Names. Blender іменує копію як "bla.L.001" та при переверті імені змінить його на "bla.R".

Перевернути Імена -- Flip Names¶

Орієнтир -- Reference

Mode:Edit Mode
Menu:Armature ‣ Flip Name, Specials ‣ Flip Name

Ви можете перевернути мітки «зліва/справа» (дивіться вище) для імен виділених кісток за допомогою «Арматура > Перевернути Імена» -- Armature ‣ Flip Name. Це може бути корисним, якщо ви вибудували половину симетричної оснастки (позначеної з лівого або правого боку), дублювали та віддзеркалили її, та хочете оновити імена на новій стороні. Blender замінить текст в іменах кісток відповідно до вище згаданих умовностей іменування та вилучить числові розширення при можливості.

АвтоІм'я -- AutoName¶

Орієнтир -- Reference

Mode:Edit Mode
Menu:Armature ‣ AutoName Left/Right, Armature ‣ AutoName Front/Back, Armature ‣ AutoName Top/Bottom

Ці три команди, що починаються з AutoName, меню «Арматура» -- Armature та «Спеціальне» -- Specials, W, дозволяють вам автоматично додавати суфікс до усіх виділених кісток на основі позиції їх кореня відносно центра арматури та її локальних координат:

АвтоІм'я Ліве/Праве -- AutoName Left/Right
will add the ".L" suffix to all bones with a positive X-coordinate root, and the ".R" suffix to all bones with a negative X-coordinate root. If the root is exactly at 0.0 on the X axis, the X coordinate of the tip is used. If both joints are at 0.0 on the X axis, the bone will just get a period suffix, with no "L"/ "R" (as Blender cannot decide whether it is a left or right bone...).
АвтоІм'я Переднє/Заднє -- AutoName Front/Back
додасть суфікс ".Bk" до імен усіх костей, корінь яких розміщено на додатній, та суфікс ".Fr" до імен усіх костей, корінь яких розміщено на від'ємній стороні локальної осі Y арматури. Подібно, для координат кісток в 0.0 по Y осі, як і для команди AutoName Left-Right...
АвтоІм'я Верхнє/Нижнє -- AutoName Top/Bottom
додасть суфікс ".Top" до імен усіх костей, корінь яких розміщено на додатній, та суфікс ".Bot" до імен усіх костей, корінь яких розміщено на від'ємній стороні локальної осі Z арматури. Подібно, для координат кісток в 0.0 по Z осі, як і для команди AutoName Left-Right...
