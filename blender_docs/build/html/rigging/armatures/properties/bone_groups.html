Групи Кісток -- Bone Groups¶

Орієнтир -- Reference

Mode:Pose Mode
Panel:Properties editor ‣ Armature ‣ Bone Groups
Menu:Pose ‣ Bone Groups ‣ ...

Панель «Групи Кісток» -- Bone Groups.

This panel allows the creation, deletion and editing of Bone Groups. Bone Groups can be used for selection or to assign a color theme to a set of bones. In example to color the left parts of the rig as blue and right parts as red.

Активна Група Кісток -- Active Bone Group
Огляд списком -- List view Груп Кісток.

Набір Кольорів -- Color Set¶

Вибірник Набору Кольорів -- Color Set для кісток у групі та кнопки кольорів.

Ви можете призначати «колірну тему» для групи (кожна кістка у ній буде мати ці кольори). Пам'ятайте, що ви повинні увімкнути стяг «Кольори» -- Colors на панелі «Показ» -- Display цієї ж вкладки, щоб бачити ці кольори.

Набір Кольорів для кісток -- Bone Color Set

Меню вибору.

  • «Стандартні Кольори» -- Default Colors: Стандартні (сірі) кольори.
  • nn - Theme Color Set: Один з двадцяти передустав тем у Blender'і.
  • «Власний Набір» -- Custom Set: Власний набір кольорів, який є специфічним для кожної групи.
Звичайно -- Normal
Перша кнопка кольору -- це колір невиділених кісток.
Виділене -- Selected
Друга колірна кнопка -- це колір контуру виділених кісток.
Активне -- Active
Третя кнопка кольору -- це колір контуру активної (останньо виділеної) кістки.

Якщо ви змінюєте один з цих кольорів, то значення перемикається на «Власний Набір» -- Custom Set.

Призначення і виділення -- Assign and Select¶

У 3D Огляді -- 3D View, використовуючи записи меню «Поза > Групи Кісток» -- Pose ‣ Bone Groups та/або спливне меню «Групи Кісток» -- Bone Groups за Ctrl-G, ви можете здійснити:

Призначання -- Assign
Призначає виділені кістки в активну групу кісток. Важливо відзначити, що кістка може входити тільки в одну групу.
Вилучення -- Remove
Вилучає виділені кістки з активної групи кісток.
Виділення -- Select
Виділяє кістки, що входять в активну групу кісток.
Зневиділення -- Deselect
Знімає виділ з кісток, що входять в активну групу кісток.
