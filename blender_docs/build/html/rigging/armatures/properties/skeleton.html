Скелет -- Skeleton¶

Орієнтир -- Reference

Mode:All Modes
Panel:Armature ‣ Skeleton

Панель «Скелет» -- Skeleton.

У цій панелі ви можете впорядковувати набори костей по різних шарах для легшого ними маніпулювання.

Позиція¶

Радіокнопка для перемикання між Позицією Пози -- Pose Position та Позицією Спокою -- Rest Position.

У той час, як у Режимі Правки -- Edit Mode ви завжди будете бачити арматуру в її Позиції Спокою -- Rest Position, тоді як у Режимі Об'єкта -- Object Mode та Режимі Пози -- Pose Mode ви бачите її в позиції пози -- Pose Position (тобто, як вона була трансформована у Режимі Пози -- Pose Mode). Якщо ви хочете бачити арматуру в її Позиції Спокою у всіх режимах, то увімкніть кнопку Rest Position у вкладці Armature (як вона є у Режимі Правки -- Edit Mode).

Шари арматури¶

Кожна арматура має 32 «шари арматури», які дозволяють вам впорядковувати вашу арматуру, «групуючи» набори кісток по шарах; це працює подібно до шарів сцени (тих, що містять ваші об'єкти). Ви можете далі «переміщати» кістку на заданий шар, ховати або вмикати показ шару чи кількох шарів тощо.

Showing/Hiding Bone Layers¶

Only bones in active layers will be visible/editable, but they will always be effective (i.e. move objects or deform geometry), whether in an active layer or not. To (de)activate a layer, you have several options, depending in which mode you are in:

  • У всіх режимах використовуйте ряд маленьких кнопок на панелі «Скелет» -- Skeleton у вкладці «Арматура» -- Armature. Якщо ви хочете вмикати/вимикати кілька шарів за раз, то як і звичайно клацайте на них з утримуваною натисненою Shift...
  • У Режимі Правки -- Edit Mode та Pose Mode ви можете робити це також з 3D Огляду -- 3D View, скориставшись меню «Арматура > Змінити Шари Арматури» -- Armature ‣ Switch Armature Layers (Change Armature Layers) або «Поза > Змінити Шари Арматури» -- Pose ‣ Switch Armature Layers (Change Armature Layers) чи за скороченням the Shift-M для виклику спливного меню з тими ж описаними вище кнопками (знову ж, клацки Shift-LMB даватимуть (зне)виділення кількох шарів за раз).

Захищені Шари -- Protected Layers¶

Ви можете блокувати певний шар кісток для усіх замісників -- proxies вашої арматури, тобто усі кістки на цьому шарі будуть не редаговані. Для цього на панелі Skeleton клацайте Ctrl-LMB на потрібній кнопці й блокування відповідного шару буде увімкнене.

Захищені шари у заміснику відновлюються до устав замісника, що були при перезавантаженні або при касуванні.
