Панель «Бібліотека Поз» -- Pose Library Panel¶

Reference

Mode:All Modes
Panel:Properties editor ‣ Object ‣ Pose Library

Панель «Бібліотека Поз» -- Pose Library.

The Pose Library panel is used to save, apply, and manage armature poses. Pose Libraries are saved to Actions. They are not generally used as actions, but can be converted to and from.

Дія -- Action
Блок даних -- Меню блоку даних для Дій та Бібліотек Поз.
Бібліотеки Поз -- Pose Libraries

Блок Огляди списків з поз для активної Бібліотеки Поз.

Додання -- Add +

If a pose is added, a pose marker is created. Keys are only stored for selected bones. If no bones are selected, all bones are keyed.

Додати Нове -- Add New
Додає нову позу в активну Бібліотеку Поз з поточної пози арматури.
Додати Нове (Поточний Кадр) -- Add New (Current Frame)
Will add a pose to the Pose Library based on the current frame selected in the Timeline. In contrast to Add New and Replace Existing which automatically allocate a pose to an action frame.
Замінити Наявне -- Replace Existing
Замінює наявну позу в активній Бібліотеці Поз поточною позою арматури.
Застосувати Позу -- Apply Pose (іконка з лупою)
Застосовує активну позу до виділених позованих кісток.
Пристосувати Дію -- Sanitize Action (іконка з рятувальним кругом)
Робить дію підхожою для використання як Бібліотеки Поз. Це використовується для конвертування Дії у Бібліотеку Поз. Поза додається у Бібліотеці Поз для кожного кадру з ключкадрами.
Move (up/down arrow icon)
Moves the pose up/down in the list.

Menu¶

Reference

  • Додати Позу -- Add Pose. Shift-L.
  • Перейменувати Позу -- Rename Pose. Shift-Ctrl-L.
