Привиди -- Ghosts¶

Орієнтир -- Reference

Mode:All Modes
Panel:Armature ‣ Ghost

Ghosts are black outlines (more or less opaque) of the bones as they are positioned at certain frames. This visualization is only shown in Pose Mode.

In traditional cartoon creation, animators use tracing paper, to see several frames preceding the one they are working on. This allows them to visualize the overall movement of their character, without having to play it back. Blender features something very similar for armatures: the "ghosts".

Приклади привидів.¶

Опції¶

Панель «Привид» -- Ghost.

The ghosts settings are found in the Armature tab.

Тип
Навколо Кадру -- Around Frame
Це дає показ заданої кількості привидів перед та після поточного кадру. Привиди показуються більш безпрозорими для кадрів біля поточного кадру і більше прозорими для кадрів, що дальше від поточного кадру.
У Діапазоні -- In Range
Це дає показ привидів кісток арматури всередині заданого діапазону кадрів. Привиди показуються більш прозорими для перших кадрів діапазону і більше безпрозорими для останніх кадрів діапазону. Цей тип має чотири опції.
На Ключкадрах -- On Keyframes
This is very similar to the In Range option, but there are ghosts only for keyframes in the armature animation (i.e. frames at which you keyed one or more of the bones). So it has the same options as above, except for the Step one (as only keyframes generate ghosts). Oddly, the shading of ghosts is reversed compared to In Range -- from opaque for the first keyframe, to transparent for the last keyframe.
Діапазон -- Range
Ця кнопка з числом визначає, скільки привидів ви будете мати на обидвох «боках» (тобто значення 5 буде давати десять привидів, п'ять перед поточним кадром та п'ять після нього).
Старт -- Start, Кінець -- End
Ця числова кнопка визначає стартовий/кінцевий кадр діапазону (виключно). Зауважте, що на жаль це не може бути нульовим або від'ємним значенням, що означає, що ви можете бачити тільки привиди, починаючи з кадру 2 включно....
Крок -- Step
Ця кнопка з числом визначає, чи буде у кожному кадрі у вас привид (стандартне значення 1), чи один на два кадри, один на три кадри тощо.

Показ -- Display¶

Лише Виділене -- Selected Only
При увімкненні, ви будете бачити привиди тільки виділених кісток (інакше, кожна кістка в арматурі буде мати привиди...).

Наостанок, ці привиди є також активними при програванні анімації за Alt-A -- це корисно тільки з опцією Around Frame, звичайно ж, ....

Ghi chú

Там не має жодного «глобального перемикача» для вимкнення цієї функції показу. Щоб її вимкнути, вам необхідно або задати значення устави Ghost як 0 (для опції Around Frame), або однакове значення для устав Start та End (для двох інших типів привидів).
