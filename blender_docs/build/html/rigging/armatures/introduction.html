Вступ¶

Арматуру -- armature -- у Blender'і можна розглядати подібно до арматури реального скелета, і як реальний скелет, Арматура може складатися з багатьох кісток. Ці кістки можуть рухатися і все, що до них прикріплене або асоційоване з ними, буде рухатися та деформуватися подібним чином.

An "armature" is a type of object used for rigging. A rig is the controls and strings that move a marionette (puppet). Armature object borrows many ideas from real-world skeletons.

Your First Armature¶

Щоб побачити, про що ми тут розмовляємо, давайте спробуймо додати стандартну арматуру у сцену Blender'а.

(Зауважте, що подробиці про редагування арматур пояснюються у відповідному розділі тут -- armatures editing section).

Відкрийте стандартну сцену, і далі:

  1. Видаліть усі об'єкти у сцені.
  2. Переконайтеся, що 3D курсор знаходиться у початку світу, за допомогою Shift-C.
  3. Натисніть Numpad1, щоб бачити світ в огляді Спереду -- Front.
  4. Додайте одиничну кістку -- Single Bone («Додання > Арматура > Одинична Кістка» -- Add ‣ Armature ‣ Single Bone).
  5. Натисніть NumpadDelete, щоб бачити арматуру у максимальному присуненні.

Стандартна арматура.

Об'єкт арматури¶

Як ви можете бачити, арматура подібна до інших типів об'єктів у Blender'і:

  • It has an origin, a position, a rotation and a scale factor.
  • Вона має блок даних «Дані Об’єкта» -- Object Data, що може редагуватися у Режимі Правки -- Edit Mode.
  • Вона може бути пов'язана в інші сцени й ці ж дані арматури можуть бути повторно використані одночасно на кількох об'єктах.
  • Все анімування ви робите у Режимі Об'єкта -- Object Mode, що працює тільки з об'єктом цілком, а не з костями арматури (використовуйте для цього Режим Пози -- Pose Mode).

Оскільки, арматури призначені для позування статичної або анімованої сцени, то вони мають специфічний стан, що зветься «позиція спокою» -- "rest position". Це стандартна «форма» арматури, стандартні позиція/оберт/масштаб її кісток, як це задано у Режимі Правки -- Edit Mode.

У Режимі Правки -- Edit Mode ви завжди будете бачити арматуру у позиції спокою, тоді як у Режимі Об'єкта -- Object Mode та Режимі Пози -- Pose Mode ви зазвичай маєте поточну «позу» арматури (допоки ви не увімкнете кнопку «Позиція Спокою» -- Rest Position на панелі «Арматура» -- Armature).
