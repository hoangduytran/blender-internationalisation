Ràng Buộc Điểm Tựa -- Pivot Constraint¶

The Pivot constraint allows the owner to rotate around a target object. It was originally intended for pivot joints found in humans e.g. fingers, feet, elbows, etc.

Tùy Chọn -- Options¶

Pivot panel.

Mục Tiêu -- Target
Data ID for the selection of the object to be used as a pivot point.
Vị Trí Điểm Tựa -- Pivot Offset
Offset of pivot from target.
Dùng Điểm Tựa Khi -- Pivot When
Always, Z Rot, Y Rot...
Nội dung cần viết thêm.
