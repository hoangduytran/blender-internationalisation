Примус «Зчленування Твердого Тіла» -- Rigid Body Joint Constraint¶

The Rigid Body Joint constraint is very special, it is used by the physics part of the Blender Game Engine to simulate a joint between its owner and its target. It offers four joint types: hinge type, ball-and-socket type, cone-twist type, and generic six-DoF type.

Quan trọng

Цей примус працює тільки з рушієм ігор -- Game Engine.

Точка та осі зчленування визначаються та фіксуються відносно володільця. Ціль рухається так, наче вона була приклеєна до центральної точки палиці, а інший кінець цієї палиці обертається навколо точки зчленування/опори...

This constraint is of no use in most "standard" static or animated projects. However, you can use its results outside of the BGE, through the Game ‣ Record Animation. (see Rigid Bodies for more info on this topic).

Демонстраційний файл, що показує деякі з цих типів дивіться тут -- BGE-Physics-RigidBodyJoints.blend.

Ghi chú

Щоб цей примус працював належно, для обидвох об'єктів (володільця та цілі) необхідне увімкнення опції «Межі Зіткнення» -- Collision Bounds.

Опції¶

Панель примусу «Зчленування Твердого Тіла» -- Rigid Body Joint.

Ціль -- Target
Поле Ідентифікатор даних --- Data ID використовується для вибору цілі для примусу, і поки вона не вибрана, то примус буде не функціональним (фон поля імені буде забарвлений червоним).
Тип Опорточки -- Pivot Type
Куля -- Ball
Works like an ideal ball-and-socket joint, i.e. allows rotations around all axes like a shoulder joint.
Шарнір -- Hinge

Works in one plane, like an elbow: the owner and target can only rotate around the X axis of the pivot (joint point).

Ліміти -- Limits
Кутові ліміти для осі X.
Закрут Конуса -- Cone Twist

Similar to Ball, this is a point-to-point joint with limits added for the cone and twist axis.

Ліміти -- Limits
Кутові ліміти.
Загальні 6 Ступенів Свободи -- Generic 6 DoF

Works like the Ball option, but the target is no longer constrained at a fixed distance from the pivot point, by default (hence the six degrees of freedom: rotation and translation around/along the three axes). In fact, there is no longer a joint by default, with this option, but it enables additional settings which allow you to restrict some of these DoF:

Ліміти -- Limits
Лінійні та кутові ліміти для заданої осі (опорточки) в Одиницях Blender та градусах відповідно.
Об'єкт Нащадка -- Child Object
Normally, leave this blank. You can reset it to blank by right-clicking and selecting Reset to Default Value.
Пов'язане Зіткнення -- Linked Collision
При увімкненні, це вимкне зіткнення, виявлюване між володільцем та ціллю (у фізичному рушії ігор BGE).
Показати Опорточку -- Display Pivot
При увімкненні, це увімкне показ опорточки зчленування у 3D Огляді -- 3D View. Найбільш корисно особливо для типу зчленування Generic 6DOF!
Опорточка -- Pivot
Ці три поля з числом дозволяють вам переміщати опорну точку, у просторі володільця.
Вісь -- Axis
Ці три числові поля дозволяють вам обертати опорну точку, у просторі володільця.
