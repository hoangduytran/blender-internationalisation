Примус «Обчислювач Об'єкта» -- Object Solver Constraint¶

Примус «Обчислювач Об'єкта» -- Object Solver дає володільцю цього примусу локацію та оберт «руху обчисленого об'єкта».

The "solved object motion" is where Blender thinks the physical, real-world (tracked) object was, relative to the camera that filmed it.

Може використовуватися для додання сіті на відео, наприклад.

Ghi chú

Цей примус працює тільки після того, як ви уклали мінімум вісім міток та натиснули кнопку «Обчислити Рух Об'єкта» -- Solve object Motion. Ця кнопка розміщена тут -- «Редактор Відеокліпів > Полиця Інструментів > Обчислення > Обчислити Рух Об'єкта» -- Movie Clip Editor ‣ Tool Shelf ‣ Solve ‣ Solve Object Motion

Якщо показується кнопка Solve Camera Motion замість кнопки Solve Object Motion, то перейдіть на «Редактор Відеокліпів > регіон Властивості > Об'єкти» -- Movie Clip Editor ‣ Properties region ‣ Objects та перемкніться з камери на потрібний об'єкт.

Опції¶

Панель «Обчислювач Об'єкта» -- Object Solver.

Активний Кліп -- Active Clip
Receive tracking data from the active movie clip in the Movie Clip editor. If unchecked, an option appears to choose from the other clips.
Об'єкт -- Object
Передбачає вибір простеженого об'єкта для отримання даних трансформування з нього.
Камера -- Camera
Select the camera to which the motion is parented to (if active empty scene camera is used).
Задати Інверсію -- Set Inverse
Moves the origin of the object to the origin of the camera.
Зчистити Інверсію -- Clear Inverse
Переміщає початок об'єкта назад на місце, задане у Редакторі Відеокліпів «Полиця Інструментів > Обчислення > Орієнтація > Задати Початок» -- Tool Shelf ‣ Solve ‣ Orientation ‣ Set Origin.
Примус у F-Криву -- Constraint to F-Curve
Застосовує цей примус, створюючи Ключкадри для трансформацій.
