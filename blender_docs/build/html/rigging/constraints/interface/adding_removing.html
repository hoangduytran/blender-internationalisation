Додання/вилучення примусу -- Adding/Removing a Constraint¶

Те, що описано на сторінці про Примуси Об'єктів, може також бути застосовано до Примусів Кісток.

Вкладка¶

Орієнтир -- Reference

Mode:Object Mode
Меню:Properties Editor ‣ Constraint tab

Для додання примусу у стек клацніть на кнопці меню «Додати Примус Об'єкта» -- Add Object Constraint у вкладці «Примуси» -- Constraints.

Для вилучення примусу зі стеку клацніть на кнопці з "X" в його заголовку -- header.

Меню¶

Додати Примус (з Цілями) -- Add Constraint (with Targets)¶

Орієнтир -- Reference

Mode:Object Mode and Pose Mode
Меню:Object ‣ Constraint ‣ Add constraint (with Targets)
Hotkey:Shift-Ctrl-C

Adds a constraint to the active object. The type of constraint must be chosen from a pop-up menu, though it can be changed later from the Add Constraint (with Targets) Operator panel. If there is an other object selected besides the active one, that object will be the constraint target (if the chosen constraint accepts targets).

Коли використовується кістка з іншої арматури як ціль для примусу, то цей засіб загляне всередину цієї іншої не-активної арматури та використає її активну кістку, що задана була, коли арматура була у Режимі Пози -- Pose Mode.

Копіювати Примуси на Виділене -- Copy Constraints to Selected¶

Орієнтир -- Reference

Mode:Object Mode and Pose Mode
Меню:Object ‣ Constraint ‣ Copy Constraints to Selected Objects

Копіює Примуси активного об'єкта на решту з виділених з ним об'єктів.

Зчищення Примусів -- Clear Constraints¶

Орієнтир -- Reference

Mode:Object Mode and Pose Mode
Panel:Object ‣ Constraint ‣ Clear Object Constraints
Hotkey:Ctrl-Alt-C

Вилучає усі Примуси з виділених об'єкта(ів).

Простеження -- Track¶

Орієнтир -- Reference

Mode:Object Mode
Panel:Object ‣ Track
Hotkey:Ctrl-T

Засоби у цьому підменю, Ctrl-T, додають примуси простеження до виділених об'єктів; об'єкт цілі такого примусу буде активний об'єкт, тому для цього слід виділяти мінімум два об'єкти.

  • Damped Track Constraint -- примус «Згасальне Стеження»
  • Track To Constraint -- примус «Стеження За»
  • Lock Track Constraint - примус «Блоковане Стеження»
Зчистити Стеження -- Clear Track Alt-T
Вилучає усі примуси стеження -- Damped Track, Track To та Lock Track -- з виділених об'єктів.
Зчистити та Утримати Трансформування (Зчистити Стеження) -- Clear and Keep Transformation (Clear Track) Alt-T
Вилучає усі примуси стеження з виділених об'єктів, зберігаючи фінальне трансформування, спричинене ними.
