Вступ¶

Constraints are a way to control an object's properties (e.g. its location, rotation, scale), using either plain static values (like the "limit" ones), or another object, called "target" (like e.g. the "copy" ones).

Незважаючи на те, що примуси є корисними у статичних проектах, їх головне використання, очевидно, здійснюється в анімації.

  • Ви можете керувати анімацією об'єкта через цілі, що використовується його примусами (це форма непрямої анімації). Дійсно, ці цілі можуть далі керувати властивостями володільця примусу, і звідси, анімування цілей буде побічно анімувати володілець.
  • Ви можете анімувати устави примусу. наприклад, «Вплив» -- Influence або коли використовується кістка арматури як ціль, анімувати, де вздовж цієї кістки (між коренем і верхівкою) лежить реальна точка цілі.

Вони можуть давати ефект відстеження очима тенісиста тенісного м'яча, що перескакує по корту, дають змогу отримати спільне обертання коліс автобуса, допомагають ногам динозавра згинатися автоматично у колінах та полегшують захват рукою руків'я меча та розмах рукою з мечем.

Примуси, у Blender'і, працюють з об'єктами -- Objects та кістками -- Bones. Читайте детальніше про використання примусів в оснащенні у главі -- Armature chapter.

Примуси об'єкта -- Object Constraints.

Примуси Кісток -- Bone Constraints.

Примуси працюють у комбінації один з одним на основі та у порядку Стеку, стосу Примусів -- Constraint Stack.

Стек Примусів обчислюється зверху вниз.

Поради¶

Примуси -- це фантастичний спосіб додання деталізації та складності в оснастку.

Але будьте обережні та надто не поспішайте, нагромаджуючи примус на примус, оскільки ви можете втратите увесь сенс їх взаємодії один з одним.

Розпочинайте з простого. Ознайомтеся зсередини та ззовні одиничного примусу. Примус «Копіювання Локації» -- Примус «Копіювання Локації» -- Copy Location Constraint є добрим на початок примусом для дослідження примусів (тут, зокрема, є анімовані приклади з ним). Приділіть час для зрозуміння кожної фундаментальної концепції в основі його дії та інші примуси набудуть для вас більше значення.
