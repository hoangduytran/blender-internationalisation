Примус «Згасальне Стеження» -- Damped Track Constraint¶

Примус «Згасальне Стеження» -- Damped Track примушує одну з локальних осей володільця завжди вказувати у напрямку цілі Target. В інших програмах створення 3D ви можете знайти його під іменем примусу «Дивитися на» -- "Look at".

Опції¶

Панель примусу «Згасальне Стеження» -- Damped Track.

Ціль -- Target
Ідентифікатор даних --- Data ID used to select the constraint's target, and is not functional (red state) when it has none.
У - To
Після того, як було покладено на об'єкт володільця примус Damped Track, ви повинні далі вибрати, яка вісь цього об'єкта має вказувати на об'єкт Цілі. Ви можете вибирати між 6-ма напрямками осей -- (-X, -Y, -Z, X, Y, Z). Від'ємний напрямок осі спричиняє те, що об'єкт буде вказувати від об'єкта Цілі вздовж обраного напрямку осі.
