Вузол «Трансформація» -- Transform Node¶

Вузол «Трансформація» -- Transform Node.

Це вузол поєднує у собі функціональність трьох інших вузлів: Масштаб -- Scale, Пересув -- Translate та Оберт -- Rotate.

Уводи -- Inputs¶

Зображення -- Image
Стандартний увід зображення.
X, Y
Використовуються для переміщення увідного зображення горизонтально і вертикально.
Кут -- Angle
Використовується для обертання зображення навколо його центру. Додатні значення обертатимуть за годинниковою стрілкою, а від'ємні -- проти годинникової стрілки.
Масштаб -- Scale
Використовується для зміни розміру зображення. Дане масштабування є відносним, тобто, значення 0.5 дає половину, а значення 2.0 дає подвоєння розміру відносно оригінального зображення.

Властивості¶

Фільтр -- Filter

Методи інтерполяції.

Найближче -- Nearest
Без інтерполяції, використовує найближчий сусідній піксель.
Білінійно -- Bilinear
Проста інтерполяція між суміжними пікселями.
Бікубічно -- Bicubic
Найвища якість інтерполяції.

Виводи -- Outputs¶

Зображення -- Image
Стандартний вивід зображення.
