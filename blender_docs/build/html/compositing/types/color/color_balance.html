Вузол «Колірний Баланс» -- Color Balance Node¶

Вузол «Колірний Баланс» -- Color Balance може коригувати колір та значення зображення.

Вузол «Колірний Баланс» -- Color Balance Node.

Уводи -- Inputs¶

Фактор -- Factor
Керує величиною впливу цього вузла на вивідне зображення.
Колір -- Color
Стандартний увід зображення.

Властивості¶

Може бути вибрана одна з двох різних формул корекції.

Підйом/Гамма/Підсил -- Lift/Gamma/Gain
Підйом -- Lift
Збільшує значення темних кольорів.
Гамма -- Gamma
Коригує середні тони.
Підсил -- Gain
Коригує світлі тони, підсвітки.
Зсув/Потужність/Схил -- Offset/Power/Slope (ASC-CDL)
Зсув -- Offset

Сумовано. (Коригує загальну яскравість).

Базис -- Basis
Додатковий зсув, дозволяє вказувати від'ємне значенння для Зсуву.
Потужність -- Power
Загальна експонента. (Коригує середні тони).
Схил -- Slope
Множник. (Коригує світлі тони, підсвітки).

Виводи -- Outputs¶

Колір -- Color
Стандартне вивідне зображення.

Просунуте¶

Формула «Зсув/Потужність/Схил» -- Offset/Power/Slope¶

out = (i × s + o)p

де:

  • out: Колір-градуйоване кодове значення пікселя.
  • i: Увідне кодове значення пікселя (від 0 до 1) (від чорного до білого).
  • s: Схил (0 або будь-яке більше число, номінальне значення -- 1.0).
  • o: Зсув (будь-яке число, номінальне значення -- 0).
  • p: Потужність (будь-яке число більше за 0, номінальне значення -- 1.0).
