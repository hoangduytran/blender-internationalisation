Вузол «Векторний Розмив» -- Vector (Motion) Blur Node¶

Вузол «Векторний Розмив» -- Vector Blur Node.

Вузол «Векторний Розмив» -- Vector Blur -- це найшвидший метод для імітування «розмиву рухом» -- Motion blur. Він використовує прохід рендера «векторна швидкість» -- vector speed для розмивання пікселів зображення у 2D.

Уводи -- Inputs¶

Зображення -- Image
Увід зображення, пов'язується з проходом рендера "Combined".
Z
Глибина Z, пов'язується з проходом рендера "Depth".
Швидкість -- Speed
Увід для проходу зображу «Вектор» -- "Vector". Дивіться проходи зображу Cycles або проходи зображу Blender internal.

Властивості¶

Вибірки -- Samples
Фактор якості.
Розмив -- Blur
Фактор масштабування для вектора руху (фактично «швидкість затвора» у кадрах).
Швидкість -- Speed

Векторний розмив може продукувати артефакти, такі як стрічки, лінії тощо. Для боротьби з цими проблемами даний фільтр застосовує затискання, яке може використовуватися для лімітування того, які пікселі повинні розмиватися. Швидкість задається в одиницях пікселів.

Максимум Швидкості -- Maximum Speed
Максимальний поріг швидкості. Більшість артефактів спричиняються пікселями, що надто швидко рухаються.
Minimum Speed
Мінімальний поріг для рухомих пікселів може відокремлювати важко рухомі пікселі від інших рухомих пікселів. Особливо, коли рухається сама камера, векторна маска може ставати цілим зображенням.

Виводи -- Outputs¶

Зображення -- Image
Вивід зображення з розмитим рухом.

Використання¶

Навіть з коректним укладом компонування з під'єднанням вузлів до всіх відповідних проходів Image, Z та Speed, все ще можливі артефакти. Ці 2D проходи рендера не містять 3D інформації, і тому ця інформація, що є позаду рухомого об'єкта або поза оглядом камери, втрачається.

Кращі результати можуть бути досягнуті шляхом рендерингу сцени у кілька проходів рендера, застосування векторного розмиву до кожного проходу рендера, а потім компонування результатів разом. Типово анімований персонаж буде рендеритися в окремих шар рендера, ніж набір фону. Особливо, якщо задіяне волосся або прозорість є важливим.

Для інших артефактів може допомогти деяке розмивання проходу Speed або задання ліміту для Maximum Speed. Це допомагає згладити рух, але надмірне розмивання призводить до своїх власних проблем.

Приклад¶

Цей speed vector у цьому прикладі було створено шляхом анімування узорчастої сфери горизонтально та використання кадру у серединній точці послідовності.

Результат рендера, без після-обробки.

Композиція, із заданими samples як 32 та blur як 1.0.
