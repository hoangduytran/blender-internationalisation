Вузол «Фільтр» -- Filter Node¶

Вузол «Фільтр» -- Filter Node.

Вузол «Фільтр» -- Filter node надає різні загальні фільтри покращення зображення.

Уводи -- Inputs¶

Фактор -- Factor
Керує величиною впливу цього вузла на вивідне результатне зображення.
Зображення -- Image
Стандартний увід зображення.

Властивості¶

Тип

Типи Soften, Laplace, Sobel, Prewitt та Kirsch здійснюють виявлення країв (дещо різними способами), базуючись на обчисленні вектора та наборі теоретичних рівнянь.

Пом'якшення -- Soften
Трохи розмиває зображення.
Різкішання -- Sharpen
Збільшує контраст, особливо на краях.
Лаплас -- Laplace
Пом'якшує навколо країв.
Собель -- Sobel
Створює негатив зображення з підсвіткою країв.
Прюїтт -- Prewitt
Намагається робити, те що робить «Собель», але краще.
Кірш -- Kirsch
Дає краще змішання, ніж «Собель» і «Прюїтт», при наближенні до країв.
Тінь -- Shadow
Здійснює рельєфний, карбувальний ефект, затемнюючи зовнішні краї.

Виводи -- Outputs¶

Зображення -- Image
Стандартний вивід зображення.

Приклад¶

Тут показані сім режимів вузла «Фільтр» -- Filter.¶

Оригінальне зображення.

Пом'якшення -- Soften.

Різкішання -- Sharpen.

Лаплас -- Laplace.

Собель -- Sobel.

Прюїтт -- Prewitt.

Кірш -- Kirsch.

Тінь -- Shadow.
