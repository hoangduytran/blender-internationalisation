Вузол «Двобічний Розмив» -- Bilateral Blur Node¶

Вузол «Двобічний Розмив» -- Bilateral Blur Node.

Вузол «Двобічний Розмив» -- Bilateral Blur Node здійснює високо-якісний адаптивний розмив на джерельному зображенні, дозволяючи розмити зображення, зберігаючи їх різкі краї.

Він може використовуватися для різних цілей, як: згладження шумнихпроходів рендера для запобігання тривалішому часу на обчислення у прикладі з промінь-простеженою загородою оточення, розмитих рефракцій/відбиттів, м'яких тіней або для виготовлення не-фотореалістичних композиційних ефектів.

Уводи -- Inputs¶

Зображення -- Image
Стандартний увід зображення. Якщо дані зображення подано лише на цей роз'єм, то розмив цим вузлом зображення залежить від країв елементів, присутніх на джерельному зображенні.
Детермінатор -- Determinator
Не обов'язковий для задання і, якщо дані подані на роз'єм «Детермінатор», то він слугує в якості джерела для визначення країв/границь для розмиву на зображенні. Це дає велику перевагу у випадку, якщо джерельне зображення є надто шумним, але нормалі у комбінації із Z-буфером все ще можуть визначити границі/краї об'єктів.

Властивості¶

Повтори -- Iterations
Визначає, скільки разів цей фільтр повинен здійснювати операцію розмиву на зображенні. Він практично визначає радіус розмиву.
Сигма Кольору -- Color Sigma
Визначає поріг, за яким відмінності у кольорі на зображенні повинні братися, як краї.
Сигма Простору -- Space Sigma
Змінна точності радіусу розмиву.

Виводи -- Outputs¶

Зображення -- Image
Стандартний вивід зображення.
