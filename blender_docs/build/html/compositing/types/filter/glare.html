Вузол «Блиск» -- Glare Node¶

Вузол «Блиск» -- Glare Node.

Вузол «Блиск» -- Glare використовується для додання відблисків в об'єктиві, туману, сяяння навколо освічених частин зображення, а також для багато чого іншого.

Уводи -- Inputs¶

Зображення -- Image
Стандартний увід зображення.

Властивості¶

Тип Блиску -- Glare Type¶

Привиди -- Ghosts
Створює ефект димки, серпанку на зображенні.
Стрічки -- Streaks

Створює яскраві стрічки, що використовуються для імітації відблисків в об'єктиві.

Стрічки -- Streaks
Загальна кількість стрічок.
Зсув Кута -- Angle Offset
Фактор зсуву обертання стрічок.
Зникання -- Fade
Фактор щезання, зникання для стрічок.
Сяяння Туману -- Fog Glow

Виглядає подібно до типу «Привиди» -- Ghosts, однак, він набагато менше за розміром та дає більше атмосферної димки, серпанку або «сяяння» по зображенню.

Розмір -- Size
Масштаб сяяння відносно розміру яскравих пікселів оригіналу.
Проста Зірка -- Simple Star

Працює подібно до типу «Стрічки» -- Streaks, але дає простішу форму, схожу на зірку.

Зникання -- Fade
Фактор щезання, зникання для стрічок.
Оберт 45 -- Rotate 45
Обертає стрічки на 45°.

Загальні Опції¶

Якість -- Quality
Якщо задано щось інше, ніж «Висока» -- High, то ефект сяяння буде застосовано тільки до низько-роздільної копії зображення. Це може бути корисним для економії часу на зображування, рендеринг, для зроблення тільки передогляду зображів, рендерів.
Повтори -- Iterations
Кількість разів виконання через алгоритм фільтра. Вищі значення дадуть більш точні результати, але й будуть довше обчислюватися. Зауважте, що це не доступно для типу «Сяяння Туману» -- Fog Glow, оскільки останній не використовує повторювано-базовий алгоритм.
Модуляція Кольору -- Color Modulation

Використовується для типів «Стрічки» -- Streaks та «Привиди» -- Ghosts для створення спеціального ефекту розсіювання, дисперсії.

Йоганнес Іттен (Johannes Itten) описує цей ефект, «Модуляція Кольору», як тонкі варіації у тонах і колірності.

Мікс -- Mix
Значення для керування тим, наскільки цей ефект буде додано у зображення. Значення -1 буде давати тільки оригінальне зображення, 0 дасть суміш 50/50, а 1 дасть тільки цей ефект.
Поріг -- Threshold
Пікселі, яскравіші за це значення, будуть задіяні цим фільтром блиску.
