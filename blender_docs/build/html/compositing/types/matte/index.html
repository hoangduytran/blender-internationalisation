Вузли типу «Маска» -- Matte Nodes¶

Ці вузли дають вам базові засоби для створення масок -- Matte для зображень, що не мають ще свого власного альфа-каналу -- Alpha Channel. Один з варіантів їх використання -- це компонування фільмоматеріалу, де дійство відзняте на синьому або зеленому тлі, із намальованою маскою або віртуальним фоном.

Загалом, підключайте ці вузли до вузла «Оглядач» -- Viewer, перемкніться на показ у Редакторі UV/Зображень -- UV/Image Editor вмісту цього вузла «Оглядач» та пограйтеся з їх повзунками, використовуючи зразок зображення з фільмоматеріалу, для отримання потрібних устав, налаштувань. У деяких випадках, незначні регулювання можуть усувати артефакти або погіршення зображення фільмоматеріалу. Надто багато зелено може дати плоскість вигляду акторів на передньоплані або їх синюватість/пурпуровість.

Ви можете і повинні з'єднувати ці вузли разом, покращувати ваше маскування та корекцію кольору послідовними уточненнями, використовуючи потужності кожного з вузлів, для оперування виводом з попередніх інших вузлів. Вузол «Ключування» -- Keying Node є найближчим «все в одному» вузлом для роботи із зеленими екранами, але найкращі результати випливають з комбінування цих технік, вузлів.

Ghi chú

«Сміттєва Маска» -- це не вузол, а техніка відбору того, що повинно бути виключеним із зображення. Тут термін маска -- Mask використовується для ідентифікації вмісту, що повинен бути вилучений із зображення, і який не може бути вилучений автоматичним процесом, таким як, ключування колірності. Ця техніка використовується для вибору специфічного вмісту для вилучення або, навпаки, для грубого виділення суб'єкта, що має залишитися, та вилучення всього іншого.

Деякі вузли приймають сміттєву маску безпосередньо. Для тих випадків, коли неможливе безпосереднє її застосування, ви все ще можете застосувати цю техніку через віднімання сміттєвої маски з маски, генерованої відповідним вузлом.

Simple garbage mattes can be created with the Box Mask or the Ellipse Mask. More complicated matte shapes using a Double Edge Mask or using a Mask.
