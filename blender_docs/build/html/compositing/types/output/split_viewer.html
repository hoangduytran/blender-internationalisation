Вузол «Розділений Оглядач» -- Split Viewer Node¶

Вузол «Розділений Оглядач» -- Split Viewer Node.

Вузол «Розділений Оглядач» -- Split Viewer node приймає два зображення та показує їх пліч-о-пліч по горизонталі чи вертикалі у тлі компонівника або у редакторі UV/Зображень, як вивід вузла «Оглядач» -- Viewer Node.

Уводи -- Inputs¶

Зображення -- Image
Показане у правій або верхній половині, залежно від заданої осі.
Зображення -- Image
Показується, відповідно, у лівій або нижній половині, залежно від обраної осі.

Властивості¶

Вісь -- Axis
X або Y використовується як вісь розділу показу.
Фактор -- Factor
Відсотковий фактор, що установлює розподіл простору між цими двома зображеннями.

Виводи -- Outputs¶

Цей вузол немає жодних вивідних роз'ємів.

Gợi ý

Цей вузол може використовуватися для планування переходів між сценами за допомогою порівняння кінцевого кадру однієї сцени зі стартовим кадром іншої з метою переконання, що вони вирівняні між собою.
