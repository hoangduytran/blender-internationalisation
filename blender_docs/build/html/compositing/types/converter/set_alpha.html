Вузол «Задання Альфа» -- Set Alpha Node¶

Вузол «Задання Альфа» -- Set Alpha Node.

Вузол «Задання Альфа» -- Set Alpha Node додає альфа-канал у зображення.

Уводи -- Inputs¶

Зображення -- Image
Стандартний увід зображення.
Альфа -- Alpha
Величина Альфа може бути задана для цілого зображення за допомогою числового поля уводу або по-піксельно через роз'єм.

Властивості¶

Цей вузол наразі немає властивостей.

Виводи -- Outputs¶

Зображення -- Image
Стандартний вивід зображення.

Ghi chú

Цей вузол не є і не повинен мати загально-призначеним рішення проблеми компонування зображення, що не містить інформації про Альфа. Ви можливо забажаєте скористатися вузлами технік «ключування колірності» -- "Chroma Keying" або «ключування відмінності» -- "Difference Keying" (що обговорюються в іншому місці цього довідника). Цей вузол найбільш часто використовується (з підхожим уводом, що забезпечується через роз'єм) у тих проблемних випадках, коли ви не можете, з певних причин, застосувати згадані вище інші техніки безпосередньо.

Приклад¶

Зникання у Чорноту -- Fade To Black¶

Поширеною технікою переводу глядачів з однієї сцени або кадру на інший є «зникання у темноту» -- "fade to black". Як випливає з назви сцена зникає у чорний екран. Ви можете також скористатися «зниканням у білизну» -- "fade to white" або у будь-який інший колір за бажанням, але чорний є добрим нейтральним кольором, що легко сприймається очима та інтелектуально «скидає» увагу і розум глядача. Карта вузлів нижче показує, як отримати цей ефект за допомогою вузла «Задання Альфа».

Ефект «Зникання у Чорноту» -- Fade to black.

У прикладі вище альфа канал зображення з вихором ігнорується. Натомість вузол «Час» -- Time node уводить фактор з діапазоном від 0.00 до 1.00 на часовий діапазон у 60 кадрів, або приблизно 2 секунди, для вузла «Задання Альфа» -- Set Alpha. Зауважте, що крива часу має експоненційну форму, так що загальна чорнота буде проявлятися повільно, а потім швидше ближче до кінця. Вузол Set Alpha не потребує увідного зображення; натомість, використовується плоский (безвідтінковий -- shadeless) чорний колір. Вузол Set Alpha використовує увідний фактор часу та колір для створення чорного зображення, що має набір альфа значень від 0.00 до 1.00 у часовому проміжку 60-ти кадрів або, іншими словами, змінюється від повної прозорості до повної безпрозорості за цей час. Думайте про альфа, як про множник для пікселя, яскравість якого ви можете бачити. Ці два зображення комбінуються вузлом «Альфа Над» - Alpha Over на повну силу («Фактор» -- Factor задано як 1.00) для продукування композитного зображення. Таким чином, вузол Set Alpha буде, залежно від рендереного кадру, продукувати чорне зображення, що має певну ступінь прозорості. Укладіть та анімуйте, і ви матимете послідовність зображень, що зникає у чорноту за 2-секундний період.

Ghi chú

Жодна Інформація зі Сцени Не Використовується

Цей приклад карти вузлів не використовує жоден вузол «Шари Зображу» -- Render Layers. Для продукування цієї 2-секундної анімації жодна інформація зі сцени Blender'а не використовується. Це приклад використання потужних здатностей компонування у Blender'і окремо від його можливостей моделювання й анімування. (Вузол «Шари Зображу» може бути підставлено замість вузла «Зображення» -- Image і ця «мережа зникання» продукуватиме такий самий ефект).

Поява Титру -- Fade In a Title¶

Для уведення у вашу анімацію ви схочете, щоб була представлена назва (титр) вашої анімації на певному задньоплані. Ви можете зробити, щоб титр влітав на екран або появлявся на ньому. Для створення ефекту появи титру на екрані скористайтеся вузлом «Задання Альфа» -- Set Alpha разом з вузлом «Час» -- Time, як показано нижче.

Використання вузла Set Alpha для створення ефекту появи титру.

У прикладі вище крива вузла Time надає значення Альфа в увідний роз'єм. Поточний вузол Render Layer, який має титр в огляді, надає зображення. Як і перед цим, вузол Alpha Over змішує (використовуючи альфа значення) задньоплановий вихор та титр з альфа для продукування композитного зображення.

Забарвлення ЧБ Зображення¶

Використання вузла Set Alpha для забарвлення зображення.

У прикладі вище, відзначте, як синя барва забарвлює уведене зображення вихору. Ви можете використати вказівник кольору на вузлі «Задання Альфа» у карті вузлів такого роду для додання суцільного кольору у чорно-біле зображення.

У карті з цього прикладу використовується значення уводу «Альфа» -- Alpha на вузлі Set Alpha для задання бажаного ступеня забарвлення. Подається увідне зображення та вузол Set Alpha у вузол «Альфа Над» -- Alpha Over для забарвлення будь-якого чорно-білого зображення таким чином.
