Виділення -- Selecting¶

Select menu.

Surface selection in Edit Mode is very similar to NURBS curve selection. The basic tools are the same as with meshes, so you can select a simple control point with an LMB-click, add to current selection with Shift-LMB clicks, B order-select, and so on.

Меню «Виділ» -- Select Menu¶

Меню Select (заголовок 3D Огляду -- 3D View) є навіть простішим, ніж для кривих...

Усі ці опції мають те ж значення і поведінку, як у режимі об'єкта -- Object Mode (особливості Border Select у режимі правки -- Edit Mode вже обговорювалися тут -- here).

Виділити Пов’язане -- Select Linked¶

L, Ctrl-L будуть додавати у виділ найближчу до курсора миші керувальну точку і всі пов'язані з нею, наприклад, усі точки, що належать до однієї й тієї ж поверхні.

Ряд Керувальних Точок -- Control Point Row¶

Орієнтир -- Reference

Mode:Edit Mode
Menu:Select ‣ Control Point Row
Hotkey:Shift-R

This option works a bit like edge loop selection for meshes, inasmuch it selects a whole row of control points, based on the active (the last selected) one. The first time you press Shift-R, the V row passing through (containing) the active point will be added to the current selection. If you use again this shortcut, you will toggle between the U and V row of this point, removing everything else from the selection.

Більше -- More та Менше -- Less¶

Орієнтир -- Reference

Mode:Edit Mode
Menu:Select ‣ More/Less
Hotkey:Ctrl-NumpadPlus / Ctrl-NumpadMinus

Ці дві опції є взаємно доповняльними та дуже подібні до аналогічних для сітей -- those for meshes. Їх призначення -- на основі поточно виділених керувальних точок -- скорочення або збільшення цього виділу.

Алгоритм є таким же самим, як для сітей:

Більше -- More
For each selected control point, select all its linked points (i.e. two, three or four).
Менше -- Less
For each selected control point, if all points linked to this point are selected, keep it selected. For all other selected control points, deselect them.

Це передбачає два моменти:

  1. Перше, коли усі керувальні точки поверхні виділені, то нічого не відбувається (оскільки для Less усі пов'язані точки завжди виділені, а, звичайно, More не може додати вже нічого). І навпаки, те ж саме відбувається, коли жодна з керувальних точок не виділена.
  2. Друге, ці засоби ніколи «не виходять за межі» поверхні (вони ніколи «не стрибатимуть» на іншу поверхню у цьому ж об'єкті).
