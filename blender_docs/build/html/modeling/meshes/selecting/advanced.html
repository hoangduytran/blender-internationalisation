Просунуте¶

Меню вибору у режимі редагування містить додаткові інструменти для виділення компонентів:

Гострі Ребра (Sharp Edges)
Цей інструмент вибирає всі ребра між двома гранями, що формують кут, більший ніж значення кута, де при його збільшенні вибираються гостріші ребра.
Linked Flat Faces Shift-Ctrl-Alt-F
Вибрати об'єднані грані на основі порога кута між ними. Це корисно для вибору плоских граней.
Віддзеркалити (Mirror)
Вибрати об'єкти сітки у дзеркальному розташуванні по вибраній осі.
Сторона Активного (Side of Active)
Вибрати всі вершини на сітці по одній осі відносно активної вершини. Тільки в Режимі Вибору Вершин (Vertex selection mode).
Пов'язані (Linked)
Selects all components that are connected to the current selection (see Select Linked).

Почергове скасування виділення (Checker Deselect)¶

Довідка

Mode:Edit Mode
Menu:Select ‣ Checker Deselect

Цей інструмент застосовує шаблон почергового виділення/скасування виділення. Це працює, тільки якщо ви вже вибрали більше одного елемента сітки.

Changes the current selection so that only every Nth elements (vertices, edges or faces, depending on the active selection mode) will remain selected, starting from the active one.

In case of islands of selected elements, this tool will affect only the island of the active element (if there is one), or the island of the first element in the order of internal storage (if there is no active element).

N-не Виділення (Nth Selection)
Skip every Nth element leaving it selected.
Пропустити (Skip)
Number of consecutive elements to skip (keep selected) at once.
Зміщення (Offset)
Offset from the starting point.

Вибрати Все за Ознаками (Select All by Traits)¶

Non Manifold Shift-Ctrl-Alt-M

Вибирає non-manifold геометрію сітки. Ця опція доступна при редагуванні сітки лише в режимах вибору Вершин і Ребер. Панель Операторів (Operator panel) надає кілька варіантів вибору:

Розширити (Extend)
Дозволяє розширити поточне виділення.
Дріт (Wire)
Вибирає всі ребра, що не належать жодній грані.
Межі (Boundaries)
Вибирає ребра на межах та дірах.
Кілька Граней (Multiple Faces)
Вибирає ребра, що належать трьом або більше граням.
Не Суміжні (Non Contiguous)
Вибирає ребра, що належать рівно двом граням з протилежними нормалями.
Вершини (Vertices)
Selects vertices that belong to wire and multiple face edges, isolated vertices, and vertices that belong to non-adjoining faces.
Внутрішні Грані (Interior Faces)
Вибирає грані, в яких усі ребра мають більше ніж дві грані.
Вибрати грані за сторонами (Select Faces by Sides)
Вибирає усі грані, що мають вказане число ребер.
Вільна геометрія (Loose Geometry)
Вибирає усі вершини або ребра, що не формують частину грані.
Розподілені вершини (Ungrouped Vertices)
Вибирає усі вершини, що не є частиною групи вершин (vertex group).

Вибрати Пов'язані (Select Linked)¶

Довідка

Mode:Edit Mode
Menu:Select ‣ Linked
Hotkey:Ctrl-L

Вибирає геометрію, що сполучена з уже вибраними елементами. Це часто корисно, коли сітка має від'єднані частини або частини, що перекриваються і їх ізолювання будь-яким іншим способом було б важко.

To give more control, you can also enable delimiters in the Operator panel, so the selection is constrained by seams, sharp edges, materials or UV islands.

За допомогою Вибрати Пов'язане (Pick Linked) ви також можете вибрати сполучену геометрію безпосередньо під курсором, використовуючи комбінацію клавіш L для вибору або Shift-L, щоб скасувати вибір.

Це працює інакше, бо використовує геометрію під курсором замість від існуючого вибору.

Вибрати подібні (Select Similar)¶

Довідка

Mode:Edit Mode
Menu:Select ‣ Similar...
Hotkey:Shift-G

Select geometry that has similar certain properties to the ones selected, based on a threshold that can be set in tool properties after activating the tool. Tool options change depending on the selection mode:

Режим виділення вершин (Vertex Selection Mode):
Нормаль (Normal)
Вибрати всі вершини, які мають нормалі, що вказують в аналогічних напрямках до поточних вибраних.
Кількість суміжних граней (Amount of Adjacent Faces)
Вибрати всі вершини, що мають однакову кількість прилеглих граней.
Групи вершин (Vertex Groups)
Вибрати всі вершини у тій же групі вершин (vertex group).
Кількість суміжних ребер (Amount of Connecting Edges)
Вибрати всі вершини, що мають однакову кількість прилеглих ребер.
Області граней (Face Regions)
Вибрати подібні особливості на сітці, яка має кілька подібних областей на основі топології.
Режим вибору ребер (Edge Selection Mode):
Довжина (Length)
Вибрати всі ребра, що мають аналогічну довжину до тих, що вже вибрані.
Напрямок (Direction)
Вибрати всі ребра, що мають аналогічний напрямок (кут) до тих, що вже вибрані.
Кількість граней навколо ребра (Amount of Faces Around an Edge)
Вибрати всі ребра, що належать однаковій кількості граней.
Кути Граней (Face Angles)
Вибрати всі ребра між двома гранями, що утворюють подібний кут, як у тих, що вже вибрані.
Згин (Crease)
Вибрати всі ребра, що мають подібне значення Згину (Crease), як у тих, що вже вибрані.
Фаска (Bevel)
Вибраті всі ребра, що мають таку ж Вагу Фаски (Bevel Weight), як у тих, що вже вибрані.
Шов (Seam)
Selects all edges that have the same Seam state as those already selected. Seam is a true/false setting used in UV texturing.
Загостреність (Sharpness)
Вибрати всі ребра, що мають такий же стан Загостреності (Sharp), як у тих, що вже вибрані. Загостреність (Sharp) - це налаштування (прапорець) правда/неправда, що використовується у Модифікаторі Розбиття Ребра (Edge Split Modifier).
Режим Вибору Граней (Face Selection Mode):
Матеріал (Material)
Вибрати всі грані, що використовують такий же матеріал, як і ті, що вже вибрані.
Зображення (Image)
Selects all faces that use the same UV texture as those already selected (see UV texturing pages).
Площа (Area)
Вибрати всі грані, що мають подібну площу до тих, що вже вибрані.
Сторони полігонів (Polygon Sides)
Вибрати всі грані, що мають однакову кількість ребер.
Периметр (Perimeter)
Вибраті всі грані, що мають подібний периметр (додані значення довжин ребер).
Нормаль (Normal)
Вибрати всі грані, що мають подібну нормаль до тих, що вже вибрані. Це спосіб вибору граней, що мають однакову орієнтацію (кут).
Копланарні (Co-planar)
Вибрати всі грані, що знаходяться у (майже) одній площині до тих, що вже вибрані.

Більше/Менше (More/Less)¶

Більше Ctrl-NumpadPlus
Розширити виділення до прилеглих елементів.
Менше Ctrl-NumpadMinus
Звузити виділення з прилеглих елементів.
Next Active Shift-Ctrl-NumpadPlus
Використовуючи історію вибору, виділити наступну вершину/ребро/грань на основі оточуючої топології.
Previous Active Shift-Ctrl-NumpadMinus
Вибір попереднього просто знімає виділення з останнього вибраного елемента.

Вибрати петлі (Select Loops)¶

Ви з легкістю можете вибирати петлі компонентів:

Петлі Ребер (Edge Loops)¶

Довідка

Mode:Edit Mode --> Vertex or Edge select mode
Menu:Select ‣ Edge Loop
Hotkey:Alt-RMB

Holding Alt while selecting an edge selects a loop of edges that are connected in a line end-to-end, passing through the edge under the mouse pointer. Holding Shift-Alt while clicking adds to the current selection.

Петлі ребер також можуть бути вибрані, виходячи з існуючого виділення, використовуючи або Вибрати (Select) ‣ Петля Ребер (Edge Loop), або опцію Вибрати петлю ребер (Edge Loop Select) із Edge Specials у меню Ctrl-E.

Ghi chú

Режим Вершин (Vertex)

У режимі вибору Вершин ви також можете вибирати петлі ребер, використовуючи ті ж гарячі клавіші, клацаючи по ребрах (не на вершинах).

Поздовжні та широтні петлі ребер (Longitudinal and latitudinal edge loops).

Ліва сфера показує ребро, що виділено поздовжньо. Зверніть увагу, що петля відкрита. Це відбувається тому, що алгоритм потрапляє на вершини на полюсах та завершується, оскільки вершини на полюсі з'єднані більш ніж чотирма ребрами. Однак, права сфера показує ребро, що було вибране широтно та сформувало замкнуту петлю. Це тому, що алгоритм повернувся до першого ребра, з якого почав.

Петлі Граней (Face Loops)¶

Довідка

Mode:Edit Mode --> Face or Vertex select modes
Hotkey:Alt-RMB

In face select mode, holding Alt while selecting an edge selects a loop of faces that are connected in a line end-to-end, along their opposite edges.

У режимі вибору вершин те ж саме можна виконати за допомогою Ctrl-Alt, щоб вибрати ребро, яке неявно вибирає петлю граней.

Вибір петлі граней (Face loop selection).

Ця петля граней була вибрана, клацаючи Alt-RMB на ребрі, у режимі вибору граней. Петля простягається перпендикулярно від обраного ребра.

Alt проти Ctrl-Alt у режимі вибору вершин.

Петля граней також може бути вибрана у режимі виділення вершин. Технічно, Ctrl-Alt-RMB вибере Петлю Ребер, однак, у режимі вибору вершин, вибираючи Петлю Ребер, неявно вибирається Петля Граней, тому що при виборі протилежних ребер грані неявно виділяється вся грань.

Граничні Ребра (Edge Boundary)¶

Довідка

Mode:Edit Mode --> Vertex or Edge select modes
Hotkey:Alt-RMB

Loop selection on edge boundaries. To extend the selection to all boundaries if the current boundary is already selected use Alt-RMB again.

Кільце Ребер (Edge Ring)¶

Довідка

Mode:Edit Mode
Menu:Select ‣ Edge Ring
Hotkey:Ctrl-Alt-RMB

У режимі вибору Ребер, утримуючи Ctrl-Alt при виборі ребра (або двох вершин), вибирає послідовність не приєднаних ребер, а тих, що знаходяться на протилежних один до одного сторонах, простягаючись вздовж петлі граней (face loop).

Як і в петлях ребер, ви також можете вибирати кільця ребер, основуючись на існуючому виділенні, використовуючи або Вибрати (Select) ‣ Кільце Ребер (Edge Ring), або опцію Вибрати кільце ребер (Edge Ring Select) із Edge Specials у меню Ctrl-E.

Ghi chú

Режим Вершин (Vertex)

У режимі вибору вершин ви можете використовувати ті ж гарячі клавіші, клацаючи на ребрах (не на вершинах), але при цьому будуть напряму вибиратися відповідні петлі граней...

Вибрана петля ребер та кільце ребер.

На мал. Вибрана петля ребер та кільце ребер. те ж саме ребро було натиснуто, але вибрано дві різні "групи ребер" на основі різних інструментів. Один базується на ребрах при обчисленні, а інший на гранях.

Найкоротший шлях (Shortest Path)¶

Довідка

Mode:Edit Mode
Menu:Select ‣ Shortest Path
Hotkey:Ctrl-RMB

Вибрати грань або вершину з допомогою Ctrl-RMB.

Виділити всю геометрію вздовж найкоротшого шляху від активної вершини/ребра/грані до вибраної.

Крокування гранями (Face Stepping)
Supports diagonal paths for vertices and faces, and selects edge rings with edges.
Топологічна відстань (Topological Distance)
Which only takes into account the number of edges of the path and not the length of the edges to calculate the distances.
Fill Region Shift-Ctrl-RMB
Вибирає всі елементи для найкоротшому шляху від активного виділення до натиснутої області.
Checker Select Options

Allows to quickly select alternate elements in a path.

N-не Виділення (Nth Selection)
Skip every Nth element, leave unselected.
Пропустити (Skip)
Number of consecutive elements to skip at once.
Зміщення (Offset)
Offset from the starting point.

Внутрішня область петлі (Loop Inner-Region)¶

Довідка

Mode:Edit Mode --> Edge select mode
Menu:Select ‣ Select Loop Inner-Region

Вибрати внутрішню область петлі (Select Loop Inner-Region) вибирає всі грані всередини закритої петлі ребер. Хоча цей оператор і можна використовувати у режимі вибору Вершин та Граней, результати можуть бути несподіваними. Зверніть увагу, що якщо вибрана петля ребер не закрита, всі пов'язані ребра сітки будуть розглядатися всередині петлі.

Петля в Область.

Як ви можете бачити, цей інструмент добре поводиться при кількох петлях.

Цей інструмент також підтримує "дірки".

Гранична Петля (Boundary Loop)¶

Довідка

Mode:Edit Mode --> Edge select mode
Menu:Select ‣ Select Boundary Loop

Вибір граничної петлі робить протилежне до Вибору внутрішньої області петлі, основуючись на всіх поточних вибраних областях, виділяє лише ребра на краю (контурі) цих островів. Може працювати у будь-якому режимі вибору, але для режиму Граней перейде до режиму Ребер після виконання.

All this is much more simple to illustrate with examples:

Вибір Граничних Ребер робить протилежне та вимушено перемикає в Режим Ввибору Ребер.
