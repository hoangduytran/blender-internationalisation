Опції Сіті -- Mesh Options¶

Орієнтир -- Reference

Mode:Edit Mode
Panel:Tool Shelf ‣ Options tab ‣ Mesh Options panel

X Mirror¶

The X Mirror option of the Mesh Options panel allows you edit symmetrical vertices on the other side of the mesh in a single action. When you transform an element (vertex, edge or face), if there is its exact X-mirrored counterpart (in local space), it will be transformed accordingly, through a symmetry along the local X axis.

Ghi chú

The conditions for X Mirror to work are quite strict, which can make it difficult to use. To have an exact mirrored version of a (half) mesh, its easier and simpler to use the Mirror Modifier.

Топологічне Дзеркалення -- Topology Mirror¶

Ghi chú

Для Topology Mirror спершу повинна бути увімкнена опція X-Mirror.

При використанні опції X-Mirror для роботи на дзеркаленій геометрії сіті вершини, що дзеркалені, повинні бути ідеально розміщені. Якщо вони не розташовані точно в їх дзеркалених локацій, то X Mirror не буде обробляти ці вершини як дзеркальні.

Topology Mirror намагається вирішити цю проблему, визначаючи які вершини є дзеркаленими вершинами не тільки за їх позиціями, а також розглядаючи як ці вершини знаходяться відносно інших у геометрії сіті. Він переглядає усю топологію для визначення, чи певні вершини будуть оброблятися як дзеркальні. Ефект від цього у тому, що дзеркалені вершини можуть бути не-симетричними та все ще оброблятися як дзеркалені, коли обидві опції X Mirror та Topology Mirror активні.

Ghi chú

Функціональність Topology Mirror буде працювати більш надійно на сіті, яка більш деталізована. Якщо ви використовуєте дуже просту геометрію, наприклад, стандартні Cube чи UV Sphere, то опція Topology Mirror часто не працює.

Приклад¶

Як приклад того, як використовувати Topology Mirror, відкрийте нову сцену у Blender'і, видаліть стандартни йкуб та додайте об'єкт Monkey у 3D Огляд.

  1. Натисніть Tab для перемикання для об'єкта Monkey у режим Edit Mode.
  2. При вимкненій опції X Mirror пересуньте одну з вершин об'єкта Мавпи трохи.
  3. Далі, увімкніть опцію X Mirror, але залиште опцію Topology Mirror вимкненою.
  4. Якщо ви тепер пересунете цю вершину знову, то X Mirror не буде працювати, а дзеркалені вершини не будуть змінені.
  5. Якщо ви далі увімкнете Topology Mirror та пересунете ці ж вершини знову, то X Mirror повинна все ще змінювати дзеркальну їй вершину, хоча вони і не ідеально розташовані.

Додаткові опції¶

Режим Виділу Ребра -- Edge Select Mode

Ця кнопка виділу вказує, що повинно робитися при виділення шляху вершин за допомогою Ctrl-RMB:

Виділ -- Select
Просто виділяє усі ребра у шляху.
Шов -- Seam
Позначає усі ребра у шляху як шви для розгортання по UV.
Гостро -- Sharp
Позначає усі ребра у шляху як гострі для модифікатора Edge Split.
Складка -- Crease
Позначає усі ребра у шляху як складки для модифікатора Subdivision Surface з вагомістю 1.0.
Скіс -- Bevel
Задає вагомість скосу 1.0 (для модифікатора Bevel) для усіх ребер у шляху.
Живе Розгортання -- Live Unwrap
Якщо увімкнене Live Unwrap, то кожен раз при зміні властивості шва для ребра буде автоматично повторно розраховувати розгортання по UV.
Поріг Дублів -- Double Threshold
Визначає максимальну відстань між вершинами, що об'єднані інструментом AutoMerge Editing.
