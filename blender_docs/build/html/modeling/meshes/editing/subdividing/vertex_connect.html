З'єднання Вершин -- Vertex Connect¶

З'єднати Вершини по Шляху -- Connect Vertex Path¶

Орієнтир -- Reference

Режим:Режим Правки -- Edit Mode
Меню:«Сіть > Вершини > З'єднати Вершини по Шляху» -- Mesh ‣ Vertices ‣ Connect Vertex Path
Скорочення:J

Цей інструмент з'єднує вершин у порядку, в якому вони були виділені, поділяючи грані між ними.

Коли виділені тільки дві вершини, то розріз буле зроблено через невиділені грані, подібно до того, як працює інструмент Knife; однак, це лімітується прямими розрізами через з'єднані грані.

Дві нез'єднані вершини.

Результат з'єднування.

Запуск у другий раз з'єднає першу/останню кінцеві точки.

Коли виділено багато вершин, то грані будуть поділені за їх виділеними вершинами.

Перед.

Після.

Вершини, не з'єднані з будь-якими гранями, будуть створювати ребра, а тому це може використовуватися як спосіб швидкого з'єднання ізольованих вершин також.

З'єднати Вершини -- Connect Vertices¶

Орієнтир -- Reference

Режим:Режим Правки -- Edit Mode
Меню:«Сіть > Вершини > З'єднати Вершини» -- Mesh ‣ Vertices ‣ Connect Vertices

Цей засіб з'єднує виділені вершини, створюючи ребра між ними та поділяючи грань.

Цей інструмент може використовуватися на багатьох гранях одночасно.

Вершини перед з'єднанням.

Після з'єднання вершин.

Результатна пара граней.

Головна відмінність між цим інструментом та Connect Vertex Path полягає у тому, що цей інструмент ігнорує порядок виділення та з'єднує усі виділені вершини, що є спільними для грані.
