Уштовх/Виштовх -- Push/Pull¶

Орієнтир -- Reference

Режим:Режими Об'єкта та Правки -- Object and Edit Modes
Панель:«Полиця Інструментів > Інструменти > Трансформація > Уштовх/Виштовх» -- Tool Shelf ‣ Tools ‣ Transform ‣ Push/Pull
Меню:«Об'єкт/Сіть > Трансформація > Уштовх/Виштовх» -- Object/Mesh ‣ Transform ‣ Push Pull

Відстань Push/Pull.

Push/Pull буде пересувати виділені елементи (об'єкти, вершини, ребра або грані) ближче один до одного (Push) або дальше один від одного (Pull). Зокрема, кожен елемент пересувається до або від центру на однакову відстань. Ця відстань керується рухом миші уверх (Push) або вниз (Pull), числовим уводом або через повзунковий керувальник.

Використання¶

Виділіть елементи, які ви хочете обробляти, та активуйте інструмент трансформації Push/Pull. Засіб Push/Pull задіюється за допомогою меню Object/Mesh ‣ Transform ‣ Push/Pull або через натискання Spacebar та використання меню пошуку оператора для знаходження Push або Pull. Величина пересування, заданого для виділу, може визначатися інтерактивно шляхом руху миші або уводом числового значення. Натискання Return підтвердить це трансформування.. Підтверджене трансформування може бути далі редаговане на панелі, яка відкривається натиском на F6 або на Полиці Інструментів, T, шляхом зміни повзунка Distance, при умові, що жодна інша дія не була здійснена між підтвердженням трансформації Push/Pull та доступом до цього повзунка.

Зауважте, що результат трансформації Push/Pull також залежить від кількості та типу виділених елементів (об'єкти, вершини, грані тощо). Дивіться нижче результат використання Push/Pull на різних елементах.

Рівновіддалені Об'єкти були уштовхані один до одного.

Випадково розміщені Об'єкти були уштовхані один до одного.

Уштовх Push (посередині) вершин навколо 3D курсора у порівнянні з масштабуванням Scale (справа).
