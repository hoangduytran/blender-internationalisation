Косування -- Shear¶

Орієнтир -- Reference

Режим:Режими Об'єкта та Правки -- Object and Edit Modes
Меню:«Об'єкт/Сіть/Крива/Поверхня > Трансформація > Косування» -- Object/Mesh/Curve/Surface ‣ Transform ‣ Shear
Скорочення:Shift-Ctrl-Alt-S

Косування -- це форма пересування, де паралельні поверхні рухаються мимо одна одної. У ході цієї трансформації пересування виділених елементів відбуватиметься уздовж горизонтальної осі поточного огляду. Локація осі буде визначатися опорною точкою -- Pivot Point. Все, що є «вище» цієї осі буде пересуватися (коситися) у тому ж напрямку, як і у вказівника миші (але завжди паралельно горизонтальній осі). Все, що є «нижче» горизонтальної осі буде пересуватися у протилежному напрямку.

Фактор скосу Offset для косування Shear.

Використання¶

Виділіть елементи, які ви хочете обробити, та активуйте інструмент трансформації Shear. Засіб Shear може бути задіяно з меню Object/Mesh/Curve/Surface ‣ Transform ‣ Shear або натисненням Shift-Ctrl-Alt-S. Величина пересування, заданого для виділу, може визначатися інтерактивно шляхом руху миші або уводом числового значення. Натискання Return підтвердить це трансформування.. Підтверджене трансформування може бути далі редаговане на панелі, яка відкривається натиском на F6 або на Полиці Інструментів, T, шляхом зміни повзунка Offset, при умові, що жодна інша дія не була здійснена між підтвердженням трансформації Shear та доступом до цього повзунка.

Зауважте, що результат трансформації Shear також залежить від кількості та типу виділених елементів (об'єкти, вершини, грані тощо). Дивіться нижче результат використання Shear на різних елементах.

Ефекти трансформації Shear з різними типами опорної точки Pivot Point.

Три зображення кадрів вище показують ефекти косування на виділених вершинах при зміні типу опорної точки. У середньому кадрі опорна точка -- Pivot Point задана як «Серединна Точка» -- Median Point, а миша у ході трансформації пересунута вліво. У правому кадрі тип Pivot Point задано як «3D Курсор», який розміщено над сіттю. Коли у ході трансформації Shear миша рухається вліво, то виділені вершини переміщуються управо, оскільки вони знаходяться нижче горизонтальної осі.

Mẹo

Величина трансформації Shear

Величина трансформації Shear, застосована до виділених елементів, є прямо пропорційною до відстані від горизонтальної осі. тобто, чи дальше від цієї осі, тим більше пересування.

Ефекти трансформації Shear на об'єктах з різними типами Pivot Point.

Три зображення кадрів вище показують ефекти косування на виділених об'єктах при зміні типу Pivot Point. У середньому кадрі тип Pivot Point задано як Median Point, а миша у ході трансформації пересунута вліво. У правому кадрі тип Pivot Point задано як «3D Cursor», який розміщено над сіттю. Коли у ході трансформації Shear миша рухається вліво, то усі виділені Об'єкти переміщуються управо, оскільки вони знаходяться нижче горизонтальної осі. Знову, зауважте, що величина трансформації є пропорційно до відстані від горизонтальної осі. У цьому випадку, нижчі Об'єкти пересунуться дальше ніж ті, що вище.
