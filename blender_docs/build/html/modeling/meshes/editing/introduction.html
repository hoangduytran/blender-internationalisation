Вступ¶

Blender надає різноманітні інструменти для правлення сітей. Це засоби, використовувані для додання, дублювання, пересування та видалення елементів.

These are available through the Mesh Tools panel in the Tool Shelf, the Mesh menu in the 3D View header, and context menus in the 3D View, as well as individual shortcut keys.

Ghi chú

All the "transform precision/snap" keys Ctrl and/or Shift work also for all these advanced operations... However, most of them do not have axis locking possibilities, and some of them do not take into account pivot point and/or transform orientation either.

Ці засоби трансформацій доступні у секції Transform меню Mesh у заголовку. Зауважте, що деякі з них можуть також використовуватися на інших редагованих об'єктах, як криві, поверхні та решітки.

Доступ до інструментів сітей¶

Інструменти сітей знаходяться у різних місцях, а також доступні за допомогою скорочень.

Полиця Інструментів: Інструменти Сітей -- Mesh Tools Shelf¶

Коли ви виділили сіть та натиснули Tab для переходу у Режим Правки, то Tool Shelf змінюється з Object Tools на Mesh Tools. Це лише деякі із засобів правлення сітей.

Меню -- Menus¶

Меню Mesh розміщується у заголовку. Деякі з цих меню можуть бути доступні за допомогою скорочень:

  • Ctrl-F викликає меню Faces
  • Ctrl-E викликає меню Edges
  • Ctrl-V викликає меню Vertices
