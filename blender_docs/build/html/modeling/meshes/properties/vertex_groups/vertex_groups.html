Панель Груп Вершин¶

Довідка

Mode:All Modes
Panel:Object Data tab ‣ Vertex Groups

The Vertex Group panel.

Групи Вершин підтримуються Редактором Властивостей (Properties Editor) Даних Об'єкта (Object Data) на панелі Груп Вершин (Vertex Groups).

Активна Група Вершин

A Огляди списків.

Заблокувати (Lock)
Locks the group from being editable. You can only rename or delete the group.
Додати +
Створити порожню групу вершин.
Видалити -
Видалити активну групу вершин.
Specials
Сортувати Групи Вершин (Sort Vertex Groups)
Сортує Групи Вершин у алфавітному порядку.
Копіювати Групу Вершин (Copy Vertex Group)
Додати Копію активної Групи Вершин як нову Групу. Вона буде називатися так само, як і оригінальна з приставкою "_copy (копія)". Та буде містити присвоєння тих самих вершин з такими ж вагами, як і вихідна група.
Копіювати Групи Вершин до Пов'язаного (Copy Vertex Groups to Linked)
Копіювати Групи Вершин цієї Сітки до всіх пов'язаних Об'єктів, що використовують ті ж дані сітки (mesh data) (всі користувачі даних).
Копіювати Групу Вершин до Вибраного (Copy Vertex Group to Selected)
Копіювати всі Групи Вершин до інших Вибраних Об'єктів з відповідними індексами (зазвичай підходить для копій сітки, що лише деформовані та не редаговані ніяк інакше).
Віддзеркалити Групу Вершин (Mirror Vertex Group)
Mirror all Vertex Groups, flip weights and/or names, editing only selected vertices, flipping when both sides are selected; otherwise copy from unselected.
Mirror Vertex Group (Topology)
The Топологічне Дзеркалення -- Topology Mirror option is enabled.
Видалити з Усіх Груп (Remove from All Groups)
(не доступно для заблокованих груп) Скасувати призначення вибраних Вершин з усіх груп. Після виконання цієї операції, вершини не будуть належати жодній групі вершин.
Очистити Активну Групу (Clear Active Group)
Видалити всі призначені вершини з активної Групи. Група стане порожньою. Зауважте, що вершини можуть досі бути призначені іншим Групам Вершин об'єкта. (не доступно для заблокованих груп).
Delete All Unlocked Groups
Remove all Vertex Groups from the Object that are not locked.
Видалити Всі Групи (Delete All Groups)
Видалити всі Групи Вершин з Об'єкта.
Заблокувати Все (Lock All)
Заблокувати всі групи.
Розблокувати Все (Unlock All)
Розблокувати всі групи.
Lock Invert All
Інвертувати Блокування Груп.

Редагування Груп Вершин (Editing Vertex Groups)¶

Довідка

Mode:All Modes
Panel:Object Data tab ‣ Vertex Groups
Menu:Mesh ‣ Vertex ‣ Vertex Groups
Hotkey:Ctrl-G

Панель Групи Вершин у Режимі Редагування або Малювання Вагів.

Коли ви переходите до Режиму Редагування (Edit Mode) або до- Режиму Малювання Вагів (Weight Paint Mode), ви можете редагувати ваги Вершин. Ті ж операції доступні і в меню 3D-переглядача Сітка (Mesh) ‣ Вершини (Vertices) ‣ Групи Вершин (Vertex Groups) або Ctrl-G.

Призначити (Assign)
Щоб призначити вибрані вершини активній групі з вагами, що визначені на Вагах (Weight) (див. нижче).
Видалити (Remove)
Щоб видалити вибрані вершини з активної групи (і, отже, видалити значення їхніх вагів).
Вибрати (Select)
Щоб вибрати всі вершини в групі.
Скасувати вибір (Deselect)
Щоб скасувати вибір всіх вершин у групі.
Вага (Weight)
Значення ваги, що призначається вибраним вершинам.
Встановити Активну Групу
Дозволяє вам вибрати групу, що стане активною (тільки в меню).

Gợi ý

Декілька об'єктів, що поділяють однакові дані, мають особливу властивість, коли назви груп зберігаються на об'єкті, а ваги - на сітці. Це дозволяє називати групи по-різному на кожному об'єкті, але будьте обережні, бо видалення групи вершин видалить цю групу з усіх об'єктів, що поділяють цю сітку.
