Модифікатор «Правка Нормалей» -- Normal Edit Modifier¶

Модифікатор Normal Edit задіює (або генерує) власні нормалі. Він використовує кілька простих параметричних методів для обчислення нормалей (досить корисних в галузях розробки ігор та архітектури) та змішує назад ці генеровані нормалі з вже наявними.

Опції¶

Модифікатор «Правка Нормалей» -- Normal Edit.

Режим -- Mode
Radial
Режим Radial -- Радіально вирівнює нормалі з вектором (початок, координати вершин), іншими словами, усі нормалі уявно виходять із заданої центральної точки, ніби вони були випромінені з еліпсоїдної поверхні.
Directional
Режим Directional -- Направлено змушує усі нормалі вказувати (збігатися) у напрямку заданого цільового об'єкта.
Цільовий Об'єкт -- Target Object

Використовує центр цього об'єкта як орієнтувальну точку при генеруванні нормалей.

Факультативно у режимі Radial, обов'язково у режимі Directional.

Паралельні Нормалі -- Parallel Normals

Робить усі нормалі паралельними до лінії між центрами обох об'єктів, замість збігання їх у напрямку центру цілі.

Актуально тільки у режимі Directional.

Зсув -- Offset

Задає центру модифікованого об'єкта зсув перед використанням його для генерування нормалей.

Актуально тільки у режимі Radial, якщо жоден Target Object не задано, та у режимі Directional, коли увімкнено опцію Parallel Normals.

Режим Міксу -- Mix Mode¶

Режим Міксу -- Mix Mode

Як впливають на наявні нормалі ново генеровані.

Зауважте, що опція Multiply не є векторним добутком, а просто множенням компонент-на-компонент.

Фактор Міксу -- Mix Factor
Наскільки генеровані нормалі змішуються з наявними.
Група Вершин -- Vertex Group
Дозволяє тонко керувати фактором міксу по кожному елементу. Вплив групи вершин може бути розвернуто, використовуючи маленьку кнопку з «двобічною стрілкою» справа.

Використання¶

Цей модифікатор може використовуватися для швидкого генерування радіальних нормалей для листя низько-полігонального дерева або «виправлення2 відтінення мультико-подібного рендерингу шляхом часткового згинання стандартних нормалей.

Єдиною обов'язковою умовою його використання є увімкнення опції «Авто Згладження» -- Auto Smooth на панелі Normals у властивостях сітей.

Mẹo

Більш складні маніпулювання нормалями можна отримати шляхом копіювання нормалей з однієї сіті на іншу, дивіться детальніше тут -- Data Transfer Modifier.

Example¶

Examples of edit custom normals to point towards a give direction, see example blend-file.

The left tree mesh has unmodified normals, while on the right one a Normal Edit modifier is used to bend them towards the camera.This shading trick is often used in games to fake scattering in trees and other vegetation.
