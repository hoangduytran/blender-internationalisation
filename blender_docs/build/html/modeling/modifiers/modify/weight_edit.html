Модифікатор «Правка Вагомостей Вершин» -- Vertex Weight Edit Modifier¶

Панель модифікатора Vertex Weight Edit.

Цей модифікатор призначений для редагування вагомостей однієї групи вершин.

Загальний процес наступний для кожної вершини:

  • (Факультативно) Він здійснює розкладання на основі однієї з наперед визначених функцій або власної кривої розкладання.
  • Він застосовує фактор впливу та факультативно групу вершин або маску текстури (0.0 означає оригінальну вагомість, 1.0 означає повну розкладену вагомість).
  • Він застосовує назад вагомість до вершини та/або може факультативно вилучати вершин з групи, якщо її вагомість нижче заданого порогу, або додавати її туди, якщо вагомість вище заданого порогу.

Quan trọng

Цей модифікатор робить неявне затискання значень вагомостей у стандартний діапазон (від 0.0 до 1.0). Усі значення нижче 0.0 будуть задані як 0.0 та усі значення вище 1.0 будуть задані як 1.0.

Опції¶

Група Вершин -- Vertex Group
Задіяна група вершин.
Стандартна Вагомість -- Default Weight
Стандартна вагомість для призначення усім вершинам, що не знаходяться у даній групі вершин.
Додати у Групу -- Group Add
Додає вершини з фінальною вагомістю Add Threshold у цю групу вершин.
Вилучити з Групи -- Group Remove
Вилучає вершини з фінальною вагомістю нижче Remove Threshold з даної групи вершин.
Тип Спаду -- Falloff Type

Тип розкладання:

Лінеарно -- Linear
Без розкладки.
Власна Крива -- Custom Curve
Дозволяє користувачу вручну визначати розкладання за допомогою кривої.
Гостро -- Sharp, Згладжено -- Smooth, Корінь -- Root та Сфера -- Sphere
Це класичні функції розкладання, від найгостріших до найбільш округлих.
Випадково -- Random
Використовує випадкове значення для кожної вершини.
Серединний Крок -- Median Step
Створює двійкові вагомості (0.0 або 1.0) з 0.5 як різальним значенням.

Опції Впливу/Маски -- Influence/Mask Options¶

Глобальний Вплив -- Global Influence

Загальний вплив модифікатора (0.0 буде залишати вагомості групи вершин недоторканими, 1.0 -- це стандартний вплив).

Quan trọng

Цей вплив впливає тільки на вагомості, доданню/вилученню вершин у/із групи вершин не перешкоджає задання цього значення як 0.0.

Маска Групи Вершин -- Vertex Group Mask¶

Маска Групи Вершин -- Vertex Group Mask
Додаткова група вершин, вагомості якої будуть помножені на значення глобального впливу кожної вершини. Якщо вершина не знаходиться у маскувальній групі вершини, то її вагомість не буде зачеплена.

Маска Текстури -- Texture Mask¶

Маска Текстури -- Texture Mask

Додаткова текстура, значення якої будуть множиться на значення глобального впливу для кожної вершини.

Це стандартний блок даних -- data-block керування текстурами. При установленні його показуються інші устави:

Координати Текстури -- Texture Coordinates

Як текстура розкладається на сіть.

Локально -- Local
Використовує локальні координати вершини.
Глобально -- Global
Вживає координати вершини у глобальному просторі.
Об'єкт -- Object

Використовує координати вершини у просторі іншого об'єкта.

Об'єкт -- Object
Об'єкт для використання як орієнтир для розкладки по Об'єкту.
UV

Використовує координати карти UV.

Карта UV -- UV Map
Карта UV, використовувана для розкладки по UV.
Вжити Канал -- Use Channel

Який канал використовувати як джерело фактора вагомості.

Червоний -- Red / Зелений -- Green / Синій -- Blue / Альфа -- Alpha
Значення одного з колірних каналів.
Інтенсивність -- Intensity
Усереднення каналів RGB (якщо значення RGB (1.0, 0.0, 0.0) є 0.33).
Значення -- Value
Найвище значення з каналів RGB (якщо значення RGB (1.0, 0.0, 0.0) є 1.0).
Відтінок -- Hue
Використовує значення відтінку зі стандартного круга кольорів (наприклад, синій має вище значення відтінку, ніж жовтий).
Насиченість -- Saturation
Вживає значення насиченості (наприклад, значення чистого червоного є 1.0, сірого - 0.0).

Ghi chú

Усі зі згаданих вище каналів є гамма-коректовані, за винятком Intensity.

Ghi chú

Ви можете переглядати модифіковані вагомості у режимі малювання вагомостей Weight Paint Mode. Це також неявно передбачає, що вам доведеться вимкнути цей модифікатор Vertex Weight Edit Modifier, якщо ви хочете бачити оригінальні вагомості групи вершин, які ви редагуєте.

Приклад¶

Використання відстані від геометрії цільового об'єкта

Ми збираємося проілюструвати це з допомогою модифікатора Displace.

Add a (10×10 BU) 100×100 vertices grid, and in Edit Mode, add to it a vertex group containing all of its vertices, as above. You can even further subdivide it with a first Subdivision Surface Modifier.

Тепер додамо коло кривої та помістимо його на 0.25 BU вище над сіттю. Збільшимо масштаб його трохи (наприклад, до 4.0 BU).

Повернімося до сітьового об'єкта, додамо для нього модифікатор Vertex Weight Proximity з режимом Geometry Distance. Увімкнемо Edge (якщо ви вживаєте тільки Vertex, і ваша крива має низьке визначення U, ви можете отримати хвилясті узори, дивіться Ілюстрацію Хвилясті узори.).

Хвилясті узори.¶

Відстані від ребер.

Відстань від вершин.

Задайте Lowest Distance як 0.2 та Highest Distance як 2.0 для розкладки назад обчислених відстаней у звичайний діапазон вагомостей.

Додайте третій модифікатор Displace та задійте бажану текстуру. Тепер, ми хочемо, щоб вершини сіті, найближчі до кола кривої, залишилися без зміщення. Оскільки вони отримають значення біля нуля, то це означає, що ви маєте задати для відстані Midlevel як 0.0. Зробіть це, використовуючи нашу задіяну групу вершин, і все! Ваші гарні гори просто звузяться до плоскої площини біля кола кривої.

Як і в попередньому прикладі. ви можете вставити модифікатор Vertex Weight Edit перед Displace та погратися з власною кривою Custom Curve розкладки для отримання ширшої/вужчої «долини»...

Варіації розкладки за допомогою Кривої.¶

Крива розкладання угнутого типу.

Без кривої розкладання (лінеарно).

Крива розкладання випуклого типу.

Вершини з обчисленою вагомістю нижче 0.1 вилучені з цієї групи вершин.

Ви можете також додати модифікатор Mask та увімкнути опцію Group Remove для Vertex Weight Edit з Remove Threshold як 0.1, щоб побачити як зникне низ вашої долини.
