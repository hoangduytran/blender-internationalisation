Модифікатор «Кеш Послідовності Сіті» -- Mesh Sequence Cache Modifier¶

Незважаючи на його назву, цей модифікатор підтримує сіті та криві. Він також обробляє послідовності файлів, а також сіті та криві з різною кількістю вершин/керувальних точок.

Опції¶

Файл Кешу -- Cache File

Меню блоку даних для вибору файлу формату Alembic.

Шлях Файлу -- File Path
Шлях до файлу Alembic.
Є Послідовність -- Is Sequence
Кеш розділений на серії файлів або ні.
Замістити Кадр -- Override Frame

Чи використовувати власний кадр для пошуку даних у файлі кешу замість використання поточного кадру сцени.

Кадр -- Frame
Час для використання у пошуку даних у файлі кешу або для визначення того, який використовується у послідовності файлів.
Масштаб Ручної Трансформації -- Manual Transform Scale
Значення, за яким збільшувати або зменшувати об'єкт відносно початку світу. (застосовне тільки через примус «Кеш Трансформацій» -- Transform Cache Constraint).
Шлях Об'єкта -- Object Path
Шлях до об'єкта Alembic всередині архіву.
Читати Дані -- Read Data

Тип даних для читання відповідно для об'єкта сіті: вершини, полігони, UV карти та шари Кольорів Вершин.

Вершини/Грані/UV/Колір -- Vertices/Faces/UV/Color
