Модифікатор «Вибух» -- Explode Modifier¶

Модифікатор Explode використовується для змінення геометрії сіті шляхом пересування/обертання її граней у спосіб, що грубо простежує частинки, випромінені цим об'єктом, роблячи вигляд її таким, ніби вона вибухає (розбивається і розлітається).

Щоб модифікатор Explode мав видимий ефект, на сіті повинна бути установлена система частинок. Система частинок на сіті -- це те, що керує тим, як сіть буде вибухати, і звідси, без системи частинок дана сіть не буде змінюватися.

Як кількість емітованих частинок, так і кількість граней визначають наскільки гранульованим буде модифікатор «Вибух». Чим більше граней і більше частинок, тим більше окремих шматків.

Here is a demo video showing a cube with a particle system and Explode Modifier. (Blend-file).

Ghi chú

Модифікатор Explode повинен стояти після модифікатора Particle System, оскільки модифікатор «Система Частинок» має інформацію, необхідну для керування модифікатором «Вибух».

Опції¶

Панель модифікатора Explode зі згорнутою панеллю модифікатора Particle System над нею.

Група Вершин -- Vertex Group

Вершини у цій групі можуть не задіюватися модифікатором Explode. Вершини з повною вагомістю взагалі не задіюються, тоді як вершин з меншою вагомістю мають вищий шанс бути задіяними.

Вершини без жодної вагомості будуть розглядатися, як такі, що не належать до цієї групи взагалі, та нормально вибухатимуть.

Захист -- Protect
Очищає краї групи вершин. Залежно від вагомостей, призначених для цієї групи вершин; або повністю захищає ці грані від задіння модифікатором Explode (що станеться, якщо ці грані мають значення вагомості 1), або повністю вилучає захист з цих граней (що відбудеться, якщо грані мають значення вагомості 0).
UV Частинок -- Particle UV
Карта UV для зміни віку частинок.
Різати Ребра -- Cut Edges
Ріже ребра граней для гарнішої шрапнелі.
Ненароджене -- Unborn
Показує сіть, коли частинки є ненароджені.
Живе -- Alive
Показує сіть, коли частинки є живими.
Мертве -- Dead
Показує сіть, коли частинки є мертвими.
Розмір -- Size
Використовує розмір частинки для шрапнелі.
Освіження -- Refresh
Освіжає дані у модифікаторі Explode.
