Модифікатор «Покрив» -- Skin Modifier¶

Модифікатор Skin використовує вершини та ребра для створення покривної поверхні на основі радіусу кожної вершини для кращого визначення форми. Вивід є здебільшого з квадратів, хоча деякі трикутники будуть з'являтися навколо самоперетинів.

Це швидкий спосіб генерувати базові сіті для ліплення та/або згладжених органічних форм з довільною топологією.

Ghi chú

Грані в оригінальній геометрії ігноруються цим модифікатором Skin.

Опції¶

Модифікатор «Покрив» -- Skin.

Створити Арматуру -- Create Armature¶

Створює арматуру поверх об'єкта. Кожне ребро стає кісткою.

Ghi chú

Якщо коренева кістка має більше, ніж одне суміжне ребро, то буде створена додаткова кістка для збереження її як кореня.

Цей засіб робить наступне:

  1. Додається новий об'єкт арматури з кістками відповідно до увідної сіті. Активне виділення перемикається на нову арматуру.
  2. Додаються групи вагомостей в увідну сіть. Модифікатор Skin поширює ці вагомості також на вивід.
  3. Додається модифікатор Armature безпосередньо нижче, під модифікатором Skin. Зауважте, що модифікатор Armature застосовується після модифікатора Skin, оскільки він повинен деформувати тільки вивід, а якщо, він буде над модифікатором Skin, то може змінювати результатну топологію.

Згладження -- Smoothing¶

Згладження Розгалуження -- Branch Smoothing
Точка розгалуження -- це вершина з трьома або більше з'єднаними ребрами. Ці області тяжіють продукувати більш ускладнену топологію, деякі з них можуть перекриватися. Устава Branch Smoothing розслабляє поверхню навколо цих точок з побічним ефектом стиснення, зморщення поверхні.
Згладжене Відтінення -- Smooth Shading
Грані виводу будуть мати скоріше згладжене відтінення, а не плоске відтінення. Згладжене/плоске відтінення граней уводу не зберігається.

Виділені Вершини -- Selected Vertices¶

Позначити/Зчистити Незв'язане -- Mark/Clear Loose
Стандартно, вершина розгалуження (вершина з трьома і більше з'єднаними ребрами) буде генерувати додаткові петлі ребер уздовж суміжних ребер, щоб зберегти щільність виводу. Відгалуження можуть бути зроблені незв'язаними клацком на кнопці Mark Loose, що дозволить виводу розтягатися між усіма суміжними вершинами. Це може бути вимкнуто знову клацком на кнопці Clear Loose з виділеною такою позначеною вершиною.
Позначити Корінь -- Mark Root

Позначення вершини як кореня спричиняє, що вершина буде використовуватися для розрахування обертань з'єднаних кінцівок. Кореневі вершини також впливають на вивід арматури; вони будуть використовуватися як початок для кореневих кісток.

Корені показують у 3D Огляді червоним пунктирним кільцем навколо вершини.

Кожен набір з'єднаних вершин повинен мати один кореневий вузол. Mark Root забезпечує правило одного кореня на набір, а тому вручну не слід знімати позначення коренів.

Вирівняти Радіуси -- Equalize Radii
Робить покривні радіуси виділених вершин рівними по кожній з осей.

Осі Симетрії -- Symmetry Axes¶

Стяги Symmetry Axes використовуються для збереження топології виводу симетричною по відповідних осях. Іншими словами, використання цього запобігає об'єднанню трикутників уздовж вказаних осей, допоки ці трикутники не формують симетричний квадрат.

Ghi chú

Ці стяги осей симетрії не додають геометрію, перевернуту через вісь. Для цього повинен використовуватися модифікатор Mirror, типово розміщений над модифікатором Skin.

Використання¶

Додайте модифікатор Skin для сіті. Вимкніть Limit selection to visible у 3D Огляді, щоб ви могли бачити вершини всередині нової геометрії. Переконайтеся, що цей модифікатор увімкнений для показу у режимі правки Edit Mode (стандартно увімкнено).

Модифікатор Skin використовує звичайні вершини та ребра як увід. Усі звичайні засоби режиму правки Edit Mode (такі як видавлення, підподілення, пересування, масштабування та обертання) можуть використовуватися при будуванні покривної сіті.

Перерозмір Покриву -- Skin Resize¶

Радіуси увідних вершин можуть індивідуально масштабувати у режимі правки Edit Mode для зміни товщини покриву через натискання Ctrl-A. Не-однорідне масштабування по осях X та Y доступне через зблокування його за допомогою X або Y. Такий радіус може також коригуватися на панелі Transform регіону Properties 3D Огляду.

Приклади¶

Просте створіння, зроблене за допомогою тільки модифікатора Skin.

  1. У меню вкладки Modifiers додайте модифікатор Modifier.
  2. Натисніть Tab для переходу у Режим Правки та почніть видавлення (extruding).
  3. Спробуйте нарисувати результат, подібний на Ілюстрації Просте створіння, зроблене за допомогою тільки модифікатора Skin., за допомогою видавлення сіті на вершинах об'єкта.
  4. Скористайтеся Ctrl-A для зміни розміру різних регіонів у межах створіння.
  5. Використайте Mark Loose у таких регіонах, як шия, для об'єднання по іншому цих граней.
  6. Для отримання більш згладженого результату, активуйте, клацніть на Smooth Shading та додайте модифікатор Subdivision Surface.
