Модифікатор «Каркас» -- Wireframe Modifier¶

Модифікатор Wireframe трансформує сіть у каркас, повторюючись по її гранях, збираючи усі ребра та перетворюючи ці ребра у чотирибічні полігони. Майте на увазі те, що ваша сіть повинна мати грані для формування каркасу. Ви можете визначити товщину, матеріал та кілька інших параметрів генерованого каркасу динамічно за допомогою опцій даного модифікатора.

Опції¶

Модифікатор «Каркас» -- Wireframe.

Товщина -- Thickness
Глибина або розмір проводів каркасу.
Зсув -- Offset
Значення між (-1 до 1) для зміни того, чи буде каркас генеровано всередині чи назовні оригінальної сіті. При заданні в нуль Offset буде центрувати каркас навколо оригінальних ребер.
Група Вершин -- Vertex Group

Обмежує дію цього модифікатора тільки в областях вказаної групи вершин.

Інвертування -- Invert
Інвертує вагомості даної групи вершин.
Фактор -- Factor
Відсоток того, який вершина має вплив на фінальний результат каркаса.
Складчасті Ребра -- Crease Edges

Ця опція призначена для використання цього модифікатора разом з модифікатором «Підподіл Поверхні» -- Subdivision Modifier. Вмикайте цю опцію для складок в ребрах на їх з'єднаннях для запобігання великим вигнутим перетинам.

Вагомість Складки -- Crease Weight
Визначає, наскільки задати складку (від 0 до 1) (від відсутності до повної), яку сполучення ребер повинні отримати.
Рівна Товщина -- Even Thickness
Підтримує товщину, коригуючи гострі кутки. Інколи підвищує якість, але також збільшує час на обчислення.
Відносна Товщина -- Relative Thickness
Визначає товщину ребра за довжиною цього ребра. Довші ребра будуть давати товстіші каркасні дроти.
Рубіж -- Boundary
Створює каркасні дроти на рубежах островів сіті.
Замінити Оригінал -- Replace Original
Якщо ця опція увімкнена, то оригінальна сіть замінюється генерованим каркасом. Якщо ні, то каркас генерується поверх неї.
Зсув Матеріалу -- Material Offset
Використовує вказаний індекс матеріалу як матеріал для каркасу; це застосовується як зсув від першого матеріалу.

Cảnh báo

Товщина каркасу є приблизною. Хоча "Even Thickness" повинна давати добрі результати у багатьох випадках, однак покривні грані можуть спричиняти потворні піки. У такому випадку ви можете або зменшити екстремальні кути в геометрії або вимкнути опцію Even Thickness.
