Модифікатор «Гачок» -- Hook Modifier¶

Модифікатор Hook використовується для деформування сітей -- Mesh, кривих -- Curve або решіток -- Lattice, використовуючи інший об'єкт (зазвичай, це порожняк -- Empty або кістка -- Bone, але це може бути будь-який об'єкт).

При русі гачка штовхаються вершини сіті. Ви можете думати про нього, як про анімовану пропорційну правку -- proportional editing.

Хоча гачки не дають тонкого керування переміщенням вершин, як це роблять ключі форм, однак вони мають перевагу у тому, що ви можете безпосередньо захоплювати вершини для маніпулювання.

Empty used as Hooks to deform a subdivided cube.

Опції¶

Модифікатор «Гачок» -- Hook.

Об'єкт -- Object
Ім'я об'єкта, який буде гачком для вершин.
Група Вершин -- Vertex Group

Allows you to define the influence per vertex.

Корисно, коли ви не робите щось інше, ніж сферичне поле впливу.

Радіус -- Radius
Розмір впливу гачка.
Сила -- Strength

Коригує вплив гачка на вершини, має значення у діапазоні (0.0 до 1.0) (без змін до повного слідування вершин за гачком).

Оскільки одночасно кілька гачків можуть працювати для одних і тих же вершин, то ви можете виважити цей вплив гачка за допомогою цієї властивості.

Тип Спаду -- Falloff Type
This can be used to adjust the kind of curve that the hook has on the mesh. You can also define a custom curve to get a much higher level of control.
Однорідний Спад -- Uniform Falloff

Ця устава корисна при використанні гачків на масштабованих об'єктах, особливо у випадках, коли не-однорідний масштаб розтягує результат гачка.

Це особливо корисно для решіток, де є поширеним використання не-однорідного масштабування.

Наступні устави доступні тільки у режимі правки Edit Mode:

Скидання -- Reset
Перераховує та зчищає зсув трансформування гачка.
Перецентрування -- Recenter
Установлює центр гачка у позицію 3D курсора.
Виділ -- Select
Виділяє вершини, що задіяні цим гачком.
Призначання -- Assign
Призначає виділені вершини цьому гачку.

Ghi chú

Модифікатор Hook зберігає індекси вершин з оригінальної сіті для визначення того, на що впливати; це означає, що модифікатори, які генерують геометрію, як модифікатор Subdivision Surface, повинен завжди застосовуватися після модифікатора «Гачок»; інакше генерована геометрія залишиться поза впливом гачка.
