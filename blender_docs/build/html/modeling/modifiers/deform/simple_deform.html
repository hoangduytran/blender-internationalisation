Модифікатор «Проста Деформація» -- Simple Deform Modifier¶

Модифікатор Simple Deform дозволяє легке застосування простого деформування об'єкта (сіті, решітки, криві, поверхні та тексти підтримуються).

Використовуючи інший об'єкт можливо визначати вісь та початок деформування, дозволяючи застосування дуже різних ефектів.

Опції¶

«Проста Деформація» -- Simple Deform.

Режим -- Mode

Ці радіокнопки визначають тип деформування, який буде застосовано.

Закрут -- Twist
Обертає навколо осі Z.
Згин -- Bend
Згинає сіть по осі Z.
Звуження -- Taper
Лінеарно масштабує уздовж осі Z.
Розтяг -- Stretch
Розтягує об'єкт уздовж осі Z (від'ємний Factor призводить до стиску), зберігаючи об'єм, масштабуючи обернено по осям X та Y.
Група Вершин -- Vertex Group
Ім'я групи вершин, що вказує на те, чи і наскільки впливає на кожну вершину це деформування.
Початок -- Origin

Ім'я об'єкта, що визначає початок деформування (зазвичай, порожняк). Цей об'єкт може бути:

  • Rotated to control the axis (its local Z axis is now used as the deformation axis).
  • Пересунуто для керування початком деформування.
  • Масштабовано для зміни фактора деформування.

Ghi chú

Коли цей об'єкт, що керує початком (той, що у полі Origin), є нащадком деформованого об'єкта, то це створює зациклення залежностей у системі даних Blender'а. Обходом цього є створення нового порожняка та приріднення обох цих об'єктів до нього.

Кут -- Angle / Фактор -- Factor
Величина деформування. Може бути від'ємною для розвертання деформування.
Ліміти -- Limits
Ці устави дозволяю вам задавати нижній та верхній ліміти деформування. Верхній ліміт не може бути нижчим, ніж нижній.
Блокувати Вісь X -- Lock X Axis / Блокувати Вісь Y -- Lock Y Axis (лише режими Taper та Stretch)
Вони керують тим, чи дозволяється змінювати координати по X та/або Y. Звідси, можливо стискати координати по X об'єкта та зберігати недоторканими координати по Y.
