Модифікатор «Згладження» -- Smooth Modifier¶

Модифікатор Smooth згладжує сіть, розпластуючи кути між сусідніми гранями в ній, подібно до однойменного засобу Specials ‣ Smooth у режимі правки Edit Mode. Він розгладжує без підподілення сіті -- кількість вершин залишається без змін.

Smooth Modifier panel.

Хоча, цей модифікатор не обмежується згладжуванням. Його керувальний фактор може конфігуруватися поза діапазоном (0.0 до 1.0) (включаючи від'ємні значення), що може давати цікаві деформування.

Опції¶

Вісь -- Axis

Перемикальні кнопки для увімкнення/вимкнення модифікатора у напрямках осей X, Y та/або Z.

X, Y, Z

Фактор -- Factor
Фактор для керування величиною згладження. Вищі значення збільшуватимуть ефект модифікатора. Значення поза цим діапазоном (понад 1.0 або нижче 0.0) спотворюють сіть.
Повтор -- Repeat
Кількість повторів згладжування, еквівалент виконання однойменного засобу Smooth кілька разів підряд.
Група Вершин -- Vertex Group
Ім'я групи вершин для обмеження ефекту тільки на її вершини. Це дозволяє вибіркове, реально-часове згладжування шляхом малювання вагомостей вершин.

Алгоритм -- Algorithm¶

Розрахування, зроблене модифікатором Smooth, є простим та логічним, і може вважатися як геометричний еквівалент розмивання зображень.

Кожна позиція нової вершини -- це просто усереднена позиція навколишніх вершин (вершин, з'єднаних з цим же ребром, що й дана вершина).
