Модифікатор «Зміщення» -- Displace Modifier¶

Модифікатор Displace зміщує вершини у сіті на основі інтенсивності текстури. Можуть використовуватися процедурні або зображеннєві текстури. Таке зміщення може бути уздовж певної локальної осі, уздовж нормалі вершини, або окремі компоненти RGB текстури можуть використовуватися для зміщення вершин у локальних напрямках X, Y та Z одночасно (інколи це називається як «векторне зміщення» -- Vector Displacement).

Опції¶

Модифікатор «Зміщення» -- Displace.

Текстура -- Texture
Ім'я текстури, на основі якої одержується зміщення для кожної вершини. Якщо це поле є порожнім, то стандартно модифікатор дає 1.0 (білий).
Напрям -- Direction

Напрям, уздовж якого зміщуються вершини. Може бути одним з наступних:

X, Y, Z
Зміщення уздовж обраної осі.
Нормаль -- Normal
Зміщення уздовж нормалі вершини.
Власна Нормаль -- Custom Normal
ToDo 2.74.
RGB у XYZ -- RGB to XYZ
Displace along local XYZ axes individually using the RGB components of the texture (Red values displaced along the X axis, Green along the Y, Blue along the Z). This is sometimes referred to as Vector Displacement.
Простір -- Space
За допомогою напрямку, заданого у X, Y, Z або XYZ, цей модифікатор може зміщувати уздовж локальних або глобальних осей.
Координати Текстури -- Texture Coordinates

Система координат текстури, що використовується при одержанні значень з текстури для кожної вершини. Може бути одним з наступних:

UV

Бере координати текстури з координат UV граней.

Карта UV -- UV Map
Карта UV, з якої беруться координати текстури. Якщо об'єкт немає жодних координат UV, то модифікатор використовує систему координат Local. Якщо це поле є порожнім, але є доступна карта UV (наприклад, зразу після додання першої карти UV для сіті), то це буде перезаписано поточною активною картою UV.

Ghi chú

Оскільки координати UV визначаються для кожної грані, то система координат UV текстури визначає координату UV для кожної вершини з першої зустрічної грані, яка використовує цю вершину; будь-які інші грані, що використовують цю вершину, ігноруються. Це може призводити до артефактів, якщо сіть має не-прилеглі координати UV.

Об'єкт -- Object

Бере координати текстури зі системи координат іншого об'єкта (що визначається полем Object).

Об'єкт -- Object

Об'єкт, з якого беруться координати текстури. Пересування такого об'єкта, тому, буде змінювати координати розкладки текстури.

Зверніть увагу, що пересування оригінального об'єкта також призведе до оновлення координат текстури. Таким чином, якщо вам необхідно зберегти систему координат зміщення у ході пересування модифікованого об'єкта, розгляньте приріднення об'єкта координат до модифікованого об'єкта.

Якщо це поле порожнє, то використовується система координат Local.

Глобально -- Global
Бере координати текстури з глобальної системи координат.
Локально -- Local
Бере координати текстури з локальної системи координат об'єкта.
Група Вершин -- Vertex Group
Ім'я групи вершин, яка використовується для керування впливом цього модифікатора. Якщо це поле залишається порожнім, то модифікатор впливає на усі вершини однаково.
Середнє -- Midlevel

Значення текстури, яке буде оброблятися, якщо немає жодного зміщення цим модифікатором. Значення текстури нижче цього значення будуть давати негативне, від'ємне зміщення уздовж обраного напрямку, тоді як значення текстури вище цього значення будуть давати позитивне, додатне зміщення.

displacement = texture_value - Midlevel

Recall that color/luminosity values are typically between (0.0 to 1.0) in Blender, and not between (0 to 255).

Сила -- Strength

Сила зміщення. Після зсування значенням Midlevel зміщення буде множитися на значення Strength для надання фінального зсуву вершин.

vertex_offset = displacement × Strength.

Негативна, від'ємна сила може використовуватися для інвертування впливу модифікатора.
