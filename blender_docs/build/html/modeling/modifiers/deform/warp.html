Модифікатор «Жолоблення» -- Warp Modifier¶

Цей модифікатор деформування може використовуватися для жолоблення частин сіті у нову локацію дуже гнучким способом, використовуючи два об'єкти для вибору регіонів «від» і «до», з опціями для використання кривої спаду, текстури та групи вершин.

Модифікатор Warp, застосований до ґратки.

Модифікатор Warp спочатку досить складний, але тут допоможемо зрозуміти, як він працює. Модифікатор вимагає двох точок, заданих початками об'єктів. Точка "From" позначає точку у просторі, що штовхається до точки "To". Це схоже на використання пропорційної правки -- Proportional Editing у режимі правки Edit Mode.

Опції¶

Модифікатор «Жолоблення» -- Warp.

Від -- From
Вказує об'єкт початку трансформації жолоблення.
До - To
Вказує об'єкт призначення трансформації жолоблення.
Зберігання Об'єму -- Preserve Volume
Вмикає зберігання об'єму при обертанні однієї з трансформ.
Група Вершин -- Vertex Group
Лімітує деформування вказаною групою вершин.
Сила -- Strength
Задає, наскільки сильним є ефект.
Радіус -- Radius
Задає відстань від трансформи, що може бути жолоблена, до трансформувальних держаків.
Тип Спаду -- Falloff Type
Задає спосіб зміни сили жолоблення від центру трансформи до значення Радіусу. Дивіться описи про типи спаду тут -- Proportional Editing.
Текстура -- Texture
Вказує текстуру, коли сила зсувається створенням варіацій зміщення.
Координати Текстури -- Texture Coordinates

Задає спосіб застосування текстури до сіті при використанні текстурованого жолоблення.

Об'єкт -- Object
Вказує об'єкт, що використовується, коли координати текстури установлено на Object.
Карта UV -- UV Map
Вказує карту UV, коли тип координат текстури задано як UV.

Використання¶

Модифікатор Warp може бути іноді незручним для використання, і випадки використання його досить незначні, однак, існує їх значна кількість. Наприклад, Warp Modifier може використовуватися для інтерактивної пропорційної правки -- Proportional Editing в анімації.

Інший спосіб вживання Warp Modifier подібний до модифікатора «Деформування Сіттю» -- Deform Modifier. Це дозволяє вам деформувати частини сіті без зроблення групи вершин.

Приклади¶

Модифікатор Warp з власною кривою спаду.
