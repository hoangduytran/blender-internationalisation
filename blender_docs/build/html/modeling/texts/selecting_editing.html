Виділення та Редагування -- Selection & Editing¶

Editing text is quite different from other object types in Blender, and happens mainly in two areas. First, the 3D View, where you type your text, and have a few shortcuts, e.g. for applying styles (see Символ -- Character) -- note however, that most Blender hotkeys you know in Edit Mode do not exist for texts. The second place is the Properties Editor, especially the Font tab.

Виділення і курсор¶

Текст у режимі правки Edit Mode.

У режимі Edit Mode ваш текст має білий курсор та, як і у будь-якому текстовому редакторі, він визначає місце, де нові символи будуть вставлятися.

Наступний/Попередній Символ -- Next/Previous Character
Ви можете пересувати курсор за допомогою клавіш зі стрілками Left або Right.
Наступне/Попереднє Слово -- Next/Previous Word
Для пересування курсора на межу слова використовуйте Ctrl-Left або Ctrl-Right.
Початок/Кінець Рядка -- Line Begin/End
Home та End пересувають курсор на початок та кінець рядка, відповідно.
Наступний/Попередній Рядок -- Next/Previous Line
Для стрибка між рядками використовуйте Up або Down.
Наступна/Попередня Сторінка -- Next/Previous Page
Для стрибання назад/вперед на десять рядків за раз скористайтеся PageUp або PageDown.

Hold Shift while using the arrow keys to select a part of the text. You can use it to specify different materials, the normal/bold/italic style, and not much more.

Редагування -- Editing¶

The menu of the 3D View header offers few options, and there is no Specials menu. You have no transform nor mirror tools, and so on. However, you can apply to texts the same modifiers as for curves.

Базово -- Basic¶

Правлення Тексту подібне до використання стандартного текстового редактора, але не настільки повнофункціональне, та яке має деякі відмінності:

Вихід з Режиму Правки -- Exit Edit Mode
Tab не вставляє символ табуляції у текст, а дає вхід та вихід з режиму правки Edit Mode, як і для інших типів об'єктів у Blender'і.
Копіювання -- Copy
To copy text to the buffer, use Ctrl-C or the Copy button in the Tool Shelf.
Вирізання і Копіювання -- Cut and Copy
To cut and copy text to the buffer, use Ctrl-X or the Cut button in the Tool Shelf.
Вставляння -- Paste
To paste text from the buffer, use Ctrl-V or the Paste button in the Tool Shelf.
Видалення усього тексту -- Delete all text
Для повного стирання або видалення усього тексту користайтеся Ctrl-Backspace.

Буфер тексту синхронізується з буфером обміну стільниці ОС. Але, якщо він використовується у межах Blender'а, то також буде копіюватися форматування тексту. Про інші способи вкладання тексту дивіться Inserting Text нижче.

Спеціальні Символи -- Special Characters¶

Орієнтир -- Reference

Mode:Edit Mode
Menu:Text ‣ Special Characters

Якщо вам потрібні спеціальні символи (такі як, акцентовані символи, увід яких недоступний з вашої клавіатури), то ви можете продукувати багато з них, використовуючи комбінацію з двох інших символів. Для цього уведіть головний символ, натисніть Alt-Backspace, а потім натисніть бажаний «модифікатор» для продукування спеціального символу. Нижче наводяться деякі приклади:

ã A, Alt-Backspace, ~
á A, Alt-Backspace, '
à A, Alt-Backspace, \
å A, Alt-Backspace, O
ë E, Alt-Backspace, "
ø O, Alt-Backspace, /

Вкладання тексту¶

Ви можете вкладати текст двома способами: з внутрішнього текстового буфера (як описано вище) або зовні з текстового файлу.

Для завантаження тексту з текстового файлу скористайтеся засобом «Текст > Вставити з Файлу» -- Text ‣ Paste File. Це викличе Оглядач Файлів -- File Browser для вказування дійсного файлу у кодуванні UTF-8. Як завжди, будьте обережні, щоб у файлі не було надто багато символів, оскільки інтерактивна реакція сповільниться.

Конвертування текстових об'єктів¶

Конвертування в об'єкт тексту¶

Using an existing text data-block, you can convert it to an object from the Text editors header, select Edit ‣ Text to 3D Object, One Object or One Object per Line depending on your needs.

Також є можливим є вставляння з буфера обміну або файлу за допомогою меню Edit у ході редагування 3D Тексту.

Конвертування у 3D сіть¶

Можливим є конвертувати Об'єкт Тексту в об'єкт 3D Сіті. Це може бути корисним тим, що ви зможете редагувати вершини у режимі правки Edit Mode, однак ви втратите можливість редагувати власне текст. Для цього перейдіть у режим об’єкта Object Mode та виділіть ваш Об'єкт Тексту. Натисніть Alt-C та виберіть Mesh From Curve/Meta/Surf/Text. Тепер ви можете перейти у режим правки Edit Mode та вручну правити вершини. Вони, зазвичай, будуть безладними, тому може бути корисним скористатися видаленням типу Limited Dissolve або модифікатором «Перебудова» -- Remesh Modifier з низьким порогом для вичищення сіті.

Призначення матеріалів¶

Орієнтир -- Reference

Mode:Edit Mode
Panel:Properties editor ‣ Materials

Кожен символ має різний індекс матеріалу Material index, що дає змогу мати різні матеріали на різних символах.

Ви можете призначати індекси як під час уведення, так і після виділення блоків тексту та клацанням кнопки «Призначення» -- Assign у вкладці Material.

Червоний Зелений Синій.

Наприклад, для створення Ілюстрації Червоний Зелений Синій. вам потрібно створити три різні матеріали та три окремі індекси матеріалів. Для кожного слова буде призначено свій Material Index шляхом виділення символів для кожного слова та клацання кнопки Assign. На Ілюстрації Червоний Зелений Синій. є тільки один єдиний об'єкт типу Text.
