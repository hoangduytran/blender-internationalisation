Властивості -- Properties¶

Форма -- Shape¶

Орієнтир -- Reference

Mode:All Modes
Panel:Properties editor ‣ Text ‣ Shape

Панель «Форма» -- Shape.

As you can see in the Shape panel, texts have most of the same options as Curves.

Роздільність -- Resolution
Передогляд -- Preview
Роздільність поверхні у напрямку U для використання в оглядвікні.
Зображ, Рендер -- Render
Роздільність поверхні у напрямку U, задана як нуль для використання роздільності згідно Preview.
Заповнення -- Fill
Mode

Визначає спосіб, яким Крива заповнюється, коли видавлюється та/або скошується.

Спереду -- Front
Заповнює передню сторону поверхні.
Позаду -- Back
Заповнює задню сторону поверхні.
Заповнити Деформоване -- Fill Deformed
Заповнює криву після застосування усіх модифікацій, що можуть деформувати криву (тобто, ключів форм та модифікаторів).
Показ -- Display
Швидке Правлення -- Fast Editing
Не заповнює полігони у ході редагування тексту.

Шрифт -- Font¶

Орієнтир -- Reference

Mode:All Modes
Panel:Properties editor ‣ Font ‣ Font

Панель Font має різні опції для зміни вигляду символів.

Завантаження та змінення шрифтів¶

Blender comes with a built-in font by default that is displayed in each of the four font style data-block menus. The built-in font is always present and shows in this list as "Bfont". The data-block menu contains a list displaying the currently loaded fonts. Select one for each font style.

To load a different Font, click one of the Load buttons in the Font panel and navigate to a font file. The File Browser will give all valid fonts a capital "F" icon.

If you select a font that is unsupported by Blender, you will get the error Not a valid font.

Ghi chú

Location of Fonts on Unix

Шрифти типово розміщуються у теці /usr/lib/fonts або, як варіант, у теці /usr/lib/X11/fonts, але не завжди. Вони можуть бути також в інших локаціях, таких як /usr/share/local або /usr/local/share, та можливо теках нижчого рівня.

Пам'ятайте, що однаковий шрифт буде застосовано до усіх символів одного і того ж стилю у тексті, але для кожного стилю вимагається окремий шрифт. Наприклад, вам доведеться завантажити шрифт Italics, щоб зробити символи або слова курсивом. Після того, як шрифт завантажено, ви можете застосувати цей «Стиль» шрифту до виділених символів або усього об'єкта. Загалом, вам потрібно завантажити мінімум чотири різних типи шрифтів для представлення кожного стилю (Звичайний -- Normal, Курсив -- Italics, Жирний -- Bold, Жирний-Курсив -- Bold-Italics).

Важливо розуміти, що Blender не цікавиться, який шрифт ви завантажуєте для «звичайного», «жирного» тощо стилів. Так ви можете мати до чотирьох різних шрифтів для використання в одному і тому ж тексті, але ви маєте вибрати між різними стилями однакового шрифту або різними шрифтами. Blender має кілька типографічних керувальників для змінення стилю та розстави тексту, що знаходяться на панелі Font.

Розмір і косування¶

Розмір -- Size
Керує розміром тексту в об'єкті цілком (неможливо керувати розміром кожного символу незалежно). Зауважте, однак, що символи з різними шрифтами (різні стилі, дивіться нижче) можуть мати різні видимі розміри.
Косування -- Shear

Керує нахилянням тексту в об'єкті цілком. На відміну від того, як це може здатися, це не подібно на стиль курсиву.

Приклад косування.

Об'єкти як шрифти¶

Ви можете також «створювати» власний «шрифт» всередині Blender'а! Це досить складний процес, але давайте розгляньмо його:

  1. First, you must create your chars. Each char, of any type, is an object (mesh, curve, meta...). They all must have a name following the schema: common prefix followed by the char name (e.g. "ft.a", "ft.b", etc.).
  2. Далі, для об'єкта Text ви повинні увімкнути кнопку Dupli Vertices (панель Object ‣ Animation Settings).
  3. На вкладці даних Font заповніть поле Object Font загальним префіксом об'єктів вашого «шрифту».

Тепер, кожен раз, коли символ у вашому тексті відповідатиме суфіксній частині імені об'єкта «шрифту», то цей об'єкт дублюватиметься на цей символ. Оригінальні символи залишаються видимими. Ці об'єкти дублюються так, що їх центр розташовується у нижньому правому кутку відповідних символів.

Текст на Кривій -- Text on Curve

Використовується для вибору кривої, по якій буде слідувати текстовий об'єкт.

Текст на кривій.

Mẹo

Ви можете використовувати модифікатор «Крива» -- Curve Modifier, що пропонує більше можливостей керування.

Підкреслення -- Underline

Перемикається кнопкою Underline перед уводом тексту. Текст може також бути установлено як Підкреслений шляхом виділення його та використання кнопки Underline на Полиці Інструментів.

Позиція -- Position
Це дозволяє вам зсувати вертикально позицію підкреслення.
Товщина -- Thickness
Це керує товщиною підкреслення.

Символ -- Character¶

Опції символу, наприклад, для уводу жирного тексту.

Жирний -- Bold
Перемикач на стиль Bold перед уводом тексту. Текст може також бути установлений як Жирний шляхом виділення його та використання кнопки Bold на Полиці Інструментів.
Курсив -- Italics
Перемикач на стиль Italic перед уводом тексту. Текст може також бути установлений як Курсив шляхом виділення його та використання кнопки Italic на Полиці Інструментів.
Підкреслення -- Underline
Вмикає підкреслення, яке керується уставами секції Underline, описаними вище.
Зменшені Заголовні -- Small Caps
Перемикає увід тексту зменшеними заголовними літерами.

Кнопки Blender'а Bold та Italic не працюють так само, як в інших застосунках, оскільки вони також служать як заповнювачі при завантаженні інших шрифтів вручну, які застосовуються, коли ви визначаєте відповідний стиль; дивіться Font.

Для застосування атрибута Bold/Italics/Underline до набору символів ви або вмикаєте стяги Bold / Italics / Underline перед уводом символів, або підсвічуєте (виділяєте) їх спершу, а потім перемикаєте кнопками Bold/Italics/Underline.

Установлення регістру символів¶

You can change the text case by selecting it then clicking the To Upper or To Lower in the Tool Shelf.

Опція Small Caps вмикає уведення символів як зменшених заголовних.

Розмір Зменшених Заголовних може бути змінено однойменною уставою Small Caps Scale. Зауважте, що устава Small Caps Scale застосовується однаково до усіх форматованих опцією Small Caps символів.

Абзац -- Paragraph¶

Орієнтир -- Reference

Mode:All Modes
Panel:Properties editor ‣ Font ‣ Paragraph

Панель Paragraph має устави для вирівнювання та задання інтервалів для тексту.

Панель «Абзац» -- Paragraph.

Вирівнення Горизонтально -- Horizontal Alignment¶

Зліва -- Left
Вирівнює текст по лівій межі кадрів при їх використанні, інакше використовує центральну точку об'єкта Text як стартову точку тесту (який зростає управо).
Центр -- Center
Центрує текст у кадрах при їх використанні, інакше використовує центральну точку об'єкта Text як серединну точку тексту (який зростає рівномірно вліво та управо).
Справа -- Right
Вирівнює текст по правій межі кадрів при їх використанні, інакше використовує центральну точку об'єкта Text як кінцеву точку тексту (який зростає вліво).
Боками -- Justify
Лише розряджає рядок, коли він закінчується завертанням слів (не за клавішею Return), вона використовує білий пропуск -- white-space замість символьного пропуску -- character spacing (кернінг -- kerning) для заповнення рядків.
Розряд -- Flush
Завжди розряджає рядок, навіть коли він закінчується символом уводу -- Enter; він використовує символьний пропуск (кернінг) для заповнення рядків.

Обидві опції Justify та Flush працюють тільки у межах кадрів.

Вирівнення Вертикально -- Vertical Alignment¶

Базова Лінія Зверху -- Top Base-Line
Вирівнює базову лінію тексту по верху кадрів при їх використанні, інакше використовує центральну точку об'єкта Text як стартову точку тексту (який зростає вниз).
Зверху -- Top
Вирівнює верх тексту по центральній точці об'єкта Text (який зростає вниз). Вона поводить себе, як і Top Base-Line при використанні кадрів. Top працює тільки без кадрів.
Центр -- Center
Центрує текст у кадрах при їх використанні, інакше використовує центральну точку об'єкта Text як серединну точку тексту (яка зростає рівномірно уверх та вниз).
Знизу -- Bottom
Вирівнює текст по нижній стороні кадрів при їх використанні, інакше використовує центральну точку об'єкта Text як кінцеву точку тексту (який зростає уверх).

Інтервал -- Spacing¶

Символ -- Character
Фактор, з яким пропуск між кожним символом масштабується у ширину.
Слово -- Word
Фактор, з яким «білий пропуск» між словами масштабується у ширину. Ви можете також керувати нею, натискаючи Alt-Left або Alt-Right для зменшення/збільшення інтервалу з кроком 0.1.
Лінія -- Line
Фактор, з яким вертикальний пропуск між рядками масштабується.

Зсув -- Offset¶

Зсув X та Y -- X offset and Y offset
Well, these settings control the X and Y offset of the text, regarding its "normal" positioning. Note that with Text Boxes, it applies to all frames' content.

Рамки Тексту -- Text Boxes¶

Орієнтир -- Reference

Mode:All Modes
Panel:Properties editor ‣ Font ‣ Text Boxes

Текстові «Рамки» або «кадри» дозволяють вам розподіляти текст по прямокутним областям у межах одиничного текстового об'єкта Дозволяє на кожен текстовий об'єкт довільна кількість вільно розташованих та з можливістю зміни розміру текстових кадрів.

Текст продовжувано тече з рамки з найменшим номером у рамку з найбільшим номером та всередині них завертається по рядках. Текст тече між кадрами, коли кадр з меншим номером не може бути заповнений більше текстом. Якщо досягнуто останній кадр, то текст витікає з нього.

Text Boxes panel.

Dimensions
Specifies the Width and Height of the frame. If set to zero no word-wrap happens.
Зсув -- Offset
Controls the X and Y offset of the frame.
Add Textbox
Inserts a new frame, just after the current one (in text flow order). The new frame will have the same size and position as the selected one.
Delete (X icon)
Delete the current frame.

Одночасно кілька стовпців, текст перетікає між рамками.
