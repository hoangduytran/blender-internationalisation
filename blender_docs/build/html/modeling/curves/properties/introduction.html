Вступ¶

Атрибути -- Attributes¶

Вагомість -- Weight¶

This sets the "goal weight" of selected control points, which is used when a curve has Soft Body physics, forcing the curve to "stick" to their original positions, based on the weight.

Радіус -- Radius¶

Радіус дозволяє вам безпосередньо керувати шириною видавлення уздовж «спінальної» кривої. Устава Radius точок задається за допомогою інструмента трансформацій Shrink/Fatten Radius, Alt-S, Curve ‣ Transform ‣ Shrink/Fatten Radius або Properties region ‣ Transform ‣ Radius.

Радіус однієї з керувальних точок установлено в нуль.

Mẹo

Пам'ятайте, ці криві можуть бути конвертовані у сіті за допомогою Alt-C у режимі об'єкта Object Mode.

Похил -- Tilt¶

Ця устава керує тим, як нормалі (візуалізація: стрілочки) скручуються навколо кожної керувальної точки -- а тому це стосується тільки 3D кривих! Ви можете задавати його за допомогою засобу трансформації Tilt на полиці інструментів Tool Shelf, Properties region ‣ Transform ‣ Tilt, Curve ‣ Transform ‣ Tilt.

Ви можете також скидати його у стандартне значення (тобто, до перпендикуляра до площини оригінальної кривої) за допомогою Alt-T, Curve ‣ Control Points ‣ Clear Tilt. Для НОРБС похил завжди інтерполюється плавно. Однак, для Безьє, ви можете вибирати алгоритм інтерполяції між «Лінеарно» -- Linear, «Легко» -- Ease, «Б-Сплайн» -- B-Spline та «Кардинально» -- Cardinal, у Properties Editor ‣ Object Data ‣ Active Spline ‣ Tilt.

30-градусний серединний похил Mean Tilt усіх керувальних точок.
