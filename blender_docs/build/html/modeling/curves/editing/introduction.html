Вступ¶

Ця сторінка розкриває основи правлення, редагування кривих.

Трансформації -- Transform¶

Крива Безьє може редагуватися шляхом трансформування локацій керувальних точок та їх держаків. Крива НОРБС, з іншого боку, має тільки керувальні точки.

Пересування -- Translation, Обертання -- Rotation, Масштабування -- Scale¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Panel:Tool Shelf ‣ Tools ‣ Transform
Menu:Curve ‣ Transform ‣ Grab/Move, Rotate, Scale, ...
Hotkey:G, R, S

Як і інші елементи у Blender'і, керувальні точки та держаки кривої можуть схоплюватися/рухатися за допомогою G, обертатися з R або масштабуватися із S як описано у підрозділі -- Basic Transformations. При знаходженні у режимі правки Edit Mode, пропорційна правка -- proportional editing також доступна для дій трансформування.

Панель «Трансформації» -- Transform Panel¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Panel:Properties region ‣ Transform

Коли нічого не виділено, то ця панель порожня. Коли більше ніж одна вершина виділена, то редагуються усереднені значення та слово "Median" додається спереду до назв засобів.

Керувальна Точка -- Control Point, Вершина -- Vertex
Перші керувальники (X, Y, Z) показують координати виділеної точки або держака (вершини). У випадку з кривими НОРБС є доступним також четвертий компонент (W), який визначає вагомість виділеної керувальної точки або серединної вагомості.
Простір -- Space

Радіокнопки Простору дозволяють вам вибирати, чи ці координати є відносними до початку об'єкта (Local) чи до глобального початку (Global).

Глобально -- Global, Локально -- Local

Дані -- Data¶

Weight, Radius, Tilt
See Curve Properties.

Інструменти -- Tools¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Menu:Curve ‣ Transform

Трансформаційні інструменти «До Сфери» -- To Sphere, «Косування» -- Shear, «Жолоблення» -- Warp та «Уштовх/Виштовх» -- Push/Pull описані у підрозділах глави Transformations. Два інших інструменти «Відхил» -- Tilt та «Стоншення/Потовщення» -- Shrink/Fatten Radius стосуються видавлення кривої -- Curve Extrusion.

Дзеркалення -- Mirror¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Menu:Curve ‣ Mirror
Hotkey:Ctrl-M

Інструмент «Дзеркалення» -- Mirror також доступний та поводиться точно так, як і для вершин сіті -- mesh vertices.

Підхоп -- Snap¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Menu:Curve ‣ Snap
Hotkey:Shift-S

Mesh snapping -- підхоплення сітей також працює з компонентами кривих. Підхоплення впливає як на керувальні точки, так і на їх держаки, за винятком підхоплення всередині себе (інші компоненти до активної кривої). Підхоплення працює з 2D кривими, але точки обмежуються локальними осями XY.

Видавити Криву та Пересунути -- Extrude Curve and Move¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Panel:Tool Shelf ‣ Tools ‣ Curve Tools ‣ Modeling: Extrude
Menu:Curve ‣ Extrude Curve and Move
Hotkey:E

Видавлює точки, дублюючи виділені точки, які далі можуть бути пересунуті. Якщо виділенням є кінцева точка, то нова точка буде з'єднана з виділеною точкою, в іншому випадку -- створиться нова нез'єднана точка.

Дублювати або Видавити до Курсора -- Duplicate or Extrude to Cursor¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Hotkey:Ctrl-LMB

Інтерактивно поміщає нові точки за допомогою Ctrl-LMB у позиції курсора. Щодо виділення, то поводиться подібним чином, як і засіб Extrude Curve and Move.

Додати Дублікат -- Add Duplicate¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Panel:Tool Shelf ‣ Tools ‣ Curve Tools ‣ Curves: Duplicate
Menu:Curve ‣ Add Duplicate
Hotkey:Shift-D

Цей інструмент дублює виділені керувальні точки разом із сегментами кривої, які неявно виділені (якщо такі є). Якщо виділено тільки держак, то дублюється також повністю його точка. Копія виділяється та поміщається у режим захопу Grab, а тому ви можете перемістити її в інше місце.

Відокремлення -- Separate¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Menu:Curve ‣ Separate
Hotkey:P

Об'єкти кривих, що складаються з кількох окремих кривих, можуть розокремлюватися в окремі для них об'єкти шлях виділення бажаних сегментів та натиснення P. Зауважте, якщо в об'єкті Кривої є тільки одна крива, то засіб Separate створить новий об'єкт Кривої без жодних керувальних точок.

Зробити Сегмент -- Make Segment¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Menu:Curve ‣ Make Segment
Hotkey:F

З'єднує дві роз'єднані керувальні точки сегментом. Виділення повинно складатися з вільних точок або з перших/останніх точок кривої, далі натискається F. Якщо ці точки належать різним кривим, то вони сполучаться сегментом, що зробить з цих кривих одну криву.

Дві криві перед.

Крива після сполучення.

Зауважте, що ви можете сполучати криві тільки однакового типу (тобто, Безьє з Безьє, НОРБС з НОРБС). Додатково ви можете закривати криву, перемкнувши зациклення.

Перемкнути Зациклення -- Toggle Cyclic¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Panel:Tool Shelf ‣ Tools ‣ Curve Tools ‣ Curves: Toggle Cyclic
Menu:Curve ‣ Toggle Cyclic
Hotkey:Alt-C

Це перемикає між відкритою та закритою кривою (зациклення). Тільки криві з принаймні однією виділеною керувальною точкою будуть закриватися/відкриватися. Форма закритого сегмента базується на стартовому та кінцевому держаках кривих Безьє і, зазвичай, на суміжних керувальних точках для НОРБС. Єдиний раз, коли держак коригується після закриття, це якщо держак має тип Auto. Ілюстрація Відкрита та Закрита криві. -- це та ж крива Безьє відкрита і закрита.

This action only works on the original starting control point or the last control point added. Deleting a segment(s) does not change how the action applies; it still operates only on the starting and last control points. This means that Alt-C may actually join two curves instead of closing a single curve! Remember that when a 2D curve is closed, it creates a renderable flat face.

Відкрита та Закрита криві.

Видалення -- Delete¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Panel:Tool Shelf ‣ Tools ‣ Curve Tools ‣ Curves: Delete
Menu:Curve ‣ Delete...
Hotkey:X, Delete; Ctrl-X

Опції спливного меню для стирання:

Вершини -- Vertices
Це видалятиме виділені керувальні точки, без розривання кривої (тобто, суміжні точки безпосередньо пов'язуються, сполучаються, після того, як проміжні були видалені). Пам'ятайте, що порядок НОРБС не може бути вищим, ніж кількість його керувальних точок, а тому він може зменшуватися, коли ви видаляєте деяку керувальну точку. Звичайно, коли залишається тільки одна точка, немає більше видимої кривої, і коли всі точки видаляються, то і крива сама видаляється.
Сегменти -- Segments
Видаляє сегмент, що з'єднує виділені керувальні точки та, та роз'єднує їх.
Розчинити Вершини -- Dissolve Vertices Ctrl-X
Видаляє виділені керувальні точки, тоді як решту сегмента заповнюється для видаленої кривої шляхом коригування його держаків.

Перед видаленням.

Видалення вершин.

Видалення сегмента.

Розчинення вершин.

Керувальні Точки -- Control Points¶

Похил -- Tilt¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Panel:Tool Shelf ‣ Tools ‣ Transform ‣ Tilt
Menu:Curve ‣ Control Points ‣ Tilt/Clear Tilt
Hotkey:Ctrl-T, Alt-T
Похил -- Tilt Ctrl-T
Дозволяє визначати вам похил виділених керувальних точок. Цей похил буде інтерполюватися від точки до точки (ви можете перевіряти це за допомогою нормалей). Кут похилу визначається інтерактивно спершу, а потім він коригується на панелі Оператора Angle.
Зчистити Похил -- Clear Tilt Alt-T
Повертає похил цих виділених керувальних точок назад у 0.

Задати Тип Держака -- Set Handle Type¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Panel:Tool Shelf ‣ Tools ‣ Curve Tools ‣ Handles:
Menu:Curve ‣ Control Points ‣ Set Handle Type
Hotkey:V

Типи держаків -- це властивість кривих Безьє -- Bézier curves, яка може використовуватися для зміни функціональних властивостей кривої. Наприклад, перемикання на держаки типу Vector може використовуватися для створення кривих з різкими кутками. Детальніше дивіться тут -- Bézier curves.

Перемкнути Вільно/Вирівняно -- Toggle Free/Align V T
Додатково, це скорочення може використовуватися для перемикання між типами держака Free та Aligned.

Задати Вагомість Цілі -- Set Goal Weight¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Menu:Specials ‣ Set Goal Weight

This sets the Soft Body Goal Weight of selected control points. The precise value can be adjusted in the Operator Panel. To adjust the Mean Weight (average) of selected control points use Properties region ‣ Transform ‣ Mean Weight.

Згладження -- Smooth¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Panel:Tool Shelf ‣ Tools ‣ Curve Tools ‣ Modeling: Smooth
Menu:Specials ‣ Smooth

Згладження кривих є доступним через меню Specials. Для кривих Безьє ця операція згладження скорочує відстань між виділеними керувальними точками та їх сусідами, утримуючи сусідів закріпленими. Не впливає на дотичні керувальних точок.

Оригінальна, незгладжена Крива.

Крива цілком згладжена понад 20 разів, утримуванням Shift-R для повторення останнього кроку.

Тільки три керувальні точки у центрі згладжені понад 20 разів.

Гачки -- Hooks¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Menu:Curve ‣ Control Points ‣ Hooks
Hotkey:Ctrl-H

Hooks -- гачки можуть додаватися для керування однією або кількома точками за допомогою інших об'єктів.

Сегменти -- Segments¶

Підподілення -- Subdivision¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Panel:Tool Shelf ‣ Tools ‣ Curve Tools ‣ Modeling: Subdivide
Menu:Curve ‣ Segments ‣ Subdivide

Підподілення кривої просто підподіляє усі виділені сегменти, додаючи одну чи кілька керувальних точок між виділеними сегментами.

Кількість Поділів -- Number of Cuts
Кількість поділів може коригуватися з панелі Оператора.

Перемкнути Напрям -- Switch Direction¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Panel:Tool Shelf ‣ Tools ‣ Curve Tools ‣ Curves: Switch Direction
Menu:Curve ‣ Segments ‣ Switch Direction, Specials ‣ Switch Direction

Цей засіб буде «розвертати» напрям будь-якої кривої з принаймні одним виділеним елементом (тобто, стартова точка буде ставати кінцевою точкою і навпаки). Це в основному корисно, коли крива використовується як шлях, або використовуються опції скошування та звуження.

Показ/Схов -- Show/Hide¶

При знаходженні у режимі правці Edit Mode ви можете ховати та розкривати елементи для показу. Ви можете показувати або ховати тільки керувальні точки, оскільки сегменти будуть показуватися до тих пір, допоки усі керувальні точки не будуть сховані, і це дасть повний схов усієї кривої.

Дивіться про Показ/Схов -- Show/Hide у режимі об'єкта -- Object Mode. Дивіться також про панель -- Показ Кривої -- Curve Display.

Задати Тип Сплайна -- Set Spline Type¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Panel:Tool Shelf ‣ Tools ‣ Curve Tools ‣ Curves: Set Spline type

Кнопка Set Spline Type.

Ви можете конвертувати сплайни в об'єкт кривої з типами Bézier, NURBS та Poly. Натисніть T для розкриття Полиці Інструментів. Клацання на кнопці Set Spline Type дозволить вам вибрати тип Сплайна (Poly, Bézier або NURBS).

Зауважте, що це не є «розумним» конвертуванням, тобто Blender не намагається підтримувати ту ж форму, ні ту ж кількість керувальних точок. Наприклад, при конвертуванні НОРБС у Безьє кожна група з трьох НОРБС керувальних точок стає однією унікальною Безьє (центральна точка та два держаки).

Конвертувати у -- Convert To/from Mesh.

Приріднення Кривої -- Curve Parenting¶

Орієнтир -- Reference

Режим -- Mode:Edit Mode
Hotkey:Ctrl-P

Ви можете робити інші виділені об'єкти нащадками -- children однієї або трьох керувальних точок за допомогою Ctrl-P, як і із сітьовими об'єктами.

Виділіть сіть, далі криву, перейдіть для кривої у режим правки та натисніть Ctrl-P. Або у режимі правки кривої виділіть одну або три керувальні точки, далі клацніть Ctrl-RMB на сіті та натисніть Ctrl-P для зроблення приріднення до вершин. Виділення трьох керувальних точок дозволить нащадку слідувати за серединною точкою між цими трьома вершинами. Альтернативою може бути використання примусу -- Child of Constraint. Дивіться також про модифікатор «Крива» -- Curve Modifier.
