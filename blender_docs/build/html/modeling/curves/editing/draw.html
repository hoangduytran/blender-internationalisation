Рисувати Криву -- Draw Curve¶

Орієнтир -- Reference

Mode:Edit Mode
Panel:Tool Shelf ‣ Create ‣ Draw Curve
Menu:Add ‣ Draw Curve
Hotkey:Shift-LMB

The Curve draw tool allows you to free-hand draw curves.

Опції штриха¶

Панель «Штрих Кривої» -- Curve Stroke.

Ці опції можуть бути знайдені в Tool Shelf ‣ Options ‣ Curve Stroke.

Тип -- Type

Використовуваний тип кривої для рисування.

Полі -- Poly
Крива Безьє з прямолінійними сегментами (держаки типу «авто»).
Безьє -- Bézier
Допуск -- Tolerance
Нижчі значення дають результат, що є ближчим до рисування штриха, тоді як вищі значення дають більш плавніші результати.
Метод -- Method
Перепідгонка -- Refit
Поступово пере-підганяє криву (дає найкращі результати).
Розділення -- Split
Розділяє криву, допоки не досягається допуск (дає кращу продуктивність рисування).
Виявляти Кутки -- Detect Corners

Виявляє кутки та використовує не-вирівняні держаки для них.

Кут Кутка -- Corner Angle
Будь-які кути, вище цього значення вважаються кутками.
Радіус Натиску -- Pressure Radius
Мін, мінімум -- Min
Мінімальний радіус, коли застосовується мінімальний натиск (також мінімум при звуженні)
Макс, максимум -- Max
Радіус для використання, коли застосовується максимальний натиск (або коли планшет не використовується).
Глибина Проекції -- Projection Depth

Опції для керування тим, де/як криві рисуються.

Курсор -- Cursor
Використовує глибину під курсором для рисування кривих.
Поверхня -- Surface

Використовується для рисування поверх інших об'єктів.

Зсув -- Offset
Відстань для зсуву кривої від поверхні.
Абсолютний Зсув -- Absolute Offset
Застосовує фіксований зсув (не масштабується радіусом кривої).
Лише Перше -- Only First

Використовує тільки початок штриха для глибини.

Нормаль/Огляд -- Normal/View
Рисує перпендикулярно до поверхні.
Нормаль/Поверхня -- Normal/Surface
Рисує вирівняно до поверхні.
Огляд -- View
Рисує вирівняно до оглядвікна.

Опції рисування¶

Панель «Рисувати Криву» -- Draw Curve.

Ці опції можна знайти внизу Полиці Інструментів на панелі -- Redo Last Panel.

Помилка -- Error
Відстань помилки в одиницях об'єкта. Це може виглядати подібно до частоти підподілення для кривої. Нижчі значення дають результат, що є ближчим до рисування штриха, тоді як вищі значення дають більш плавніші результати.
Метод Підгонки -- Fit Method
Перепідгонка -- Refit
Поступово пере-підганяє криву (дає найкращі результати).
Розділення -- Split
Розділяє криву, допоки не досягається допуск (дає кращу продуктивність рисування).
Кут Кутка -- Corner Angle
Будь-які кути, вище цього значення вважаються кутками.
Зациклення -- Cyclic
Перемикає зациклення -- Cyclic кривої.
