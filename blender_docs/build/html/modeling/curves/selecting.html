Chọn Lựa -- Selecting¶

Curve selection in Edit Mode has fewer options than with meshes. Mainly this is, because there is only one selectable element type, the control points (no select mode needed here...). These points are a bit more complex than simple vertices, however, especially for Béziers, as there is the central vertex, and its two handles...

The basic tools are the same as with meshes, so you can select a simple control point with the RMB, add to current selection with Shift-RMB, Border Select B, and so on.

One word about the Bézier control points: when you select the main central vertex, the two handles are automatically selected too, so you can grab it as a whole, without creating an angle in the curve. However, when you select a handle, only this vertex is selected, allowing you to modify this control vector...

Note that, unlike mesh edges, you cannot directly select a segment. Instead, select all of the control points that make up the segment you want to edit.

Trình Đơn Lựa Chọn -- Select Menu¶

With curves, all "advanced" selection options are regrouped in the Select menu of the 3D Views header. Let us detail them:

  • Random...
  • Đảo Nghịch -- Inverse
  • Select/Deselect All
Border/Circle Select
All these options have the same meaning and behavior as in Object Mode (and the specifics of Border Select in Edit Mode have already been discussed here).

Chọn cái Kết Nối -- Select Linked¶

Tham Chiếu -- Reference

Chế Độ -- Mode:Chế Độ Biên Soạn -- Edit Mode
Trình Đơn -- Menu:
 Select ‣ Select Linked
Phím Tắt -- Hotkey:
 L, Ctrl-L, Shift-L

L (or Ctrl-L for all) will add to the selection the cursor's nearest control point, and all the linked ones, i.e. all points belonging to the same curve. Note that for Bézier, using L with a handle selected will select the whole control point and all the linked ones.

Chọn Tương Đồng -- Select Similar¶

Tham Chiếu -- Reference

Chế Độ -- Mode:Chế Độ Biên Soạn -- Edit Mode
Trình Đơn -- Menu:
 Select ‣ Select Similar
Phím Tắt -- Hotkey:
 Shift-G

Selects control points that have certain similar properties to the active one. The Operator panel provides several selection options:

Thể Loại -- Type
Thể Loại -- Type
Selects splines that have the same spline Type i.e. Bézier, NURBS or Poly.
Bán Kính -- Radius
Selects control points that have a similar Radius value.
Trọng Lượng -- Weight
Selects all points that have a similar Weight value.
Chiều Hướng -- Direction
Selects control points that have a similar handles direction.
So Sánh -- Compare
Equal, Greater, Less. (only for Radius, Weight) (ToDo 2.76)
Giới Hạn -- Threshold
Precision (ToDo 2.76)

Đường Ngắn Nhất -- Shortest Path¶

Tham Chiếu -- Reference

Chế Độ -- Mode:Chế Độ Biên Soạn -- Edit Mode
Trình Đơn -- Menu:
 Operator Search ‣ Pick Shortest Path
Phím Tắt -- Hotkey:
 Ctrl-RMB

Selects the curve segments between two control points: the active and the one under the cursor. In the case of a closed curve, the shortest path will be selected.

Select/Deselect First/Last¶

Tham Chiếu -- Reference

Chế Độ -- Mode:Chế Độ Biên Soạn -- Edit Mode
Trình Đơn -- Menu:
 Select ‣ Select/Deselect First, Select ‣ Select/Deselect Last

These operators will toggle the selection of the first or last control point(s) of the curve(s) in the object. This is useful to quickly find the start of a curve (e.g. when using it as path...).

Select Next/Previous¶

Tham Chiếu -- Reference

Chế Độ -- Mode:Chế Độ Biên Soạn -- Edit Mode
Trình Đơn -- Menu:
 Select ‣ Select Next, Select ‣ Select Previous

These operators will select the next or previous control point(s), based on the current selection (i.e. the control points following or preceding the selected ones along the curve). In case of a cyclic curve, the first and last points are not considered as neighbors.

Mở Rộng/Thu Nhỏ vùng Chọn Lựa -- Select More/Less¶

Tham Chiếu -- Reference

Chế Độ -- Mode:Chế Độ Biên Soạn -- Edit Mode
Trình Đơn -- Menu:
 Select ‣ More/Less
Phím Tắt -- Hotkey:
 Ctrl-NumpadPlus, Ctrl-NumpadMinus

Their purpose, based on the currently selected control points, is to reduce or enlarge this selection.

Nhiều Hơn -- More
For each selected control point, select all its linked points (i.e. one or two...).
Nhỏ Hơn -- Less
For each selected control point, if all points linked to this point are selected, keep this one selected. Otherwise, de-select it.

This implies two points:

  1. When all control points of a curve are selected, nothing will happen (as for Less, all linked points are always selected, and of course, More cannot add any). Conversely, the same goes when no control points are selected.
  2. Second, these tools will never "go outside" of a curve (they will never "jump" to another curve in the same object).
