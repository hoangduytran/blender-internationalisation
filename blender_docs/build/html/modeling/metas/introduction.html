Вступ¶

Орієнтир -- Reference

Mode:Object or Edit Modes
Menu:Add ‣ Meta
Hotkey:Shift-A

Мета об'єкти -- це неявні поверхні, що означає, що вони не явно визначаються вершинами (як сіті) або керувальними точками (як поверхні): вони існують процедурно. Мета об’єкти -- це буквально математичні формули, що розраховуються на льоту Blender'ом.

Дуже відмінні візуальні характеристики мета полягають у тому, що вони є рідкими рухливими або глино-подібними формами, що мають «округлий» обрис. Крім того, коли два мета об'єкти наближаються один до одного, вони починають взаємодіяти між собою. Вони «змішуються» або «об'єднуються», як роблять краплі води, особливо у нульовій точці (що, між іншим, робить їх дуже зручним для моделювання потоків води, коли ви не хочете робити імітацію рідини). Якщо вони згодом рухаються один від одного, то вони відновлюють свою оригінальну форму.

Кожен з них визначається його власною підлежачою математичною структурою (Структура -- Structure), та ви можете у будь-який час перемикатися між ними за допомогою панелі Active Element.

Typically Meta objects are used for special effects or as a basis for modeling. For example, you could use a collection of metas to form the initial shape of your model and then convert it to a mesh for further modeling. Meta objects are also very efficient for ray tracing.

Ghi chú

Meta об'єкти мають дещо іншу поведінку у режимі об'єкта Object Mode.

Візуалізація¶

У режимі об'єкта Object Mode показується розрахована сіть разом з чорним «кільцем виділення» (яке стає рожевим, коли об'єкт виділено).

Приклад мета кулі Meta Ball.

У режимі правки Edit Mode (Ілюстрація Приклад мета кулі Meta Ball.), мета рисується як сіть (відтінена або чорно-каркасна, але без будь-яких вершин, звичайно) з двома кольоровими кільцями -- червоним для виділення (рожеве, коли виділено) та зеленим для прямого керування тугістю мета (світло-зелене, коли активне). Зауважте, що за винятком трансформації Scale S, підсвічування кільця зеленим є еквівалентом підсвічування червоним.
